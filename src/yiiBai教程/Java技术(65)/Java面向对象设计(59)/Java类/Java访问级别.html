Java访问级别

类的一个简单名称是在class关键字和{符号之间的名称。
当通过简单的类名称来引用一个类时，编译器在引用类所在的同一个包中查找该类声明。

当然，还可以使用全名来引用一个类，如下所示。

com.yiibai.Dog aDog;

指定类的访问级别的一般语法是

<access level modifier>class <class name> {
    // Body of the class
}

类声明中<access level modifier>只有两个有效值：

  • 无值
  • public

无值则称为包级访问。具有包级别访问的类只能在声明它的包中访问。

具有公共访问（public）级别修改器的类可以从应用程序中的任何包访问。

package  com.yiibai;

public class Dog  {

}

Java main()方法

现在来了解一下一个类中的main（）方法。main（）方法声明如下：

public static  void  main(String[]  args)  {

}

在main（）方法的声明中使用了两个修饰符public和static。public修饰符使得它可以从应用程序中的任何地方访问，只要声明的类是可访问的。

static修饰符使它成为一个类方法，因此可以使用类名来调用它。它的返回类型是void，这意味着它不返回一个值给它的调用者。

它的名字是main，接受一个类型为String array（String []）的参数。

main（）方法是Java应用程序的入口方法。 例如，可使用以下命令运行Main类：

java  com.yiibai.Main

在上面的代码中，当运行一个类时，JVM调用main（）方法。

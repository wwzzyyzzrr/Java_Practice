JSON教程

JSON或JavaScript对象表示法是一个轻量级的基于文本的开放式标准，旨在为人类可读的数据交换。 JSON格式最初是由Douglas Crockford规定，在RFC4627中描述。 JSON的官方互联网媒体类型application/json。 JSON文件扩展名是 .json.

本教程将帮助理解 JSON 并如何在各种编程语言使用它，如PHP，Perl，Python，Ruby 和 Java 等

读者

本教程帮助初学者了解基本的 JavaScript 对象符号（JSON）功能，开发数据交换格式。完成本教程后，希望对 Javscript，Ajax，PERL等使用JSON 能够有一个较深入的理解和认识。

必备条件

在继续本教程之前，你应该有一个基本的了解如何通过HTTP的Web应用程序的工作，我们假设你有基本的 JavaScript 知识。

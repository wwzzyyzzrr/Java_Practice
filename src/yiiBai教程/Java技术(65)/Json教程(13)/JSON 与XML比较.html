JSON 与XML比较

JSON和XML是人类可读的格式和语言无关。他们都有在现实世界中创建，读取和解码的支持。我们可以比较JSON与XML基于以下因素：

详细

XML比JSON更详细的，因此它的速度更快写，而JSON友好（人性化，易读写）

数组使用

XML是用来描述的结构化数据，其中不包括数组而JSON包括数组。

解析

JavaScript的eval方法解析JSON。当应用到JSON，eval返回所描述的对象。

例子

这显示XML和JSON的个别例子：

JSON

{
   "company": Volkswagen,
   "name": "Vento",
   "price": 800000
}

XML

<car>
   <company>Volkswagen</company>
   <name>Vento</name>
   <price>800000</price>
</car>

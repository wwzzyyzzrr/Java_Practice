Struts2 <s:debug>标签示例

在Struts2在 <s:debug> 标签是一个非常有用的调试标记输出“值栈”的内容，并在网页中“堆栈上下文”的详细信息。在本教程，在JSP页面中显示如何使用<s:debug>标签。


1. 动作

一个简单的Action类，带有 propertyInStack 属性，显示叠加后的值。

DebugTagAction.java 

package com.yiibai.common.action;

import com.opensymphony.xwork2.ActionSupport;
 
public class DebugTagAction extends ActionSupport{
	
	public String propertyInStack;

	public String getPropertyInStack() {
		return propertyInStack;
	}

	public void setPropertyInStack(String propertyInStack) {
		this.propertyInStack = propertyInStack;
	}

}

2. <s:date>标签示例

在JSP页面使用<s:debug>标签输出系统的“值栈”和“栈上下文”。

debug.jsp 

<%@ taglib prefix="s" uri="/struts-tags" %>
 <html>
<head>
</head>
 
<body>
<h1>Struts2 <s:debug>标签示例 - www.yiibai.com</h1>

<s:debug />

</body>
</html>
将生成一个名为 debug 文字链接，点击文字链接获得调试的详细信息。

3. struts.xml 

<?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE struts PUBLIC
"-//Apache Software Foundation//DTD Struts Configuration 2.0//EN"
"http://struts.apache.org/dtds/struts-2.0.dtd">
 
<struts>
 	<constant name="struts.devMode" value="true" />
	<package name="default" namespace="/" extends="struts-default">
	
		<action name="debugTagAction" 
			class="com.yiibai.common.action.DebugTagAction" >
			<result name="success">pages/debug.jsp</result>
		</action>
		
	</package>
</struts>

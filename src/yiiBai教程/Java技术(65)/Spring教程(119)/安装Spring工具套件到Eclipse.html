安装Spring工具套件到Eclipse

1- 介绍

本教程是基于：
  • Eclipse 4.4 (Mars)
Spring工具套件(STS)就是Eclipse的扩展IDE。它专注于开发Spring应用程序。这篇文指导您如何在Eclipse中安装STS插件。

2- 安装Spring工具套件到Eclipse


输入STS进行搜索。为 Eclipse4.4 安装Spring 工具套件。


安装完成后，Eclipse会要求你重新启动IDE。单击 Yes 重新启动。



3- 开始使用Spring工具套件

切换视角到Spring。



  • File/New/Other...

OK，你可以使用Spring工具套件并从这里开始一个Spring项目了。

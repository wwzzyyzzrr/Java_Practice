java.lang.Math.log1p(double x)方法实例

java.lang.Math.log1p(double x) 返回自变量的总和及1。注意的自然对数，对于小的值x， log1p(x)的结果更接近ln(1+x)的真实结果比数的浮点计算log(1.0+ X)。特殊情况：

  • 如果参数为NaN或小于-1，那么结果为NaN。

  • 如果参数为正无穷大，那么结果为正无穷大。

  • 如果参数为负数，那么结果为负无穷大。

  • 如果参数是零，那么结果是相同的符号参数为零。

声明

以下是java.lang.Math.log1p()方法的声明

public static double log1p(double x)

参数

  • x -- 一个double值

返回值

此方法返回值 ln(x + 1)的自然对数  x + 1

异常

  • NA

例子

下面的例子显示lang.Math.log1p()方法的使用。

package com.yiibai;

import java.lang.*;

public class MathDemo {

   public static void main(String[] args) {

      // get two double numbers
      double x = 60984.1;
      double y = 1000;

      // call log1p and print the result
      System.out.println("Math.log1p(" + x + ")=" + Math.log1p(x));

      // call log1p and print the result
      System.out.println("Math.log1p(" + y + ")=" + Math.log1p(y));


   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Math.log1p(60984.1)=11.018384851023473
Math.log1p(1000)=6.90875477931522

java.lang.Float.longValue()方法实例

java.lang.Float.longValue() 方法(通过强制转换为long类型)返回当前Float作为一个long值。

声明

以下是java.lang.Float.longValue()方法的声明

public long longValue()

参数

  • NA

返回值

此方法返回当前对象表示的float值转换为long类型。

异常

  • NA

例子

下面的例子显示java.lang.Float.longValue()方法的使用。

package com.yiibai;

import java.lang.*;

public class FloatDemo {

   public static void main(String[] args) {

     /* returns the float value represented by this object
     converted to type long */
     Float obj = new Float("9873267.90");
     long l = obj.longValue();
     System.out.println("Value = " + l);
    
     obj = new Float("10.0");
     l = obj.longValue();
     System.out.println("Value = " + l);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Value = 9873268
Value = 10

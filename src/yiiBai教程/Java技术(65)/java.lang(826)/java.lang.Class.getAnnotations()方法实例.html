java.lang.Class.getAnnotations()方法实例

java.lang.Class.getAnnotations() 方法返回当前这个元素上的所有注释。它返回一个长度为零的数组，如果这个元素没有注释。该方法的调用者可以随意修改返回的数组;这对其他调用者返回的数组产生任何影响。

声明

以下是java.lang.Class.getAnnotations()方法的声明

public Annotation[] getAnnotations()

参数

  • NA

返回值

此方法返回当前这个元素上的所有注释。

异常

  • NA

例子

下面的例子显示java.lang.Class.getAnnotations()方法的使用。

package com.yiibai;

import java.lang.annotation.Annotation;
   
public class ClassDemo {

   public static void main(String []args) {
        
     ClassDemo cls = new ClassDemo();
     Class c = cls.getClass();
          
     Annotation[] a = c.getAnnotations();
     if(a.length != 0) {
        for(Annotation val : a) {
           System.out.println(val.toString());
        }
     }
     else {
        System.out.println("Annotations is not present...");
     }
   }
}

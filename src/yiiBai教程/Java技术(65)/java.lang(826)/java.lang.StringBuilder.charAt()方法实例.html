java.lang.StringBuilder.charAt()方法实例

java.lang.StringBuilder.charAt() 方法返回此序列中指定索引处的char值。
第一个char值在索引0的位置，下一个是索引1，依此类推，如数组索引。索引参数必须大于或等于0，且小于该序列的长度。

声明

以下是java.lang.StringBuilder.charAt()方法的声明

public char charAt(int index)

参数

  • index -- 所期望的字符值的索引。

返回值

此方法返回指定索引处的char值。

异常

  • IndexOutOfBoundsException -- 如果索引是负值或大于或等于length()。

例子

下面的例子显示java.lang.StringBuilder.charAt()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBuilderDemo {

  public static void main(String[] args) {
  
    StringBuilder str= new StringBuilder("Java");
    System.out.println("string = " + str);

    // returns the char at index 2
    System.out.println("character = " + str.charAt(2));
    
    str = new StringBuilder("amrood admin ");
    System.out.println("string = " + str);
    // returns the char at index 6, whitespace gets printed here
    System.out.println("character = " + str.charAt(6));
  }
}

character = v

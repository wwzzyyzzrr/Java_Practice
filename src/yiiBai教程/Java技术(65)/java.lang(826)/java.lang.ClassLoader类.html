java.lang.ClassLoader类

java.lang.ClassLoader 类是一个对象，它负责加载类。这个类是一个抽象类。它可用于通过安全管理器，以指示安全域。

类声明

以下是java.lang.ClassLoader类的声明：

  extends Object

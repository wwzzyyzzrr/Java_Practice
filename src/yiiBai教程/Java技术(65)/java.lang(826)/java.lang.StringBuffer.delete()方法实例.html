java.lang.StringBuffer.delete()方法实例

java.lang.StringBuffer.delete() 方法删除这个序列中的一个子串字符。子串开始在指定的开始，并延伸到该字符在索引end - 1，或如果没有这样的字符存在于序列的末端。如果 start 等于end，不进行任何更改。

声明

以下是java.lang.StringBuffer.delete()方法的声明

public StringBuffer delete(int start, int end)

参数

  • start --这是开始索引，包括。

  • end -- 这就是结尾索引(不包涵)。

返回值

此方法返回这个对象。

异常

  • StringIndexOutOfBoundsException -- 如果start为负或大于length()，或大于end。

例子

下面的例子显示java.lang.StringBuffer.delete()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBufferDemo {

  public static void main(String[] args) {
  
    StringBuffer buff = new StringBuffer("Java lang package");
    System.out.println("buffer = " + buff);
 
    // deleting characters from index 4 to index 9
    buff.delete(4, 9);
    System.out.println("After deletion = " + buff);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

buffer = Java lang package
After deletion = Java package

java.lang.String.charAt()方法实例

java.lang.String.charAt() 方法返回指定索引处的char值。范围从0索引到length()- 1序列，第一个char值在索引0，下一个是索引1，依此类推，对于数组索引。

声明

以下是java.lang.String.charAt()方法的声明

public char charAt(int index)

参数

  • index -- 这是char值的索引。

返回值

此方法返回这个字符串的指定索引处的char值。第一个char值在索引0。

异常

  • IndexOutOfBoundsException -- 如果索引参数比该串的长度小或为负。

例子

下面的例子显示java.lang.String.charAt()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str = "This is yiibai";
                   
    // prints character at 1st location
    System.out.println(str.charAt(0));
          
    // prints character at 5th location i.e white-space character
    System.out.println(str.charAt(4));
          
    // prints character at 18th location 
    System.out.println(str.charAt(17));
  }
}

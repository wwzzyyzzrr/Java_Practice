java.lang.StringBuffer.insert(int dstOffset, CharSequence s)方法实例

java.lang.StringBuffer.insert(int dstOffset, CharSequence s) 方法插入指定的CharSequence到这个序列。
CharSequence参数字符插入到这个顺序，在这个顺序中指定的位置，向上移动原来的字符的位置和参数s的长度增加该序列的长度。

声明

以下是java.lang.StringBuffer.insert()方法的声明

public StringBuffer insert(int dstOffset, CharSequence s)

参数

  • dstOffset -- 这是偏移量。

  • s -- 这是要被插入的序列。

返回值

该方法返回一个引用对象。

异常

  • IndexOutOfBoundsException -- 如果偏移量是无效的。

java.lang.StringBuffer.append(CharSequence s, int start, int end)方法实例

java.lang.StringBuffer.append(CharSequence s) 方法追加指定的CharSequence这个序列。 CharSequence参数中的字符追加，按顺序由参数的长度增加到该序列的长度。

声明

以下是java.lang.StringBuffer.append()方法的声明

public StringBuffer append(CharSequence s)

参数

  • s -- 这是要追加的CharSequence。

返回值

该方法返回一个此对象引用。

异常

  • NA

例子

下面的例子显示java.lang.StringBuffer.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBufferDemo {

  public static void main(String[] args) {
  
    StringBuffer buff = new StringBuffer("compile ");
    System.out.println("buffer = " + buff);

    CharSequence cSeq = "online";
    // appends the CharSequence
    buff.append(cSeq);

    // print the string buffer after appending
    System.out.println("After append = " + buff);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

buffer = compile
After append = compile online

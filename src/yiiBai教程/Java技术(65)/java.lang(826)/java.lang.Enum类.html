java.lang.Enum类

java.lang.Enum 类是所有Java语言枚举类型的公共基类。

类声明

以下是java.lang.Enum类的声明：

public abstract class Enum<E extends Enum<E>>
   extends Object
      implements Comparable<E>, Serializable

java.lang.Character.reverseBytes()方法实例

 java.lang.Character.reverseBytes(char ch) 返回通过反转指定char值的字节的顺序而获得的值。

声明

以下是java.lang.Character.reverseBytes()方法的声明

public static char reverseBytes(char ch)

参数

  • NA

返回值

此方法返回在指定的字符值的字节通过反转(或等价地，交换)得到的值。

异常

  • NA

例子

下面的例子显示lang.Character.reverseBytes()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 2 char primitives ch1, ch2
      char ch1, ch2;

      // assign value to ch1
      ch1 = 'u4d00';

      // assign result of reverseBytes on ch1 to ch2
      ch2 = Character.reverseBytes(ch1);

      String str = "Reversing bytes on ch1 gives " + ch2;

      // print ch2 value
      System.out.println( str );
   }
}

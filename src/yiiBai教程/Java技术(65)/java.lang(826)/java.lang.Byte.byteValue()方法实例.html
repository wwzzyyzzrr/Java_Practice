java.lang.Byte.byteValue()方法实例

java.lang.Byte.byteValue() 返回字节一个字节的值。

声明

以下是java.lang.Byte.byteValue()方法的声明

public byte byteValue()

Overrides

byteValue 在类 Number

参数

  • NA

返回值

此方法将返回转换后该对象表示输入字节的数值。

异常

  • NA

例子

下面的例子显示lang.Byte.byteValue()方法的使用。

package com.yiibai;

import java.lang.*;

public class ByteDemo {

   public static void main(String[] args) {

      // create a Byte object b
      Byte b;

      // assign value to b
      b = new Byte("100");

      // create a byte primitive bt
      byte bt;

      // assign primitive value of b to bt
      bt = b.byteValue();

      String str = "Primitive byte value of Byte object " + b + " is " + bt;

      // print bt value
      System.out.println( str );
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Primitive byte value of Byte object 100 is 100

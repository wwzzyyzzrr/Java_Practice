java.lang.Class.getSigners()方法实例

java.lang.Class.getSigners() 得到这个类的签名。

声明

以下是java.lang.Class.getSigners()方法的声明

public Object[] getSigners()

参数

  • NA

返回值

此方法返回这个类的签名，或者为null，如果没有签名，如果这个对象表示一个基本类型或void，则返回null。

异常

  • NA

例子

下面的例子显示java.lang.Class.getSigners()方法的使用。

package com.yiibai;

import java.lang.*;

public class ClassDemo {

   public static void main(String[] args) {

     try {  
        Class cls = Class.forName("ClassDemo"); 
          
        // returns the name of the class
        System.out.println("Class = " + cls.getName());
        
        Object[] obj = cls.getSigners();
        System.out.println("Value = " + obj); 
     }
     catch(ClassNotFoundException ex) {
        System.out.println(ex.toString());
     }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Class = ClassDemo
Value = null

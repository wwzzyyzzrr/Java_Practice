java.lang.String.split(String regex, int limit)方法实例

java.lang.String.split(String regex, int limit) 方法根据给定的正则表达式匹配拆分这个字符串。

此方法返回的数组包含此字符串由另一个子给定的表达式匹配，或由字符串的末尾终止的每个子字符串。

如果表达式不匹配任何部分的输入，然后将得到的数组只有一个元素，即此字符串。

声明

以下是java.lang.String.split()方法的声明

public String[] split(String regex, int limit)

参数

  • regex -- 这是在限定的正则表达式。

  • limit -- 这个控制模式施加的数的次数，因此，会影响所得到的数组的长度

返回值

此方法返回分拆围绕给定的正则表达式匹配这个字符串计算字符串数组。

异常

  • PatternSyntaxException -- 如果正则表达式的语法无效。

例子

下面的例子显示java.lang.String.split()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str = "a d, m, i.n";
    String delimiters = "\s+|,\s*|\.\s*";

    // analyzing the string
    String[] tokensVal = str.split(delimiters);

    // prints the count of tokens
    System.out.println("Count of tokens = " + tokensVal.length);
    
    for(String token : tokensVal) {
      System.out.print(token);
    } 
    
    // analyzing the string with limit as 3
    tokensVal = str.split(delimiters, 3);

    // prints the count of tokens 
    System.out.println("
Count of tokens = " + tokensVal.length);
    
    for(String token : tokensVal) {
      System.out.print(token);
    }    
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Count of tokens = 5
admin
Count of tokens = 3
adm, i.n

java.lang.Package.getImplementationVersion()方法实例

java.lang.Package.getImplementationVersion() 方法返回这个实现的版本。它包括由本实施供应商分配的任何字符串和不具有由Java运行时指定的或预期的任何特定的语法。这可能是比较与此供应商为这个包用于本实施其他软件包的版本字符串相等。

声明

以下是java.lang.Package.getImplementationVersion()方法的声明

public String getImplementationVersion()

参数

  • NA

返回值

此方法返回执行的版本，如果尚不清楚，则返回null。

异常

  • NA

例子

下面的例子显示了lang.Object.getImplementationVersion()方法的使用。

package com.yiibai;

public class PackageDemo {

   public static void main(String[] args) {

      // create a package object for java.lang package
      Package pack = Package.getPackage("java.lang");

   }
}

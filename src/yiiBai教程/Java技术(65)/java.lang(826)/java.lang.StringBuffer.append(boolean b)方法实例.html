java.lang.StringBuffer.append(boolean b)方法实例

java.lang.StringBuffer.append(boolean b) 方法追加布尔参数序列到字符串的表示形式。

声明

以下是java.lang.StringBuffer.append()方法的声明

public StringBuffer append(boolean b)

参数

  • b -- 这是布尔值。

返回值

该方法返回一个引用对象。

异常

  • NA

例子

下面的例子显示java.lang.StringBuffer.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBufferDemo {

  public static void main(String[] args) {
  
    StringBuffer buff = new StringBuffer("tuts ");
    System.out.println("buffer = " + buff);

    // appends the boolean argument as string to the string buffer
    buff.append(true);
    // print the string buffer after appending
    System.out.println("After append = " + buff);
    
    buff = new StringBuffer("abcd ");
    System.out.println("buffer = " + buff);
    // appends the boolean argument as string to the string buffer    
    buff.append(false);
    // print the string buffer after appending
    System.out.println("After append = " + buff);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

buffer = tuts
After append = tuts true
buffer = abcd
After append = abcd false

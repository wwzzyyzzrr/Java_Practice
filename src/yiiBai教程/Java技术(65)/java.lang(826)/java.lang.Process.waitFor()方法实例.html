java.lang.Process.waitFor()方法实例

java.lang.Process.waitFor() 方法导致当前线程等待，如果有必要，直到由该Process对象表示的进程已经终止。此方法返回，如果立刻子进程已经结束。如果子进程还没有终止，则调用线程将被阻塞，直到子进程退出。

声明

以下是java.lang.Process.waitFor()方法的声明

public abstract int waitFor()

参数

  • NA

返回值

该方法将处理返回的退出值。按照惯例，0表示正常终止。

异常

  • NA

例子

下面的例子显示lang.Process.waitFor()方法的使用。

package com.yiibai;

public class ProcessDemo {

   public static void main(String[] args) {
      try {
         // create a new process
         System.out.println("Creating Process...");
         Process p = Runtime.getRuntime().exec("notepad.exe");

         // cause this process to stop until process p is terminated
         p.waitFor();

         // when you manually close notepad.exe program will continue here
         System.out.println("Waiting over.");

      } catch (Exception ex) {
         ex.printStackTrace();
      }

   }
}

Waiting over.

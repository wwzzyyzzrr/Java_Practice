java.lang.Math.min(float a, float b)方法实例

java.lang.Math.min(float a, float b) 返回两个float值的最小那个参数。即，其结果是该值越接近负无穷大。如果该参数具有相同的值，其结果是相同的值。如果任一值为NaN，那么结果为NaN。与数值比较运算不同，该方法认为负零严格小于正零。如果一个参数是正零，而另一个是负零，则结果为负零。

声明

以下是java.lang.Math.min()方法的声明

public static float min(float a,  float b)

参数

  • a -- 参数

  • b -- 另一个参数

返回值

这个方法返回a和b之间的最小值。

异常

  • NA

例子

下面的例子显示lang.Math.min()方法的使用。

package com.yiibai;

import java.lang.*;

public class MathDemo {

   public static void main(String[] args) {

      // get two float numbers
      float x = 9875.875f;
      float y = 154.134f;

      // print the smaller number between x and y
      System.out.println("Math.min(" + x + "," + y + ")=" + Math.min(x, y));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Math.min(9875.875f, 154.134f)=154.134

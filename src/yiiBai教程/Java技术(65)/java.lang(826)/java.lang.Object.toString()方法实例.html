java.lang.Object.toString()方法实例

java.lang.Object.toString() 方法返回该对象的字符串表示形式。一般情况下，toString方法返回一个字符串，该字符串“文本方式表示”此对象。其结果应是一个简明而丰富的表示，很容易对一个人阅读。我们建议所有子类重写此方法。

声明

以下是java.lang.Object.toString()方法的声明

public String toString()

参数

  • NA

返回值

此方法返回该对象的字符串表示形式。

异常

  • NA

例子

下面的例子显示了lang.Object.toString()方法的使用。

package com.yiibai;

import java.util.ArrayList;

public class ObjectDemo {

   public static void main(String[] args) {

      // get an integer
      Integer i = new Integer(50);

      // get a list
      ArrayList list = new ArrayList();

      // add some elements in list
      list.add(50);
      list.add("Hello World");

      // print their string representation
      System.out.println("" + i.toString());
      System.out.println("" + list.toString());
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

50
[50, Hello World]

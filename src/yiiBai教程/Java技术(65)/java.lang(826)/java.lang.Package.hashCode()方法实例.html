java.lang.Package.hashCode()方法实例

java.lang.Package.hashCode() 方法返回包名称计算的哈希码。

声明

以下是java.lang.Package.hashCode()方法的声明

public int hashCode()

参数

  • NA

返回值

此方法返回包名称计算的哈希码。

异常

  • NA

例子

下面的例子显示了lang.Object.hashCode()方法的使用。

package com.yiibai;

public class PackageDemo {

   public static void main(String[] args) {

      // get the java lang package
      Package pack = Package.getPackage("java.lang");

      // print the hash code computed for this package
      System.out.println("" + pack.hashCode());

   }
}

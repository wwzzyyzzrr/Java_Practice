java.lang.Character.charValue()方法实例

 java.lang.Character.charValue() 返回字符对象的值。

声明

以下是java.lang.Character.charValue()方法的声明

public char charValue()

参数

  • NA

返回值

此方法将返回该对象表示原始字符值。

异常

  • NA

例子

下面的例子显示lang.Character.charValue()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create a Character object c
      Character c;

      // assign value to c
      c = new Character('a');

      // create a char primitive ch
      char ch;

      // assign primitive value of c to ch
      ch = c.charValue();

      String str = "Primitive char value is " + ch;

      // print ch value
      System.out.println( str );
   }
}

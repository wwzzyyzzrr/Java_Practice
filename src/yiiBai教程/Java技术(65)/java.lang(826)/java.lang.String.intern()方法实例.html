java.lang.String.intern()方法实例

java.lang.String.intern() 方法返回字符串对象的规范化表示。字符串，初始为null，池是由String类私有维护。

对于任意字符串s和t，有 s.intern() == t.intern() 为true当且仅当s.equals(t)是true。
所有的文字字符串和字符串值常量表达式扣留。

声明

以下是java.lang.String.intern()方法的声明

public String intern()

参数

  • NA

返回值

此方法返回具有相同的内容正如这个字符串，但保证从字符串池中是唯一的。

异常

  • NA

例子

下面的例子显示java.lang.String.intern()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str1 = "This is YiiBai";
    
    // returns canonical representation for the string object
    String str2 = str1.intern();
   
    // prints the string str2
    System.out.println(str2);
    
    // check if str1 and str2 are equal or not
    System.out.println("Is str1 equal to str2 ? = " + (str1 == str2));
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

This is YiiBai
Is str1 equal to str2 ? = true

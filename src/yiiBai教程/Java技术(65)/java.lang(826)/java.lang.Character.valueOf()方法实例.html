java.lang.Character.valueOf()方法实例

 java.lang.Character.valueOf(char c) 返回表示指定字符值的字符实例。如果不需要新的字符实例，此方法一般应优先于构造字符(char)的使用，因为此方法是有可能通过缓存经常请求的值，得到显著更好的空间和时间性能。

此方法将在'u0000'到'u007f'包函，并可以高速缓存等在此范围之外的值总是缓存值。

声明

以下是java.lang.Character.valueOf()方法的声明

public static Character valueOf(char c)

参数

  • c - 一个char值

返回值

此方法返回表示c字符实例

异常

  • NA

例子

下面的例子显示lang.Character.valueOf()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 2 Character objects c1, c2
      Character c1, c2;

      // create 2 char primitives and assign values 
      char ch1 = 'i';
      char ch2 = 65;

      // assign Character values of ch1, ch2 to c1, c2
      c1 = Character.valueOf(ch1);
      c2 = Character.valueOf(ch2);

      String str1 = "Character value of " + ch1 + " is " + c1;
      String str2 = "Character value of " + ch2 + " is " + c2;

      // print c1, c2 values
      System.out.println( str1 );
      System.out.println( str2 );
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Character value of i is i
Character value of A is A

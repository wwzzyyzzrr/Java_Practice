java.lang.Character.UnicodeBlock.of()方法实例

java.lang.Character.UnicodeBlock.of(char c) 方法返回表示包含给定字符，如果字符不是定义块的成员的Unicode块的对象则返回null。

声明

以下是java.lang.Character.UnicodeBlock.of()方法的声明

public static Character.UnicodeBlock of(char c)

参数

  • c -- 这是一个字符

返回值

此方法返回UnicodeBlock实例，表示的Unicode块，这个字符是一个成员，如果字符不是任何Unicode模块中的员成则返回null。

异常

  • NA

例子

下面的例子显示java.lang.Character.UnicodeBlock.of()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterUnicodeBlockDemo {
 
   public static void main(String[] args) {

     /* returns the object representing the Unicode block containing
     the given character */ 
     System.out.println(Character.UnicodeBlock.of('u20ac'));
     System.out.println(Character.UnicodeBlock.of('-'));
     System.out.println(Character.UnicodeBlock.of('='));
     System.out.println(Character.UnicodeBlock.of('Z'));
   }
}

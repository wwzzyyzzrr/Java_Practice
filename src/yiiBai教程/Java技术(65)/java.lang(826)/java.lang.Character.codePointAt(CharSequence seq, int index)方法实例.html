java.lang.Character.codePointAt(CharSequence seq, int index)方法实例

java.lang.Character.codePointAt(CharSequence seq, int index) 返回CharSequence给定索引上的代码点。

如果给定的索引中的CharSequence处的char值在高代理范围，下面的索引小于所述CharSequence的长度，并在以下索引处的字符值是在低代理范围，则该补充对应于此代理对代码点返回。否则，返回给定索引处的char值。

声明

以下是java.lang.Character.codePointAt()方法的声明

public static int codePointAt(CharSequence seq, int index)

参数

  • seq - char值序列(Unicode代码单元)

  • index - 索引为char值(Unicode代码单元)要转换的char数组

返回值

此方法将返回给定索引处的Unicode代码点。

异常

  • NullPointerException - 如果seq 为 null.

  • IndexOutOfBoundsException - 如果该值指数为负或不小于seq.length()。

例子

下面的例子显示lang.Character.codePointAt()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create a CharSequence seq and assign value
      CharSequence seq = "Hello";

      // create and assign value to index
      int index  = 4;

      // create an int res
      int res;

      // assign result of codePointAt on seq at index to res
      res = Character.codePointAt(seq, index);

      String str = "Unicode code point is " + res;

      // print res value
      System.out.println( str );
   }
}

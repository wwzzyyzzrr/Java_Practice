java.lang.StringBuilder.append(double d)方法实例

java.lang.StringBuilder.append(double d) 方法追加double参数到这个序列的字符串表示形式。

声明

以下是java.lang.StringBuilder.append()方法的声明

public StringBuilder append(double d)

参数

  • d -- 这是double值。

返回值

该方法返回这个对象的引用。

异常

  • NA

例子

下面的例子显示java.lang.StringBuilder.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBuilderDemo {

  public static void main(String[] args) {
  
    StringBuilder str = new StringBuilder("tuts ");
    System.out.println("string = " + str);

    // appends the double argument as string to the StringBuilder
    str.append(30.100000001d);
    // print the StringBuilder after appending
    System.out.println("After append = " + str);
    
    str = new StringBuilder("abcd ");
    System.out.println("string = " + str);
    // appends the double argument as string to the StringBuilder
    str.append(15.12d);
    // print the string Builder after appending
    System.out.println("After append = " + str);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

string = tuts
After append = tuts 30.100000001
string = abcd
After append = abcd 15.12

java.lang.Class.isInterface()方法实例

java.lang.Class.isInterface() 确定指定的Class对象表示一个接口类型。

声明

以下是java.lang.Class.isInterface()方法的声明

public boolean isInterface()

参数

  • NA

返回值

如果此对象表示一个接口此方法返回true，否则返回false。

异常

  • NA

例子

下面的例子显示java.lang.Class.isInterface()方法的使用。

package com.yiibai;

import java.lang.*;

public class ClassDemo {

   public static void main(String[] args) {

     ClassDemo c = new ClassDemo();
     Class cls = c.getClass();

     // determines if the specified Class object represents an interface type
     boolean retval = cls.isInterface();
     System.out.println("It is an interface ? " + retval);       
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

It is an interface ? false

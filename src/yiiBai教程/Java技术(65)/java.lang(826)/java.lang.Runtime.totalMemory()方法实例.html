java.lang.Runtime.totalMemory()方法实例

java.lang.Runtime.totalMemory() 方法返回存储器中的Java虚拟机的总量。通过此方法返回的值可以随时间变化，这取决于宿主环境。注意，存储器来保存任何给定类型的一个对象所需的时间量依赖于实现。

声明

以下是java.lang.Runtime.totalMemory()方法的声明

public long totalMemory()

参数

  • NA

返回值

这个方法没有返回值。

异常

  • NA

例子

下面的例子显示lang.Runtime.totalMemory()方法的使用。

package com.yiibai;

public class RuntimeDemo {

   public static void main(String[] args) {

      // print the state of the program
      System.out.println("Program is starting...");

      // print the total memory
      System.out.println("" + Runtime.getRuntime().totalMemory());

   }
}

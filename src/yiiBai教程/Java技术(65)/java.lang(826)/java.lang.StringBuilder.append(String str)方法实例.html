java.lang.StringBuilder.append(String str)方法实例

java.lang.StringBuilder.append(String str) 方法将指定字符串追加到此字符序列。 String参数字符追加并按顺序由参数的长度增加该序列的长度。

声明

以下是java.lang.StringBuilder.append()方法的声明

public StringBuilder append(String str)

参数

  • str -- 这是一个字符串。

返回值

该方法返回这个对象的引用。

异常

  • NA

例子

下面的例子显示java.lang.StringBuilder.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBuilderDemo {

  public static void main(String[] args) {
  
    StringBuilder str = new StringBuilder("tutorials ");
    System.out.println("string = " + str);
    
    // appends the string argument to the StringBuilder
    str.append("yiibai");
    // print the StringBuilder after appending
    System.out.println("After append = " + str);
    
    str = new StringBuilder("1234 ");
    System.out.println("string = " + str);
    // appends the string argument to the StringBuilder 
    str.append("!#$%");
    // print the StringBuilder after appending
    System.out.println("After append = " + str);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

string = tutorials
After append = tutorials yiibai
string = 1234
After append = 1234 !#$%

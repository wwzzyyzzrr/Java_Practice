java.lang.Runtime.maxMemory()方法实例

java.lang.Runtime.maxMemory() 方法返回的内存，Java虚拟机将尝试使用的最大数量。如果不存在固有的限制，则Long.MAX_VALUE值将被返回。

声明

以下是java.lang.Runtime.maxMemory()方法的声明

public long maxMemory()

参数

  • NA

返回值

此方法返回虚拟机将尝试使用以字节为单位的内存的最大数量。

异常

  • NA

例子

下面的例子显示lang.Runtime.maxMemory()方法的使用。

package com.yiibai;

public class RuntimeDemo {

   public static void main(String[] args) {

      // print when the program starts
      System.out.println("Program starting...");

      // print the maximum memory
      System.out.println("" + Runtime.getRuntime().maxMemory());

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Program starting...
935854080

java.lang.Character.getNumericValue(int codePoint)方法实例

java.lang.Character.getNumericValue(int codePoint) 返回int值指定的字符(Unicode代码点)表示。例如，字符' u216C'(罗马数字50)将返回一个int值50。

(通过' u005A'' u0041')，小写(' u0061'到' u007A')，和全宽度的变体(“ uFF21'到' uFF3A'和' uFF41'中的大写字母AZ通过' uFF5A')形式的具有数字值从10到35，这是独立的Unicode规范，这并不对这些char值分配数值。

如果字符没有一个数字值，则返回-1。如果字符具有不能被表示为一个非负整数(例如，分数值)的数字值，那么返回-2。

声明

以下是java.lang.Character.getNumericValue()方法的声明

public static int getNumericValue(int codePoint)

参数

  • codePoint - 字符(Unicode代码点)转换

返回值

此方法返回字符的数值，作为一个非负的int值; -2如果字符有一个数值，不是一个非负整数; -1，如果字符没有数值。

异常

  • NA

例子

下面的例子显示lang.Character.getNumericValue()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 2 int primitives cp1, cp2
      int cp1, cp2;

      // assign values to cp1, cp2
      cp1 = 0x0068;
      cp2 = 0xff30;

      // create 2 int primitives i1, i2
      int i1, i2;

      // assign numeric values of cp1, cp2 to i1, i2
      i1 = Character.getNumericValue(cp1);
      i2 = Character.getNumericValue(cp2);

      String str1 = "Numeric value of code point cp1 is " + i1;
      String str2 = "Numeric value of code point cp2 is " + i2;

      // print i1, i2 values
      System.out.println( str1 );
      System.out.println( str2 );
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Numeric value of code point cp1 is 17
Numeric value of code point cp2 is 25

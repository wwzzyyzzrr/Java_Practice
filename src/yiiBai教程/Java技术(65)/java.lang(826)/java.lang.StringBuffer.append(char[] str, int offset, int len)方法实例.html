java.lang.StringBuffer.append(char[] str, int offset, int len)方法实例

java.lang.StringBuffer.append(char[] str, int offset, int len) 方法将char数组参数追加到此序列的子数组的字符串表示形式。将char和arraystr字符，从索引offset，被附加在该序列顺序的内容。此序列的长度增加len。

声明

以下是java.lang.StringBuffer.append()方法的声明

public StringBuffer append(char[] str, int offset, int len)

参数

  • str -- 这是要追加的字符。

  • offset -- 这是第一个字符追加的索引。

  • len -- 字符追加的数量。

返回值

该方法返回一个此对象引用。

异常

  • NA

例子

下面的例子显示java.lang.StringBuffer.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBufferDemo {

  public static void main(String[] args) {
  
    StringBuffer buff = new StringBuffer("amrood ");
    System.out.println("buffer = " + buff);
    
    // char array
    char[] str = new char[]
    {'a','d','m','i','n','i','s','t','r','a','t','o','r'};
	
    /* appends the string representation of char array argument to
    this string buffer with offset at index 0 and length as 5 */
    buff.append(str, 0, 5);
	
    // print the string buffer after appending
    System.out.println("After append = " + buff);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

buffer = amrood
After append = amrood admin

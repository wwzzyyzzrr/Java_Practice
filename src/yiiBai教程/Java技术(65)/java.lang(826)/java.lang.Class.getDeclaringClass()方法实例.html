java.lang.Class.getDeclaringClass()方法实例

java.lang.Class.getDeclaringClass() 方法返回表示在它被声明的类的Class对象，如果此Class对象所表示的类或接口是另一个类成员。

如果这个类或接口不属于任何其他类成员此方法返回null。如果此Class对象表示一个数组类，基本类型或void，则此方法返回null。

声明

以下是java.lang.Class.getDeclaringClass()方法的声明

public Class<?> getDeclaringClass()

参数

  • NA

返回值

此方法返回这个类的类声明。

异常

  • NA

例子

下面的例子显示java.lang.Class.getDeclaringClass()方法的使用。

package com.yiibai;

import java.lang.reflect.*;

public class ClassDemo {

   public static void main(String[] args) {

     // class object associated with class Long
     Class cls = Long.class;

     Method[] m = cls.getMethods();
     for(int i = 0; i < m.length; i++) {
        // returns te declaring class
        Class dec = m[i].getDeclaringClass();
        // displays all methods
        System.out.println("Method = " + m[i].toString());
        System.out.println(" Declaring class: " + dec.toString());
     }
   }
}

java.lang.Class.getName()方法实例

java.lang.Class.getName() 返回Class对象所表示的实体(类，接口，数组类，基本类型或void)的名字，作为一个字符串。

声明

以下是java.lang.Class.getName()方法的声明

public String getName()

参数

  • NA

返回值

此方法返回该对象表示的类或接口的名称。

异常

  • NA

例子

下面的例子显示java.lang.Class.getName()方法的使用。

package com.yiibai;

import java.lang.*;

public class ClassDemo {

   public static void main(String[] args) {

     // returns the Class object associated with this class
     ClassDemo cl = new ClassDemo();
     Class c1Class = cl.getClass();

     // returns the name of the class
     String name = c1Class.getName();
     System.out.println("Class Name = " + name);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Class Name = ClassDemo

java.lang.Enum.toString()方法实例

java.lang.Enum.toString() 方法返回枚举常量包含在声明中的名称。

声明

以下是java.lang.Enum.toString()方法的声明

public String toString()

参数

  • NA

返回值

此方法返回枚举常量的名称。

异常

  • NA

例子

下面的例子显示java.lang.Enum.toString()方法的使用。

package com.yiibai;

import java.lang.*;

// enum showing programming languages
enum Language {
   C, Java, PHP;
}

public class EnumDemo {

   public static void main(String args[]) {
       
     // returns the name of this enum constant
     System.out.println("Programming in " + Language.C.toString());       
     System.out.println("Programming in " + Language.Java.toString());  
     System.out.println("Programming in " + Language.PHP.toString());     
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Programming in C
Programming in Java
Programming in PHP

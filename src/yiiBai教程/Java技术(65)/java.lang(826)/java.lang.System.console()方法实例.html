java.lang.System.console()方法实例

java.lang.System.console() 方法返回与当前Java虚拟机关联的唯一Console对象(如果有)。

声明

以下是java.lang.System.console()方法的声明

public static Console console()

参数

  • NA

返回值

此方法返回系统控制台(如果有的话)，否则返回null。

异常

  • NA

例子

下面的例子显示java.lang.System.console()方法的使用。

package com.yiibai;

import java.lang.*;
import java.io.Console;

public class SystemDemo {

   public static void main(String[] args) {

     Console ob = System.console();

     ob.printf("Welcome to yiibai...");

     // flushes the console
     ob.flush();
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Welcome to yiibai...

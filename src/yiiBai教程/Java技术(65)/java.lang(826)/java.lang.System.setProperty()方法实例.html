java.lang.System.setProperty()方法实例

java.lang.System.setProperty() 方法设置指定键指定的系统属性。

声明

以下是java.lang.System.setProperty()方法的声明

public static String setProperty(String key, String value)

参数

  • key -- 此是该系统属性的名称。

  • value -- 是该系统属性的值。

返回值

此方法返回系统属性之前的值，则返回null，如果它没有一个。

异常

  • SecurityException -- 如果安全管理器存在并且其checkPermission方法不允许指定属性的设置。

  • NullPointerException -- 如果键或值为null。

  • IllegalArgumentException -- 如果key是空的。

例子

下面的例子显示java.lang.System.setProperty()方法的使用。

package com.yiibai;

import java.lang.*;

public class SystemDemo {

   public static void main(String[] args) {

     // prints Java Runtime Version before property set
     System.out.print("Previous : ");
     System.out.println(System.getProperty("java.runtime.version" ));
     System.setProperty("java.runtime.version", "Java Runtime 1.6.0");
     
     // prints Java Runtime Version after property set
     System.out.print("New : ");
     System.out.println(System.getProperty("java.runtime.version" ));
   }
}

让我们编译并运行上述程序，这将产生以下结果：

Previous : 1.6.0_22-b22
New : Java Runtime 1.6.0

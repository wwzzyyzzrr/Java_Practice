java.lang.Compiler.compileClasses()方法实例

java.lang.Compiler.compileClasses() 方法编译，其名称与指定字符串匹配的所有类。

声明

以下是java.lang.Compiler.compileClasses()方法的声明

public static boolean compileClasses(String string)

参数

  • string -- 这是类编译的名称。

返回值

此方法返回true，如果编译成功;返回false如果编译失败或没有编译器可用。

异常

  • NullPointerException -- 如果 string 为 null.

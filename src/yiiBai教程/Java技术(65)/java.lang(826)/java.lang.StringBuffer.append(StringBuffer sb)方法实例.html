java.lang.StringBuffer.append(StringBuffer sb)方法实例

java.lang.StringBuffer.append(StringBuffer sb) 方法将指定的StringBuffer 追加到此序列。
StringBuffer的参数中的字符按顺序追加到这个StringBuffer，通过参数的长度增加到这个StringBuffer的长度。如果对象为null，则这四个字符“null”被附加到这个StringBuffer。

声明

以下是java.lang.StringBuffer.append()方法的声明

public StringBuffer append(StringBuffer sb)

参数

  • sb -- 这是用来追加的StringBuffer。

返回值

该方法返回一个此对象引用。

异常

  • NA

例子

下面的例子显示java.lang.StringBuffer.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBufferDemo {

  public static void main(String[] args) {
  
    StringBuffer buff1 = new StringBuffer("tutorials ");
    System.out.println("buffer1 = " + buff1);
    StringBuffer buff2 = new StringBuffer("yiibai ");
    System.out.println("buffer2 = " + buff2);
     
    // appends stringbuffer2 to stringbuffer1
    buff1.append(buff2);
    // print the string buffer after appending
    System.out.println("After append = " + buff1);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

buffer1 = tutorials
buffer2 = yiibai
After append = tutorials yiibai

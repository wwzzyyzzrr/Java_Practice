java.lang.Math.max(float a, float b)方法实例

java.lang.Math.max(float a, float b) 返回两个float值较大值。即，其结果该参数接近正无穷大。如果该参数具有相同的值，其结果是相同的值。如果任一值为NaN，那么结果为NaN。与数值比较运算不同，该方法认为负零严格小于正零。如果一个参数是正零和其他负0，那么结果为正零。

声明

以下是java.lang.Math.max()方法声明

public static float max(float a, float b)

参数

  • a -- 一个参数

  • b -- 另一个参数

返回值

此方法返回a和b之中的最大值。

异常

  • NA

例子

下面的例子显示lang.Math.max()方法的使用。

package com.yiibai;

import java.lang.*;

public class MathDemo {

   public static void main(String[] args) {

      // get two float numbers
      float x = 60984.1f;
      float y = 1000f;

      // print the larger number between x and y
      System.out.println("Math.max(" + x + "," + y + ")=" + Math.max(x, y));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Math.max(60984.1f, 1000f)=60984.1

java.lang.Character.codePointBefore(CharSequence seq, int index)方法实例

java.lang.Character.codePointBefore(CharSequence seq, int index) 返回CharSequence的给定索引前面的代码点。如果在(index - 1)的char值在CharSequence是在低代理项范围，(index - 2)不为负，char值(index - 2)在CharSequence的是在高代理项范围，则对应于此代理项对的增补代码点返回。

否则，在(index - 1)处的char值返回。

声明

以下是java.lang.Character.codePointBefore()方法的声明

public static int codePointBefore(CharSequence seq, int index)

参数

  • seq - 该实例的CharSequence

  • index - 下面返回代码点的索引

返回值

此方法将返回给定索引之前的Unicode代码点值。

异常

  • NullPointerException - 如果a为null。

  • IndexOutOfBoundsException - 如果索引参数比seq.length()大或小于1

例子

下面的例子显示lang.Character.codePointBefore()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create a CharSequence seq and assign value
      CharSequence seq = "Hello";

      // create and assign value to index
      int index  = 4;

      // create an int res
      int res;

      // assign result of codePointBefore on seq at index to res
      res = Character.codePointBefore(seq, index);

      String str = "Unicode code point is " + res;

      // print res value
      System.out.println( str );
   }
}

java.lang.Character.forDigit(int digit, int radix)方法实例

java.lang.Character.forDigit(int digit, int radix) 确定在指定基数中的一个特定数字以字符表示。如果基数的值不是一个有效的基数，或者数字的值是不是在指定基数的有效位数，空字符('u0000“)返回。

radix参数是有效的，如果它大于或等于MIN_RADIX和小于或等于MAX_RADIX。数字参数是有效的，如果0 ≤ digit < radix。

如果该数字小于10，则“0”+位被返回。否则，值'a'+ 数字- 10返回。

声明

以下是java.lang.Character.forDigit()方法的声明

public static char forDigit(int digit, int radix)

参数

  • digit - 数字转换为字符

  • radix - 基数

返回值

此方法返回指定基数数字的字符表示。

异常

  • NA

例子

下面的例子显示lang.Character.forDigit()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 2 character primitives ch1, ch2
      char ch1, ch2;

      // create 2 int primitives i1, i2 and assign values
      int i1 = 3;
      int i2 = 14;

      // assign char representation of i1, i2 to ch1, ch2 using radix
      ch1 = Character.forDigit(i1, 10);
      ch2 = Character.forDigit(i2, 16);

      String str1 = "Char representation of " + i1 + " in radix 10 is " + ch1;
      String str2 = "Char representation of " + i2 + " in radix 16 is " + ch2;

      // print ch1, ch2 values
      System.out.println( str1 );
      System.out.println( str2 );
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Char representation of 3 in radix 10 is 3
Char representation of 14 in radix 16 is e

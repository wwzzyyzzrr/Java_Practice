java.lang.Class类

java.lang.Class 类的实例表示正在运行的Java应用程序中的类和接口。它没有公共的构造函数。

类声明

以下是java.lang.Class类的声明：

  extends Object
    implements Serializable, GenericDeclaration, Type, AnnotatedElement

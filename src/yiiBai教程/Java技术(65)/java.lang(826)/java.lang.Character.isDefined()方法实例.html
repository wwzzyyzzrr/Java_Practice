java.lang.Character.isDefined()方法实例

java.lang.Character.isDefined(char ch) 判断一个字符在Unicode中有定义。

字符被定义，如果下列中至少一个为真时：

  • 它具有在UnicodeData文件的一个条目。

  • 它是由UnicodeData文件的定义在范围内的值。

声明

以下是java.lang.Character.isDefined()方法的声明

public static boolean isDefined(char ch)

参数

  • ch - 要测试的字符

返回值

如果字符在Unicode定义的含义，此方法返回true，否则返回false。

异常

  • NA

例子

下面的例子显示lang.Character.isDefined()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create a char primitive ch
      char ch;

      // assign value to ch
      ch = '@';

      // create a boolean primitives b
      boolean b;

      // assign isDefined result of ch to b
      b = Character.isDefined(ch);

      String str = "Character " +ch+ " has defined meaning in Unicode is " +b;

      // print b value
      System.out.println( str );
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Character @ has defined meaning in Unicode is true

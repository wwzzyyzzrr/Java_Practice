java.lang.Long.parseLong()方法实例

java.lang.Long.parseLong(String s) 方法解析字符串参数s为符号的十进制long类型。

声明

以下是java.lang.Long.parseLong()方法的声明

public static long parseLong(String s) throws NumberFormatException

参数

  • s -- 这是要分析包含long 表示形式的字符串。

返回值

此方法返回long ，十进制参数表示。

异常

  • NumberFormatException -- 如果字符串中不包含一个可分析的long。

例子

下面的例子显示java.lang.Long.parseLong()方法的使用。

package com.yiibai;

import java.lang.*;

public class LongDemo {

   public static void main(String[] args) {

     // parses the string argument
     long a = Long.parseLong("1452");
     long b = Long.parseLong("26");
     long c = Long.parseLong("54");

     long m = a * b * c;
     System.out.print("Value after multiplying = " + m);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Value after multiplying = 2038608

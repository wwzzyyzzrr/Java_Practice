java.lang.Character.offsetByCodePoints(CharSequence seq, int index, int codePointOffset)方法实例

java.lang.Character.offsetByCodePoints(CharSequence seq, int index, int codePointOffset) 返回给定的字符序列，该序列是从给定的索引由codePointOffset代码点偏移中的索引。

通过索引和codePointOffset给定的文本范围内未配对的代理算作为每一个代码点。

声明

以下是java.lang.Character.offsetByCodePoints()方法的声明

public static int offsetByCodePoints(CharSequence seq, int index, int codePointOffset)

参数

  • seq - 该字符序列

  • index - 索引被抵消

  • codePointOffset -偏移中的代码点

返回值

此方法返回的字符序列中的索引

异常

  • NullPointerException - 如果 seq 为 null

  • IndexOutOfBoundsException - 如果指数index为负或大于char序列的长度，或如果codePointOffset为正和开始索引的子序列具有比codePointOffset码点更少，或者如果codePointOffset是负和索引之前的子序列具有比codePointOffset的绝对值少代码点。

例子

下面的例子显示lang.Character.offsetByCodePoints()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create a CharSequence seq and assign value
      CharSequence seq = "Hello World";

      // create an int primitive res
      int res;

      // assign result of offsetByCodePoints on seq to res
      res = Character.offsetByCodePoints(seq, 3, 8);

      String str = "The index within the char sequence seq is " + res;

      // print res value
      System.out.println( str );
   }
}

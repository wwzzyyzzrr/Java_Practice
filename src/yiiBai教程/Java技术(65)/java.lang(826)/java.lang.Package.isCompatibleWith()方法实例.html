java.lang.Package.isCompatibleWith()方法实例

java.lang.Package.isCompatibleWith(String desired) 方法所需的版本进行比较这个包的规范版本。它返回true，如果此包规范的版本号是否大于或等于期望的版本号。

版本号被依次进行比较所需的和说明书中的字符串的对应元件相比。每个组件都被转换为十进制整数和值进行比较。如果说明书值大于所需的值，则返回true。如果该值小于则返回false。如果这两个值相等的周期被跳过，下一对组件进行比较。

声明

以下是java.lang.Package.isCompatibleWith()方法的声明

public boolean isCompatibleWith(String desired)

参数

  • desired -- 所需版本的版本字符串。

返回值

如果这个包的版本号是否大于或等于期望的版本号此方法返回true

异常

  • NumberFormatException -- 如果所需的或当前版本是不正确的虚线形式.

例子

下面的例子显示lang.Object.isCompatibleWith()方法的使用。

package com.yiibai;

public class PackageDemo {

   public static void main(String[] args) {

      // get the java lang package
      Package pack = Package.getPackage("java.lang");

      // check if this package is compatible with version 1.4.6
      System.out.println("" + pack.isCompatibleWith("1.4.6"));

   }
}

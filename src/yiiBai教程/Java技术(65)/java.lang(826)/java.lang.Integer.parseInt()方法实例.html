java.lang.Integer.parseInt()方法实例

java.lang.Integer.parseInt(String s) 方法解析字符串参数s为有符号十进制整数。

声明

以下是java.lang.Integer.parseInt()方法的声明

public static int parseInt(String s) throws NumberFormatException

参数

  • s -- 这是要分析包含整型表示的字符串。

返回值

此方法返回由十进制参数表示的整数值。

异常

  • NumberFormatException -- 如果字符串中不包含一个可分析的整数。

例子

下面的例子显示java.lang.Integer.parseInt()方法的使用。

package com.yiibai;

import java.lang.*;

public class IntegerDemo {

   public static void main(String[] args) {

     // parses the string argument
     int a = Integer.parseInt("12");
     int b = Integer.parseInt("26");
     int c = Integer.parseInt("54");

     int m = a * b * c;
     System.out.print("Value after multiplying = " + m);
   }
}

Value after multiplying = 16848

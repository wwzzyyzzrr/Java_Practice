java.lang.Math.tanh(double x)方法实例

java.lang.Math.tanh(double x) 返回double值的双曲正切值。 x的双曲正切定义为 (ex - e-x)/(ex + e-x), 换句话说，sinh(x)/cosh(x)。需要注意的是准确的tanh的绝对值总是小于1，特殊情况是：

  • 如果参数为NaN，那么结果为NaN。

  • 如果参数是零，那么结果是相同的符号参数为零。

  • 如果参数为正无穷大，那么结果是+1.0。

  • 如果参数为负无穷大，那么结果为-1.0。

计算结果必须在2.5 ulp准确结果。双曲正切对于任何有限的输入的结果必须有一个绝对值小于或等于1。注意，一旦双曲正切的确切结果1/2 限值ulp 的一半，正确+1.0 被返回。

声明

以下是java.lang.Math.tanh()方法的声明

public static double tanh(double x)

参数

  • x -- 返回数的双曲正切值。

返回值

此方法返回x的双曲正切值。

异常

  • NA

例子

下面的例子显示lang.Math.tanh()方法的使用。

package com.yiibai;

import java.lang.*;

public class MathDemo {

   public static void main(String[] args) {

      // get two double numbers numbers
      double x = 45;
      double y = -180;

      // convert them in radians
      x = Math.toRadians(x);
      y = Math.toRadians(y);

      // print the hyperbolic tangent of these doubles
      System.out.println("Math.tanh(" + x + ")=" + Math.tanh(x));
      System.out.println("Math.tanh(" + y + ")=" + Math.tanh(y));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Math.tanh(0.7853981633974483)=0.6557942026326724
Math.tanh(-3.141592653589793)=-0.99627207622075

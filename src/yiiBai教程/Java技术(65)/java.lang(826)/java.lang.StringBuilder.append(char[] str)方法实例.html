java.lang.StringBuilder.append(char[] str)方法实例

java.lang.StringBuilder.append(char[] str) 方法追加char数组参数到该序列的字符串表示形式。字符数组参数按顺序追加到这个序列的内容。这个序列的长度增加为该参数的长度。

声明

以下是java.lang.StringBuilder.append()方法的声明

public StringBuilder append(char[] str)

参数

  • str -- 将要追加的字符。

返回值

该方法返回这个对象的引用。

异常

  • NA

例子

下面的例子显示java.lang.StringBuilder.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBuilderDemo {

  public static void main(String[] args) {
  
    StringBuilder str = new StringBuilder("compile ");
    System.out.println("string = " + str);
    
    // char array
    char[] cArr = new char[]{'o','n','l','i','n','e'};
	
    /* appends the string representation of char array argument to
    this StringBuilder */
    str.append(cArr);
    // print the StringBuilder after appending
    System.out.println("After append = " + str);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

string = compile
After append = compile online

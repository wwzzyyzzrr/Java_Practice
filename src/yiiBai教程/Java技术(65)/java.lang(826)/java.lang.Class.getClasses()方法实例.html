java.lang.Class.getClasses()方法实例

java.lang.Class.getClasses() 方法返回一个包含代表所有的公共类和属于此Class对象所表示的类的成员接口的Class对象数组。这包括公共类和接口的成员从超类和公共类和接口成员类中声明继承。

声明

以下是java.lang.Class.getClasses()方法的声明

public Class<?>[] getClasses()

参数

  • NA

返回值

此方法返回一个代表这个类的公共成员的Class对象的数组。

异常

  • SecurityException -- 如果安全管理存在。

例子

下面的例子显示java.lang.Class.getClasses()方法的使用。

package com.yiibai;

import java.lang.*;

public class ClassDemo {

   public static void main(String[] args) {

     try {
        // returns the Class object associated with this class
        Class cls = Class.forName("ClassDemo");

        /* returns the array of Class objects representing the public 
        members of this class */
        Class[] classes = cls.getClasses();
        for (int i = 0; i < classes.length; i++) {
           System.out.println("Class found = " + classes[i].getName());
        }
     }
     catch (ClassNotFoundException e) {
        System.out.println(e.toString());
     }
   }

   public class InnerClass1 {
      public InnerClass1() {
         System.out.println("Inner Class1");
      }
   }

   public class InnerClass2 {
      public InnerClass2() {
         System.out.println("Inner Class2");
      }
   }

   private class InnerPrivateClass {
      public InnerPrivateClass() {
         System.out.println("Inner Private Class");
      }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Class found = ClassDemo$InnerClass2
Class found = ClassDemo$InnerClass1

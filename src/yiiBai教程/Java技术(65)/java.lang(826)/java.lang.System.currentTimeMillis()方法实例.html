java.lang.System.currentTimeMillis()方法实例

java.lang.System.currentTimeMillis() 方法返回当前时间(毫秒)。返回值的时间单位是1毫秒，该值的粒度取决于底层操作系统。

例如，许多操作系统测量时间在几十毫秒为单位。

声明

以下是java.lang.System.currentTimeMillis()方法的声明

public static long currentTimeMillis()

参数

  • NA

返回值

此方法返回以毫秒为单位，当前时间和午夜，1970年1月1日UTC(协调世界时)之间的时间差。

异常

  • NA

例子

下面的例子显示java.lang.System.currentTimeMillis()方法的使用。

package com.yiibai;

import java.lang.*;

public class SystemDemo {

   public static void main(String[] args) {

      // returns the current time in milliseconds
      System.out.print("Current Time in milliseconds = ");
      System.out.println(System.currentTimeMillis());
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Current Time in milliseconds = 1349333576093

java.lang.Character.isSurrogatePair()方法实例

java.lang.Character.isSurrogatePair(char high, char low) 确定指定的char对值是否是一个有效的Unicode代理对。

声明

以下是java.lang.Character.isSurrogatePair()方法的声明

public static boolean isSurrogatePair(char high, char low)

参数

  • high - 要测试的高代理代码值

  • low - 要测试的低代理代码值

返回值

如果指定的高和低代理项代码值代表一个有效的代理对此方法返回true，否则返回false。

异常

  • NA

例子

下面的例子显示lang.Character.isSurrogatePair()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 2 char primitives ch1, ch2
      char ch1, ch2;

      // assign values to ch1, ch2
      ch1 = 'ud800';
      ch2 = 'udc00';

      // create a boolean primitive b
      boolean b;

      // check if ch1, ch2 is a surrogate pair and assign result to b
      b = Character.isSurrogatePair(ch1, ch2);

      String str = "ch1, ch2 is a valid Unicode surrogate pair is " + b;

      // print b value
      System.out.println( str );
   }
}

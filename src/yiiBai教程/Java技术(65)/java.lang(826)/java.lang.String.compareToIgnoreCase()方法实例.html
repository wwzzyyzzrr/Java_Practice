java.lang.String.compareToIgnoreCase()方法实例

java.lang.String.compareToIgnoreCase() 方法按字典顺序比较两个字符串，不区分大小写的差异。

声明

以下是java.lang.String.compareToIgnoreCase()方法的声明

public int compareToIgnoreCase(String str)

参数

  • str -- 这是进行比较字符串。

返回值

此方法返回一个负整数，零或正整数；如果指定字符串大于，等于或小于这个字符串(忽略大小写)。

异常

  • NA

例子

下面的例子显示java.lang.String.compareToIgnoreCase()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str1 = "tutorials", str2 = "TUTORIALS";

    // comparing str1 and str2 with case ignored
    int retval = str1.compareToIgnoreCase(str2);

    // prints the return value of the comparison
    if (retval > 0) {
       System.out.println("str1 is greater than str2");
    }
        
    else if (retval == 0) {
       System.out.println("str1 is equal to str2");
    }
        
    else {
       System.out.println("str1 is less than str2");
    }
  }
}

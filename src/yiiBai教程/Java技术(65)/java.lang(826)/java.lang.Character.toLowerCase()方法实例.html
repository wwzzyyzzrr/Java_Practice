java.lang.Character.toLowerCase()方法实例

java.lang.Character.toLowerCase(char ch) 字符参数使用来自UnicodeData文件的大小写映射信息转换成小写。

需要注意的是Character.isLowerCase(Character.toLowerCase(ch))在字符某些范围内并不总是返回true，尤其是那些符号和象形文字。

声明

以下是java.lang.Character.toLowerCase()方法的声明

public static char toLowerCase(char ch)

参数

  • ch - 要转换字符

返回值

此方法返回的字符，如果有相应的小写; 否则返回字符本身。

异常

  • NA

例子

下面的例子显示lang.Character.toLowerCase()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 4 char primitives
      char ch1, ch2, ch3, ch4;

      // assign values to ch1, ch2
      ch1 = 'P';
      ch2 = 's';

      // assign lowercase of ch1, ch2 to ch3, ch4
      ch3 = Character.toLowerCase(ch1);
      ch4 = Character.toLowerCase(ch2);

      String str1 = "Lowercase of " + ch1 + " is " + ch3;
      String str2 = "Lowercase of " + ch2 + " is " + ch4;

      // print ch3, ch4 values
      System.out.println( str1 );
      System.out.println( str2 );
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Lowercase of P is p
Lowercase of s is s

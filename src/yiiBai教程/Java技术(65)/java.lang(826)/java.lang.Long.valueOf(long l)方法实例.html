java.lang.Long.valueOf(long l)方法实例

java.lang.Long.valueOf(long l) 方法返回一个Long实例，表示指定的long值l。

声明

以下是java.lang.Long.valueOf()方法的声明

public static Long valueOf(long l)

参数

  • l -- 这是一个long 值.

返回值

此方法返回一个Long实例，表示l。

异常

  • NA

例子

下面的例子显示java.lang.Long.valueOf()方法的使用。

package com.yiibai;

import java.lang.*;

public class LongDemo {

   public static void main(String[] args) {

     Long l = new Long(30);
   
     // returns a Long instance representing the specified long value 
     System.out.println("Value = " + l.valueOf(27666));
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Value = 27666

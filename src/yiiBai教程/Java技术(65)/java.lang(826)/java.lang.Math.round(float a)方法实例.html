java.lang.Math.round(float a)方法实例

 java.lang.Math.round(float a) 返回最接近参数的整数。其结果是通过添加1/2取其结果为地板除，并且将结果转换为int类型四舍五入为整数。特殊情况：

  • 如果参数为NaN，则结果为0。

  • 如果参数为负无穷大，或小于或等于Integer.MIN_VALUE的值的任何值，则结果等于Integer.MIN_VALUE值。

  • 如果参数是正无穷大或大于或等于Integer.MAX_VALUE的值的任何值，其结果等于Integer.MAX_VALUE值。

声明

以下是java.lang.Math.round()方法的声明

public static int round(float a)

参数

  • a -- 浮点数值被舍入为整数。

返回值

此方法返回四舍五入到最接近的int值的参数的值。

异常

  • NA

例子

下面的例子显示lang.Math.round()方法的使用。

package com.yiibai;

import java.lang.*;

public class MathDemo {

   public static void main(String[] args) {

      // get two float numbers
      float x = 1654.9874f;
      float y = -9765.134f;

      // find the closest int for these floats
      System.out.println("Math.round(" + x + ")=" + Math.round(x));
      System.out.println("Math.round(" + y + ")=" + Math.round(y));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Math.round(1654.9874f)=1655
Math.round(-9765.134f)=-9765

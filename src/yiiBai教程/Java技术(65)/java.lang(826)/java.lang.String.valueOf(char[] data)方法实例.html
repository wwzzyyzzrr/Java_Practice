java.lang.String.valueOf(char[] data)方法实例

java.lang.String.valueOf(char[] data) 方法返回将char数组参数的字符串表示形式。字符数组的内容被复制和字符数组的后续修改不会影响新创建的字符串。

声明

以下是java.lang.String.valueOf()方法的声明

public static String valueOf(char[] data)

参数

  • data -- 这是一个char数组。

返回值

这个方法返回一个代表包含在字符数组参数相同的字符序列新分配的字符串。

异常

  • NA

例子

下面的例子显示java.lang.String.valueOf()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    // character array chararray1
    char[] chararr1 = new char[] { 't', 'u', 't', 's' };
    String str1 = String.valueOf(chararr1);

    // character array chararray2
    char[] chararr2 = new char[] { '2', '1', '5' };
    String str2 = String.valueOf(chararr2);

    // prints the string representations   
    System.out.println(str1);
    System.out.println(str2);
  }
}

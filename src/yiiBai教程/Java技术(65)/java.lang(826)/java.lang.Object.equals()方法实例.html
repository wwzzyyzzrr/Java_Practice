java.lang.Object.equals()方法实例

java.lang.Object.equals(Object obj) 表示某个其他对象是否“等于”这一个对象。

类的equals方法是对象实现对象上可能差别最大的相等关系;也就是说，对于任何非空引用值x和y，当且仅当x和y引用同一个对象，这个方法返回true(x== y的值为true)。

需要注意的是，通常需要重写这个方法每当被重写hashCode方法，以保持hashCode方法，要指出，相等的对象必须具有相同的哈希码。

声明

以下是java.lang.Object.equals()方法的声明

public boolean equals(Object obj)

参数

  • obj -- 与该参照对象进行比较。

返回值

如果这个对象是与obj参数是一样的，此方法返回true; 否则为false。

异常

  • NA

例子

下面的例子显示lang.Object.equals()方法的使用。

package com.yiibai;

public class ObjectDemo {

   public static void main(String[] args) {

      // get an integer, which is an object
      Integer x = new Integer(50);

      // get a float, which is an object as well
      Float y = new Float(50f);

      // check if these are equal,which is 
      // false since they are different class
      System.out.println("" + x.equals(y));

      // check if x is equal with another int 50
      System.out.println("" + x.equals(50));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

false
true

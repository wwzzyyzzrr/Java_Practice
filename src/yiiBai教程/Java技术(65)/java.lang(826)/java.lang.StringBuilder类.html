java.lang.StringBuilder类

java.lang.StringBuilder 类是字符的可变序列。这提供了一个API和StringBuffer兼容，但不保证同步。

类声明

以下是java.lang.StringBuilder类的声明：

public final class StringBuilder
  extends Object
    implements Serializable, CharSequence

java.lang.Class.getDeclaredClasses()方法实例

java.lang.Class.getDeclaredClasses() 方法返回一个Class对象，包括公共，保护，默认(包)访问和私有类和类中声明的接口的数组，但不包括继承的类和接口。

如果类没有声明的类或接口的成员，或如果此Class对象表示一个基本类型，一个数组类或void此方法返回一个长度为0的数组。

声明

以下是java.lang.Class.getDeclaredClasses()方法的声明

public Class<?>[] getDeclaredClasses() throws SecurityException

参数

  • NA

返回值

此方法返回一个代表这个类的所有成员声明Class对象的数组。

异常

  • SecurityException -- 如果安全管理存在。

例子

下面的例子显示java.lang.Class.getDeclaredClasses()方法的使用。

package com.yiibai;

import java.lang.*;

public class ClassDemo {

   public static void main(String[] args) {
 
     try {
        // returns the Class object associated with this class
        Class cls = Class.forName("ClassDemo");

        /* returns the array of Class objects representing all the declared 
        members of this class */
        Class[] classes = cls.getDeclaredClasses();
        for (int i = 0; i < classes.length; i++) {
           System.out.println("Class = " + classes[i].getName());
        }
     }
     catch (ClassNotFoundException e) {
        System.out.println(e.toString());
     }
   }

   public class InnerClass1 {
      public InnerClass1() {
         System.out.println("Inner Class1");
      }
   }

   public class InnerClass2 {
      public InnerClass2() {
         System.out.println("Inner Class2");
      }
   }

   private class InnerPrivateClass {
      public InnerPrivateClass() {
         System.out.println("Inner Private Class");
      }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Class = ClassDemo$InnerPrivateClass
Class = ClassDemo$InnerClass2
Class = ClassDemo$InnerClass1

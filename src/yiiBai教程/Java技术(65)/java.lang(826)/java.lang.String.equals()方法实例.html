java.lang.String.equals()方法实例

 java.lang.String.equals() 方法此字符串指定的对象进行比较。其结果是true，当且仅当参数不为null，并且是一个String对象，它代表的字符与此对象有相同的顺序。

声明

以下是java.lang.String.equals()方法的声明

public boolean equals(Object anObject)

参数

  • anObject -- 这是比较对象的字符串。

返回值

如果给定对象表示的字符串等于该字符串此方法返回true，否则返回false。

异常

  • NA

例子

下面的例子显示java.lang.String.equals()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str1 = "sachin tendulkar";
    String str2 = "amrood admin";
    String str3 = "amrood admin";

    // checking for equality
    boolean retval1 = str2.equals(str1);
    boolean retval2 = str2.equals(str3);

    // prints the return value
    System.out.println("str2 is equal to str1 = " + retval1);
    System.out.println("str2 is equal to str3 = " + retval2);        
 }
}

让我们来编译和运行上面的程序，这将产生以下结果：

str2 is equal to str1 = false
str2 is equal to str3 = true

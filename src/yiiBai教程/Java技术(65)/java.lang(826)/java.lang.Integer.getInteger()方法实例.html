java.lang.Integer.getInteger()方法实例

java.lang.Integer.getInteger(String nm) 方法确定具有指定名称的系统属性的整数值。

声明

以下是java.lang.Integer.getInteger()方法的声明

public static Integer getInteger(String nm)

参数

  • nm -- 这是属性的名称。

返回值

此方法返回属性的整数值。

异常

  • NA

例子

下面的例子显示java.lang.Integer.getInteger()方法的使用。

package com.yiibai;

import java.lang.*;

public class IntegerDemo {

   public static void main(String[] args) {

     // determines the integer value of the system property 
     String str = "sun.arch.data.model";
     System.out.println(Integer.getInteger(str));

     // prints null
     str = "java";
     System.out.println(Integer.getInteger(str));
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

64
null

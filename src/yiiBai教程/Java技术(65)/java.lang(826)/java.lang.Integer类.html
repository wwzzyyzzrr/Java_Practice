java.lang.Integer类

java.lang.Integer 类封装了基本类型int的值在一个对象。 Integer类型的对象包含一个字段，它的类型是int。

类声明

以下是声明了java.lang.Integer类：

public final class Integer
  extends Number
    implements Comparable<Integer>

java.lang.StringBuilder.insert(int offset, char[] str)方法实例

java.lang.StringBuilder.insert(int offset, char[] str) 方法将char数组参数插入到这个序列的字符串表示形式。
数组参数中的字符插入此序列中offset所指示的位置的内容。这个序列的长度增加该参数的长度。

声明

以下是java.lang.StringBuilder.insert()方法的声明

public StringBuilder insert(int offset, char[] str)

参数

  • offset -- 这是偏移量。

  • str -- 这是字符数组。

返回值

该方法返回这个对象的一个引用。

异常

  • StringIndexOutOfBoundsException -- 如果偏移量是无效的。

例子

下面的例子显示java.lang.StringBuilder.insert()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBuilderDemo {

  public static void main(String[] args) {
  
    StringBuilder str = new StringBuilder("tutorials website");
    System.out.println("string = " + str);
        
    // character array to be inserted
    char cArr[] = { 'y', 'i', 'i', 'b', 'a' };
    // insert character array at offset 9
    str.insert(9, cArr);
        
    // prints StringBuilder after insertion
    System.out.print("After insertion = ");
    System.out.println(str.toString());
  }      
}

让我们来编译和运行上面的程序，这将产生以下结果：

string = tutorials website
After insertion = yiibai website

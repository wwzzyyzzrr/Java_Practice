java.lang.StringBuffer.append(String str)方法实例

java.lang.StringBuffer.append(String str) 方法将指定的字符串追加此字符序列。 String参数中的字符追加，按顺序由参数的长度增加到该序列的长度。如果str为null，则这四个字符“null”被追加。

声明

以下是java.lang.StringBuffer.append()方法的声明

public StringBuffer append(String str)

参数

  • str -- 这是一个字符串的值。

返回值

该方法返回一个此对象引用。

异常

  • NA

例子

下面的例子显示java.lang.StringBuffer.append()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBufferDemo {

  public static void main(String[] args) {
  
    StringBuffer buff = new StringBuffer("tutorials ");
    System.out.println("buffer = " + buff);
    
    // appends the string argument to the string buffer
    buff.append("yiibai");
    // print the string buffer after appending
    System.out.println("After append = " + buff);
    
    buff = new StringBuffer("1234 ");
    System.out.println("buffer = " + buff);
    // appends the string argument to the string buffer 
    buff.append("!#$%");
    // print the string buffer after appending
    System.out.println("After append = " + buff);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

buffer = tutorials
After append = tutorials yiibai
buffer = 1234
After append = 1234 !#$%

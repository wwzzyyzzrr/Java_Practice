java.lang.Float.doubleValue()方法实例

java.lang.Float.doubleValue() 方法返回该Float对象的double值。

声明

以下是java.lang.Float.doubleValue()方法的声明

public double doubleValue()

参数

  • NA

返回值

此方法返回当前对象表示的float值转换为double类型和转换的结果。

异常

  • NA

例子

下面的例子显示java.lang.Float.doubleValue()方法的使用。

package com.yiibai;

import java.lang.*;

public class FloatDemo {

   public static void main(String[] args) {

     /* returns the float value represented by this object
     converted to type double */
     Float obj = new Float("22.10f");
     double d = obj.doubleValue();
     System.out.println("Value = " + d);
    
     obj = new Float("5.0");
     d = obj.doubleValue();
     System.out.println("Value = " + d);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Value = 22.100000381469727
Value = 5.0

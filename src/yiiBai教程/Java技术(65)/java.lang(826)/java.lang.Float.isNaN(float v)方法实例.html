java.lang.Float.isNaN(float v)方法实例

java.lang.Float.isNaN(float v) 如果指定的数不是非数字(NAN)的值方法返回true，否则返回false。该参数v是要测试的值。

声明

以下是java.lang.Float.isNaN()方法的声明

public static boolean isNaN(float v)

参数

  • v -- 这是要测试的值。

返回值

如果参数为NaN此方法返回true，否则返回false。

异常

  • NA

例子

下面的例子显示java.lang.Float.isNaN()方法的使用。

package com.yiibai;

import java.lang.*;

public class FloatDemo {

   public static void main(String[] args) {

     Float f1 = new Float(-1.0/0.0);
     Float f2 = new Float(0.0/0.0);
  
     // returns true if this Float value is a Not-a-Number(NaN), else false 
     System.out.println(f1 + " = " + f1.isNaN(f1));
  
     System.out.println(f2 + " = " + f2.isNaN(f2));
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

-Infinity = false
NaN = true

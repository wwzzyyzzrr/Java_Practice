java.lang.Integer.toOctalString()方法实例

java.lang.Integer.toOctalString() 方法返回一个整数参数的作为以8为基数下列字符的无符号整数的字符串表示形式作为八进制数字：01234567

声明

以下是java.lang.Integer.toOctalString()方法的声明

public static String toOctalString(int i)

参数

  • i -- 这是要转换为字符串的整数。

返回值

此方法返回由八进制的参数(基数8)表示的无符号整型值的字符串表示形式。

异常

  • NA

例子

下面的例子显示java.lang.Integer.toOctalString()方法的使用。

package com.yiibai;

import java.lang.*;

public class IntegerDemo {

   public static void main(String[] args) {

     int i = 170;
     System.out.println("Number = " + i);
    
     /* returns the string representation of the unsigned integer value
     represented by the argument in octal (base 8) */
     System.out.println("Octal = " + Integer.toOctalString(i));
   }  
}

让我们来编译和运行上面的程序，这将产生以下结果：

Number = 170
Octal = 252

java.lang.String.compareTo()方法实例

java.lang.String.compareTo() 方法按字典顺序比较两个字符串。该比较是基于所述字符串的每个字符的Unicode值。这个String对象表示的字符序列按字典顺序进行比较的参数字符串所表示的字符序列。

  • 其结果是负整数，如果该对象的字符串按字典顺序前面的参数字符串。
  • 其结果是正整数，如果这个String对象字典参数字符串顺序。
  • 其结果是零，如果两个字符串相等，compareTo返回0。

声明

以下是java.lang.String.compareTo()方法的声明

public int compareTo(String anotherString)

参数

  • anotherString -- 这是字符串进行比较。

返回值

此方法如果该参数字符串等于此字符串返回值0，如果该字符串是按字典顺序比字符串参数少，返回值小于0，如果这个字符串是按字典顺序比字符串参数大则返回一个大于0的值。

异常

  • NA

例子

下面的例子显示java.lang.String.compareTo()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str1 = "tutorials", str2 = "point";

    // comparing str1 and str2
    int retval = str1.compareTo(str2);

    // prints the return value of the comparison
    if (retval < 0) {
       System.out.println("str1 is greater than str2");
    }
        
    else if (retval == 0) {
       System.out.println("str1 is equal to str2");
    }
        
    else {
       System.out.println("str1 is less than str2");
    }
  }
}

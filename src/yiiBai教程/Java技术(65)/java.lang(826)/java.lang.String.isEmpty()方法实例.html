java.lang.String.isEmpty()方法实例

java.lang.String.isEmpty() 只有length()为0方法返回true。

声明

以下是java.lang.String.isEmpty()方法的声明

public boolean isEmpty()

参数

  • NA

返回值

如果length()为0此方法返回true，否则返回false。

异常

  • NA

例子

下面的例子显示java.lang.String.isEmpty()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str = "yiibai";
   
    // prints length of string
    System.out.println("length of string = " + str.length());
    
    // checks if the string is empty or not
    System.out.println("is this string empty? = " + str.isEmpty());
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

length of string = 14
is this string empty? = false

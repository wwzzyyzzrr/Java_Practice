java.lang.Long.rotateLeft()方法实例

java.lang.Long.rotateLeft() 方法返回通过旋转，由位指定数左移指定long值的二进制补码表示法得到的值。 (位移出左手的，或高次，再输入侧在右边，或低的顺序。)

声明

以下是java.lang.Long.rotateLeft()方法的声明

public static long rotateLeft(long i, int distance)

参数

  • i -- 这是long 值。

  • distance -- 这是旋转的距离。

返回值

下面的例子显示java.lang.Long.rotateLeft()方法的使用。

异常

  • NA

例子

下面的例子显示java.lang.Long.rotateLeft()方法的使用。

package com.yiibai;

import java.lang.*;

public class LongDemo {

   public static void main(String[] args) {

     long n = 64;

     // returns the value obtained by rotating left
     for(int i = 0; i < 4; i++) {
        n = Long.rotateLeft(n, 4);
        System.out.println(n);
     }
   }
}

java.lang.StringBuilder.substring(int start)方法实例

java.lang.StringBuilder.substring(int start) 方法返回一个新的String，它包含的字符目前包含在此字符序列子序列。该子字符串从指定的索引处，并延伸至该序列的结尾。

声明

以下是java.lang.StringBuilder.substring()方法的声明

public String substring(int start)

参数

  • start -- 这是开始索引(包括)。

返回值

此方法返回新的字符串。

异常

  • StringIndexOutOfBoundsException -- 如果起始小于零,或比该对象的长度大。

例子

下面的例子显示java.lang.StringBuilder.substring()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBuilderDemo {

  public static void main(String[] args) {
  
    StringBuilder str = new StringBuilder("admin");
    System.out.println("string = " + str);
    
    // prints substring from index 2
    System.out.println("substring = "+ str.substring(2));
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

string = admin
substring = min

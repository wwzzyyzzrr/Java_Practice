java.lang.String.concat()方法实例

java.lang.String.concat() 方法将指定的字符串 拼接该字符串的结尾。

声明

以下是java.lang.String.concat()方法的声明

public String concat(String str)

参数

  • str -- 这是连结到该字符串的结尾的字符串。

返回值

此方法返回表示由字符串参数的字符此对象字符拼接字符串。

异常

  • NA

例子

下面的例子显示java.lang.String.concat()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    // print str1
    String str1 = "self";
    System.out.println(str1);
 
    // print str2 concatenated with str1
    String str2 = str1.concat(" learning");
    System.out.println(str2);
   
    // prints str3 concatenated with str2(and str1)
    String str3 = str2.concat(" center");
    System.out.println(str3);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

self
self learning
self learning center

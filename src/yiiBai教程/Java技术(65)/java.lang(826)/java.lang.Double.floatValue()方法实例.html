java.lang.Double.floatValue()方法实例

java.lang.Double.floatValue() 方法返回当前Double对象的浮点值。

声明

以下是java.lang.Double.floatValue()方法的声明

public float floatValue()

参数

  • NA

返回值

此方法返回该对象表示double值转换为float类型。

异常

  • NA

例子

下面的例子显示java.lang.Double.floatValue()方法的使用。

package com.yiibai;

import java.lang.*;

public class DoubleDemo {

   public static void main(String[] args) {

     // returns the float value of this Double object
     Double obj = new Double("5.60");
     float f = obj.floatValue();
     System.out.println("Value = " + f);
    
     obj = new Double("32");
     f = obj.floatValue();
     System.out.println("Value = " + f);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Value = 5.6
Value = 32.0

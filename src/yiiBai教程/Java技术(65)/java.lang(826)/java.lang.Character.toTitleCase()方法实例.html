java.lang.Character.toTitleCase()方法实例

java.lang.Character.toTitleCase(char ch) 字符参数使用来自UnicodeData文件的大小写映射信息标题字符转换。

如果字符没有明确的首字母大写映射，不是本身根据UnicodeData就是一个首字母大写字符，则大写字母映射返回为一个等效的首字母大写映射。如果char参数已经是一个首字母大写字符，相同的字符值将被返回。

需要注意的是Character.isTitleCase(Character.toTitleCase(ch)) 在某些范围并不总是返回true字符。

声明

以下是java.lang.Character.toTitleCase()方法的声明

public static char toTitleCase(char ch)

参数

  • ch - 要转换字符

返回值

此方法返回首字母大写等同的字符，如果有的话;否则返回该字符本身。

异常

  • NA

例子

下面的例子显示lang.Character.toTitleCase()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 4 char primitives
      char ch1, ch2, ch3, ch4;

      // assign values to ch1, ch2
      ch1 = 'p';
      ch2 = '+';

      // assign titlecase of ch1, ch2 to ch3, ch4
      ch3 = Character.toTitleCase(ch1);
      ch4 = Character.toTitleCase(ch2);

      String str1 = "Titlecase of " + ch1 + " is " + ch3;
      String str2 = "Titlecase of " + ch2 + " is " + ch4;

      // print ch3, ch4 values
      System.out.println( str1 );
      System.out.println( str2 );
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Titlecase of p is P
Titlecase of + is +

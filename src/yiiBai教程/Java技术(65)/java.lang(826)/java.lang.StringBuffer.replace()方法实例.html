java.lang.StringBuffer.replace()方法实例

java.lang.StringBuffer.replace() 方法按照这个顺序，在指定的字符串中的子字符串替换字符。
子串开始在指定位置开始，并延伸到该字符的索引end - 1，或如果没有这样的字符存在到达序列的末端。在子字符串的字符首先被删除，然后在指定的字符串被插入在start的位置。

声明

以下是java.lang.StringBuffer.replace()方法的声明

public StringBuffer replace(int start, int end, String str)

参数

  • start -- 这是开始索引(包括)。

  • end -- 这就是结束索引(不包括)。

  • str -- 这是字符串，它将取代以前的内容。

返回值

此方法返回这个对象。

异常

  • StringIndexOutOfBoundsException --如果start为负，end大于length()，或大于end。

例子

下面的例子显示java.lang.StringBuffer.replace()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringBufferDemo {

  public static void main(String[] args) {
  
    StringBuffer buff = new StringBuffer("Java Util Package");
    System.out.println("buffer = " + buff);
    
    // replace substring from index 5 to index 9
    buff.replace(5, 9, "Lang");
    // prints the stringbuffer after replacing
    System.out.println("After replacing: " + buff);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

buffer = Java Util Package
After replacing: Java Lang Package

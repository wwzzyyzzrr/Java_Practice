java.lang.Math.acos()方法实例

java.lang.Math.acos(double a) 返回角度的余弦值，范围在0.0到pi之间。如果参数为NaN或它的绝对值大于1，那么结果为NaN。一个结果必须在1 ulp正确舍入的结果。结果必须具有半单调性。

声明

以下是声明java.lang.Math.acos()方法

public static double acos(double a)

参数

  • a -- 要返回其余弦值的参数。

返回值

此方法返回参数的反余弦值。

异常

  • NA

例子

下面的例子显示lang.Math.acos()方法的使用。

package com.yiibai;

import java.lang.*;

public class MathDemo {

   public static void main(String[] args) {

      // get a variable x which is equal to PI/2
      double x = Math.PI / 2;

      // convert x to radians
      x = Math.toRadians(x);

      // get the arc cosine of x
      System.out.println("Math.acos(" + x + ")=" + Math.acos(x));
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Math.acos(0.027415567780803774)=1.5433773235341761

java.lang.ProcessBuilder.redirectErrorStream(boolean redirectErrorStream)方法实例

java.lang.ProcessBuilder.redirectErrorStream(boolean redirectErrorStream) 方法设置此进程生成器redirectErrorStream的属性。如果此属性为true，则通过子进程所产生的任何错误输出随后由该对象的start()方法启动将与标准输出合并，这样既可以用Process.getInputStream()方法来读取。此使得更容易与对应的输出相关的错误消息。初始值是false。

声明

以下是java.lang.ProcessBuilder.redirectErrorStream()方法的声明

public ProcessBuilder redirectErrorStream(boolean redirectErrorStream)

参数

  • redirectErrorStream -- 新的属性值

返回值

此方法返回此进程生成器

异常

  • NA

例子

下面的例子显示lang.ProcessBuilder.redirectErrorStream()方法的使用。

package com.yiibai;

public class ProcessBuilderDemo {

   public static void main(String[] args) {

      // create a new list of arguments for our process
      String[] list = {"notepad.exe", "test.txt"};

      // create the process builder
      ProcessBuilder pb = new ProcessBuilder(list);

      // Redirect the errorstream
      pb.redirectErrorStream(true);
      System.out.println(""+pb.redirectErrorStream());
   }
}

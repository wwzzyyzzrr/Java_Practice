java.lang.ClassLoader.getResource()方法实例

java.lang.ClassLoader.getResource() 方法找到具有给定名称的资源。资源是一些数据(图像，音频，文本等)，可以通过类代码中的方式，是独立的代码的位置的访问。资源的名称是“/” - 标识资源分离路径名。

声明

以下是java.lang.ClassLoader.getResource()方法的声明

public URL getResource(String name)

参数

  • name -- 资源的名称。

返回值

此方法返回URL对象读取资源，则返回null如果资源无法找到或调用者没有足够的权限来获取资源。

异常

  • NA

例子

下面的例子显示java.lang.ClassLoader.getResource()方法的使用。

package com.yiibai;

import java.lang.*;
import java.net.URL;

public class ClassLoaderDemo {

   public static void main(String[] args) throws Exception {
     
     Class cls = Class.forName("ClassLoaderDemo");

     // returns the ClassLoader object associated with this Class
     ClassLoader cLoader = cls.getClassLoader();
    
     System.out.println(cLoader.getClass());
    
     // finds resource with the given name
     URL url = cLoader.getResource("file.txt");
     System.out.println("Value = " + url);

     // finds resource with the given name
     url = cLoader.getResource("newfolder/a.txt");
     System.out.println("Value = " + url);  
   }
}

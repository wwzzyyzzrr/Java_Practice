java.lang.Short.floatValue()方法实例

java.lang.Short.floatValue() 方法返回这个Short的浮点数值。

声明

以下是java.lang.Short.floatValue()方法的声明

public float floatValue()

参数

  • NA

返回值

此方法返回该对象转换后表示float类型的数值。

异常

  • NA

例子

下面的例子显示java.lang.Short.floatValue()方法的使用。

package com.yiibai;

import java.lang.*;

public class ShortDemo {

   public static void main(String[] args) {

     // create short object and assign value to it
     short sval = 100;
     Short sobj = new Short(sval);
    
     // returns float value of Short
     double floatValue = sobj.floatValue();
    
     // printing float value
     System.out.println("float value of Short is = " + floatValue);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

float value of Short is = 100.0

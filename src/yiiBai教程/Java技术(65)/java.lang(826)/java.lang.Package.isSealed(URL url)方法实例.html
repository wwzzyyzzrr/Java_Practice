java.lang.Package.isSealed(URL url)方法实例

java.lang.Package.isSealed(URL url) 方法返回true，如果对于指定的代码源URL这个包是密封。

声明

以下是java.lang.Package.isSealed()方法的声明

public boolean isSealed(URL url)

参数

  • url -- 该代码源URL

返回值

如果相对URL这个包是密封的此方法返回true

异常

  • NA

例子

下面的例子显示lang.Object.isSealed(​​)方法的使用。

package com.yiibai;

public class PackageDemo {

   public static void main(String[] args) {
      try {
         // get the java lang package
         Package pack = Package.getPackage("java.lang");

         // check if this package is sealed
         URL url = new URL("http://www.yiibai.com");
         System.out.println("" + pack.isSealed(url));
      } catch (MalformedURLException ex) {
         ex.printStackTrace();
      }

   }
}

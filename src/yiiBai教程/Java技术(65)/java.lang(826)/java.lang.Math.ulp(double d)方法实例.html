java.lang.Math.ulp(double d)方法实例

java.lang.Math.ulp(double d) 返回参数的ulp大小。 double值的ulp是该浮点值与下一个数值较大double值之间的正距离。请注意，对于非NaN x, ulp(-x) == ulp(x)。特殊情况：

  • 如果参数为NaN，那么结果为NaN。

  • 如果参数为正或负无穷大，那么结果为正无穷大。

  • 如果参数为正或负0，那么结果是Double.MIN_VALUE。

  • 如果参数为Double.MAX_VALUE，那么结果等于 2971.

声明

以下是java.lang.Math.ulp()方法的声明

public static double ulp(double d)

参数

  • d -- 要返回浮点的ulp值

返回值

此方法返回参数的ulp大小

异常

  • NA

例子

下面的例子显示lang.Math.ulp()方法的使用。

package com.yiibai;

import java.lang.*;

public class MathDemo {

   public static void main(String[] args) {

      // get two double numbers numbers
      double x = 956.294;
      double y = 123.1;

      // print the ulp of these doubles
      System.out.println("Math.ulp(" + x + ")=" + Math.ulp(x));
      System.out.println("Math.ulp(" + y + ")=" + Math.ulp(y));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Math.ulp(956.294)=1.1368683772161603E-13
Math.ulp(123.1)=1.4210854715202004E-14

java.lang.Class.getConstructors()方法实例

java.lang.Class.getConstructors() 方法返回一个包含某些Constructor对象反映此Class对象所表示的类所有公共构造函数的数组。如果类没有公共的构造函数那么返回长度为0的数组，或者类是一个数组类，或者类反映了一个基本类型或void。

声明

以下是java.lang.Class.getConstructors()方法的声明

public Constructor<?>[] getConstructors() throws SecurityException

参数

  • NA

返回值

此方法返回一个代表这个类的公共构造函数构造对象的数组。

异常

  • SecurityException -- 如果安全管理存在。

例子

下面的例子显示java.lang.Class.getConstructors()方法的使用。

package com.yiibai;

import java.lang.reflect*;

public class ClassDemo {

   public static void main(String[] args) {

     try {
        Class cls = Class.forName("java.awt.Panel");
        System.out.println("Panel Constructors =");

        /* returns the array of Constructor objects representing the public 
        constructors of this class */
        Constructor c[] = cls.getConstructors();
        for(int i = 0; i < c.length; i++) {
           System.out.println(c[i]);
        }
     }
     catch (Exception e) {
        System.out.println("Exception: " + e);
    }
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Panel Constructors =
public java.awt.Panel()
public java.awt.Panel(java.awt.LayoutManager)

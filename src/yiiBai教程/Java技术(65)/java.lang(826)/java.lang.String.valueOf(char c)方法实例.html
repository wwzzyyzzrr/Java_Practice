java.lang.String.valueOf(char c)方法实例

 java.lang.String.valueOf(char c) 方法返回char参数的字符串表示形式。

声明

以下是java.lang.String.valueOf()方法的声明

public static String valueOf(char c)

参数

  • c -- 这是一个char值。

返回值

此方法返回长度为1，包含作为其单个字符参数c的字符串。

异常

  • NA

例子

下面的例子显示java.lang.String.valueOf()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str1 = String.valueOf('k');
    String str2 = String.valueOf('m');

    // print the string representation of char
    System.out.println(str1);
    System.out.println(str2);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

k
m

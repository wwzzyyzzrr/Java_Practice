java.lang.InheritableThreadLocal类

java.lang.InheritableThreadLocal 类扩展了ThreadLocal从父线程提供的值来继承子线程：创建一个子线程的时候，子的所有可继承线程局部变量可以是其父有接收值的初始值。

   extends ThreadLocal<T>

java.lang.Long.decode()方法实例

java.lang.Long.decode() 方法解码一个字符串转化成一个Long。它接受十进制，十六进制和八进制数。

声明

以下是java.lang.Long.decode()方法的声明

public static Long decode(String nm) throws NumberFormatException

参数

  • nm -- 这是字符串进行解码。

返回值

该方法返回一个Long对象持有nm所代表的Long 值。

异常

  • NumberFormatException -- 如果该串不包含一个可分析的long。

例子

下面的例子显示java.lang.Long.decode()方法的使用。

package com.yiibai;

import java.lang.*;

public class LongDemo {

   public static void main(String[] args) {

     Long l = new Long(10);
    
     String str = "57820";
     /* returns Long object holding the long value represented by 
     string str */
     System.out.println("Number = " + l.decode(str)); 
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Number = 57820

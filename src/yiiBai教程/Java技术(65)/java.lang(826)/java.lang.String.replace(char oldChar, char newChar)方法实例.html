java.lang.String.replace(char oldChar, char newChar)方法实例

java.lang.String.replace(char oldChar, char newChar) 方法返回通过用newChar更换oldChar所有出现在此字符串产生一个新的字符串。

声明

以下是java.lang.String.replace()方法声明

public String replace(char oldChar, char newChar)

参数

  • oldChar -- 这是旧字符。

  • newChar -- 这是新的字符。

返回值

此方法返回通过更换oldChar每次出现通过用newChar来替换此字符串的字符串。

异常

  • NA

例子

下面的例子显示java.lang.String.replace()方法的使用。

package com.yiibai;

import java.lang.*;

public class StringDemo {

  public static void main(String[] args) {
  
    String str = "aacdeaa";
    System.out.println("string = " + str);

    // replace all 'a' characters with 'b' characters.
    String replaceStr = str.replace('a', 'b');

    // prints the strings after replacement
    System.out.println("new string = " + replaceStr);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

string = aacdeaa
new string = bbcdebb

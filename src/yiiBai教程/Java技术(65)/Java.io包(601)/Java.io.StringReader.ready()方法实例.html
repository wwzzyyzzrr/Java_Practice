Java.io.StringReader.ready()方法实例

java.io.StringReader.ready() 方法通知此流是否已准备好被读取。

声明

以下是java.io.StringReader.ready()方法的声明

public boolean ready()

参数

  • NA

返回值

如果下一个read()保证输入不阻塞此方法返回true

异常

  • IOException -- 如果发生I/O错误

例子

下面的示例演示java.io.StringReader.ready()方法的用法。

package com.yiibai;

import java.io.*;

public class StringReaderDemo {

   public static void main(String[] args) {

      String s = "Hello World";

      // create a new StringReader
      StringReader sr = new StringReader(s);

      try {
         // check if reader is ready
         System.out.println("" + sr.ready());

         // read the first five chars
         for (int i = 0; i < 5; i++) {
            char c = (char) sr.read();
            System.out.print("" + c);
         }

         // close the stream
         sr.close();

      } catch (IOException ex) {
         ex.printStackTrace();
      }
   }
}

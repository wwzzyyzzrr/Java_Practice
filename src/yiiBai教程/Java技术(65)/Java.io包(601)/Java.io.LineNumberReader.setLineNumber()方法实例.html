Java.io.LineNumberReader.setLineNumber()方法实例

java.io.LineNumberReader.setLineNumber(int lineNumber) 方法将当前行号。

声明

以下是java.io.LineNumberReader.setLineNumber(int lineNumber) 方法的声明：

public void setLineNumber(int lineNumber)

参数

  • lineNumber -- 为一个int值，指定要设置的当前行号。

返回值

该方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.LineNumberReader.setLineNumber(int lineNumber) 方法的用法。

package com.yiibai;

import java.io.FileReader;
import java.io.IOException;
import java.io.LineNumberReader;

public class LineNumberReaderDemo {
   public static void main(String[] args) throws IOException {
      
      FileReader fr = null;
      LineNumberReader lnr = null;
      int i;
      
      try{
         // create new reader
         fr = new FileReader("C:/test.txt");
         lnr = new LineNumberReader(fr);
         
         // set current line number
         lnr.setLineNumber(100);
         System.out.println("setLineNumber(100) is invoked");
         
         // get current line number
         i = lnr.getLineNumber();
         System.out.print("Current line number: "+i);
         
      }catch(Exception e){
         
         // if any error occurs
         e.printStackTrace();
      }finally{
         
         // closes the stream and releases system resources
         if(fr!=null)
            fr.close();
         if(lnr!=null)
            lnr.close();
      }
   }
}

假设我们有一个文本文件c:/ test.txt，它具有以下内容。该文件将被用作输入到我们的示例程序：

ABCDE

让我们编译和运行上面的程序，这将产生以下结果：

setLineNumber(100) is invoked
Current line number: 100

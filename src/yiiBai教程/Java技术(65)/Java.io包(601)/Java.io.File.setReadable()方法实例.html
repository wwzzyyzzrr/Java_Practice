Java.io.File.setReadable()方法实例

java.io.File.setReadable(boolean readable) 方法设置此抽象路径名的存取权限。

声明

以下是java.io.File.setReadable(boolean readable) 方法的声明：

public boolean setReadable(boolean readable)

参数

  • readable -- true时是设置访问在未经许可允许读操作。

返回值

如果操作成功此方法返回true，否则返回false。

异常

  • SecurityException -- 如果安全管理器存在并且其方法拒绝写访问无论是旧的或新的路径名。

例子

下面的示例演示java.io.File.setReadable(boolean readable) 方法的用法。

package com.yiibai;

import java.io.File;

public class FileDemo {
   public static void main(String[] args) {
      
      File f = null;
      boolean bool = false;
      
      try{      
         // create new File object
         f = new File("C:/test.txt");
         
         // returns true if file exists
         bool = f.exists();
         
         // if file exists
         if(bool)
         {
            // set read permission
            bool = f.setReadable(true);
            
            // print
            System.out.println("setReadable() succeeded?: "+bool);
            
            // checks whether the file is readable
            bool  = f.canRead();
            
            // prints
            System.out.print("Is file readable?: "+bool);
         }      
      }catch(Exception e){
         // if any error occurs
         e.printStackTrace();
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

setReadable() succeeded?: true
Is file readable?: true

Java.io.PrintWriter.append()方法实例

java.io.PrintWriter.append() 方法将指定字符追加到这个writer。

声明

以下是java.io.PrintWriter.append()方法声明

public PrintWriter append(char c)

参数

  • c -- 16位字符以附加

返回值

此方法返回这个写入器

异常

  • NA

例子

下面的示例演示java.io.PrintWriter.append()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintWriterDemo {

   public static void main(String[] args) {
      try {

         // create a new stream at system
         PrintWriter pw = new PrintWriter(System.out);

         // append chars
         pw.append('H');
         pw.append('e');
         pw.append('l');
         pw.append('l');
         pw.append('o');

         // flush the writer
         pw.flush();

      } catch (Exception ex) {
         ex.printStackTrace();
      }
   }
}

Java.io.SerializablePermission类

Java.io.SerializablePermission类是可序列化的权限。SerializablePermission包含一个名称(也称为“target name”)，但没有动作列表;要么有命名的权限，也可以没有。目标名称是可序列化权限的名称。

类声明

以下是Java.io.SerializablePermission类的声明：

public final class SerializablePermission
  extends BasicPermission

类构造函数

S.N. 构造函数 & 描述
1 SerializablePermission(String name) 
这将创建一个新的SerializablePermission具有指定名称。
2 SerializablePermission(String name, String actions) 
这将创建具有指定名称的新的SerializablePermission对象。

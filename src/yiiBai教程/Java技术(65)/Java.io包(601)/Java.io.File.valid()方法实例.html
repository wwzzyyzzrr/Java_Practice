Java.io.File.valid()方法实例

java.io.File.valid() 如果方法测试文件描述符对象是有效的。

声明

以下是java.io.File.valid()方法的声明：

public boolean valid()

参数

  • NA

返回值

如果文件描述符对象是有效的该方法返回true，否则该方法返回false。

异常

  • NA

例子

下面的示例演示java.io.File.valid()方法的用法。

package com.yiibai;

import java.io.FileDescriptor;
import java.io.FileInputStream;
import java.io.IOException;

public class FileDemo {
   public static void main(String[] args) throws IOException {
      
      FileInputStream fis = null;
      FileDescriptor fd = null;
      boolean bool = false;   
      
      try{
         // create input stream
         fis = new FileInputStream("c:/java test.txt");
         
         // get file descriptor
         fd = fis.getFD();
         
         // tests file descriptor object's validity
         bool = fd.valid();
         
         // print
         System.out.print("is file descriptor valid?: "+bool);
         
      }catch(Exception e){
         // if any error occurs
         e.printStackTrace();
      }finally{
         // releases systems resources
         if(fis!=null)
            fis.close();   
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

is file descriptor valid?: true

Java.io.DataOutputStream.writeByte()方法实例

java.io.BufferedInputStream.writeByte(int v) 方法写入一个字节到底层流为1个字节的值。成功执行此方法该计数器加1递增。

声明

以下是java.io.DataOutputStream.writeByte(int v)方法的声明：

public final void writeByte(int v)

参数

  • v -- 一个字节值写入到流。

返回值

此方法不返回任何值。

异常

  • IOException -- 如果发生I/ O错误。

例子

下面的示例演示java.io.DataOutputStream.writeByte(int v)方法的用法。

package com.yiibai;

import java.io.ByteArrayOutputStream;
import java.io.DataOutputStream;
import java.io.IOException;

public class DataOutputStreamDemo {
   public static void main(String[] args) throws IOException {
      
      ByteArrayOutputStream baos = null;
      DataOutputStream dos = null;
      byte[] buf = {12, 15, 30, 40, 5, 7};
      
      try{
         // create byte array output stream
         baos = new ByteArrayOutputStream();
         
         // create data output stream
         dos = new DataOutputStream(baos);
         
         // write to the output stream from the buffer
         for(byte b: buf)
         {
            dos.writeByte(b);
         }
         // flushes bytes to underlying output stream
         dos.flush();
   
         // for each byte in the buffer content
         for(byte b:baos.toByteArray())
         {   
            // print character
            System.out.print(b + " ");
         }
      }catch(Exception e){
         // if any error occurs
         e.printStackTrace();
      }finally{
         
         // releases all system resources from the streams
         if(baos!=null)
            baos.close();
         if(dos!=null)
            dos.close();
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

12 15 30 40 5 7

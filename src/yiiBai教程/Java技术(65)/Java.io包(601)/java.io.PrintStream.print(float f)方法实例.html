java.io.PrintStream.print(float f)方法实例

java.io.PrintStream.print() 方法打印一个浮点数。通过String.valueOf(float)生成的字符串转换成字节按照平台的默认字符编码，这些字节跟被写在write(int)方法方式相同。

声明

以下是java.io.PrintStream.print()方法的声明

public void print(float f)

参数

  • f -- 要被打印的float

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.PrintStream.print()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo {

   public static void main(String[] args) {
      float x = 1.89f;

      // create printstream object
      PrintStream ps = new PrintStream(System.out);

      // print float
      ps.print(x);

      // flush the stream
      ps.flush();

   }
}

让我们编译和运行上面的程序，这将产生以下结果：

1.89

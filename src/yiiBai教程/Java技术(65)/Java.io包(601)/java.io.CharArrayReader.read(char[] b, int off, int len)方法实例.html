java.io.CharArrayReader.read(char[] b, int off, int len)方法实例

java.io.CharArrayReader.read(char[] b, int off, int len) 方法读取字符到指定数组的一部分。

声明

以下是java.io.CharArrayReader.read(char[] b, int off, int len) 方法的声明：

public int read(char[] b, int off, int len)

参数

  • b -- 目标数组。

  • off -- 偏移量开始存储字符。

  • len -- 要读取的字符数。

返回值

实际读取字符的数目，-1为流的末尾。

异常

  • IOException -- 如果发生任何I/ O错误。

例子

下面的例子显示java.io.CharArrayReader.read(char[] b, int off, int len) 方法。

package com.yiibai;

import java.io.CharArrayReader;
import java.io.IOException;

public class CharArrayReaderDemo {
   public static void main(String[] args) {

      CharArrayReader car = null;
      char[] ch = {'H', 'E', 'L', 'L', 'O'};
      
      char[] d = new char[5];
      try{
         
         // create new character array reader
         car = new CharArrayReader(ch);
         
         // read character to the destination buffer
         car.read(d, 3, 2);
         
         // for every character in the buffer
         for (char c : d)
         {
            int i = (int)c;
            if(i==0)
            {
               System.out.println("0");
            }
            else
            {
               System.out.println(c);
            }
         }
      }catch(IOException e){
         
         // if I/O error occurs
         System.out.print("Stream is already closed");
      }finally{
         
         // releases any system resources associated with the stream
         if(car!=null)
            car.close();
      }
   }
}

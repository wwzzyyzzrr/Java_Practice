Java.io.File.canRead()方法实例

java.io.File.canRead() 如果文件可以被读记为它的抽象名称方法返回true。

声明

以下是java.io.File.canRead()方法的声明：

public boolean canRead()

参数

  • NA

返回值

此方法返回布尔值-true，如果指定的文件存在的路径名和文件允许应用程序读取。

异常

  • SecurityException -- 如果SecurityManager.checkRead(java.lang.String)方法，通过应用程序拒绝读取权限。

例子

下面的示例演示java.io.File.canRead()方法的用法。

package com.yiibai;

import java.io.File;

public class FileDemo {
   public static void main(String[] args) {
      
      File f = null;
      
      try{
         // create new file
         f = new File("c:/test.txt");
         
         // returns true if the file can be read
         boolean bool = f.canRead();
         
         // print
         System.out.print("File can be read: "+bool);
         
      }catch(Exception e){
         // if any I/O error occurs
         e.printStackTrace();
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

File can be read: true

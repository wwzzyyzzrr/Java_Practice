Java.io.PrintStream.println()方法实例

java.io.PrintStream.println() 方法通过写入行分隔符字符串终止当前行。行分隔符字符串由系统属性line.separator定义，并且不一定是单个换行符(' ').

声明

以下是java.io.PrintStream.println()方法的声明

public void println()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.PrintStream.println()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo {

   public static void main(String[] args) {


      // create printstream object
      PrintStream ps = new PrintStream(System.out);

      // print this string
      ps.print("This is a String");

      // print new line
      ps.println();

      // print a new string
      ps.println("New line");

      // flush the stream
      ps.flush();

   }
}

让我们编译和运行上面的程序，这将产生以下结果：

This is a String
New line

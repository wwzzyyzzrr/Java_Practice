Java.io.ObjectInputStream.readClassDescriptor()方法实例

java.io.ObjectInputStream.readClassDescriptor() 方法读取序列化流的类描述符。这种方法被称为当ObjectInputStream预期一类描述符中的序列化流中的下一个项目。 ObjectInputStream子类可以重写此方法来读取已写入的非标准格式(通过对象输出流的已重写writeClassDescriptor方法，子类)类描述符。默认情况下，此方法根据对象序列化规范中定义的格式读取类描述符。

声明

以下是java.io.ObjectInputStream.readClassDescriptor()方法的声明

protected ObjectStreamClass readClassDescriptor()

参数

  • NA

返回值

此方法返回的类描述读取

异常

  • ClassNotFoundException -- 如果无法找到该类库中的类描述符用于表示一个序列化的对象

  • IOException -- 如果发生I/ O错误。

java.io.FilterWriter.write(int c)方法实例

java.io.FilterWriter.write(int c) 方法写入单个字符的过滤写入器。

声明

以下是java.io.FilterWriter.write(int c) 方法的声明：

public void write(int c)

参数

  • c -- 整数，指定要写入到这个过滤写入器的源字符。

返回值

该方法不返回任何值。

异常

  • IOException -- 如果发生I/ O错误。

例子

下面的示例演示java.io.FilterWriter.write(int c) 方法的用法。

package com.yiibai;

import java.io.FilterWriter;
import java.io.StringWriter;
import java.io.Writer;

public class FilterWriterDemo {
   public static void main(String[] args) throws Exception {
      
      FilterWriter fw = null;
      Writer w = null;
      String s=null;
      
      try{
         // create new reader
         w = new StringWriter(6);
          
         // filter writer
         fw = new FilterWriter(w) {
         };
         
         // write to filter writer
         fw.write(65);
         fw.write(66);
         fw.write(67);

         // get the string
         s = w.toString();
         
         // print
         System.out.print("String: "+s);
         
      }catch(Exception e){
         
         // if any I/O error occurs
         e.printStackTrace();
      }finally{
         
         // releases system resources associated with this stream
         if(w!=null)
            w.close();
         if(fw!=null)
            fw.close();
      }
   }
}

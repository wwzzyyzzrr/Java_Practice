Java.io.OutputStream类

Java.io.OutputStream类是表示输出字节流的所有类的超类。一个输出流接受输出字节并将其发送到一些下沉。需要定义OutputStream的子类的应用程序必须始终提供至少是将输出写入一个字节的方法。

类声明

以下是java.io.OutputStream类的声明：

public abstract class OutputStream
  extends Object
    implements Closeable, Flushable

类方法

S.N. 方法 & 描述
1 void close() 
此方法关闭此输出流并释放与此流有关的所有系统资源。
2 void flush() 
此方法刷新此输出流并强制将所有缓冲的输出字节被写出。
3 void write(byte[] b) 
此方法从指定的字节数组输出流写入b.length个字节。
4 void write(byte[] b, int off, int len) 
此方法从指定的字节数组开始到当前输出流关闭写入len字节。
5 abstract void write(int b) 
此方法将指定字节写入此输出流。

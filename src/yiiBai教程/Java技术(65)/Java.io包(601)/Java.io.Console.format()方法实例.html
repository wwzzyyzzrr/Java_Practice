Java.io.Console.format()方法实例

java.io.Console.format(String fmt, Object... args) 方法输出格式化字符串使用指定的格式字符串参数到该控制台的输出流。

声明

以下是java.io.Console.format(String fmt, Object... args)方法的声明：

public Console format(String fmt, Object... args)

参数

  • fmt -- 格式字符串在格式的语法描述

  • args -- 参数的格式说明符在引用的格式字符串。

返回值

这个方法返回当前控制台(console)。 

异常

  • IllegalFormatException -- 如果格式字符串包含非法语法，格式说明符与给定的格式字符串给定的，参数不足，或其他非法条件不符

例子

下面的示例演示java.io.Console.format(String fmt, Object... args)方法的用法。

package com.yiibai;

import java.io.Console;

public class ConsoleDemo {
   public static void main(String[] args) {
      
      Console cnsl = null;
      
      try{
         cnsl = System.console();

         if (cnsl != null) {
            String fmt = "%1$4s %2$10s %3$10s%n";
            
            // format
            cnsl.format(fmt, "Items", "Quanity", "Price");
            cnsl.format(fmt, "-----", "-----", "-----");
            cnsl.format(fmt, "Tomato", "1Kg", "15");
            cnsl.format(fmt, "Potato", "5Kg", "50");
            cnsl.format(fmt, "Onion", "2Kg", "30");
            cnsl.format(fmt, "Apple", "4Kg", "80");
         }      
      }catch(Exception ex){
         
         // if any error occurs
         ex.printStackTrace();      
      }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Items      Quantity      Price
-----      --------      -----
Tomato     1Kg           15
Potato     5Kg           50
Onion      2Kg           30
Apple      4Kg           80

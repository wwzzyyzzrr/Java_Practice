java.io.BufferedInputStream.write(byte[] b, int off, int len)方法实例

java.io.BufferedInputStream.write(byte[] b, int off, int len) 方法从指定的字节数组b的起始位置，到底层输出流写入len个字节。

声明

以下是 java.io.DataOutputStream.write(byte[] b, int off, int len)方法的声明：

public void write(byte[] b, int off, int len)

参数

  • b -- 源缓冲区。

  • off -- 起始位置。

  • len -- 写入到流的字节数。

返回值

此方法不返回任何值。

异常

  • IOException -- 如果发生I/ O错误。

例子

下面的例子显示java.io.DataOutputStream.write(byte[] b, int off, int len)方法的用法。

package com.yiibai;

import java.io.ByteArrayOutputStream;
import java.io.DataOutputStream;
import java.io.IOException;

public class DataOutputStreamDemo {
   public static void main(String[] args) throws IOException {
         
      ByteArrayOutputStream baos = null;
      DataOutputStream dos = null;
      byte[] buf = {87,64,72,31,90};
      
      try{
         // create byte array output stream
         baos = new ByteArrayOutputStream();
         
         // create data output stream
         dos = new DataOutputStream(baos);
         
         // write to the stream from the source buffer
         dos.write(buf, 2, 3);
         
         // flushes bytes to underlying output stream
         dos.flush();
   
         // for each byte in the baos buffer content
         for(byte b:baos.toByteArray())
         {
            System.out.println(b);
         }
      }catch(Exception e){
         // if any error occurs
         e.printStackTrace();
      }finally{
         
         // releases all system resources from the streams
         if(dos!=null)
            dos.close();
         if(baos!=null)
            baos.close();
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

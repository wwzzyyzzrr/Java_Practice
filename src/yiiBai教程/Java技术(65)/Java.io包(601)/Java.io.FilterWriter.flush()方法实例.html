Java.io.FilterWriter.flush()方法实例

java.io.FilterWriter.flush() 方法刷新出来的字符串。

声明

以下是java.io.FilterWriter.flush()方法的声明：

public void flush()

参数

  • NA

返回值

该方法不返回任何值。

异常

  • IOException -- 如果发生 I/O 错误。

例子

下面的示例演示java.io.FilterWriter.flush()方法的用法。

package com.yiibai;

import java.io.FilterWriter;
import java.io.StringWriter;
import java.io.Writer;

public class FilterWriterDemo {
   public static void main(String[] args) throws Exception {
      
      FilterWriter fw = null;
      Writer w = null;
      String s=null;
      
      try{
         // create new reader
         w = new StringWriter(6);
          
         // filter writer
         fw = new FilterWriter(w) {
         };
         
         // write to filter writer
         fw.write(65);

         // flushes the content
         fw.flush();
         System.out.println("flush() invoked");
         System.out.println("flushes content to the writer");

         // get the string
         s = w.toString();
         
         // print
         System.out.print("String: "+s);
         
      }catch(Exception e){
         
         // if any I/O error occurs
         e.printStackTrace();
      }finally{
         
         // releases system resources associated with this stream
         if(w!=null)
            w.close();
         if(fw!=null)
            fw.close();
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

flush() invoked
flushes content to the writer
String: A

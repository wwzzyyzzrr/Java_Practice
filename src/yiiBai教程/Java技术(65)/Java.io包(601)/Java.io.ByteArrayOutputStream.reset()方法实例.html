Java.io.ByteArrayOutputStream.reset()方法实例

调用 java.io.ByteArrayOutputStream.reset() 方法重置流，使流计数=0。重置该流丢弃所有当前累积输出。

声明

以下是java.io.ByteArrayOutputStream.reset()方法的声明：

public void reset()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.ByteArrayOutputStream.reset()方法的用法。

package com.yiibai;

import java.io.ByteArrayOutputStream;
import java.io.IOException;

public class ByteArrayOutputStreamDemo {
   public static void main(String[] args) throws IOException {
      
      ByteArrayOutputStream baos = null;
      
      try{
         String str = "";
         
         // create new ByteArrayOutputStream
         baos = new ByteArrayOutputStream();
         
         // writing byte to output stream
         baos.write(75);
         
         // output stream to string
         str = baos.toString();
         System.out.println("Before Resetting : "+str);
         
         // reset() method invocation 
         baos.reset();
         
         // writing byte to output stream
         baos.write(65);
         
         // output stream to string()
         str = baos.toString();
         System.out.println("After Resetting : "+ str);
      
      }catch(Exception e){

         // if I/O error occurs
         e.printStackTrace();
      }finally{
         if(baos!=null)
            baos.close();
      }   
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Before Resetting : K
After Resetting : A

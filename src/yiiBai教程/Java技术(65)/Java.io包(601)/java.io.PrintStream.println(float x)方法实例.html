java.io.PrintStream.println(float x)方法实例

java.io.PrintStream.println() 方法打印一个浮点数，然后终止该行。此方法的行为就像先调用print(float)然后调用println()。

声明

以下是java.io.PrintStream.println()方法的声明

public void println(float x)

参数

  • x -- 要打印的浮点数。

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.PrintStream.println()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo {

   public static void main(String[] args) {
      float c = 1.76348f;

      // create printstream object
      PrintStream ps = new PrintStream(System.out);

      // print a float and change line
      ps.println(c);
      ps.print("New Line");

      // flush the stream
      ps.flush();


   }
}

让我们编译和运行上面的程序，这将产生以下结果：

1.76348
New Line

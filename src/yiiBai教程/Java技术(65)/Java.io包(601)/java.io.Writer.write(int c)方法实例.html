java.io.Writer.write(int c)方法实例

java.io.Writer.write(int c) 方法写入单个字符。要写入的字符被包含在给定整数值的低16位; 16个高序位被忽略。

声明

以下是java.io.Writer.write()方法的声明

public void write(int c)

参数

  • c -- int，指定一个写入的字符

返回值

此方法没有返回值

异常

  • IOException -- 如果发生I/ O错误

例子

下面的示例演示java.io.Writer.write()方法的用法。

package com.yiibai;

import java.io.*;

public class WriterDemo {

   public static void main(String[] args) {
      int c = 70;

      // create a new writer
      Writer writer = new PrintWriter(System.out);

      try {
         // write an int that will be printed as ASCII
         writer.write(c);

         // flush the writer
         writer.flush();

         // write another int that will be printed as ASCII
         writer.write(71);

         // flush the stream again
         writer.flush();

      } catch (IOException ex) {
         ex.printStackTrace();
      }
   }
}

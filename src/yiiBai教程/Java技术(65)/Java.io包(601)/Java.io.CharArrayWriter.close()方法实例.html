Java.io.CharArrayWriter.close()方法实例

java.io.CharArrayWriter.close() 方法关闭该流，但不释放该缓冲区。

声明

以下是java.io.CharArrayWriter.close()方法的声明：

public void close()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.CharArrayWriter.close()方法的用法。

package com.yiibai;

import java.io.CharArrayWriter;

public class CharArrayWriterAppendDemo {
   public static void main(String[] args) {
      
      CharArrayWriter chw = null;
      
      try{
         // create character array writer
         chw = new CharArrayWriter();
         
         // declare character sequence
         CharSequence csq = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
         
         // closes the stream
         chw.close();
         
         // append character sequence to the writer
         chw.append(csq);
         
         // prints out the character sequences
         System.out.println(chw.toString());
         
      }catch(Exception e){
         
         // for any error
         e.printStackTrace();
      }finally{
         
         // releases all system resources from writer
         if(chw!=null)
            chw.close();
      }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

ABCDEFGHIJKLMNOPQRSTUVWXYZ

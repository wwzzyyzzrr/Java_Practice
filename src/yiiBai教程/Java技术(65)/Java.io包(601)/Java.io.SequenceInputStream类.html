Java.io.SequenceInputStream类

Java.io.SequenceInputStream 类代表的其他输入流的逻辑连接。它开始时与输入流的有序集合，并从第一个到文件结束读取为止，之后将其从第二个读出，依此类推，直到文件的末尾的最后的包含的输入流。

  extends InputStream

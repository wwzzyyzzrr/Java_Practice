java.io.PrintWriter.print(char c)方法实例

java.io.PrintWriter.print() 方法打印的字符。该字符转换为一个或多个字节根据平台的默认字符编码，这些字节被写入由write(int) 方法进行。

声明

以下是java.io.PrintWriter.print()方法的声明

public void print(char c)

参数

  • c -- 要打印的字符

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.PrintWriter.print()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintWriterDemo {

   public static void main(String[] args) {
      char c = 'a';

      // create a new writer
      PrintWriter pw = new PrintWriter(System.out);

      // print chars
      pw.print(c);

      // change line
      pw.println();

      // print another char
      pw.print('b');

      // flush the writer
      pw.flush();

   }
}

让我们编译和运行上面的程序，这将产生以下结果：

a
b

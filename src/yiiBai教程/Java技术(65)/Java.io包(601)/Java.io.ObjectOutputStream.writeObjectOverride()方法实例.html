Java.io.ObjectOutputStream.writeObjectOverride()方法实例

java.io.ObjectOutputStream.writeObjectOverride(Object obj) 方法由子类重写默认writeObject方法。这种方法被称为由ObjectInputStream受信任子类使用受保护的无参数构造函数构造对象输入。该子类预计将提供一个覆盖方法的修饰符“final”。

声明

以下是java.io.ObjectOutputStream.writeObjectOverride()方法的声明

protected void writeObjectOverride(Object obj)

参数

  • obj -- 对象被写入到基础流

返回值

此方法无返回值。

异常

  • IOException --如果在写入底层流有I / O错误

Java.io.SequenceInputStream.available()方法实例

java.io.SequenceInputStream.available()方法从当前底层输入流中可通过一个方法的下一次调用阻塞当前底层输入流返回可以读取(或跳过)的字节数的估计值。下一次调用可能是同一个线程或其他线程。一个读或跳过的字节数目将不会阻止，但可以读取或跳过的字节。

声明

以下是java.io.SequenceInputStream.available()方法的声明

public int available()

参数

  • NA

返回值

此方法返回可从当前底层输入流读取(或跳过)，而不阻塞或0字节数的估计，如果此输入流已关闭通过调用close()方法

异常

  • IOException -- 如果发生I/O错误。

例子

下面的示例演示java.io.SequenceInputStream.available()方法的用法。

package com.yiibai;

import java.io.*;

public class SequenceInputStreamDemo {

   public static void main(String[] args) {

      // create two  new strings with 5 characters each
      String s1 = "Hello";
      String s2 = "World";

      // create 2 input streams
      byte[] b1 = s1.getBytes();
      byte[] b2 = s2.getBytes();
      ByteArrayInputStream is1 = new ByteArrayInputStream(b1);
      ByteArrayInputStream is2 = new ByteArrayInputStream(b2);

      // create a new Sequence Input Stream
      SequenceInputStream sis = new SequenceInputStream(is1, is2);
      try {
         
         // determine how many bytes are available in the first stream
         System.out.println("" + sis.available());
         
         // read 10 characters, 5 from each stream
         for (int i = 0; i < 10; i++) {
            char c = (char) sis.read();
            System.out.print("" + c);
         }

         // close the stream
         sis.close();

      } catch (IOException ex) {
         ex.printStackTrace();
      }
   }
}

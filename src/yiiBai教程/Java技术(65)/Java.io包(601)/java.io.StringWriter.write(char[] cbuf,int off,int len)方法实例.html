java.io.StringWriter.write(char[] cbuf,int off,int len)方法实例

java.io.StringWriter.write() 方法写入字符数组的一部分。

声明

以下是java.io.StringWriter.write()方法的声明

public void write(char[] cbuf,int off,int len)

参数

  • cbuf -- 字符数组

  • off -- 从开始写入字符偏移量

  • len -- 要写入的字符数

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.StringWriter.write()方法的用法。

package com.yiibai;

import java.io.*;

public class StringWriterDemo {

   public static void main(String[] args) {

      char c = 'b';
      // create a new writer
      StringWriter sw = new StringWriter();

      // write chars
      sw.write('a');
      sw.write(c);

      // print result by converting to string
      System.out.println("" + sw.toString());


   }
}

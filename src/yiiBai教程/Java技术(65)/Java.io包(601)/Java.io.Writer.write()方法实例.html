Java.io.Writer.write()方法实例

java.io.Writer.write(char[] cbuf) 方法写入字符数组。

声明

以下是java.io.Writer.write()方法的声明

public void write(char[] cbuf)

参数

  • cbuf -- 写入的字符数组

返回值

此方法没有返回值

异常

  • IOException -- 如果发生I/ O错误

例子

下面的示例演示java.io.Writer.write()方法的用法。

package com.yiibai;

import java.io.*;

public class WriterDemo {

   public static void main(String[] args) {
      char[] c1 = {'h', 'e', 'l', 'l', 'o'};
      char[] c2 = {'w', 'o', 'r', 'l', 'd'};

      // create a new writer
      Writer writer = new PrintWriter(System.out);

      try {
         // write a char array
         writer.write(c1);

         // flush the writer
         writer.flush();

         // write a new char array
         writer.write(c2);

         // flush the stream again
         writer.close();

      } catch (IOException ex) {
         ex.printStackTrace();
      }
   }
}

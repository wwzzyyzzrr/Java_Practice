java.io.PrintStream.print(String s)方法实例

java.io.PrintStream.print() 方法打印一个字符串。如果参数为null，则字符串“null”的字样。否则，该字符串的字符转换为字节按照平台的默认字符编码，这些字节被写在write(int)方法的方式完全相同。

声明

以下是java.io.PrintStream.print()方法的声明

public void print(String s)

参数

  • s -- 要被打印的字符串

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.PrintStream.print()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo {

   public static void main(String[] args) {
      String s = "Hello World";

      // create printstream object
      PrintStream ps = new PrintStream(System.out);

      // print string
      ps.print(s);
      ps.print(" This is an example");

      // flush the stream
      ps.flush();


   }
}

让我们编译和运行上面的程序，这将产生以下结果：

Hello World This is an example

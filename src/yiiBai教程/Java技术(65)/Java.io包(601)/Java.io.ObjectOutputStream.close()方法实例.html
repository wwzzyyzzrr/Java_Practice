Java.io.ObjectOutputStream.close()方法实例

java.io.ObjectOutputStream.close() 方法关闭该流。此方法必须调用来释放任何与该流关联的资源。

声明

以下是java.io.ObjectOutputStream.close()方法的声明

public void close()

参数

  • NA

返回值

这个方法没有返回值

异常

  • IOException -- 如果发生I/ O错误。

例子

下面的示例演示java.io.ObjectOutputStream.close()方法的用法。

package com.yiibai;

import java.io.*;

public class ObjectOutputStreamDemo {

   public static void main(String[] args) {

      int i = 319874;
      try {

         // create a new file with an ObjectOutputStream
         FileOutputStream out = new FileOutputStream("test.txt");
         ObjectOutputStream oout = new ObjectOutputStream(out);

         // write something in the file
         oout.writeInt(i);
         oout.writeInt(1653984);
         oout.flush();

         // close the stream
         oout.close();

         // create an ObjectInputStream for the file we created before
         ObjectInputStream ois =
                 new ObjectInputStream(new FileInputStream("test.txt"));

         // read and print an int
         System.out.println("" + ois.readInt());

         // read and print an int
         System.out.println("" + ois.readInt());

      } catch (Exception ex) {
         ex.printStackTrace();
      }

   }
}

让我们编译和运行上面的程序，这将产生以下结果：

1653984

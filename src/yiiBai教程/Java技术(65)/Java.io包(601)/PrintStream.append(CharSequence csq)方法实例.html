PrintStream.append(CharSequence csq)方法实例

java.io.PrintStream.append() 方法将指定的字符序列来此输出流。

声明

以下是java.io.PrintStream.append()方法的声明

public PrintStream append(CharSequence csq)

参数

  • csq -- 字符序列追加。如果csq为null，则这四个字符“null”被追加到当前输出流。

返回值

这个方法返回当前输出流

异常

  • NA

例子

下面的示例演示java.io.PrintStream.append()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo {

   public static void main(String[] args) {
      String s = "Hello World. ";

      // create printstream object
      PrintStream ps = new PrintStream(System.out);

      // append our strings
      ps.append(s);
      ps.append("This is an example.");

      // print the result
      ps.flush();
      ps.close();

   }
}

让我们编译和运行上面的程序，这将产生以下结果：

Hello World. This is an example.

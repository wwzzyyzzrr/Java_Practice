Java.io.PrintStream.append()方法实例

java.io.PrintStream.append() 方法将指定字符追加到当前输出流。

声明

以下是java.io.PrintStream.append()方法的声明

public PrintStream append(char c)

参数

  • c -- 16位字符以附加

返回值

这个方法返回当前输出流

异常

  • NA

例子

下面的示例演示java.io.PrintStream.append()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo {

   public static void main(String[] args) {
      char c = 'A';

      // create printstream object
      PrintStream ps = new PrintStream(System.out);

      // append our characters
      ps.append(c);
      ps.append('y');
      ps.append('m');

      // print the result
      ps.flush();
      ps.close();

   }
}

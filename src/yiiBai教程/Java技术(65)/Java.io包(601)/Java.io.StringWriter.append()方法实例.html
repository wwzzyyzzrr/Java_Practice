Java.io.StringWriter.append()方法实例

java.io.StringWriter.append() 方法将指定字符追加到这个writer。

声明

以下是java.io.StringWriter.append()方法的声明

public StringWriter append(char c)

参数

  • c -- 16位字符以附加

返回值

此方法返回这个writer。

异常

  • NA

例子

下面的示例演示java.io.StringWriter.append()方法的用法。

package com.yiibai;

import java.io.*;

public class StringWriterDemo {

   public static void main(String[] args) {

      // create a new writer
      StringWriter sw = new StringWriter();

      // append a char       
      System.out.println("" + sw.append("a"));
      System.out.println("" + sw.append("b"));

   }
}

Let us compile and run the above program, this will produce the following result

a
ab

Java.io.ObjectStreamClass.getSerialVersionUID()方法实例

java.io.ObjectStreamClass.getSerialVersionUID()方法返回serialVersionUID类。serialVersionUID定义了一组类中的所有与已经从一个共同的根类进化相同的名称，并同意被序列化并使用通用格式反序列化。序列化的类有serialVersionUID的0L。

声明

以下是java.io.ObjectStreamClass.getSerialVersionUID()方法的声明

public long getSerialVersionUID()

参数

  • NA

返回值

此方法返回通过这个描述符所描述的类的SUID

异常

  • NA

例子

下面的示例演示java.io.ObjectStreamClass.getSerialVersionUID()方法的用法。

package com.yiibai;

import java.io.*;
import java.util.Calendar;

public class ObjectStreamClassDemo {

   public static void main(String[] args) {

      // create a new object stream class for Integers
      ObjectStreamClass osc = ObjectStreamClass.lookup(Integer.class);

      // get the serial for Integers
      System.out.println("" + osc.getSerialVersionUID());

      // create a new object stream class for Calendar
      ObjectStreamClass osc2 = ObjectStreamClass.lookup(Calendar.class);

      // get the serial for Calendar
      System.out.println("" + osc2.getSerialVersionUID());
   }
}

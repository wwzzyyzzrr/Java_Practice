Java.io.File.isHidden()方法实例

java.io.File.isHidden() 检查命名此抽象路径名的文件是否是一个隐藏文件。

声明

以下是java.io.File.isHidden()方法的声明：

public boolean isHidden()

参数

  • NA

返回值

当且仅当此抽象路径名的文件是一个隐藏的文件该方法返回true，否则该方法返回false。

异常

  • SecurityException -- 如果安全管理器存在并且其SecurityManager.checkRead(java.lang.String) 方法拒绝对文件的读访问

例子

下面的示例演示java.io.File.isHidden()方法的用法。

package com.yiibai;

import java.io.File;

public class FileFreeSpace {
   public static void main(String[] args) {
      
      File f = null;
      String path;
      boolean bool = false;
      
      try{      
         // create new file
         f = new File("c:/test.txt");
         
         // true if the file path is a hidden file
         bool = f.isHidden();
         
         // get the path
         path = f.getPath();
         
         // prints
         System.out.print(path+" is file hidden? "+bool);
         
      }catch(Exception e){
         // if any error occurs
         e.printStackTrace();
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

c:	est.txt is file hidden? false

Java.io.ObjectOutputStream.writeClassDescriptor()方法实例

java.io.ObjectOutputStream.writeClassDescriptor(ObjectStreamClass desc) 方法指定的类描述符的对象输出流。类描述符被用来标识写入流对象的类。 ObjectOutputStream的子类可以重写此方法以自定义在类描述符被写入序列化流的方式。在ObjectInputStream，readClassDescriptor相应的方法应该被重写以从其自定义流表示重组类描述符。默认情况下，此方法根据对象序列化规范中定义的格式写入类描述符。

声明

以下是java.io.ObjectOutputStream.writeClassDescriptor()方法的声明

protected void writeClassDescriptor(ObjectStreamClass desc)

参数

  • desc -- 类描述符写入到流

返回值

此方法没有返回值。

异常

  • IOException -- 如果在写入底层流发生I / O错误

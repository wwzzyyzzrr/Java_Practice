Java.io.PrintStream.setError()方法实例

java.io.PrintStream.setError() 方法设置流的错误状态为true。

声明

以下是java.io.PrintStream.println()方法的声明

protected void setError()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.PrintStream.setError()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo extends PrintStream {

   public PrintStreamDemo(OutputStream out) {
      super(out);
   }

   public static void main(String[] args) {
      byte c[] = {70, 71, 72, 73, 74, 75, 76};

      // create printstream object
      PrintStreamDemo ps = new PrintStreamDemo(System.out);

      // write bytes 1-3
      ps.write(c, 1, 3);

      // flush the stream
      ps.flush();

      // set an internal error
      ps.setError();
   }
}

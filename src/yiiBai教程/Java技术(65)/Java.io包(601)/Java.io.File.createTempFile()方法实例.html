Java.io.File.createTempFile()方法实例

java.io.File.createTempFile(String prefix, String suffix) 方法自动创建在默认的临时文件夹中的空文件.

声明

以下是java.io.File.createTempFile(String prefix, String suffix)方法的声明：

public static File createTempFile(String prefix, String suffix)

参数

  • prefix -- 前缀字符串定义的文件名;必须至少有三个字符长

  • suffix -- 后缀字符串定义文件的扩展名;如果为null后缀".tmp" 将被使用

返回值

一个抽象路径名新创建的空文件。

异常

  • IllegalArgumentException -- 如果前缀参数包含少于三个字符

  • IOException -- 如果文件创建失败

  • SecurityException -- 如果SecurityManager.checkWrite(java.lang.String)方法不允许创建一个文件

例子

下面的示例演示java.io.File.createTempFile(String prefix, String suffix) 方法的用法。

package com.yiibai;

import java.io.File;

public class FileDemo {
   public static void main(String[] args) {
      
      File f = null;
            
      try{
         // creates temporary file
         f = File.createTempFile("tmp", ".txt");
         
         // prints absolute path
         System.out.println("File path: "+f.getAbsolutePath());
         
         // creates temporary file
         f = File.createTempFile("tmp", null);
         
         // prints absolute path
         System.out.print("File path: "+f.getAbsolutePath());
         
      }catch(Exception e){
         // if any error occurs
         e.printStackTrace();
      }
   }
}

File path: C:UsersTPAppDataLocalTemp	mp1783337266599428081.tmp

Java.io.OutputStreamWriter.getEncoding()方法实例

java.io.OutputStreamWriter.getEncoding() 方法返回正在使用此流的字符编码的名称。如果编码有一个历史的名称，那么返回其名;否则编码的规范名称被返回。如果此实例用OutputStreamWriter(OutputStream, String) 构造方法创建则返回的名称，是唯一的编码，可能与传递给构造函数的名称。如果流已关闭，此方法可能返回null。

声明

以下是java.io.OutputStreamWriter.getEncoding()方法声明

public String getEncoding()

参数

  • NA

返回值

此方法返回当前编码的历史名称，或如果流已关闭，可能为null。

异常

  • IOException -- 如果发生I/ O错误。

例子

下面的示例演示java.io.OutputStreamWriter.getEncoding()方法的用法。

package com.yiibai;

import java.io.*;

public class OutputStreamWriterDemo {

   public static void main(String[] args) {

      try {

         // create a new OutputStreamWriter
         OutputStream os = new FileOutputStream("test.txt");
         OutputStreamWriter writer = new OutputStreamWriter(os);

         // create a new FileInputStream to read what we write
         FileInputStream in = new FileInputStream("test.txt");

         // write something in the file
         writer.write(70);

         // flush the stream
         writer.flush();

         // get and print the encoding for this stream
         System.out.println("" + writer.getEncoding());

         // read what we write
         System.out.println("" + (char) in.read());


      } catch (Exception ex) {
         ex.printStackTrace();
      }
   }
}

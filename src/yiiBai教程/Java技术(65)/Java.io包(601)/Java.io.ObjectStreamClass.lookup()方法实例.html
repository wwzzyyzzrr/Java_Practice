Java.io.ObjectStreamClass.lookup()方法实例

java.io.ObjectStreamClass.lookup(Class<?> cl) 方法找到该描述符为可序列化的类。创建一个ObjectStreamClass实例，如果一个尚不存在的类。如果指定的类没有实现java.io.Serializable或java.io.Externalizable，则返回null。

声明

以下是java.io.ObjectStreamClass.lookup()方法的声明

public static ObjectStreamClass lookup(Class<?> cl)

参数

  • cl -- 获得该描述符的类

返回值

此方法返回类描述符指定的类

异常

  • NA

例子

下面的示例演示java.io.ObjectStreamClass.lookup()方法的用法。

package com.yiibai;

import java.io.*;
import java.util.Calendar;

public class ObjectStreamClassDemo {

   public static void main(String[] args) {

      // create a new object stream class for Integers
      ObjectStreamClass osc = ObjectStreamClass.lookup(Integer.class);

      // get the name for Integers
      System.out.println("" + osc.getName());

      // create a new object stream class for Calendar
      ObjectStreamClass osc2 = ObjectStreamClass.lookup(Calendar.class);

      // get the name for Calendar
      System.out.println("" + osc2.getName());
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

java.lang.Integer
java.util.Calendar

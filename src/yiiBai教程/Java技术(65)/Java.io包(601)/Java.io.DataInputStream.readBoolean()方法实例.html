Java.io.DataInputStream.readBoolean()方法实例

java.io.DataInputStream.readBoolean() 如果该字节不为零返回true，如果该字节是零方法返回false。

声明

以下是java.io.DataInputStream.readBoolean()方法的声明：

public final boolean readBoolean()

参数

  • NA

返回值

读取布尔值

异常

  • IOException -- 如果流已关闭。

  • EOFException -- 如果输入流已经到达末尾。

例子

下面的示例演示java.io.DataInputStream.readBoolean()方法的用法。

package com.yiibai;

import java.io.ByteArrayInputStream;
import java.io.DataInputStream;
import java.io.IOException;
import java.io.InputStream;

public class DataInputStreamDemo {
   public static void main(String[] args) throws IOException {
      
      InputStream is = null;
      DataInputStream dis = null;
      byte[] buf = {65, 0, 0, 68, 69};
      
      try{
         
         // create new byte array input stream
         is = new ByteArrayInputStream(buf);
         
         // create data input stream
         dis = new DataInputStream(is);
         
         // readBoolean till the data available to read
         while( dis.available() >0)
         {
            System.out.println(dis.readBoolean());
         }
      }catch(Exception e){
         // if any I/O error occurs
         e.printStackTrace();
      }finally{
         
         // releases any associated system files with this stream
         if(is!=null)
            is.close();
         if(dis!=null)
            dis.close();
      }   
   }
}

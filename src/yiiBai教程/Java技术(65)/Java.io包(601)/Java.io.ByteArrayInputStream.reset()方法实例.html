Java.io.ByteArrayInputStream.reset()方法实例

java.io.ByteArrayInputStream.reset() 方法重设缓冲到最后标记的位置。这需要标志位置为0，除非显式指定了标记。

声明

以下是java.io.ByteArrayInputStream.reset()方法的声明：

public void reset()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.ByteArrayInputStream.reset()方法的用法。

package com.yiibai;

import java.io.ByteArrayInputStream;
import java.io.IOException;

public class ByteArrayInputStreamDemo {
   public static void main(String[] args) throws IOException {
      
      byte[] buf = {65, 66, 67, 68, 69};
      ByteArrayInputStream bais = null;
      
      try{
         
         // create new byte array input stream
         bais = new ByteArrayInputStream(buf);
         
         // print bytes
         System.out.println("Byte read "+ bais.read());
         System.out.println("Byte read "+ bais.read());
         System.out.println("Byte read "+ bais.read());
         System.out.println("Byte read "+ bais.read());
         System.out.println("Byte read "+ bais.read());
         
         System.out.println("Reset() invocation");
         
         // reset() invocation
         bais.reset();
         System.out.println("Byte read "+ bais.read());
         System.out.println("Byte read "+ bais.read());
      
      }catch(Exception e){

         // if I/O error occurs
         e.printStackTrace();
      }finally{
         if(bais!=null)
            bais.close();
      }   
   }
}

Byte read 67
Byte read 68

Byte read 66

java.io.PipedOutputStream.write(byte[] b, int off, int len)方法实例

java.io.PipedOutputStream.write(byte[] b, int off, int len) 方法从指定的字节数组开始到这个管道输出流偏移量off写入len字节。此方法将阻塞，直到所有的字节写入到输出流。

声明

以下是java.io.PipedOutputStream.write()方法的声明

public void write(byte[] b, int off, int len)

参数

  • b -- 数据

  • off -- 在数据偏移的开始。

  • len -- 要写入的字节数。

返回值

此方法无返回值。

异常

  • IOException -- 如果发生I/ O错误。

例子

下面的示例演示java.io.PipedOutputStream.write()方法的用法。

package com.yiibai;

import java.io.*;

public class PipedOutputStreamDemo extends PipedInputStream {

   public static void main(String[] args) {

      byte[] b = {'h', 'e', 'l', 'l', 'o'};

      // create a new Piped input and Output Stream
      PipedOutputStream out = new PipedOutputStream();
      PipedInputStreamDemo in = new PipedInputStreamDemo();

      try {
         // connect input and output
         out.connect(in);

         // write something 
         out.write(b, 0, 3);

         // flush the stream
         out.flush();

         // print what we wrote
         for (int i = 0; i < 3; i++) {
            System.out.print("" + (char) in.read());
         }


      } catch (IOException ex) {
         ex.printStackTrace();
      }


   }
}

Java.io.Console.readPassword()方法实例

java.io.Console.readPassword() 方法从控制台读取密码，禁用回显。

声明

以下是java.io.Console.readPassword()方法的声明：

public char[] readPassword()

参数

  • NA

返回值

此方法返回一个字符数组，包含从控制台读取密码，不包括行终止符，或者如果流的末尾已到达返回null。

异常

  • IOError -- 如果发生I/O错误。

例子

下面的示例演示java.io.Console.readPassword()方法的用法。

package com.yiibai;

import java.io.Console;

public class ConsoleDemo {
   public static void main(String[] args) {
      
      Console cnsl = null;
      String alpha = null;
      
      try{
         // creates a console object
         cnsl = System.console();

         // if console is not null
         if (cnsl != null) {
            
            // read line from the user input
            alpha = cnsl.readLine("Name: ");
            
            // prints
            System.out.println("Name is: " + alpha);
            
            // read password into the char array
            char[] pwd = cnsl.readPassword("Password: ");
            
            // prints
            System.out.println("Password is: "+pwd);
         }      
      }catch(Exception ex){
         
         // if any error occurs
         ex.printStackTrace();      
      }
   }
}

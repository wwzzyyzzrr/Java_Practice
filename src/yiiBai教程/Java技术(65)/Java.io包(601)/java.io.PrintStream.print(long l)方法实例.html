java.io.PrintStream.print(long l)方法实例

java.io.PrintStream.print() 方法打印一个长整数。通过String.valueOf(long)生成的字符串转换成字节按照平台的默认字符编码，这些字节被写在write(int)方法完全相同的方式。

声明

以下是java.io.PrintStream.print()方法的声明

public void print(long l)

参数

  • l -- 要打印的long

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.PrintStream.print()方法的用法。

package com.yiibai;

import java.io.*;

public class PrintStreamDemo {

   public static void main(String[] args) {
      long x = 50000000000l;
      try {
         // create printstream object
         PrintStream ps = new PrintStream(System.out);

         // print long
         ps.print(x);

         // flush the stream
         ps.flush();

      } catch (Exception ex) {
         ex.printStackTrace();
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

50000000000

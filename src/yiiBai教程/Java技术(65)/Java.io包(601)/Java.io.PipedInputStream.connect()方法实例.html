Java.io.PipedInputStream.connect()方法实例

java.io.PipedInputStream.connect(PipedOutputStream src) 方法导致此管道输入流连接到管道输出流src中。如果此对象已经连接到其他某个管道输出流，则抛出IOException。

声明

以下是java.io.PipedInputStream.connect()方法声明

public void connect(PipedOutputStream src)

参数

  • src -- 该管道连接的输出流。

返回值

此方法无返回值

异常

  • IOException -- 如果发生I/ O错误。

例子

下面的示例演示java.io.PipedInputStream.connect()方法的用法。

package com.yiibai;

import java.io.*;

public class PipedInputStreamDemo {

   public static void main(String[] args) {

      // create a new Piped input and Output Stream
      PipedOutputStream out = new PipedOutputStream();
      PipedInputStream in = new PipedInputStream();

      try {
         // connect input and output
         in.connect(out);

         // write something 
         out.write(70);
         out.write(71);

         // read what we wrote
         for (int i = 0; i < 2; i++) {
            System.out.println("" + (char) in.read());
         }

      } catch (IOException ex) {
         ex.printStackTrace();
      }


   }
}

Java.io.Console.flush()方法实例

java.io.Console.flush() 方法刷新控制台，并强制立即写入任何输出。

声明

以下是java.io.Console.flush()方法的声明：

public void flush()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.io.Console.flush()方法的用法。

package com.yiibai;

import java.io.Console;

public class ConsoleDemo {
   public static void main(String[] args) {
      
      Console cnsl = null;
      
      try{
         //Create a console object.
           cnsl = System.console();
           
           // test for console not null
           if (cnsl != null) {
              
              // read line from the console
               String name = cnsl.readLine("Enter  name  : ");
               
               // print
               System.out.println("You have entered : " + name);
           }
           
           // flushes console and forces output to be written
           cnsl.flush();   
      }catch(Exception ex){
         
         // if any error occurs
         ex.printStackTrace();      
      }
   }
}

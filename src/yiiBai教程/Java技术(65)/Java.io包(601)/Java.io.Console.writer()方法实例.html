Java.io.Console.writer()方法实例

java.io.Console.writer() 方法检索与此控制台关联的唯一PrintWriter对象。

声明

以下是java.io.Console.writer()方法的声明：

public PrintWriter writer()

参数

  • NA

返回值

此方法返回与此控制台关联的PrintWriter。

异常

  • NA

例子

下面的示例演示java.io.Console.writer()方法的用法。

package com.yiibai;

import java.io.Console;
import java.io.PrintWriter;

public class ConsoleDemo {
   public static void main(String[] args) {
      
      Console cnsl = null;
      PrintWriter out = null;
      
      try{
         // creates a console object
         cnsl = System.console();

         // if console is not null
         if (cnsl != null) {
            
            // creates new print writer
            out = cnsl.writer();
            
            // prints
            out.println("Here is The Optimus Prime!!");
         }      
      }catch(Exception ex){
         
         // if any error occurs
         ex.printStackTrace();      
      }
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

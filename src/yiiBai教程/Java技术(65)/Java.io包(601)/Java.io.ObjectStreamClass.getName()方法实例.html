Java.io.ObjectStreamClass.getName()方法实例

java.io.ObjectStreamClass.getName() 方法返回由这个描述符描述的类的名称。此方法中所使用Class.getName()方法的格式返回类的名称。

声明

以下是java.io.ObjectStreamClass.getName()方法的声明

public String getName()

参数

  • NA

返回值

此方法返回表示类的名称的字符串

异常

  • NA

例子

下面的示例演示java.io.ObjectStreamClass.getName()方法的用法。

package com.yiibai;

import java.io.*;
import java.util.Calendar;

public class ObjectStreamClassDemo {

   public static void main(String[] args) {

      // create a new object stream class for Integers
      ObjectStreamClass osc = ObjectStreamClass.lookup(Integer.class);

      // get the name of the class and print it
      System.out.println("" + osc.getName());

      // create a new object stream class for Calendar
      ObjectStreamClass osc2 = ObjectStreamClass.lookup(Calendar.class);

      // get the name of the class and print it
      System.out.println("" + osc2.getName());
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

java.lang.Integer
java.util.Calendar

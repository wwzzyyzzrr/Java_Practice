Java国际化(i18n)SimpleDateFormat类

java.text.SimpleDateFormat类根据给定的模式格式化日期。 它也用于解析日期从字符串，其中字符串包含提到的格式的日期。 请参阅以下使用SimpleDateFormat类的示例。

文件:IOTester.java - 

import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;

public class I18NTester {
   public static void main(String[] args) throws ParseException {

      String pattern = "dd-MM-yyyy";

      SimpleDateFormat simpleDateFormat = new SimpleDateFormat(pattern);

      Date date = new Date();

      System.out.println(date);
      System.out.println(simpleDateFormat.format(date));

      String dateText = "29-11-2017";

      date = simpleDateFormat.parse(dateText);

      System.out.println(simpleDateFormat.format(date));
   }
}

执行上面示例代码，得到以下结果 - 

Wed Nov 29 17:01:22 IST 2017
29-11-2017
29-11-2017

Java泛型环境设置

由于此Java泛型教程是基于Java语言来讲解的，所以在学习此教程之前，必须要安装好Java开发环境，也就是安装好JDK。要查看你的电脑上有没有安装好 Java ， 可直接打开命令提示符，然后输入：

C:\Users\Administrator> java -version

执行上面命令后，输出结果如下 -

如果您执行命令 java -version 后没有看到类似的输出，那么需要安装 JDK ，有关 JDK 的安装，请参考以下文章： http://www.yiibai.com/java/java_environment_setup.html

提示：建议安装 jdk 1.8 及以上版本，以获得更好的功能支持，也能更好的与本教程中的示例同步。

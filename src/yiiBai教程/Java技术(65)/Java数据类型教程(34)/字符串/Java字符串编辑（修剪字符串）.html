替换字符串的一部分

replace()方法将旧字符和新字符作为参数。它通过将新字符替换所有的旧字符并返回一个替换后新的String对象。 例如，

public class Main {
  public static void main(String[] args) {
    String oldStr = new String("tooth");
    String newStr  = oldStr.replace('o', 'e');
    System.out.println(newStr);
  }
}

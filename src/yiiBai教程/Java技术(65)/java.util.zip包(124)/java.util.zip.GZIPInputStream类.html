java.util.zip.GZIPInputStream类

java.util.zip.GZIPInputStream类实现了用于读取GZIP文件格式的压缩数据的流过滤器。

字段

以下是java.util.zip.GZIPInputStream类的字段 -

  • protected boolean eos - 表示输入流的结束。

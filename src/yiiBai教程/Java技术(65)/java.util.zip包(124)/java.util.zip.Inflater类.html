java.util.zip.Inflater类

java.util.zip.Inflater类为使用流行的ZLIB压缩库的通用解压缩提供支持。 ZLIB压缩库最初是作为PNG图形标准的一部分开发的，不受专利保护。 它在java.util.zip包描述的规范中有详细描述。

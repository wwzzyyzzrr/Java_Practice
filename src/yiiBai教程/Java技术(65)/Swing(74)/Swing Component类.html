Swing Component类

非菜单用户界面控件 AWT Component 是抽象基类。组件表示图形表示的对象。

类声明

以下是声明的java.awt.Component类：

public abstract class Component
   extends Object
      implements ImageObserver, MenuContainer, Serializable

字段域

以下是 java.awt.Component 类的字段：

  • static float BOTTOM_ALIGNMENT -- 易于使用常量getAlignmentY。

  • static float CENTER_ALIGNMENT -- 易于使用的常量为getAlignmentY 和 getAlignmentX。

  • static float LEFT_ALIGNMENT -- 易于使用常量getAlignmentX。

  • static float RIGHT_ALIGNMENT -- 易于使用常量getAlignmentX。

  • static float TOP_ALIGNMENT -- 易于使用常量为getAlignmentY()。

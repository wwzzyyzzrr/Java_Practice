SWING InputEvent事件类

InputEvent类是所有组件级别输入事件的根事件类。输入事件被传递到处理前的监听器通常由他们起源的源。这让监听器和组件的子类“消费”事件源不会使他们以默认方式处理。例如，一个按钮组件，将防止消耗mousePressed事件按钮被激活。

类声明

以下是声明 java.awt.event.InputEvent类：

public abstract class InputEvent
   extends ComponentEvent

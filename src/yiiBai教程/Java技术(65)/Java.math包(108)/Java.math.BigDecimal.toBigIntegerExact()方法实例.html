Java.math.BigDecimal.toBigIntegerExact()方法实例

java.math.BigDecimal.toBigIntegerExact() 此BigDecimal转换为BigInteger，检查丢失的信息。抛出一个异常，如果此BigDecimal具有非零小数部分。

声明

以下是java.math.BigDecimal.toBigIntegerExact()方法的声明

public BigInteger toBigIntegerExact()

参数

  • NA

返回值

此方法返回BigDecimal对象转换为BigInteger的值

异常

  • ArithmeticException - 如果具有非零小数部分

例子

下面的例子显示math.BigDecimal.toBigIntegerExact()方法的用法

package com.yiibai;

import java.math.*;

public class BigDecimalDemo {

     public static void main(String[] args) {

        // create a BigDecimal object
        BigDecimal bg1;

        // create a BigInteger object
        BigInteger i1;

        bg1 = new BigDecimal("2426");

        // assign the BigIntegerExact value of bg1 to i1
        i1 = bg1.toBigIntegerExact();

	String str = "BigInteger value of " + bg1 + " is " + i1;

        // print i1 value
        System.out.println( str );
    }
}

让我们编译和运行上面的程序，这将产生以下结果：

BigInteger value of 2426 is 2426

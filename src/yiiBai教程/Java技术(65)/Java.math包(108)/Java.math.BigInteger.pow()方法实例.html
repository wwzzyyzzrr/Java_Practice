Java.math.BigInteger.pow()方法实例

 java.math.BigInteger.pow(int exponent) 返回一个BigInteger，其值是 (thisexponent)，该指数是一个整数，而不是一个BigInteger。

声明

以下是java.math.BigInteger.pow()方法的声明

public BigInteger pow(int exponent)

参数

  • exponent - 指数以这此BigInteger要引发或提高

返回值

该方法返回一个BigInteger对象的值是 thisexponent.

异常

  • ArithmeticException - 指数为负。这会导致操作以产生一个非整数值。

例子

下面的例子显示math.BigInteger.pow()方法的用法

package com.yiibai;

import java.math.*;

public class BigIntegerDemo {

public static void main(String[] args) {

	// create 2 BigInteger objects
	BigInteger bi1, bi2;

	// create and assign value to exponent
	int exponent = 2;

	// assign value to bi1
	bi1 = new BigInteger("6");

	// perform pow operation on bi1 using exponent
	bi2 = bi1.pow(exponent);

	String str = "Result is " + bi1 + "^" +exponent+ " = " +bi2;

	// print bi2 value
	System.out.println( str );
   }
}

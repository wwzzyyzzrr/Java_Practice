Java.math.BigDecimal.round()方法实例

java.math.BigDecimal.round(MathContext mc)根据MathContext设置舍入返回一个BigDecimal。如果精度设置为0，则四舍五入不发生。这种方法的效果与plus(MathContext) 方法是相同。

声明

以下是java.math.BigDecimal.round()方法的声明

public BigDecimal round(MathContext mc)

参数

  • mc - 使用上下文

返回值

此方法根据MathContext设置舍入返回一个BigDecimal。

异常

  • ArithmeticException - 如果舍入模式是UNNECESSARY，BigDecimal操作需要四舍五入。

例子

下面的例子显示math.BigDecimal.round()方法的用法

package com.yiibai;

import java.math.*;

public class BigDecimalDemo {

    public static void main(String[] args) {

        // create 2 BigDecimal Objects
        BigDecimal bg1, bg2;

	bg1 = new BigDecimal("5.46497");

        MathContext mc = new MathContext(3); // 3 precision

        // bg1 is rounded using mc
        bg2 = bg1.round(mc);

	String str = "The value " + bg1 + " after rounding is " + bg2;

        // print bg2 value
        System.out.println( str );
    }
}

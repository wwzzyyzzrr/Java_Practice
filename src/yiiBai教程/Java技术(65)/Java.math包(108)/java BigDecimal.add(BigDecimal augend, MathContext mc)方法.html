java BigDecimal.add(BigDecimal augend, MathContext mc)方法

java.math.BigDecimal.add(BigDecimal augend, MathContext mc) 返回一个BigDecimal，其值为 (this + augend)，根据MathContext设置进行舍入。如果任一数为零，并且精度设置为非零的话，其他数，如果需要舍入，作为结果。

声明

以下是java.math.BigDecimal.add()方法的声明

public BigDecimal add(BigDecimal augend, MathContext mc)

参数

  • augend - 值被添加到当前的BigDecimal

  • mc - 使用上下文

返回值

此方法返回一个BigDecimal对象的值是this + augend，舍入是必要的。

Exception

  • ArithmeticException - 如果结果不准确，但是舍入模式为 UNNECESSARY.

例子

下面的示例演示math.BigDecimal.add()方法的用法。

package com.yiibai;

import java.math.*;

public class BigDecimalDemo {

   public static void main(String[] args) {

      // create 3 BigDecimal objects
      BigDecimal bg1,bg2,bg3,bg4;

      // assign value to bg1 and bg2
      bg1 = new BigDecimal("40.732");
      bg2 = new BigDecimal("30.12");

      // print bg1 and bg2 value
      System.out.println("Object Value is " + bg1);
      System.out.println("Augend value is " + bg2);

      // create MathContext object with 4 precision
      MathContext mc = new MathContext(4);

      // perform add operation on bg1 with augend bg2 and context mc
      bg3=bg1.add(bg2,mc);

      // print bg3 value with
      System.out.println("Result is " + bg3);
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

Object Value is 40.732
Augend value is 30.12
Result is 70.85

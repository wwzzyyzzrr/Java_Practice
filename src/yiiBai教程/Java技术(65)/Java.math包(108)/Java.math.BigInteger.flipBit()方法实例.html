Java.math.BigInteger.flipBit()方法实例

java.math.BigInteger.flipBit(int n) 返回一个BigInteger，其值相当于此BigInteger指定位翻转。它计算 (this ^ (1<<n))。

声明

以下是java.math.BigInteger.flipBit()方法的声明

public BigInteger flipBit(int n)

参数

  • n - 位的指数翻转

返回值

该方法返回一个BigInteger对象的值是, this ^ (1<<n)

异常

  • ArithmeticException - n是负数

例子

下面的例子显示math.BigInteger.flipBit()方法的用法

package com.yiibai;

import java.math.*;

public class BigIntegerDemo {

public static void main(String[] args) {

        // create 2 BigInteger objects
	BigInteger bi1, bi2;

	// assign value to bi1
	bi1 = new BigInteger("8");//1000

	// perform flipbit operation on bi1 with index 1
	bi2 = bi1.flipBit(1);

	String str = "Flipbit operation on " +bi1+ " at index 1 gives " +bi2;

	// print bi2 value
	System.out.println( str );
    }
}

让我们编译和运行上面的程序，这将产生以下结果：

Flipbit operation on 8 at index 1 gives 10

Java.math.BigInteger.floatValue()方法实例

 java.math.BigInteger.floatValue() 此BigInteger转换为浮点数float。这种转换是类似的double的限制原语转换。

如果此BigInteger有太大的幅度表示为float值，它会被转换为Float.NEGATIVE_INFINITY或Float.POSITIVE_INFINITY(如适用)。即使当返回值是有限的，此转换会丢失关于BigInteger值的精度信息。

声明

以下是java.math.BigInteger.floatValue()方法的声明

public float floatValue()

Specified by

在类 Number 中的floatValue

参数

  • NA

返回值

此方法返回此BigInteger转换为float。

异常

  • NA

例子

下面的例子显示math.BigInteger.floatValue()方法的用法

package com.yiibai;

import java.math.*;

public class BigIntegerDemo {

public static void main(String[] args) {

        // create 2 BigInteger objects
	BigInteger bi1, bi2;

	// create 2 Float objects
	Float f1, f2;

	// assign values to bi1, bi2
	bi1 = new BigInteger("123");
	bi2 = new BigInteger("-123");

	// assign float values of bi1, bi2 to f1, f2
	f1 = bi1.floatValue();
	f2 = bi2.floatValue();

	String str1 = "Float value of " + bi1 + " is " +f1;
	String str2 = "Float value of " + bi2 + " is " +f2;

	// print f1, f2 values
	System.out.println( str1 );
	System.out.println( str2 );
    }
}

让我们编译和运行上面的程序，这将产生以下结果：

Float value of 123 is 123.0
Float value of -123 is -123.0

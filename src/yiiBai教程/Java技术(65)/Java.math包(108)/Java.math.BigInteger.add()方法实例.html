Java.math.BigInteger.add()方法实例

java.math.BigInteger.add(BigInteger val) 返回一个BigInteger对象的值是 (this + val).

声明

以下是java.math.BigInteger.add()方法的声明

public BigInteger add(BigInteger val)

参数

  • val - 此值被相加到当前的BigInteger

返回值

此方法返回BigIntger对象的值, this + val

异常

  • NA

例子

下面的例子显示math.BigInteger.add()方法的用法

package com.yiibai;

import java.math.*;

public class BigIntegerDemo {

public static void main(String[] args) {

       // create 3 BigInteger objects
	BigInteger bi1, bi2, bi3;  

	// assign values to bi1, bi2
	bi1 = new BigInteger("123");
	bi2 = new BigInteger("50");

	// perform add operation on bi1 using bi2
	bi3 = bi1.add(bi2);
	  
	String str = "Result of addition is " +bi3;;

	// print bi3 value
	System.out.println( str );
    }
}

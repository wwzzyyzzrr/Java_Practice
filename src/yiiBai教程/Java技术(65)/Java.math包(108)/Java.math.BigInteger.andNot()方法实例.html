Java.math.BigInteger.andNot()方法实例

java.math.BigInteger.andNot(BigInteger val) 返回一个BigInteger，其值是(this & ~val). 该方法中，这相当于and(val.not()), 是作为一个方便的屏蔽操作。该方法返回一个负BigInteger，当且仅当这是消极和val为正值。

声明

以下是java.math.BigInteger.andNot()方法的声明

public BigInteger andNot(BigInteger val)

参数

  • val - 值加以补充，并与此BigInteger作AND运算

返回值

此方法返回BigIntger对象的值, this & ~val

异常

  • NA

例子

下面的例子显示math.BigInteger.andNot()方法的用法

package com.yiibai;

import java.math.*;

public class BigIntegerDemo {

   public static void main(String[] args) {

	// create 3 BigInteger objects
	BigInteger bi1, bi2, bi3;  

	// assign values to bi1, bi2
	bi1 = new BigInteger("6"); //110
	bi2 = new BigInteger("3"); //011

	// perform andNot operation on bi1 using bi2
	bi3 = bi1.andNot(bi2);
	  
	String str = "Result of andNot operation is " +bi3;;

	// print bi3 value
	System.out.println( str );
   }
}

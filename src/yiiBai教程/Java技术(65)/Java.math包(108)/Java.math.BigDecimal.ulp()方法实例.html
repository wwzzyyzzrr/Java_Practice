Java.math.BigDecimal.ulp()方法实例

 java.math.BigDecimal.ulp() 返回此BigDecimal的ulp的大小，在最后一位的单位。非零BigDecimal值的ulp是该值和BigDecimal值与数字相同的数量级下一个较大的正距离。

一个零值的ulp在数值上等于1与此精度。其结果存储具有相同的规模，因为这使结果为零和非零值等于 [1, this.scale()].

声明

以下是java.math.BigDecimal.ulp()方法的声明

public BigDecimal ulp()

参数

  • NA

返回值

此方法返回BigDecimal的ulp大小。

异常

  • NA

例子

下面的例子显示math.BigDecimal.ulp()方法的用法

package com.yiibai;

import java.math.*;

public class BigDecimalDemo {

   public static void main(String[] args) {

      // create 4 BigDecimal objects
      BigDecimal bg1, bg2, bg3, bg4;

      bg1 = new BigDecimal("123");
      bg2 = new BigDecimal("1.23");

      // assign ulp value of bg1, bg2 to bg3, bg4
      bg3 = bg1.ulp();
      bg4 = bg2.ulp();

      String str1 = "ULP value of " + bg1 + " is " + bg3;
      String str2 = "ULP value of " + bg2 + " is " + bg4;

      // print bg3, bg4 values
      System.out.println( str1 );
      System.out.println( str2 );
   }
}

让我们编译和运行上面的程序，这将产生以下结果：

ULP value of 123 is 1
ULP value of 1.23 is 0.01

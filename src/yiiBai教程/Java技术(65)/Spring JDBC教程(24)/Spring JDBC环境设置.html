Spring JDBC环境设置

本章将介绍在Windows和Linux系统上设置Spring-AOP的过程。Spring-AOP可以通过以下几个简单安装的步骤，和集成到当前的Java环境和MAVEN中，而无需任何复杂的安装过程。

必要安装工具 -

  • Java(JDK 8) - 参考：http://www.yiibai.com/java/java_environment_setup.html
  • Maven 3 - 参考： http://www.yiibai.com/maven/maven_environment_setup.html
  • Eclipse - 参考： http://www.yiibai.com/eclipse/eclipse_installation.html

需要注意的是：

本系列教程以 Eclipse + Maven 模式作为基础开发环境，所以您需要熟悉 Eclipse + Maven 模式开发环境。

现在假设您已经安装完成了上面三个工具。然后可以继续进行下一章：第一个AOP示例。

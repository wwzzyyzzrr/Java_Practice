Java9开发环境安装

如果您想为Java编程语言设置自己的开发环境，请按照以下步骤设置您的Java环境。

Java SE可以免费下载。 要下载请点击这里，请下载与您的操作系统兼容的版本。

注:在书写本教程时，使用的是Windows 10 64位操作系统，因此教程以此环境编译和执行代码。

按照说明下载Java(dk-9.0.4_windows-x64_bin.exe)，然后运行.exe在您的机器上安装Java。 当机器上成功地安装了Java 9，还需要设置环境变量来指向正确的安装目录。

双击下载的文件:jdk-9.0.4_windows-x64_bin.exe ， 进入安装过程 -

点击是，进入下一步 -

点击下一步(N)，进入下一步，选择安装目录，这里选择安装到D:\Program Files\Java\jdk-9.0.4\ -

选择JIR安装的目录，这里安装到:D:\Program Files\Java\jir-9.0.4\，如下图所示 -

安装完成，如下图所示 -

注:默认情况下程序自动将java安装路径添加到系统环境变量中。

验证安装结果，打开命令提示符，执行 - 

C:\Users\Administrator>java -version
java version "9.0.4"
Java(TM) SE Runtime Environment (build 9.0.4+11)
Java HotSpot(TM) 64-Bit Server VM (build 9.0.4+11, mixed mode)

可以看到有安装了最新的Java版本为:"9.0.4"

Java9 JavaDocs

Java文档可以使用javadoc工具生成。 它目前以html 4.0格式生成文档。 在java 9中，可以通过在命令行参数中使用-html5选项生成html 5格式的文档。

旧式的java文档

考虑F:\worksp\java9文件夹中有一个Java文件:Tester.java，其代码如下 - 

/**
  * @author 小马哥(Maxsu)
  * @version 0.1
*/
public class Tester {
   /**
      * Default method to be run to print 
      * <p>Hello world</p>
      * @param args command line arguments
   */
   public static void main(String []args) {
      System.out.println("Hello World");
   }
}

可以运行jdk 7 的javadoc工具来生成文档。

使用-html5标志运行jdk 9的javadoc工具以生成新类型的文档。

Java并发开发环境配置

Java SE可以从Java官方网站免费获得。可以根据您的操作系统下载一个版本。

  • http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

以在Windows系统上为例，按照说明下载Java并运行.exe在您的机器上安装Java。在机器上安装了Java之后，还需要设置环境变量来指向正确的Java安装目录。

流行的Java编辑器

要编写Java程序，您需要一个文本编辑器。市场上很多，很复杂的IDE。建议您考虑以下之一 -

  • 记事本 - 在Windows机器上，您可以使用任何简单的文本编辑器，如记事本(本教程推荐)，TextPad。

  • Netbeans - 一个开放源代码和免费的Java IDE，可从 http://www.netbeans.org/index.html 下载。

  • Eclipse - 由Eclipse开源社区开发的Java IDE，可从 http://www.eclipse.org/ 下载。(Eclipse是Java相关教程推荐使用)

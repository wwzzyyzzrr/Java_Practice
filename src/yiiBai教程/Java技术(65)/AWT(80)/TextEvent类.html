TextEvent类

这个类的对象表示文本事件。TextEvent 输入字符在文本字段或文本区域时产生的。 TextEvent并将实例不包含字符的文本组件产生的事件，而我们提供了其他的方法来检索信息。

类的声明

以下是声明的 java.awt.event.TextEvent类：

public class TextEvent
   extends AWTEvent

字段域

字段类java.awt.event.TextEvent：

  • static int TEXT_FIRST -- 用于文本事件的id范围内的第一个数字。

  • static int TEXT_LAST -- 用于文本事件的ID范围内的最后一个数字。

  • static int TEXT_VALUE_CHANGED --此事件ID表示对象的文本改变。

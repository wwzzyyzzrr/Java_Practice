AWT ComponentEvent类

介绍

ComponentEvent 类表示，组件被移动，改变大小，更改或可见性

类的声明

以下是声明的java.awt.event.ComponentEvent类：

public class ComponentEvent
   extends AWTEvent

字段类

以下是java.awt.Component的 类的字段：

  • static int COMPONENT_FIRST -- 第一个数字范围内用于组件事件的ID。

  • static int COMPONENT_HIDDEN -- 此事件表示该组件被渲染不可见。

  • static int COMPONENT_LAST -- 最后一个数字范围内用于组件事件的ID。

  • static int COMPONENT_MOVED -- 此事件表示组件的位置改变。

  • static int COMPONENT_RESIZED -- 此事件表示组件的大小改变。

  • static int COMPONENT_SHOWN -- 此事件表示该组件可见。

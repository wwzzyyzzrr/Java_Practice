indexOf(Object elem,int index)方法实例

这是之前的 indexOf()方法的另一变体。唯一的区别是，在搜索给定参数的第一次出现开始于提到的第二个参数的索引位置。

声明

以下是java.util.Vector.indexOf()方法的声明

public int indexOf(Object elem,int index)

参数

  • elem--输入参数是一个对象

  • index--这是从非负指数开始搜索

返回值

该方法调用的位置索引或更新版本的矢量返回此向量的对象参数的第一个匹配项的索引。

异常

  • IndexOutOfBoundsException--如果该指数为负，抛出此异常

Java.util.Calendar.getInstance(Locale locale)方法实例

 java.util.Calendar.getInstance() 使用目前的时区和指定语言环境的方法得到一个日历。

声明

以下是java.util.Calendar.getInstance()方法的声明

public static Calendar getInstance(Locale locale)

参数

  • locale -- 语言环境的日历数据

返回值

该方法返回一个日历。

异常

  • NA

例子

下面的示例演示java.util.calendar.getInstance()方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {
      
      // create a calendar
      Calendar cal = Calendar.getInstance(Locale.CANADA);

      // print the date for canada 
      System.out.println("Date And Time in Canada Is: " + cal.getTime());
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Date And Time in China Is: Tue May 01 19:05:34 EEST 2012

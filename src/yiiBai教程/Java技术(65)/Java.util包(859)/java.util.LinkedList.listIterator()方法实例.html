java.util.LinkedList.listIterator()方法实例

java.util.LinkedList.listIterator(int index) 方法返回一个列表迭代器在此列表中的元素(按适当顺序)，从列表中的指定位置。

声明

以下是java.util.LinkedList.listIterator()方法的声明

public ListIterator<E> listIterator(int index)

参数

  • index -- 从列表迭代器返回的第一个元素的索引

返回值

此方法返回此列表中的元素的ListIterator(按适当顺序)，从列表中的指定位置

异常

  • IndexOutOfBoundsException -- 如果索引超出范围

例子

下面的示例演示java.util.LinkedList.listIterator()方法的用法。

package com.yiibai;

import java.util.*;

public class LinkedListDemo {

   public static void main(String[] args) {

      // create a LinkedList
      LinkedList list = new LinkedList();

      // add some elements
      list.add("Hello");
      list.add(2);
      list.add("Chocolate");
      list.add("10");

      // print the list
      System.out.println("LinkedList:" + list);

      // set Iterator at specified index
      Iterator x = list.listIterator(1);

      // print list with the iterator
      while (x.hasNext()) {
         System.out.println(x.next());
      }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

2
Chocolate
10

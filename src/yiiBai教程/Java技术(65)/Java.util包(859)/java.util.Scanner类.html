java.util.Scanner类

java.util.Scanner 类是一个简单的文本扫描器可以分析基本类型和字符串使用正则表达式。以下是关于扫描器的要点：

  • 一个扫描器使用分隔符模式分解它的输入，默认情况下与空白匹配。

  • 扫描操作可能阻塞等待输入。

  • 扫描器是不是安全的，无需外部同步多线程使用。

类声明

以下是java.util.Scanner类的声明：

public final class Scanner
   extends Object
      implements Iterator<String>

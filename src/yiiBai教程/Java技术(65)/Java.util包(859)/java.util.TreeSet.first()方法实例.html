java.util.TreeSet.first()方法实例

first() 方法用于返回这个集合中当前的第一个(最小的)元素。

声明

以下是 java.util.TreeSet.first()方法的声明。

public E first()

参数

  • NA

返回值

该方法调用返回的第一个(最低)元素当前在此set。

异常

  • NoSuchElementException--如果集合为空，抛出此异常。

例子

下面的例子展示java.util.TreeSet.first()方法的使用。

package com.yiibai;

import java.util.TreeSet;

public class TreeSetDemo {
   public static void main(String[] args) {
     // creating a TreeSet 
     TreeSet <Integer>treeadd = new TreeSet<Integer>();
     
     // adding in the tree set
     treeadd.add(12);
     treeadd.add(11);
     treeadd.add(16);
     treeadd.add(15);
     
     // getting the first lowset element
     System.out.println("First lowest element: "+treeadd.first());   
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

First lowest element: 11

java.util.Hashtable.containsValue()方法实例

containsValue(Object value) 方法是用来测试是否该哈希映射一个或多个键指向值。

声明

以下是java.util.Hashtable.containsValue()方法的声明。

public boolean containsValue(Object value)

参数

  • value--这是一个进行搜索的值。

返回值

该方法调用返回true，如果此映射将一个或多个键映射到指定值。

异常

  • NullPointerException--如果该值为null，则此异常。

例子

下面的例子显示java.util.Hashtable.containsValue()方法的使用

package com.yiibai;

import java.util.*;

public class HashTableDemo {
   public static void main(String args[]) {
      // create hash table 
      Hashtable htable = new Hashtable();
      
      // put values into the table
      htable.put(1, "A");
      htable.put(2, "B");
      htable.put(3, "C");
      htable.put(4, "D");
      
      // check if table contains value "D"
      boolean isavailable=htable.containsValue("D");

      // display search result
      System.out.println("Hash table contains value 'D': "+isavailable);      
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Hash table contains value 'D': true

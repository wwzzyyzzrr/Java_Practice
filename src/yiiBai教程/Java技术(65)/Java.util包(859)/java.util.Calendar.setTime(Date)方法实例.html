java.util.Calendar.setTime(Date)方法实例

java.util.Calendar.setTime(Date) 方法设置日历的时间为给定的日期 Date.

声明

以下是java.util.Calendar.setTime()方法的声明

public final void setTime(Date date)

参数

  • date --给定的日期

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.util.calendar.setTime()方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {

      // create a calendar
      Calendar cal = Calendar.getInstance();

      // get the current time
      System.out.println("Current time is :" + cal.getTime());

      // create new date and set it
      Date date = new Date(95, 10, 10);
      cal.setTime(date);
	  
      // print the new time
      System.out.println("After setting Time:  " + cal.getTime());
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Current time is :Wed May 02 20:33:17 EEST 2012
After setting Time:  Fri Nov 10 00:00:00 EET 1995

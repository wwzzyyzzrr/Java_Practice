java.util.EnumSet.of()方法实例

java.util.EnumSet.of(E e) 方法创建一个最初包含指定元素的枚举set。

声明

以下是java.util.EnumSet.of()方法的声明

public static <E extends Enum<E>> EnumSet<E> of(E e)

参数

  • e-- 这个集合是最初要包含的元素。

返回值

此方法返回最初包含指定元素的枚举set。

异常

  • NullPointerException-- 如果 e 为 null

例子

下面的示例演示java.util.EnumSet.Of()方法的用法。

package com.yiibai;

import java.util.*;

public class EnumSetDemo {

   // create an enum
   public enum Numbers {

      ONE, TWO, THREE, FOUR, FIVE
   };

   public static void main(String[] args) {

      // create a set
      EnumSet<Numbers> set;

      // add one element
      set = EnumSet.of(Numbers.FIVE);

      // print the set
      System.out.println("Set:" + set);

      // add another element which replaces the previous
      set = EnumSet.of(Numbers.THREE);

      // print the set. Notice that it has one element
      System.out.println("Set:" + set);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Set:[FIVE]
Set:[THREE]

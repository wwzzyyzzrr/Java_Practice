java.util.StringTokenizer类

java.util.StringTokenizer 类允许应用程序将字符串分解令牌。

  • 由于类保留兼容性的原因，不鼓励使用新的代码遗留下来的类。

  • 其方法不识别，数字和引用字符串中区分开来。

  • 这个类的方法甚至不承认和跳过注释。

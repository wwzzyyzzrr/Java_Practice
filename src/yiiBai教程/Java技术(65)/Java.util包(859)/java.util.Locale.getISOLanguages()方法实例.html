java.util.Locale.getISOLanguages()方法实例

java.util.Locale.getISOLanguages() 方法返回ISO 639中定义的所有两字母语言代码。可用于创建语言环境的列表。

声明

以下是java.util.Locale.getISOLanguages()方法的声明

public static String[] getISOLanguages()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.util.Locale.getISOLanguages​​()方法的用法。

package com.yiibai;

import java.util.*;

public class LocaleDemo {

   public static void main(String[] args) {

      // create a new locale
      Locale locale = new Locale("en", "US", "WIN");

      // print locale
      System.out.println("Locale:" + locale);

      // get ISO languages
      String[] languages = Locale.getISOLanguages();

      // print countries
      System.out.println("Languages are:");
      for (int i = 0; i < languages.length; i++) {
         System.out.println(i + ":" + languages[i]);
      }
   }
}

addAll(int index,Collection<? extends E> c)方法实例

addAll(int index,Collection<? extends E> c) 方法是先前addAll()方法的另一种变型。它插入所有指定集合中的元素融入到这个列表中。插入启动指定的位置/索引和移动元素目前在该位置(如果有的话)。对应的索引还增加转向后续元素向右移动。因为它们是由指定集合的迭代器返回的新元素将出现在列表中的顺序相同。

声明

以下是java.util.Vector.addAll()方法的声明

public boolean addAll(int index,Collection<? extends E> c)

参数

  • index -- 这是该指数/位置，从指定集合的第一个元素将被插入。

  • c -- 这是要添加到该列表中包含的元素的集合。

返回值

返回类型是true如果该列表被更改为调用的结果。

异常

  • NullPointerException -- 如果指定集合为null，则方法调用将抛出此异常。

    IndexOutOfBoundsException -- 如果访问的索引/位置超出范围这将引发异常 (index < 0 || index > size()).

例子

下面的例子显示java.util.Vector.addAll(int index,Collection c) 方法的使用。

package com.yiibai;

import java.util.Vector;

public class VectorDemo {
   public static void main(String[] args) {
      // create two empty Vectors firstvec and secondvec      
      Vector<Integer> firstvec = new Vector<Integer>(4);      
      Vector<Integer> secondvec = new Vector<Integer>(4);
      
      // use add() method to add elements in the secondvec vector
      secondvec.add(5);
      secondvec.add(6);
      secondvec.add(7);
      secondvec.add(8);

      // use add() method to add elements in the firstvec vector
      firstvec.add(1);
      firstvec.add(2);
      firstvec.add(3);
      firstvec.add(4);
      
      /** use addAll() method to add elements of the 2nd vector at 
      1st element position of the first vector */
              
      firstvec.addAll(1,secondvec);

      // let us print all the elements available in vector
      System.out.println("Added numbers are :- "); 
      for (Integer number : firstvec) {         
         System.out.println("Number = " + number);
      }           
   } 
}

java.util.Collections.checkedSortedMap()方法实例

checkedSortedMap(SortedMap<, V>, Class<K>, Class<V>) 方法用于获取指定有序映射的一个动态类型安全视图。

声明

以下是java.util.Collections.checkedSortedMap()方法的声明。

public static <K,V> SortedMap<K,V> checkedSortedMap(SortedMap<K,V> m,Class<K> keyType,Class<V> valueType)

参数

  • m--这对于一个动态类型安全视图是要返回的地图。

  • keyType--键类型m被允许保持的。

  • valueType --类型的值m被允许保持。

返回值

该方法调用返回指定映射的一个动态类型安全视图。

异常

  • NA

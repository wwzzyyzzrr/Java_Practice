java.util.EnumSet.allOf()方法实例

 java.util.EnumSet.allOf(Class<E> elementType) 方法创建一个包含所有在指定元素类型的元素的枚举set。

声明

以下是java.util.EnumSet.allOf()方法的声明

public static <E extends Enum<E>> EnumSet<E> allOf(Class<E> elementType)

参数

  • elementType -- 这是个枚举set的元素类型的类对象

返回值

此方法不返回任何值。

异常

  • NullPointerException-- 如果elementType 为 null

例子

下面的示例演示java.util.EnumSet.allOf()方法的用法。

package com.yiibai;

import java.util.*;

public class EnumSetDemo {

   // create an enum
   public enum Numbers {

      ONE, TWO, THREE, FOUR, FIVE
   };

   public static void main(String[] args) {

      // create an empty EnumSet 
      EnumSet<Numbers> set = null;

      // print the set
      System.out.println(set);

      // create the set by getting all elements from Numbers
      set = EnumSet.allOf(Numbers.class);

      // print the updated set
      System.out.println("Updated set:" + set);

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

null
Updated set[ONE, TWO, THREE, FOUR, FIVE]

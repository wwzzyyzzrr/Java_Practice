java.util.Calendar.getDisplayNames()方法实例

java.util.Calendar.getDisplayNames() 方法返回一个Map包含日历字段在给定的风格和语言环境和相应的字段值的所有名称。

声明

以下是java.util.Calendar.getDisplayNames()方法的声明

public Map<String,Integer> getDisplayNames(int field,int style,Locale locale)

参数

  • field -- 日历字段。

  • style -- 将被应用到字符串表示形式风格

  • locale -- 该字符串表示形式语言环境

返回值

该方法返回一个Map包含所有显示名称的风格和语言环境和字段值，或者null，如果没有字符串表示形式。

异常

  • IllegalArgumentException - 如果field或style都无效，如果此日历是不宽松，任何字段具有无效值

  • NullPointerException - 如果 locale 为 null

例子

下面的示例演示java.util.calendar.getDisplayNames()方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {

      // create calendar and locale
      Calendar now = Calendar.getInstance();
      Locale locale = Locale.getDefault();

      // call the getdisplaynames method
      Map< String, Integer> representations = 
	  now.getDisplayNames(Calendar.DAY_OF_WEEK, Calendar.LONG, locale);
      NavigableMap< String, Integer> navMap = 
	  new TreeMap< String, Integer>(representations);

      // print the results
      System.out.printf("Whole list:%n%s%n", navMap);
   }
}

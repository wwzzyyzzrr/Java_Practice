java.lang.Character.equals()方法实例

java.lang.Character.equals(Object obj) 这个对象与指定对象比较。其结果是true，当且仅当参数不为 null；是一个字符对象，表示同样的char值作为此对象。

声明

以下是java.lang.Character.equals()方法的声明

public boolean equals(Object obj)

Overrides

equals 在类 Object

参数

  • obj - 与比较对象

返回值

如果对象是相同的，此方法返回true，否则返回false。

异常

  • NA

例子

下面的例子显示lang.Character.equals()方法的使用。

package com.yiibai;

import java.lang.*;

public class CharacterDemo {

   public static void main(String[] args) {

      // create 2 Character objects c1, c2
      Character c1, c2;

      // create a boolean primitive res
      boolean res;

      // assign values to c1, c2
      c1 = new Character('a');
      c2 = new Character('A');

      // assign the result of equals method on c1, c2 to res
      res = c1.equals(c2);

      String str = c1+ " and " + c2 + " are equal is " + res;

      // print res value
      System.out.println( str );
   }
}

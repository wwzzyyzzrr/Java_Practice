java.util.EnumSet.of(E first, E rest)方法实例

java.util.EnumSet.of(E first, E... rest) 方法创建一个最初包含指定元素的枚举set。此工厂参数列表使用参数功能，可用于创建一个枚举set最初包含元素的任意数字，但它很可能运行速度比不使用变量参数的重载运行得慢。

声明

以下是java.util.EnumSet.of()方法的声明

public static <E extends Enum<E>> EnumSet<E> of(E first, E... rest)

参数

  • first -- 该集最初要包含的元素。

  • rest -- 其余的元素集合最初要包含的。

返回值

此方法返回最初包含指定元素的枚举set。

异常

  • NullPointerException-- 如果e 是 null

例子

下面的示例演示java.util.EnumSet.Of()方法的用法。

/*This example is using a method called main2
to simulate calling the main method using args
from a command line.*/

package com.yiibai;

import java.util.*;

public class EnumSetDemo {

   // create an enum
   public enum Numbers {

      ONE, TWO, THREE, FOUR, FIVE
   };

   public static void main(String[] args) {

      // create a fake list that will be used like args
      Numbers[] list = {Numbers.ONE, Numbers.THREE, Numbers.FOUR, Numbers.FIVE};

      // call the fake main
      main2(list);

   }

   // This is a fake main. This is used as an example
   public static void main2(Numbers[] fakeargs) {

      // create a set
      EnumSet<Numbers> set;

      // add first element and the rest of fakeargs
      set = EnumSet.of(Numbers.ONE, fakeargs);

      // print the set
      System.out.println("Set:" + set);

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Set:[ONE, THREE, FOUR, FIVE]

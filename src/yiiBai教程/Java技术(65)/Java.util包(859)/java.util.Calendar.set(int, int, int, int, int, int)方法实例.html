java.util.Calendar.set(int, int, int, int, int, int)方法实例

java.util.Calendar.set(int, int, int, int, int, int) 方法设置为日历中的值的字段YEAR，MONTH，DAY_OF_MONTH，HOUR_OF_DAY，MINUTE 和 SECOND。

声明

以下是java.util.Calendar.set()方法的声明

public final void set(int year,int month,int day,int hourOfDay, int minute,int second)

参数

  • year -- 用于YEAR字段的值

  • month -- 用于MONTH字段的值。 0表示一月

  • day -- 用于DAY字段的值

  • hourOfDay -- 被用于HOUR_OF_DAY字段的值

  • minute -- 用于MINUTE字段的值

  • secpnd -- 用于SECOND字段的值

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的例子显示了java.util.calendar.set()方法的用法。

package com.yiibai;
import java.util.Calendar;
public class CalendarDemo {
   public static void main(String[] args) {
      // create a calendar
      Calendar cal = Calendar.getInstance();
      // print current time
      System.out.println("Current year is :" + cal.getTime());
      // set the year,month and day to something else
      cal.set(1995, 5, 25, 04, 15, 20);
      // print the result
      System.out.println("Altered year is :" + cal.getTime());
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Current year is :Wed May 02 19:38:06 EEST 2012
Altered year is :Sun Jun 25 04:15:20 EEST 1995

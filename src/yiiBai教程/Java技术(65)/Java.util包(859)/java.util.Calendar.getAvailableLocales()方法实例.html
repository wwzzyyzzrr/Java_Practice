java.util.Calendar.getAvailableLocales()方法实例

java.util.Calendar.getAvailableLocales() 方法返回的语言环境的数组，它由此类的getInstance方法可以返回本地化的实例。

声明

以下是java.util.Calendar.getAvailableLocales()方法的声明

public static Locale[] getAvailableLocales()

参数

  • NA

返回值

返回语言环境的本地化的日历实例可用的数组。

异常

  • NA

例子

下面的示例演示java.util.calendar.getAvailableLocales()方法的用法。

package com.yiibai;

import java.util.*;

class CalendarGetAvailableLocalesEx {

   public static void main(String args[]) {

      // create an object of locale class 
      Locale[] array = new Locale[10];

      // get available locales and assign them to array
      array = Locale.getAvailableLocales();

      // print the results
      System.out.println("The first 10 locales installed are :-
");
      for (int i = 0; i < 10; i++) {
         System.out.println(array[i].getISO3Country());
      }
   }
}

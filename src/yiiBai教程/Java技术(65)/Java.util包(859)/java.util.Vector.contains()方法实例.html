java.util.Vector.contains()方法实例

contains(Object elem) 方法是用来测试向量中的元素是否存在。

声明

以下是java.util.Vector.contains()方法的声明

public boolean contains(Object elem)

参数

  • elem-- 这是一个作为输入的对象。

返回值

true-- 当且仅当指定的对象是相同的在此向量的分量，则返回true。否则返回false。

异常

  • ClassCastException-- 如果指定元素的类型与此集合(可选)不兼容。

    NullPointerException-- 如果指定的元素为null，并且此collection不支持null元素(可选)。

例子

下面的例子显示java.util.Vector.contains()方法的使用。

package com.yiibai;

import java.util.Vector;

public class VectorDemo {
   public static void main(String[] args) {
      // create an empty Vector vec with an initial capacity of 4      
      Vector<Integer> vec = new Vector<Integer>(4);

      // use add() method to add elements in the vector
      vec.add(4);
      vec.add(3);
      vec.add(2);
      vec.add(1);

      // let us check the existence of number 4 in the vector
      System.out.println("Checking the existence of number 4 :- "+vec.contains(4));                 
   }
}

让我们编译并运行上述程序，这将产生以下结果。向量存在数量，方法调用返回true

Checking the existence of number 4 :- true

java.util.Scanner.nextBigDecimal()方法实例

java.util.Scanner.nextBigDecimal() 方法扫描输入的下一个标记为一个BigDecimal。如果下一个标记与上述定义，则令牌通过移除所有组分隔符，通过Character.digit映射非ASCII数字为ASCII数字，以及将得到的字符串到BigDecimal转换成一个BigDecimal值，如果正则表达式匹配BigDecimal(String) 构造。

声明

以下是java.util.Scanner.nextBigDecimal()方法的声明

public BigDecimal nextBigDecimal()

参数

  • NA

返回值

此方法返回从输入信息扫描的BigDecimal

异常

  • InputMismatchException -- 如果下一个标记不匹配小数正则表达式，或者是超出范围

  • NoSuchElementException -- 如果输入被耗尽

  • IllegalStateException -- 如果此scanner 已关闭

例子

下面的示例演示java.util.Scanner.nextBigDecimal()方法的用法。

package com.yiibai;

import java.util.*;

public class ScannerDemo {

   public static void main(String[] args) {

      String s = "Hello World! 3 + 3.0 = 6  true";

      // create a new scanner with the specified String Object
      Scanner scanner = new Scanner(s);

      // find the next BigDecimal token and print it
      // loop for the whole scanner
      while (scanner.hasNext()) {

         // if the next is BigDecimal, print found and the decimal
         if (scanner.hasNextBigDecimal()) {
            System.out.println("Found :" + scanner.nextBigDecimal());
         }

         // if a BigDecimal is not found, print "Not Found" and the token
         System.out.println("Not Found :" + scanner.next());
      }

      // close the scanner
      scanner.close();
   }
}

java.util.Vector.add(int index,E element)方法实例

add(int index,E element) 方法用于将指定的元素(E)在列表中的指定位置(索引)。当插入新的元素，它改变了现有元素在该位置(如果有的话)和所有后续元素向右移动与变化索引。这是add()方法的另一个变化。

声明

以下是java.util.Vector.add()方法的声明

public void add(int index,E element)

参数

  • index -- 这是索引(position)，在该位置指定的元素将被插入。

  • element -- 这是要在指定位置插入的元素。

返回值

返回类型为void因此该方法不返回任何值。

异常

  • IndexOutOfBoundsException - 该方法将抛出这个异常，如果该索引(元素的位置)，我们尝试访问超出了范围(index < 0 || index > size()).

例子

下面的例子显示java.util.Vector.add(int index,E element)方法的使用。

package com.yiibai;

import java.util.Vector;

public class VectorDemo {
    
   public static void main(String[] args) {
      // create an empty Vector vec with an initial capacity of 4      
      Vector<Integer> vec = new Vector<Integer>(4);

      // use add() method to add elements in the vector
      vec.add(0,4);
      vec.add(1,3);
      vec.add(2,2);
      vec.add(3,1);

      // let us print all the elements available in vector
      System.out.println("Added numbers are :- "); 
      for (Integer number : vec) {         
         System.out.println("Index :"+vec.indexOf(number) +" Number: " + number);
      }
      
      // added new number 10 at 3rd position/index
      vec.add(3,10);
      
      // let us print all the elements available in vector after insertion
      System.out.println("Added numbers after insertion are :- "); 
      for (Integer number : vec) {         
         System.out.println("Index :"+vec.indexOf(number) +" Number: " + number);
      }
   }  
}

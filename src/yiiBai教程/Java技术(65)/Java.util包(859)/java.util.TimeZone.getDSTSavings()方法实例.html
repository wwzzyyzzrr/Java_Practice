java.util.TimeZone.getDSTSavings()方法实例

getDSTSavings() 方法用于返回的时候要添加到本地标准时间以获取本地挂钟时间。

声明

以下是java.util.TimeZone.getDSTSavings()方法的声明。

public int getDSTSavings()

参数

  • NA

返回值

方法调用返回以毫秒为单位的节约时间。

异常

  • NA

例子

下面的例子显示java.util.TimeZone.getDSTSavings()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
    public static void main( String args[] ){
       
      // create time zone object     
      TimeZone timezone = TimeZone.getTimeZone("Europe/Paris");
      
      // checking DST saving         
      System.out.println("DST saving is :" + timezone.getDSTSavings());
   }    
}

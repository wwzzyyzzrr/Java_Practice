Java.util.ArrayDeque.peekFirst()方法实例

java.util.ArrayDeque.peekFirst() 方法检索此deque队列的第一个元素(但不删除)，则返回null如果此deque队列为空。

声明

以下是java.util.ArrayDeque.peekFirst()方法的声明

public E peekFirst()

参数

  • NA

返回值

此方法返回此deque队列的头部，或如果此deque队列为空返回null。

异常

  • NA

例子

下面的示例演示java.util.ArrayDeque.peekFirst()方法的用法。

package com.yiibai;

import java.util.ArrayDeque;
import java.util.Deque;

public class ArrayDequeDemo {
   public static void main(String[] args) {
      
      // create an empty array deque with an initial capacity
      Deque<Integer> deque = new ArrayDeque<Integer>(8);

      // use add() method to add elements in the deque
      deque.add(25);
      deque.add(30);
      deque.add(20);
      deque.add(18);        
           
      // this will retrieve head of the queue
      int retval = deque.peekFirst();
      System.out.println("Retrieved Element is " + retval);
	  
      // printing all the elements available in deque
      for (Integer number : deque) {
         System.out.println("Number = " + number);
      }
   }
}

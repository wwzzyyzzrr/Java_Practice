java.util.Locale.getDisplayName(Locale inLocale)方法实例

java.util.Locale.getDisplayName(Locale inLocale) 方法返回的语言环境适合用户显示的名称。这将是getDisplayLanguage()，getDisplayCountry()，和getDisplayVariant()的返回值组装成一个字符串。

声明

以下是java.util.Locale.getDisplayName()方法的声明

public String getDisplayName(Locale inLocale)

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NullPointerException -- 如果 inLocale 为 null

例子

下面的示例演示java.util.Locale.getDisplayName()方法的用法。

package com.yiibai;

import java.util.*;

public class LocaleDemo {

   public static void main(String[] args) {

      // create a new locale
      Locale locale = new Locale("ENGLISH", "US");

      // print locale
      System.out.println("Locale:" + locale);

      // print display name for locale - based on inLocale
      System.out.println("Name:"
              + locale.getDisplayName(new Locale("GERMAN", "GERMANY")));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Locale:english_US
Name:english (United States)

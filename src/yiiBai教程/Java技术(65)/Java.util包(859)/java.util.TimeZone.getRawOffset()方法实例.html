java.util.TimeZone.getRawOffset()方法实例

getRawOffset() 方法用于得到以毫秒为单位的时间量添加到UTC以获取标准时间在这个时间段。

声明

以下是java.util.TimeZone.getRawOffset()方法的声明。

public abstract int getRawOffset()

参数

  • NA

返回值

方法调用返回原始的偏移时间量以毫秒为单位，增加到UTC。

异常

  • NA

例子

下面的例子显示java.util.TimeZone.getRawOffset()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
   public static void main( String args[] ){
       
      // create time zone object     
      TimeZone timezone = TimeZone.getDefault();
      
      // checking raw offset value     
      System.out.println("Raw Offset value:" + timezone.getRawOffset());
   }    
}

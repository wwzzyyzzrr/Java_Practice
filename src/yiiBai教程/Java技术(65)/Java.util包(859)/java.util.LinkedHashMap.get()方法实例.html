java.util.LinkedHashMap.get()方法实例

java.util.LinkedHashMap.get() 方法返回指定键所映射的值，或者如果此映射不包含该键，返回null。

声明

以下是java.util.LinkedHashMap.get()方法的声明

public V get(Object key)

参数

  • key -- 要返回与其关联键的值

返回值

此方法返回的值来指定键映射，或者null，如果此映射不包含该键

异常

  • NA

例子

下面的示例演示java.util.LinkedHashMap.get()方法的用法。

package com.yiibai;

import java.util.*;

public class LinkedHashMapDemo {

   public static void main(String[] args) {

      // create a new linked hash map
      LinkedHashMap map = new LinkedHashMap(5);

      // add some values in the map
      map.put("One", 1);
      map.put("Two", 2);
      map.put("Three", 3);

      // print the map
      System.out.println("Map:" + map);

      // get key "Three"
      System.out.println("" + map.get("Three"));

      // get key "Five"
      System.out.println("" + map.get("Five"));
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Map:{One=1, Two=2, Three=3}
3
null

min(Collection<? extends T>, Comparator<? super T>)方法实例

min(Collection<? extends T>, Comparator<? super T>) 方法用来返回给定集合中的最小元素，根据诱发指定比较器的顺序。

声明

以下是java.util.Collections.min()方法的声明。

public static <T> T min(Collection<? extends T> coll,Comparator<? super T> comp)

参数

  • coll-- 该集合的最小元素为待定。

  • comp-- 比较器，用以确定最小元素。

返回值

方法调用返回给定collection的最小元素，根据指定的比较器。

异常

  • ClassCastException-- 这被抛出如果集合包含不可相互比较的元素使用指定的比较。

  • NoSuchElementException--如果集合为空的，这是抛出。

例子

下面的例子显示java.util.Collections.min()方法的使用

package com.yiibai;

import java.util.*;

public class CollectionsDemo {
   public static void main(String args[]) { 
      // create link list object 
      LinkedList<Integer> list = new LinkedList<Integer>();
	  
      // populate the list  
      list.add(-18);  
      list.add(40);  
      list.add(-45);  
      list.add(12); 
      
      // comparing using natural ordering
      System.out.println("Min val: " + Collections.min(list,null));          
   }  
}

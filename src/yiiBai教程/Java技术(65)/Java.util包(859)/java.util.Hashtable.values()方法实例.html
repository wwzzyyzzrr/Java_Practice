java.util.Hashtable.values()方法实例

values() 方法是用来获取包含在这个哈希表中的值的Collection视图。

声明

以下是java.util.Hashtable.values()方法的声明。

public Collection<V> values()

参数

  • NA

返回值

在方法调用返回此映射中包含的值的collection视图。

异常

  • NA

例子

下面的例子显示java.util.Hashtable.values()方法的使用

package com.yiibai;

import java.util.*;

public class HashTableDemo {
   public static void main(String args[]) {
      // create hash table 
      Hashtable htable = new Hashtable(3); 
      
      // populate the table
      htable.put(1, "TP");
      htable.put(2, "IS");
      htable.put(3, "THE");
      htable.put(4, "BEST");
      htable.put(5, "TUTORIAL");
      
      System.out.println("Collection view of hash table: "+htable.values()); 
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Collection view of hash table: [TUTORIAL, BEST, THE, IS, TP]

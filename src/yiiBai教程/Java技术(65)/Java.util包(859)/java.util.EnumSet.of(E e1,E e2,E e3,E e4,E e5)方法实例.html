java.util.EnumSet.of(E e1,E e2,E e3,E e4,E e5)方法实例

java.util.EnumSet.of(E e1,E e2,E e3,E e4,E e5) 方法创建一个最初包含五个指定元素的枚举set。

声明

以下是java.util.EnumSet.of()方法的声明

public static <E extends Enum<E>> EnumSet<E> of(E e1,E e2,E e3,E e4,E e5)

参数

  • e1-- 一个元素，这个集合初始化所要包含的。

  • e2-- 另一个元素，这个集合初始化所要包含的。

  • e3-- 另一个元素，这个集合初始化所要包含的。

  • e4-- 另一个元素，这个集合初始化所要包含的。

  • e5-- 另一个元素，这个集合初始化所要包含的。

返回值

下面的示例演示java.util.EnumSet.Of()方法的用法。

异常

  • NullPointerException-- 如果其中一个参数为 null

例子

下面的示例演示java.util.EnumSet.Of()方法的用法。

package com.yiibai;

import java.util.*;

public class EnumSetDemo {

   // create an enum
   public enum Numbers {

      ONE, TWO, THREE, FOUR, FIVE
   };

   public static void main(String[] args) {

      // create a set
      EnumSet<Numbers> set;

      // add one element
      set = EnumSet.of(Numbers.FIVE,Numbers.FOUR,Numbers.THREE,Numbers.TWO,Numbers.ONE);

      // print the set
      System.out.println("Set:" + set);

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Set:[ONE,TWO,THREE,FOUR,FIVE]

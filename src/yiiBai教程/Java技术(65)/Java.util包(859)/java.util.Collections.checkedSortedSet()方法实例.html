java.util.Collections.checkedSortedSet()方法实例

checkedSortedSet(SortedSet<E>, Class<E>) 方法用于获取指定有序set的一个动态类型安全视图。

声明

以下是java.util.Collections.checkedSortedSet()方法的声明。

public static <E> SortedSet<E> checkedSortedSet(SortedSet<E> s, Class<E> type)

参数

  • s--这是有序集合，有关的动态类型安全视图将被返回。

  • type--这是元素的s被允许保持的类型。

返回值

在方法调用返回指定有序set的一个动态类型安全视图。

异常

  • NA

例子

下面的例子显示java.util.Collections.checkedSortedSet()方法的使用

package com.yiibai;

import java.util.*;

public class CollectionsDemo {
   public static void main(String args[]) {
      // create sorted set    
      SortedSet<String> sset = new TreeSet<String>();
      
      // populate the set
      sset.add("Java");
      sset.add("is");
      sset.add("best");
      
      // get typesafe view of the sorted set
      SortedSet<String> tsset;
      tsset = Collections.checkedSortedSet(sset,String.class);     
      
      System.out.println("Dynamically typesafe view: "+tsset);
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Dynamically typesafe view: [Java, best, is]

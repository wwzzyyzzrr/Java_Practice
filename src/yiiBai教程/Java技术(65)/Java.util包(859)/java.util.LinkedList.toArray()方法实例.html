java.util.LinkedList.toArray()方法实例

java.util.LinkedList.toArray() 方法返回一个包含所有在此列表中正确的序列中元素的数组(从第一个到最后一个元素)。此方法充当基于数组的和集合的API之间的桥梁。

声明

以下是java.util.LinkedList.toArray()方法的声明

public Object[] toArray()

参数

  • NA

返回值

这个方法返回一个包含正确的顺序所有在此列表中的元素的数组

异常

  • NA

例子

下面的示例演示java.util.LinkedList.toArray()方法的用法。

package com.yiibai;

import java.util.*;

public class LinkedListDemo {

   public static void main(String[] args) {

      // create a LinkedList
      LinkedList list = new LinkedList();

      // add some elements
      list.add("Hello");
      list.add(2);
      list.add("Chocolate");
      list.add("10");

      // print the list
      System.out.println("LinkedList:" + list);

      // create an array and copy the list to it
      Object[] array = list.toArray();

      // print the array
      for (int i = 0; i < list.size(); i++) {
         System.out.println("Array:" + array[i]);
      }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

LinkedList:[Hello, 2, Chocolate, 10]
Array:Hello
Array:2
Array:Chocolate
Array:10

java.util.Dictionary.put(K key,V value)方法实例

java.util.Dictionary.put(K key,V value) 方法映射到指定键到这个字典中指定的值。

声明

以下是java.util.Dictionary.keys()方法的声明

public abstract V put(K key,V value)

参数

  • key --哈希表键

  • value -- 值

返回值

此方法返回到该键被映射到这个字典中，则返回null如果该键没有映射值。

异常

  • NullPointerException -- 如果该值或key为null

例子

下面的示例演示java.util.Dictionary.put()方法的用法。

package com.yiibai;

import java.util.*;

public class DictionaryDemo {

   public static void main(String[] args) {

      // create a new hasthtable
      Dictionary d = new Hashtable();

      // put some elements
      d.put("1", "Chocolate");
      d.put("2", "Cocoa");
      d.put("5", "Coffee");

      // print how many times put was called
      System.out.println("Number of times put was called:" + d.size());
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Number of times put was called:3

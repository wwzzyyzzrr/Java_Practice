java.util.Vector.clear()方法实例

clear() 方法用于从这个向量删除所有的元素。该向量会是空在这个方法调用之后，除非它抛出一个异常。

声明

以下是java.util.Vector.clear()方法的声明

public void clear()

参数

  • 此方法不带任何参数作为输入。

返回值

返回类型为void，因此不会返回任何值。

异常

  • UnsupportedOperationException-- 如果不支持集合的明确方法则抛出该异常。

例子

下面的例子显示java.util.Vector.clear()方法的使用。

package com.yiibai;

import java.util.Vector;

public class VectorDemo {
   public static void main(String[] args) {
      // create an empty Vector vec with an initial capacity of 4      
      Vector<Integer> vec = new Vector<Integer>(4);

      // use add() method to add elements in the vector
      vec.add(4);
      vec.add(3);
      vec.add(2);
      vec.add(1);

      // let us print the size of the vector
      System.out.println("Size of the vector after addition :"+vec.size()); 
      System.out.println("Added numbers are :- "); 
      for (Integer number : vec) {         
         System.out.println("Number = " + number);
      }
      
      // let us clear the vector
      vec.clear();
      
       // let us print the size of the vector
      System.out.println("Size of the vector after clearing :"+vec.size());       
   } 
}

让我们编译并运行上述程序，这将产生以下结果。矢量调用clear()方法后的大小为0。

Added numbers are :-

Number = 3
Number = 2
Number = 1

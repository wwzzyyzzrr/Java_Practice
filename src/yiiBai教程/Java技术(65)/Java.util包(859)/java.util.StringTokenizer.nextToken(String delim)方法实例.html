java.util.StringTokenizer.nextToken(String delim)方法实例

nextToken(String delim) 方法用于返回在此字符串标记生成的字符串中的下一个标记。首先，设定由这StringTokenizer对象认为是分隔符的字符改变为字符串delim中的字符。那么当前位置之后的字符串中的下一个标记返回。

声明

以下是java.util.StringTokenizer.nextToken()方法的声明。

public String nextToken(String delim)

参数

  • delim--这是新的分隔符。

返回值

在方法调用返回的下一个标记，切换到新的分隔符集后。

异常

  • NoSuchElementException--如果在此标记生成的字符串没有更多的令牌，这将被抛出。

例子

下面的例子显示java.util.StringTokenizer.nextToken()方法的使用

package com.yiibai;

import java.util.*;

public class StringTokenizerDemo {
   public static void main(String[] args) {
      // creating string tokenizer with delimeter
      StringTokenizer st = new StringTokenizer("Come/to/learn");
      
      // checking next token
      System.out.println("Next token is : " + st.nextToken("/"));
   }    
}

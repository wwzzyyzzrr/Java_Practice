java.util.Date.clone()方法实例

 java.util.Date.clone() 返回此Date对象的浅表副本。

声明

以下是java.util.Date.clone()方法声明

public Object clone()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.util.Date.clone()方法的用法。

package com.yiibai;

import java.util.*;

public class DateDemo {

   public static void main(String[] args) {

      // create a date
      Date date = new Date(98, 5, 21);

      // clone it to a second date
      Object date2 = date.clone();

      // print the results
      System.out.println("Original Date:" + date.toString());
      System.out.println("Cloned date :" + date2.toString());
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Original Date:Sun Jun 21 00:00:00 EEST 1998
Cloned date :Sun Jun 21 00:00:00 EEST 1998

java.util.LinkedList.set()方法实例

java.util.LinkedList.set(int index,E element) 方法替换在与指定元素在此列表中指定位置的元素。

声明

以下是java.util.LinkedList.set()方法的声明

public E set(int index,E element)

参数

  • index -- 替换元素的索引

  • element -- 元素将被存储在指定的位置

返回值

这种方法在指定位置返回之前元素

异常

  • IndexOutOfBoundsException -- 如果索引超出范围

例子

下面的示例演示java.util.LinkedList.set()方法的用法。

package com.yiibai;

import java.util.*;

public class LinkedListDemo {

   public static void main(String[] args) {

      // create a LinkedList
      LinkedList list = new LinkedList();

      // add some elements
      list.add("Hello");
      list.add(2);
      list.add("Chocolate");
      list.add("10");

      // print the list
      System.out.println("LinkedList:" + list);

      // set "Coffee" at index 1
      System.out.println("Object to be replaced:" + list.set(1, "Coffee"));

      // print the list
      System.out.println("LinkedList:" + list);

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

LinkedList:[Hello, 2, Chocolate, 10]
Object to be replaced:2
LinkedList:[Hello, Coffee, Chocolate, 10]

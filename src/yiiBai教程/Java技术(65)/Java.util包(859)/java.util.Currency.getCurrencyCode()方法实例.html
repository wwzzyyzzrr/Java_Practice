java.util.Currency.getCurrencyCode()方法实例

java.util.Currency.getCurrencyCode() 方法获取此货币的ISO4217货币代码。

声明

以下是java.util.Currency.getCurrencyCode()方法的声明

public String getCurrencyCode()

参数

  • NA

返回值

这种方法获取此货币的ISO4217货币代码。

异常

  • NA

例子

下面的示例演示java.util.Currency.getCurrencyCode()方法的用法。

package com.yiibai;

import java.util.*;

public class CurrencyDemo {

   public static void main(String args[]) {

      // create a currency with eur code
      Currency curr = Currency.getInstance("EUR");

      // get currency code and print it
      String curCode = curr.getCurrencyCode();
      System.out.println("Currency Code is = " + curCode);
   }
}

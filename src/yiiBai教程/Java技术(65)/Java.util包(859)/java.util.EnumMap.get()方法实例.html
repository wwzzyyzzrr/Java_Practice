java.util.EnumMap.get()方法实例

java.util.EnumMap.get(Object key) 方法返回指定键所映射的值，或者null，如果此映射不包含该键。

声明

以下是java.util.EnumMap.get()方法的声明

public V get(Object key)

参数

  • key -- 要返回其关联值的键

返回值

这个方法返回指定键所映射的值，或者如果此映射不包含该键那么为null。

异常

  • NA

例子

下面的示例演示java.util.EnumMap.get()方法的用法。

package com.yiibai;

import java.util.*;

public class EnumMapDemo {

   // create an enum
   public enum Numbers {

      ONE, TWO, THREE, FOUR, FIVE
   };

   public static void main(String[] args) {
      EnumMap<Numbers, String> map =
              new EnumMap<Numbers, String>(Numbers.class);

      // associate values in map1
      map.put(Numbers.ONE, "1");
      map.put(Numbers.TWO, "2");
      map.put(Numbers.THREE, "3");
      map.put(Numbers.FOUR, "4");


      // print the map
      System.out.println(map);

      // get the value for Numbers.ONE
      String value = map.get(Numbers.ONE);

      // print the result
      System.out.println("Numbers.ONE value:" + value);
      System.out.println("Numbers.FIVE value:" + map.get(Numbers.FIVE));


   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

{ONE=1, TWO=2, THREE=3, FOUR=4}
Numbers.ONE value:1
Numbers.FIVE value:null

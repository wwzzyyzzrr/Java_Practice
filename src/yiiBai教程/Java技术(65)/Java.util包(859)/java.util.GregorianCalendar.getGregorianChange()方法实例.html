java.util.GregorianCalendar.getGregorianChange()方法实例

java.util.GregorianCalendar.getGregorianChange() 方法获取公历更改日期。这是一点，当从儒略的开关日期，以公历日期发生。默认值是1582年10月15日(公历)。在此之前，日期将在罗马儒略历。

声明

以下是java.util.GregorianCalendar.getGregorianChange()方法的声明

public final Date getGregorianChange()

参数

  • NA

返回值

此方法返回公历日期切换此GregorianCalendar对象。

异常

  • NA

例子

下面的示例演示java.util.GregorianCalendar.getGregorianChange()方法的用法。

package com.yiibai;

import java.util.*;

public class GregorianCalendarDemo {

   public static void main(String[] args) {

      // create a new calendar
      GregorianCalendar cal =
              (GregorianCalendar) GregorianCalendar.getInstance();

      // print the current date and time
      System.out.println("" + cal.getTime());

      // get Gregorian change and print it
      System.out.println("Date change:" + cal.getGregorianChange());

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Fri May 18 02:59:54 EEST 2012
Date change:Fri Oct 15 02:00:00 EET 1582

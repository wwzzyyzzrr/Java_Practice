java.util.TimeZone.setID()方法实例

setID(String ID) 方法被用于设置时区ID。这不会改变的时区对象中的任何其他数据。

声明

以下是java.util.TimeZone.setID()方法的声明。

public void setID(String ID)

参数

  • ID--这是新的时区ID。

返回值

NA

异常

  • NA

例子

下面的例子显示java.util.TimeZone.setID()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
   public static void main( String args[] ){
       
      // create time zone object 
      TimeZone tzone = TimeZone.getDefault();
      
      // set time zone ID
      tzone.setID("GMT+08:00");

      // checking time zone ID
      System.out.println("Time zone ID:" +tzone.getID());
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Time zone ID:GMT+08:00

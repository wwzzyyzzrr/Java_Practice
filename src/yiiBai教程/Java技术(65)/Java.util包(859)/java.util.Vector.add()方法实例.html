java.util.Vector.add()方法实例

java.util.Vector.add(E e) 方法用在Vector的末尾插入指定的元素e。这种方法是可行从Java 2平台v1.2开始。

声明

以下是java.util.Vector.add()方法的声明

public boolean add(E e)

参数

  • e --  要被添加到这个向量的元素。

返回值

如果指定的元素被成功添加到向量该方法返回true，否则返回false。

异常

  • NA.

例子

下面的例子显示java.util.Vector.add(E)方法的使用。

package com.yiibai;

import java.util.Vector;

public class VectorDemo {
   public static void main(String[] args) {
      // create an empty Vector vec with an initial capacity of 4      
      Vector<Integer> vec = new Vector<Integer>(4);

      // use add() method to add elements in the vector
      vec.add(4);
      vec.add(3);
      vec.add(2);
      vec.add(1);

      // let us print all the elements available in vector
      System.out.println("Added numbers are :- "); 
      for (Integer number : vec) {         
         System.out.println("Number = " + number);
      }
   }    
}

让我们来编译和运行上面的程序，这将产生以下结果：

Added numbers are :- 
Number = 4
Number = 3
Number = 2
Number = 1

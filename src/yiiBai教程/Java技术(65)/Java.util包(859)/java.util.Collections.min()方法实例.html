java.util.Collections.min()方法实例

min(Collection<? extends T>) 方法用于返回给定collection的最小元素，根据其元素的自然顺序。

声明

以下是java.util.Collections.min()方法的声明。

public static <T extends Object & Comparable<? super T>> T min(Collection<? extends T> coll)

参数

  • coll-- 该集合的最小元素(待定)。

返回值

方法调用返回给定collection的最小元素，根据其元素的自然顺序。

异常

  • ClassCastException--这被抛出，如果集合中包含不可相互比较的(例如，字符串和整数)个元素。

  • NoSuchElementException--如果集合是null，这将被抛出。

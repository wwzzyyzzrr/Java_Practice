java.util.Calendar.getMinimum()方法实例

java.util.Calendar.getMinimum() 方法返回给定日历字段的最小值。

声明

以下是java.util.Calendar.getMinimum()方法的声明

public abstract int getMinimum(int field)

参数

  • field -- 日历字段。

返回值

该方法返回输入字段的最小值。

异常

  • NA

例子

下面的示例演示java.util.calendar.getMinimum()方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {

      // create a calendar
      Calendar cal = Calendar.getInstance();

      // print the minimum value for years
      int min=cal.getMinimum(Calendar.YEAR);
      System.out.println("Minimum Years: "+ min);
   }
}

Minimum Years: 1

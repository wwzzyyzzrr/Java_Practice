java.util.Locale.getISOCountries()方法实例

java.util.Locale.getISOCountries() 方法返回ISO 3166中定义的所有2个字母的国家代码的列表，这可以用来创建语言环境。

声明

以下是java.util.Locale.getISOCountries()方法的声明

public static String[] getISOCountries()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的例子显示java.util.Locale.getISOCountries()方法的用法。

package com.yiibai;

import java.util.*;

public class LocaleDemo {

   public static void main(String[] args) {

      // create a new locale
      Locale locale = new Locale("en", "US", "WIN");

      // print locale
      System.out.println("Locale:" + locale);

      // get ISO countries
      String[] countries = Locale.getISOCountries();

      // print countries
      System.out.println("Countries are:");
      for (int i = 0; i < countries.length; i++) {
         System.out.println(i + ":" + countries[i]);
      }
   }
}

java.util.TreeSet.addAll()方法实例

addAll(Collection c) 方法用于将所有指定的集合中的元素添加到该集合。

声明

以下是java.util.TreeSet.addAll()方法的声明。

public boolean addAll(Collection c)

参数

  • c--这些将被添加的元素。

返回值

如果此set改变调用的结果，方法调用返回true。

异常

  • ClassCastException--如果所提供的元素不能与set中的当前元素进行比较，抛出此异常。

  • NullPointerException--如果指定集合为null，抛出此异常。

例子

下面的例子显示java.util.TreeSet.addAll()方法的使用。

package com.yiibai;

import java.util.Iterator;
import java.util.TreeSet;

public class TreeSetDemo {
   public static void main(String[] args) {
     // creating a TreeSet 
     TreeSet <Integer>treeone = new TreeSet<Integer>();
     TreeSet <Integer>treetwo = new TreeSet<Integer>();
     
     // adding in the treeone
     treeone.add(12);
     treeone.add(13);
     treeone.add(14);
     
     // adding in the treetwo
     treetwo.add(15);
     treetwo.add(16);
     treetwo.add(17);  
     
     // adding treetwo to treeone
     treeone.addAll(treetwo);
     
     // create an iterator
     Iterator iterator;
     iterator = treeone.iterator();
     
     // displaying the Tree set data
     System.out.print("Tree set data: ");     
     while (iterator.hasNext()){
        System.out.print(iterator.next() + " ");
     }
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Tree set data: 12 13 14 15 16 17

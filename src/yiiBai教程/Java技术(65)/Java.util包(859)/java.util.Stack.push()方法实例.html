java.util.Stack.push()方法实例

push(Object item) 方法是用来推(压入)项到该堆栈的顶部。

声明

以下是java.util.Stack.push()方法的声明。

public Object push(Object item)

参数

  • item--此是要被压入该堆栈中的项。

返回值

方法调用返回参数项。

异常

  • NA

例子

下面的例子显示java.util.Stack.push()方法的使用

package com.yiibai;

import java.util.*;

public class StackDemo {
   public static void main(String args[]) {
      // creating stack
      Stack st = new Stack();
      
      // populating stack
      st.push("Java");
      st.push("Source");
      st.push("code");

      // checking elements
      System.out.println("Elements in the stack: "+st);
   }     
}

现在编译和运行上面的代码示例，将产生以下结果。

Elements in the stack: [Java, Source, code]

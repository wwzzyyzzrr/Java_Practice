java.util.Stack类

 java.util.Stack 类代表对象的后进先出(LIFO)堆栈。

  • 当创建一个堆栈，它不包含任何项。

  • 在这个类中，插入的最后一个元素会是第一个被访问。

类声明

以下是java.util.Stack类的声明：

public class Stack<E>
    extends Vector<E>

类构造函数

S.N. 构造函数 & 描述
1 Stack() 
此构造函数创建一个空栈。

类方法

S.N. 方法 & 描述
1 boolean empty() 
此方法测试堆栈是否为空。
2 E peek() 
此方法会查看该对象在该堆栈的顶部而不从栈中删除。
3 E pop() 
此方法会删除在该堆栈的顶部的对象，并返回该对象作为该函数的值。
4 E push(E item) 
此方法推的项目到这个堆栈的顶部。
5 int search(Object o) 
此方法返回从1开始的位置，一个对象在栈中。

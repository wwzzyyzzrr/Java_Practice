java.util.TimeZone.useDaylightTime()方法实例

useDaylightTime() 方法被查询此时区是否使用夏令时。

声明

以下是java.util.TimeZone.useDaylightTime()方法的声明。

public abstract boolean useDaylightTime()

参数

  • NA

返回值

如果此时区使用夏令时，返回false，否则该方法调用返回true。

异常

  • NA

例子

下面的例子显示java.util.TimeZone.useDaylightTime()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
   public static void main( String args[] ){
       
      // create time zone object 
      TimeZone tzone = TimeZone.getDefault();      

      // checking daylight time
      System.out.println("Using day light time :" +tzone.useDaylightTime());
   }    
}

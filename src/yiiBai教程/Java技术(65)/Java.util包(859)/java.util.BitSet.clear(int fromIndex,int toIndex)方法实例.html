java.util.BitSet.clear(int fromIndex,int toIndex)方法实例

java.util.BitSet.clear(int fromIndex,int toIndex) 方法从指定的fromIndex(包括)到指定的toIndex(不包括)设置位为false。

声明

以下是java.util.BitSet.clear()方法的声明

public void clear(int fromIndex,int toIndex)

参数

  • fromIndex -- 第一比特的索引将被清除

  • toIndex -- 最后一个比特的索引将被清除

返回值

此方法不返回任何值。

异常

  • IndexOutOfBoundsException -- 如果fromIndex为负，则为toIndex是负，或fromIndex比toIndex大。

例子

下面的示例演示java.util.BitSet.clear()方法的用法。

package com.yiibai;

import java.util.*;

public class BitSetDemo {

   public static void main(String[] args) {

      // create 2 bitsets
      BitSet bitset1 = new BitSet(8);
      BitSet bitset2 = new BitSet(8);

      // assign values to bitset1
      bitset1.set(0);
      bitset1.set(1);
      bitset1.set(2);
      bitset1.set(3);
      bitset1.set(4);
      bitset1.set(5);

      // assign values to bitset2
      bitset2.set(2);
      bitset2.set(4);
      bitset2.set(6);
      bitset2.set(8);
      bitset2.set(10);
	  
	  // print the sets
      System.out.println("Bitset1:" + bitset1);
      System.out.println("Bitset2:" + bitset2);

      // clear from index 2 to index 4 in bitset1
      bitset1.clear(1, 4);

      // clear index 2 to index 8 in bitset2
      bitset2.clear(2, 8);

      // print new bitsets
      System.out.println("" + bitset1);
      System.out.println("" + bitset2);

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Bitset2:{2, 4, 6, 8, 10}
{0, 4, 5}
{8, 10}

Java.util.Arrays.asList()方法实例

java.util.Arrays.asList(T... a) 返回由指定数组支持的固定大小的列表(更改返回列表“写”到数组)。此方法充当基于数组的集合和基于API之间的桥梁。

声明

以下是java.util.Arrays.asList()方法的声明

public static <T> List<T> asList(T... a)

参数

  • a -- 这是该列表将要备份的数组。

返回值

此方法返回指定数组的列表视图。

异常

  • NA

例子

下面的示例演示java.util.Arrays.asList()方法的用法。

package com.yiibai;

import java.util.Arrays;
import java.util.List;

public class ArrayDemo1 {
  public static void main (String args[]) {

   // create an array of strings
   String a[] = new String[]{"abc","klm","xyz","pqr"};
   
   List list1 = Arrays.asList(a);

   // printing the list
   System.out.println("The list is:" + list1);
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

The list is:[abc, klm, xyz, pqr]

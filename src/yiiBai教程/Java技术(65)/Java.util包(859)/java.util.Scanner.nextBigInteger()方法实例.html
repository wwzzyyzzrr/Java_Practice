java.util.Scanner.nextBigInteger()方法实例

java.util.Scanner.nextBigInteger() 方法扫描输入的下一个标记为一个BigInteger。形式nextBigInteger()这种方法的调用，完全相同的行为方式和调用nextBigInteger(radix)，其中radix是此scanner的默认基数。

声明

以下是java.util.Scanner.nextBigInteger()方法的声明

public BigInteger nextBigInteger()

参数

  • NA

返回值

此方法返回从输入信息扫描的BigInteger

异常

  • InputMismatchException -- 如果下一个标记不匹配的Integer正则表达式，或者是超出范围

  • NoSuchElementException -- 如果输入被耗尽

  • IllegalStateException -- 如果此scanner 已关闭

例子

下面的示例演示java.util.Scanner.nextBigInteger()方法的用法。

package com.yiibai;

import java.util.*;

public class ScannerDemo {

   public static void main(String[] args) {

      String s = "Hello World! 3 + 3.0 = 6  true";

      // create a new scanner with the specified String Object
      Scanner scanner = new Scanner(s);

      // find the next BigInteger token and print it
      // loop for the whole scanner
      while (scanner.hasNext()) {

         // if the next is BigInteger, print "Found" and the Integer
         if (scanner.hasNextBigInteger()) {
            System.out.println("Found :" + scanner.nextBigInteger());
         }

         // if a BigInteger is not found, print "Not Found" and the token
         System.out.println("Not Found :" + scanner.next());
      }

      // close the scanner
      scanner.close();
   }
}

Found :6
Not Found :true

getDisplayName(Locale locale)方法实例

getDisplayName(Locale locale) 方法用于获取适合于展示给指定区域的用户的时区名称。

声明

以下是java.util.TimeZone.getDisplayName()方法的声明。

public final String getDisplayName(Locale locale)

参数

  • locale--这是提供的显示名称的区域设置。

返回值

方法调用返回给定区域中时区的可读名称。它也可以返回默认的语言环境，如果给定语言环境无法识别。

异常

  • NA

例子

下面的例子显示java.util.TimeZone.getDisplayName()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
   public static void main( String args[] ){
       
      // create default time zone object
      TimeZone timezone = TimeZone.getTimeZone("Europe/Paris");
      
      // create locale
      Locale locale = new Locale("CHINESE", "CHINA");
      
      // get display name for specific locale
      String disname=timezone.getDisplayName(locale);       
         
      // checking display name         
      System.out.println("Display name is :" + disname);
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Display name is :Central European Time

java.util.IdentityHashMap.containsValue()方法实例

containsValue(Object value) 方法是用来测试指定的对象引用是否为此标识哈希映射中的值。

声明

以下是java.util.IdentityHashMap.containsValue()方法的声明。

public boolean containsValue(Object value)

参数

  • value--这是要检查的可能值。

返回值

该方法调用返回true，如果此映射将一个或多个键映射到指定的对象引用。

异常

  • NA

例子

下面的例子显示java.util.IdentityHashMap.containsValue()方法的使用

package com.yiibai;

import java.util.*;

public class IdentityHashMapDemo {
   public static void main(String args[]) {
      // create identity hash map
      IdentityHashMap ihmap = new IdentityHashMap();
      
      // populate the map
      ihmap.put(1, "java");
      ihmap.put(2, "util");
      ihmap.put(3, "package");
      
      // check if value exists at key 3 
      boolean isavailable=ihmap.containsValue(3);

      System.out.println("Is value exists at key '3': " + isavailable);
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Is value exists at key '3': false

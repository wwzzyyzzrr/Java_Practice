java.util.Vector.toArray(T[] a)方法实例

toArray(T[] a) 方法用来返回一个包含在此向量中以正确的顺序的所有元素的数组。

声明

以下是java.util.Vector.toArray()方法的声明

public <T> T[] toArray(T[] a)

参数

  • a--这是向量的元素被存储的数组，如果它是足够大;否则，同样的运行时类型的新数组分配用于这一目的。

返回值

该方法调用返回一个包含Vector的元素的数组。

异常

  • ArrayStoreException--该异常被抛出，如果a的运行时类型不是每个元素的向量中运行时类型的超类。

  • NullPointerException--如果如果给定的数组为null，抛出此异常。

例子

下面的例子显示java.util.Vector.toArray()方法的使用。

package com.yiibai;

import java.util.*;
import java.util.Arrays;

public class VectorDemo {
   public static void main(String[] args) {
      // create an empty Vector vec with an initial capacity of 4      
      Vector<Integer> vec = new Vector<Integer>(4);
      // create an array
      Integer[] anArray = new Integer[4];

      // use add() method to add elements in the vector
      vec.add(4);
      vec.add(3);
      vec.add(2);
      vec.add(1);
      
      // fill the array from the vector
      vec.toArray(anArray); 

      // check the content of the array
      System.out.println("Elements are: ");  
      for (int i = 0; i < anArray.length; i++) {
         System.out.println(anArray[i]);
      }
   }    
}

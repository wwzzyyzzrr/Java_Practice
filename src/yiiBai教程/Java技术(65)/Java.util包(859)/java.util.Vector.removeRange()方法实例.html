java.util.Vector.removeRange()方法实例

removeRange(int fromIndex,int toIndex) 方法用于删除列表索引fromIndex(包括)和为toIndex(不包括)之间的所有元素。

声明

以下是java.util.Vector.removeRange()方法的声明

protected void removeRange(int fromIndex,int toIndex)

参数

  • fromIndex --这是要移除的第一个元素的索引。

  • toIndex --这是要去除最后一个元素之后的索引

返回值

NA

异常

  • NA

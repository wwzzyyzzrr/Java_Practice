java.util.EnumSet.complementOf()方法实例

java.util.EnumSet.complementOf(EnumSet<E> s) 方法创建一个枚举设置相同的元素类型与指定枚举set。它包含此类型的所有未包含指定集合中的元素。

声明

以下是java.util.EnumSet.complementOf()方法的声明

public static <E extends Enum<E>> EnumSet<E> complementOf(EnumSet<E> s)

参数

  • s -- 从其补充初始化枚举设置此枚举set。

返回值

此方法不返回任何值。

异常

  • NullPointerException-- 如果 s 为 null

例子

下面的示例演示java.util.EnumSet.complementOf()方法的用法。

package com.yiibai;

import java.util.*;

public class EnumSetDemo {

   // create an enum
   public enum Numbers {

      ONE, TWO, THREE, FOUR, FIVE
   };

   public static void main(String[] args) {

      // create an EnumSet that has only Numbers.FOUR as element 
      EnumSet<Numbers> set1 = EnumSet.of(Numbers.FOUR);

      // print the set
      System.out.println("Set1:" + set1);

      // create a set2 which has all elements that set1 doesn't have
      EnumSet<Numbers> set2 = EnumSet.complementOf(set1);

      // print the updated set
      System.out.println("Set2:" + set2);

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Set1:[FOUR]
Set2:[ONE, TWO, THREE, FIVE]

java.util.LinkedHashMap.containsValue()方法实例

java.util.LinkedHashMap.containsValue(Object value) 如果此映射一个或多个键映射到指定值，方法返回true。

声明

以下是java.util.LinkedHashMap.containsValue()方法的声明

public boolean containsValue(Object value)

参数

  • value -- 要测试是否存在于此映射的值

返回值

如果此映射的一个或多个键映射到指定值，此方法返回true

异常

  • NA

例子

下面的示例演示java.util.LinkedHashMap.containsValue()方法的用法。

package com.yiibai;

import java.util.*;

public class BitSetDemo {

   public static void main(String[] args) {

      // create a new linked hash map
      LinkedHashMap map = new LinkedHashMap(5);

      // add some values in the map
      map.put("One", 1);
      map.put("Two", 2);
      map.put("Three", 3);

      // print the map
      System.out.println("Map:" + map);

      // check if map contains 3
      System.out.println("" + map.containsValue(3));

      // check if map contains 5
      System.out.println("" + map.containsValue(5));
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

java.util.Hashtable.rehash()方法实例

rehash() 方法是用来增加容量和内部对其进行重组这个哈希表。这样做是为了容纳并更有效地访问其条目。当在哈希表中的键的数量超过这个哈希表的容量此方法被自动调用。

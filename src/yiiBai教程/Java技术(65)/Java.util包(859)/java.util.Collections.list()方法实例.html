java.util.Collections.list()方法实例

list(Enumeration<T>) 方法用于获取包含由指定枚举它们是由枚举返回的顺序返回元素的数组列表。

声明

以下是java.util.Collections.list()方法的声明。

public static <T> ArrayList<T> list(Enumeration<T> e)

参数

  • e--这是枚举提供了返回的数组列表中的元素。

返回值

方法调用返回指定目标列表的最后出现的起始位置指定源列表中，或者-1，如果没有这样发生。

异常

  • NA

例子

下面的例子显示java.util.Collections.list()方法的使用

package com.yiibai;

import java.util.*;

public class CollectionsDemo {
   public static void main(String args[]) {
      // create vector and array list
      List arrlist = new ArrayList();
      Vector v = new Vector();
      
      // populate the vector
      v.add("A");       
      v.add("B");
      v.add("C");
      v.add("D");
      v.add("E");
      
      // create enumeration
      Enumeration e = v.elements();
      
      // get the list
      arrlist = Collections.list(e);
      
      System.out.println("Value of returned list: "+arrlist);
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Value of returned list: [A, B, C, D, E]

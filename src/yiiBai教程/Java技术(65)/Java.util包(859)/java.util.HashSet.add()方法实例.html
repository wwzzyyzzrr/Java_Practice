java.util.HashSet.add()方法实例

add(E e) 方法用于为指定的元素添加到这个组，如果它是不存在的。如果此set已经包含该元素，则调用设定不变，并返回false。

声明

以下是java.util.HashSet.add()方法的声明。

public boolean add(E e)

参数

  • e--这是被添加到当前组的元素。

返回值

该方法调用返回true，如果此set不已经包含了指定的元素。

异常

  • NA

例子

下面的例子显示java.util.HashSet.add()方法的使用

package com.yiibai;

import java.util.*;

public class HashSetDemo {
   public static void main(String args[]) {
      // create hash set
      HashSet <String> newset = new HashSet <String>();      
      
      // populate hash set
      newset.add("Learning"); 
      newset.add("Easy");
      newset.add("Simply");  
            
      // checking elements in hash set
      System.out.println("Hash set values: "+ newset);
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Hash set values: [Learning, Simply, Easy]

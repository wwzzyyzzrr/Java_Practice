java.util.Vector.remove(Object o)方法实例

remove(Object o) 方法用于删除在这个矢量指定第一次出现的元素。如果向量不包含该元素，它保持不变。

声明

以下是java.util.Vector.remove()方法的声明

public boolean remove(Object o)

参数

  • o--这是准备从向量中移除(如果存在)的元素。

返回值

如果向量包含指定元素，方法调用返回true。

异常

  • NA

java.util.Calendar.getMinimalDaysInFirstWeek()方法实例

java.util.Calendar.getMinimalDaysInFirstWeek() 方法获取所需要的一年中第一个星期所需的最少天数。

声明

以下是java.util.Calendar.getMinimalDaysInFirstWeek()方法的声明

public int getMinimalDaysInFirstWeek()

参数

  • NA

返回值

该方法返回在今年的第一个星期所需的最少天数。

异常

  • NA

例子

下面的示例演示java.util.calendar.getMinimalDaysInFirstWeek()方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {

      // create a calendar
      Calendar cal = Calendar.getInstance();

      // get what the minimal days required in the first week of the year 
      int i = cal.getMinimalDaysInFirstWeek();

      // print the result
      System.out.println("Minimal days required :" + i);
   }
}

Minimal days required :4

java.util.Properties.load(Reader reader)方法实例

java.util.Properties.load(Reader reader) 方法从一个简单的面向行的格式输入字符流中读取属性列表(键和元素对)。

声明

以下是java.util.Properties.load()方法的声明

public void load(Reader reader)

参数

  • reader -- 输入字符流。

返回值

此方法不返回任何值。

异常

  • IOException -- 如果从输入流中读取数据时发生错误。

  • IllegalArgumentException -- 如果输入流中包含错误的Unicode转义序列。

例子

下面的示例演示java.util.Properties.list()方法的用法。

package com.yiibai;

import java.io.IOException;
import java.io.StringReader;
import java.util.*;

public class PropertiesDemo {

   public static void main(String[] args) {
      Properties prop = new Properties();
      String s = "Height=200
Width=15";

      // create a new reader
      StringReader reader = new StringReader(s);

      try {
         // load from input stream
         prop.load(reader);

         // print the properties list from System.out
         prop.list(System.out);


      } catch (IOException ex) {
         ex.printStackTrace();
      }
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

-- listing properties --
Width=15
Height=200

java.util.Control.getFallbackLocale()方法实例

java.util.ResourceBundle.Control.getFallbackLocale(String baseName, Locale locale) 方法返回由ResourceBundle.getBundle工厂方法被用来作为一个备用的区域进行进一步资源包搜索一个区域设置。这种方法是从工厂方法每次调用时没有造成资源包已经发现baseName和地区，其中locale是ResourceBundle.getBundle的参数，或者此方法返回之前的回退区域设置。

声明

以下是java.util.Control.getFallbackLocale()方法的声明

public Locale getFallbackLocale(String baseName, Locale locale)

参数

  • baseName -- 该资源包的基本名称，其中ResourceBundle.getBundle无法找到任何资源包(除基本包)，一个完全合格的类名

  • locale -- ResourceBundle.getBundle无法找到任何资源包(除基本包)，使用此Locale

返回值

该方法返回一个区域设置为回退搜索，如果没有进一步的回退搜索，返回null。

异常

  • NullPointerException -- 如果 baseName 或 locale 为 null

例子

下面的示例演示java.util.ResourceBundle.Control.getFallbackLocale()方法的用法。

package com.yiibai;

import java.util.Locale;
import java.util.ResourceBundle;
import java.util.ResourceBundle.Control;

public class ResourceBundleControlDemo {

   public static void main(String[] args) {

      // create a new ResourceBundle.Control with default format
      ResourceBundle.Control rbc =
              ResourceBundle.Control.getControl(Control.FORMAT_DEFAULT);

      // print fallback locale
      System.out.println("" + rbc.getFallbackLocale("hello", Locale.US));
   }
}

假设在你的CLASSPATH中，资源文件hello_en_US.properties可用，包含以下内容。该文件将被用作输入到示例程序：

hello=Hello World!

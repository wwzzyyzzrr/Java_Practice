java.util.Collections.reverse()方法实例

reverse(List<?>) 方法被用来反转指定列表中的元素的顺序。

声明

以下是java.util.Collections.reverse()方法的声明。

public static void reverse(List<?> list)

参数

  • list--这是一个列表，其元素将被扭转。

返回值

NA

异常

  • UnsupportedOperationException--这就是，如果指定列表或其列表迭代器不支持set操作。

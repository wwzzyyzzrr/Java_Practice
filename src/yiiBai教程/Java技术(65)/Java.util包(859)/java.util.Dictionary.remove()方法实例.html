java.util.Dictionary.remove()方法实例

java.util.Dictionary.remove(Object key) 方法删除的值和它的从这个字典中对应的按键。

声明

以下是java.util.Dictionary.remove()方法的声明

public abstract V remove(Object key)

参数

  • key -- 要去除的键

返回值

此方法返回到该键存在映射，则返回null如果该键没有映射值。

异常

  • NullPointerException - 如果key 为 null.

例子

下面的示例演示java.util.Dictionary.remove()方法的用法。

package com.yiibai;

import java.util.*;

public class DictionaryDemo {

   public static void main(String[] args) {

      // create new hashtable
      Dictionary d = new Hashtable();
	  
      // add some elements
      d.put("1", "Chocolate");
      d.put("2", "Cocoa");
      d.put("5", "Coffee");

      // remove one element
      System.out.println(d.get("5"));
      System.out.println(d.remove("5") + " has been removed");
      System.out.println(d.get("5"));
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Coffee
Coffee has been removed
null

java.util.Vector.capacity()方法实例

capacity() 方法用于返回一个向量的当前容量。容量是该阵列的长度保持在此向量。

声明

以下是java.util.Vector.capacity()方法的声明

public int capacity()

参数

  • 此方法不带任何参数作为输入。

返回值

该方法返回向量作为一个整数值的当前容量。这里的容量是指其内部数据数组的长度，保持在该矢量的字段elementData。

异常

  • NA.

例子

下面的例子显示java.util.Vector.capacity()方法的使用。

package com.yiibai;

import java.util.Vector;

public class VectorDemo {
   public static void main(String[] args) {
      // create an empty Vector vec   
      Vector<Integer> vec = new Vector<Integer>();

      // use add() method to add elements in the vector
      vec.add(14);
      vec.add(13);
      vec.add(12);
      vec.add(11);
      vec.add(4);
      vec.add(3);
      vec.add(2);
      vec.add(1);

      // let us print the capacity of the vector
      System.out.println("Capacity of the vector is :"+vec.capacity());                 
   }     
}

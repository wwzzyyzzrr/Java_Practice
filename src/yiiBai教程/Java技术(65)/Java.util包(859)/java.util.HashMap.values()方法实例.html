java.util.HashMap.values()方法实例

values() 方法用于返回此映射中包含的值的Collection视图。

声明

以下是java.util.HashMap.values()方法的声明。

public Collection<V> values()

参数

  • NA

返回值

在方法调用返回此映射中包含的值的collection视图。

异常

  • NA

例子

下面的例子显示java.util.HashMap.values()方法的使用

package com.yiibai;

import java.util.*;

public class HashMapDemo {
   public static void main(String args[]) {
      // create hash map
      HashMap newmap = new HashMap();      
      
      // populate hash map
      newmap.put(1, "tutorials");
      newmap.put(2, "point");
      newmap.put(3, "is best");
      
      // checking collection view of the map
      System.out.println("Collection view is: "+ newmap.values());
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Collection view is: [tutorials, point, is best]

java.util.Calendar.getFirstDayOfWeek()方法实例

java.util.Calendar.getFirstDayOfWeek() 方法返回一周的第一天。

声明

以下是java.util.Calendar.getFirstDayOfWeek()方法的声明

public int getFirstDayOfWeek()

参数

  • NA

返回值

该方法返回一周的第一天。

异常

  • NA

例子

下面的示例演示java.util.calendar.getFirstDayOfWeek()方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {

      // create a new calendar
      Calendar cal = Calendar.getInstance();

      // print the first day of the week
      System.out.println("First day is :" + cal.getFirstDayOfWeek());
      int day = cal.getFirstDayOfWeek();
      switch (day) {
         case (1):
            System.out.println("Sunday");
            break;
         case (2):
            System.out.println("Monday");
            break;
         case 3:
            System.out.println("Tuesday");
            break;
         case 4:
            System.out.println("Wednesday");
            break;
         case 5:
            System.out.println("Thrusday");
            break;
         case 6:
            System.out.println("Friday");
            break;
         case 7:
            System.out.println("Saturday");
            break;
      }
   }
}

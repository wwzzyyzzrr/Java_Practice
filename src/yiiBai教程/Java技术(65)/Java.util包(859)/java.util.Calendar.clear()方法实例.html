java.util.Calendar.clear()方法实例

java.util.Calendar.clear() 方法将所有的日历字段值和Calendar的时间值为undefined。Calendar实现类可能会使用默认字段值的日期和时间计算。

声明

以下是java.util.Calendar.clear()方法声明

public final void clear()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.util.Calendar.clear()方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {
   
      // create calendar object
      Calendar cal = Calendar.getInstance();

      // displays the current date and time
      System.out.println("Current Calendar Date: " + cal.getTime());

      // use clear method to set all field values and time value as undefined.
      cal.clear();

      // print the result
      System.out.println("The calendar shows : " + cal.getTime());
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Current Calendar Date: Sat May 05 15:31:36 EEST 2012
The calendar shows : Thu Jan 01 00:00:00 EET 1970

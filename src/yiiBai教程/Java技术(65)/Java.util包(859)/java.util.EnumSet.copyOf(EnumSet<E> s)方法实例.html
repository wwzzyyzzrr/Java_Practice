java.util.EnumSet.copyOf(EnumSet<E> s)方法实例

java.util.EnumSet.copyOf(EnumSet<E> s) 方法创建一个枚举设置相同的元素类型与指定枚举set。它包含相同的元素(如果有的话)。

声明

以下是java.util.EnumSet.copyOf()方法的声明

public static <E extends Enum<E>> EnumSet<E> copyOf(EnumSet<E> s)

参数

  • s-- 枚举集合，从初始化此枚举set。

返回值

此方法不返回任何值。

异常

  • NullPointerException-- 如果 s 为 null

例子

下面的示例演示 java.util.EnumSet.copyOf()方法的用法。

package com.yiibai;

import java.util.*;

public class EnumSetDemo {

   // create an enum
   public enum Numbers {

      ONE, TWO, THREE, FOUR, FIVE
   };

   public static void main(String[] args) {

      // create a set that contains all Numbers class
      EnumSet<Numbers> set1 = EnumSet.allOf(Numbers.class);

      // print set1
      System.out.println("Set1:" + set1);

      // create a second set which is a copy of set1
      EnumSet<Numbers> set2 = EnumSet.copyOf(set1);

      // print the new set
      System.out.println("Set2:" + set2);

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Set1:[ONE, TWO, THREE, FOUR, FIVE]
Set2:[ONE, TWO, THREE, FOUR, FIVE]

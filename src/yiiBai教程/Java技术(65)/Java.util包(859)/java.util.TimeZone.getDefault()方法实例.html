java.util.TimeZone.getDefault()方法实例

getDefault() 方法是用来获取这台主机的默认时区。默认时区的来源可能与实施有所不同。

声明

以下是java.util.TimeZone.getDefault()方法的声明。

public static TimeZone getDefault()

参数

  • NA

返回值

方法调用返回一个默认的时区。

异常

  • NA

例子

下面的例子显示java.util.TimeZone.getDefault()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
   public static void main( String args[] ){
       
      // create default time zone object
      TimeZone timezonedefault = TimeZone.getDefault();      
   
      // checking default time zone value          
      System.out.println("Default time zone is :
" + timezonedefault);
   }    
}

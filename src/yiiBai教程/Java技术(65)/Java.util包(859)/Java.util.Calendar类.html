Java.util.Calendar类

java.util.calendar 类是一个抽象类，它提供了与在某一特定时刻和一组日历字段如YEAR，MONTH，DAY_OF_MONTH，HOUR等，并为操作日历字段，如获取的日期转换方法下周。以下是关于日历的要点：

  • 这个类还提供了额外的字段和实施具体的日历系统外面包的方法。

  • 日历定义了某些日历字段返回值的范围。

   extends Object
      implements Serializable, Cloneable, Comparable<Calendar>

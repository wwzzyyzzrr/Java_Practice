java.util.TimeZone.inDaylightTime()方法实例

inDaylightTime(Date date) 方法用于查询，是否给定的日期是在夏令时在该时区。

声明

以下是java.util.TimeZone.inDaylightTime()方法的声明。

public abstract boolean inDaylightTime(Date date)

参数

  • date--这是给定的日期

返回值

如果给定的日期是在夏令时，返回false，否则该方法调用返回true。

异常

  • NA

例子

下面的例子显示java.util.TimeZone.inDaylightTime()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
   public static void main( String args[] ){
       
      // create time zone object     
      TimeZone timezoneone = TimeZone.getDefault();
      
      // create date object
      Date date = new Date(2008 , 05, 28);
      
      // checking day light
      System.out.println("In daylight time:" +timezoneone.inDaylightTime(date));
   }    
}

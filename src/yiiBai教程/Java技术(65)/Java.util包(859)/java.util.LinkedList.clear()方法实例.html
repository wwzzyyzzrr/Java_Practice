java.util.LinkedList.clear()方法实例

java.util.LinkedList.clear() 方法移除此列表中的所有元素。

声明

以下是java.util.LinkedList.clear()方法的声明

public void clear()

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NA

例子

下面的示例演示java.util.LinkedList.clear()方法的用法。

package com.yiibai;

import java.util.*;

public class LinkedListDemo {

   public static void main(String[] args) {

      // create a LinkedList
      LinkedList list = new LinkedList();

      // add some elements
      list.add("Hello");
      list.add(2);
      list.add("Chocolate");
      list.add("10");

      // print the list
      System.out.println("LinkedList:" + list);
      
      // clear the list
      list.clear();

      // print the list
      System.out.println("LinkedList:" + list);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

LinkedList:[Hello, 2, Chocolate, 10]
LinkedList:[]

java.util.Collections.singletonMap()方法实例

singletonMap(K, V) 方法用来得到一个不变的映射，只映射指定键为指定的值。

声明

以下是java.util.Collections.singletonMap()方法的声明。

public static <K,V> Map<K,V> singletonMap(K key,V value)

参数

  • key-- 唯一的键被存储在返回的映射。

  • value--里边反值到返回的映射中映射键

返回值

在方法调用返回一个只包含指定键 - 值映射的不可变映射。

异常

  • NA

例子

下面的例子显示java.util.Collections.singletonMap()方法的使用

package com.yiibai;

import java.util.*;

public class CollectionsDemo {
   public static void main(String[] a){
      // create singleton map
      Map map = Collections.singletonMap("key","Value");
      
      System.out.println("Singleton map is: "+map);
   }
}

现在编译和运行上面的代码示例，将产生以下结果。

Singleton map is: {key=Value}

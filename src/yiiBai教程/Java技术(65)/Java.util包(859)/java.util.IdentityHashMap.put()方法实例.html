java.util.IdentityHashMap.put()方法实例

put(K key, V value) 方法用于为指定的值，在此标识哈希映射中的指定键关联。如果映射以前包含此键的映射关系，则旧值被替换。

声明

以下是java.util.IdentityHashMap.put()方法的声明。

public V put(K key, V value)

参数

  • key--这是与指定值将要关联的键。

  • value--这是与指定键关联的值。

返回值

该方法调用返回与key映射关联的前一个值；返回null，如果没有键相关联的。

异常

  • NA

例子

下面的例子显示java.util.IdentityHashMap.put()方法的使用

package com.yiibai;

import java.util.*;

public class IdentityHashMapDemo {
   public static void main(String args[]) {
      // create identity hash map
      IdentityHashMap ihmap = new IdentityHashMap();
           
      // populate the map
      ihmap.put(1, "java");
      ihmap.put(2, "util");
      ihmap.put(3, "package");
      
      System.out.println("Initial map: " + ihmap);
      
      // put value at key 2
      ihmap.put(2, "awt");
      
      System.out.println("Map after modification: " + ihmap);
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Initial map: {1=java, 3=package, 2=util}
Map after modification: {1=java, 3=package, 2=awt}

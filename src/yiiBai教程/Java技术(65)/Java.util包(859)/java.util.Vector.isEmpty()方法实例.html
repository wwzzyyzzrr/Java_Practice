java.util.Vector.isEmpty()方法实例

isEmpty() 方法用于测试此向量是否不包含组件(是一个空向量)。

声明

以下是java.util.Vector.isEmpty()方法的声明

public boolean isEmpty()

参数

  • NA

返回值

返回值是true当且仅当此向量没有组件，也就是说，它的大小是零。否则返回false。

异常

  • NA

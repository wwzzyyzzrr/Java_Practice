java.util.Arrays.deepHashCode(Object[])方法实例

java.util.Arrays.deepHashCode(Object[]) 方法返回基于指定数组的“深层内容”返回哈希码。对于任何两个数组a和b，使得Arrays.deepEquals(a, b)，它也是下面情况Arrays.deepHashCode(a) == Arrays.deepHashCode(b)所示。

声明

以下是java.util.Arrays.deepHashCode()方法的声明

public static int deepHashCode(Object[] a)

参数

  • a -- 这是其深基于内容的哈希码来计算数组。

返回值

此方法返回一个深基于内容的哈希码。

异常

  • NA

例子

下面的示例演示java.util.Arrays.deepHashCode()方法的用法。

package com.yiibai;

import java.util.Arrays;

public class ArrayDemo {

  public static void main(String[] args) {
    
   //initializing an object array
   Object[] ob={ "tuts","point" };
    
   // deephashcode for object ob
   int retval=Arrays.deepHashCode(ob);
    
   // printing value
   System.out.println("The Hash Code of ob is:" + retval);
  }
}

getDisplayName(boolean daylight,int style)方法实例

getDisplayName(boolean daylight,int style) 方法用于获取适合于展示给默认区域的用户的时区名称。

声明

以下是java.util.TimeZone.getDisplayName()方法的声明。

public final String getDisplayName(boolean daylight,int style)

参数

  • daylight--如果该值为true，则返回夏令时名称。

  • style-- 或LONG 或 SHORT.

返回值

方法调用返回该时区的默认语言环境的可读名称。

异常

  • NA

例子

下面的例子显示java.util.TimeZone.getDisplayName()方法的使用

package com.yiibai;

import java.util.*;

public class TimeZoneDemo {
   public static void main( String args[] ){
       
      // create default time zone object
      TimeZone timezonedefault = TimeZone.getDefault();
      
      // get display name
      String disname=timezonedefault.getDisplayName(true, 1); 
   
      // checking display name         
      System.out.println("Display name is :" + disname);
   }    
}

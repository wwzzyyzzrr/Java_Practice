java.util.UUID.hashCode()方法实例

hashCode() 方法用于返回此UUID的一个哈希码。

声明

以下是java.util.UUID.hashCode()方法的声明。

public int hashCode()

参数

  • NA

返回值

该方法调用返回此UUID的哈希码值。

异常

  • NA

例子

下面的例子显示java.util.UUID.hashCode()方法的使用

package com.yiibai;

import java.util.*;

public class UUIDDemo {
   public static void main(String[] args) {
      // creating UUID      
      UUID uid = UUID.fromString("38400000-8cf0-11bd-b23e-10b96e4ef00d");     
        
      // checking hash code value
      System.out.println("Hash code value: "+uid.hashCode());    
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Hash code value: 1757475081

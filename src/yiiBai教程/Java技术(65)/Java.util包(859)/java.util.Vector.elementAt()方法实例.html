java.util.Vector.elementAt()方法实例

elementAt(int index) 方法用于获取组件的向量的指定索引/位置。

声明

以下是java.util.Vector.elementAt()方法的声明

public Object elementAt(int index)

参数

  • index--这是一个索引该向量

返回值

它返回指定索引处的组件。

异常

  • ArrayIndexOutOfBoundsException-- 如果该索引比这个Vector对象的当前大小要小或为负数。

例子

下面的例子显示java.util.Vector.elementAt()方法的使用。

package com.yiibai;

import java.util.Vector;

public class VectorDemo {
   public static void main(String[] args) {
      // create an empty Vector vec with an initial capacity of 4      
      Vector<Integer> vec = new Vector<Integer>(4);

      // use add() method to add elements in the vector
      vec.add(4);
      vec.add(3);
      vec.add(2);
      vec.add(1);

      // let us print the element at 1st position in the vector
      System.out.println("Element at 1st position :- "+vec.elementAt(1));                 
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Element at 1st position :- 3

Java.util.BitSet类

java.util.BitSet 类实现位向量作为一个按需增长。以下是关于BitSet中的要点：

  • BitSet 是不安全的，除非外部同步多线程使用。

  • 集合中的所有位的初始值false。

  • 传递一个null参数到BitSet中的任何方法会导致一个NullPointerException。

类声明

以下是java.util.BitSet类的声明：

      implements Cloneable, Serializable

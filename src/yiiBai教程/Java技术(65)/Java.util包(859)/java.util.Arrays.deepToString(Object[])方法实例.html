java.util.Arrays.deepToString(Object[])方法实例

java.util.Arrays.deepToString(Object[]) 方法返回指定数组的“深层内容”的字符串表示形式。如果数组包含其他数组作为元素的字符串表示形式包含其内容等。这种方法是专为转换多维数组为字符串。该字符串表示形式由数组的元素，括在方括号(“[]”)的列表。相邻元素由字符分隔“，”(逗号后面有一个空格)。

声明

以下是java.util.Arrays.deepToString()方法的声明

public static String deepToString(Object[] a)

参数

  • a -- 这是它的字符串表示形式返回该数组

返回值

此方法返回的字符串表示形式。

异常

  • NA

例子

下面的示例演示java.util.Arrays.deepToString()方法的用法。

package com.yiibai;

import java.util.Arrays;

public class ArrayDemo {

  public static void main(String[] args) {
    
    // initializing an object array
    Object[] ob={"tuts","point"};

    // let us print all the values available in list
    System.out.println("The array is:");
    for (Object value : ob) {
      System.out.println("Value = " + value);
    }
	
    System.out.println("The string representation of array is:");
    System.out.println(Arrays.deepToString(ob));    
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

The array is:
Value = tuts
Value = point
The string representation of array is:
[tuts, point]

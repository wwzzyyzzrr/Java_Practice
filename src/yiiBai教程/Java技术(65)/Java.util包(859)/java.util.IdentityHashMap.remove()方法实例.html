java.util.IdentityHashMap.remove()方法实例

remove(Object key) 方法用于如果存在，则从该映射中移除了映射此键。

声明

以下是java.util.IdentityHashMap.remove()方法的声明。

public V remove(Object key)

参数

  • key--这是其映射关系要从映射中移除的键。

返回值

在方法调用返回与指定键相关联的前一个值，返回null，如果有关键的项。

异常

  • NA

例子

下面的例子显示java.util.IdentityHashMap.remove()方法的使用

package com.yiibai;

import java.util.*;

public class IdentityHashMapDemo {
   public static void main(String args[]) {
      // create identity hash 
      IdentityHashMap ihmap = new IdentityHashMap();
                 
      // populate the ihmap
      ihmap.put(1, "java");
      ihmap.put(2, "util");
      ihmap.put(3, "package");
      
      System.out.println("Value of ihmap before: " + ihmap);
      
      // remove element at key 2
      ihmap.remove(2);
      
      System.out.println("Value of ihmap after remove: " + ihmap);
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Value of ihmap before: {1=java, 3=package, 2=util}
Value of ihmap after remove: {1=java, 3=package}

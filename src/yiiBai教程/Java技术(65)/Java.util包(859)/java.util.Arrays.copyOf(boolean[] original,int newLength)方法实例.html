java.util.Arrays.copyOf(boolean[] original,int newLength)方法实例

java.util.Arrays.copyOf(boolean[] original,int newLength) 方法复制指定的数组，截取或使用false填充(如有必要)，以使副本具有指定的长度。对于这一切都有效同时在原数组和副本索引，这两个数组将包含相同的值。对于在副本中有效，但不是原来的任何索引，副本将包含错误。这些索引存在，当且仅当指定长度大于原数组。

声明

以下是java.util.Arrays.copyOf()方法的声明

public static boolean[] copyOf(boolean[] original,int newLength)

参数

  • original -- 这是要被复制的数组。

  • newLength -- 这是要返回的副本长度。

返回值

此方法返回原始数组，截取或填充用false元素的副本，以获得指定的长度。

异常

  • NegativeArraySizeException -- 如果newLength为负。

  • NullPointerException --如果original是空。

例子

下面的示例演示java.util.Arrays.copyOf()方法的用法。

package com.yiibai;

import java.util.Arrays;

public class ArrayDemo {

  public static void main(String[] args) {
    
    // intializing an array arr1
    boolean[] arr1 = new boolean[] {true, false};

    // printing the array
    System.out.println("Printing 1st array:");
    for (int i = 0; i < arr1.length; i++) {
      System.out.println(arr1[i]);  
    }

    // copying array arr1 to arr2 with newlength as 4
    boolean[] arr2 = Arrays.copyOf(arr1, 4);
    arr2[2] = false;
    arr2[3] = true;

    // printing the array arr2
    System.out.println("Printing new array:");
    for (int i = 0; i < arr2.length; i++) {
      System.out.println(arr2[i]);
    }
  }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Printing 1st array:
true
false
Printing new array:
true
false
false
true

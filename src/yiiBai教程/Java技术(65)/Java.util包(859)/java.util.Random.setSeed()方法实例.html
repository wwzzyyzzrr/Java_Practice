java.util.Random.setSeed()方法实例

setSeed(long seed) 方法用于使用单个long种子设置此随机数发生器的种子。

声明

以下是java.util.Random.setSeed()方法的声明。

public void setSeed(long seed)

参数

  • seed--这是最初的种子。

返回值

NA

异常

  • NA

例子

下面的例子显示java.util.Random.setSeed()方法的使用

package com.yiibai;

import java.util.*;

public class RandomDemo {
   public static void main( String args[] ){
      // create random object
      Random randomno = new Random();
      
      // setting seed
      randomno.setSeed(20);
      
      // value after setting seed
      System.out.println("Object after seed: " + randomno.nextInt());
   }    
}

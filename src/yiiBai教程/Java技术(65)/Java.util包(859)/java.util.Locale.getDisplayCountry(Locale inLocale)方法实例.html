java.util.Locale.getDisplayCountry(Locale inLocale)方法实例

java.util.Locale.getDisplayCountry(Locale inLocale) 方法返回适合向用户显示的名称语言环境的国家。如果可能的话，返回的名称将根据inLocale本地化。

声明

以下是java.util.Locale.getDisplayCountry()方法的声明

public String getDisplayCountry(Locale inLocale)

参数

  • NA

返回值

此方法不返回任何值。

异常

  • NullPointerException -- 如果inLocale 是 null

例子

下面的示例演示java.util.Locale.getDisplayCountry()方法的用法。

package com.yiibai;

import java.util.*;

public class LocaleDemo {

   public static void main(String[] args) {

      // create a new locale
      Locale locale = new Locale("ENGLISH", "US");

      // print locale
      System.out.println("Locale:" + locale);

      // print locale display name - result is based on inLocale
      System.out.println("Display Name:"
              + locale.getDisplayCountry(new Locale("GERMAN", "GERMANY")));

   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Locale:english_US
Display Name:United States

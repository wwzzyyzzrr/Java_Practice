java.util.Calendar.getMaximum()方法实例

java.util.Calendar.getMaximum() 方法返回给定日历字段的最大值。

声明

以下是java.util.Calendar.getMaximum()方法的声明

public abstract int getMaximum(int field)

参数

  • field -- 日历字段。

返回值

该方法返回输入的日历字段的最大值。

异常

  • NA

例子

下面的示例演示java.util.calendar.getMaximum(field) 方法的用法。

package com.yiibai;

import java.util.*;

public class CalendarDemo {

   public static void main(String[] args) {

      // create a calendar
      Calendar cal = Calendar.getInstance();

      // return the maximum value for days
      int max = cal.getMaximum(Calendar.DAY_OF_WEEK);
      System.out.println("Maximum days  :" + max);

      // return the maximum value for months
      max = cal.getMaximum(Calendar.MONTH);
      System.out.println("Maximum months  :" + max);
   }
}

让我们来编译和运行上面的程序，这将产生以下结果：

Maximum days  :7
Maximum months  :11

java.util.HashSet.remove()方法实例

remove(Object o)方法用于从该组中移除指定元素(如果存在)。

声明

以下是java.util.HashSet.remove()方法的声明。

public boolean remove(Object o)

参数

  • o--这是将要从这个集合中移除(如果存在)的对象。

返回值

该方法调用返回true，如果集合包含指定的元素。

异常

  • NA

例子

下面的例子显示java.util.HashSet.remove()方法的使用

package com.yiibai;

import java.util.*;

public class HashSetDemo {
   public static void main(String args[]) {
      // create hash set
      HashSet <String> newset = new HashSet <String>();
                  
      // populate hash set
      newset.add("Learning"); 
      newset.add("Easy");
      newset.add("Simply");  
      
      System.out.println("Values before remove: "+newset);
      
      // remove "Easy" from set
      boolean isremoved = newset.remove("Easy"); 
      
      System.out.println("Return value after remove: "+isremoved); 
      
      System.out.println("Values after remove: "+newset);     
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

Values before remove: [Learning, Simply, Easy]
Return value after remove: true
Values after remove: [Learning, Simply]

java.util.Locale.getAvailableLocales()方法实例

java.util.Locale.getAvailableLocales() 方法返回所有已安装的语言环境的数组。返回的数组表示Java运行时环境和安装LocaleServiceProvider实现支持的语言环境的结合。它必须至少包含一个Locale实例等于Locale.US。

声明

以下是java.util.Locale.getAvailableLocales()方法的声明

public static Locale[] getAvailableLocales()

参数

  • NA

返回值

此方法返回安装的语言环境的数组。

异常

  • NA

例子

下面的示例演示java.util.Locale.getAvailableLocales()方法的用法。

package com.yiibai;

import java.util.*;

public class LocaleDemo {

   public static void main(String[] args) {

      // create a new array and get all installed locales
      Locale[] locales = Locale.getAvailableLocales();

      // print locales
      System.out.println("Installed locales are:");
      for (int i = 0; i < locales.length; i++) {
         System.out.println(i + ":" + locales[i]);
      }
   }
}

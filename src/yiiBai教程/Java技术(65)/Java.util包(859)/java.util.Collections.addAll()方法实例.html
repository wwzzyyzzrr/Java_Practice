java.util.Collections.addAll()方法实例

addAll(Collection<? super T>, T..) 方法用于所有指定元素添加到指定的集合。

声明

以下是java.util.Collections.addAll()方法的声明。

public static <T> boolean addAll(Collection<? super T> c, T.. a)

参数

  • c-- 这就是其中元素将被插入的集合。

  • a--这是插入到c中的元素

返回值

该方法调用返回true，如果集合更改为调用的结果

异常

  • UnsupportedOperationException-- 这被抛出，如果c不支持add方法。

  • NullPointerException-- 这被抛出，如果元素包含一个或多个null值和c不支持null元素。

  • IllegalArgumentException-- 这被抛出，如果在元素值的某些方面不允许它添加到c。

例子

下面的例子显示java.util.Collections.addAll()方法的使用

package com.yiibai;
import java.util.*;
public class CollectionsDemo {
   public static void main(String args[]) {
      // create array list object       
      List arrlist = new ArrayList();
      // populate the list
      arrlist.add("A");
      arrlist.add("B");
      arrlist.add("C");  
      
      System.out.println("Initial collection value: "+arrlist);
      // add values to this collection
      boolean b = Collections.addAll(arrlist, "1","2","3");
      System.out.println("Final collection value: "+arrlist);
   }    
}

让我们来编译和运行上面的程序，这将产生以下结果。

Initial collection value: [A, B, C]
Final collection value: [A, B, C, 1, 2, 3]

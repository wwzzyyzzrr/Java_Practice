java.util.Hashtable.toString()方法实例

toString() 方法用来得到这个Hashtable对象的一组条目的形式的字符串表示形式。

声明

以下是java.util.Hashtable.toString()方法的声明。

public String toString()

参数

  • NA

返回值

在方法调用返回此哈希表中的字符串表示形式。

异常

  • NA

例子

下面的例子显示java.util.Hashtable.toString()方法的使用

package com.yiibai;

import java.util.*;

public class HashTableDemo {
   public static void main(String args[]) {
      // create hash table 
      Hashtable htable = new Hashtable(3); 
      
      // populate the table
      htable.put(1, "TP");
      htable.put(2, "IS");
      htable.put(3, "THE");
      htable.put(4, "BEST");
      htable.put(5, "TUTORIAL");
      
      System.out.println("String form of hash table is: "+htable.toString()); 
   }    
}

现在编译和运行上面的代码示例，将产生以下结果。

String form of hash table is: {5=TUTORIAL, 4=BEST, 3=THE, 2=IS, 1=TP}

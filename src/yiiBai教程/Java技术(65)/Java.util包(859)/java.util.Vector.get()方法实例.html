java.util.Vector.get()方法实例

get(int index) 方法用于返回在向量中指定索引位置的元素。

声明

以下是java.util.Vector.get()方法的声明

public E get(int index)

参数

  • index--这是该索引位置的元素被返回

返回值

该方法调用返回指定索引处的对象。

异常

  • ArrayIndexOutOfBoundsException--如果访问的索引/位置超出范围时抛出此异常。

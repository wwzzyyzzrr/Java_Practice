java.util.Vector.setSize()方法实例

setSize(int newSize) 方法是用来设置这个向量的大小。如果新的大小大于当前大小，新的空项目被添加到该向量的末端。如果新的大小小于该当前大小所有的组件在索引newSize和更大的被丢弃。

声明

以下是java.util.Vector.setSize()方法的声明

public void setSize(int newSize)

参数

  • newSize--这是向量的新的大小

返回值

NA

异常

  • ArrayIndexOutOfBoundsException--如果新的大小为负，抛出此异常。

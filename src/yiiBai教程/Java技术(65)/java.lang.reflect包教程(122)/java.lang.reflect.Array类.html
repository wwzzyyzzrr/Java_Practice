java.lang.reflect.Array类

java.lang.reflect.Array类提供静态方法来动态创建和访问Java数组。 Array允许在get或set操作期间扩展转换，但如果发生缩小转换，则会抛出IllegalArgumentException异常。

类声明

以下是java.lang.reflect.Array类的声明 -

   extends Object

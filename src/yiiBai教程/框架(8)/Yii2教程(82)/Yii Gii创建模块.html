Yii Gii创建模块

现在让我们来看看如何生成模块。
第1步 - 要生成模块，打开模块生成接口并填写表格字段内容，如下所示：


第2步 - 然后，单击“Preview” 和 “Generate” 按钮。

步骤3 - 我们需要激活该模块。修改 config/web.php 文件中的模块(modules)应用程序组件。 
'modules' => [
   'admin' => [
      'class' => 'app\modules\admin\Admin',
   ],
],

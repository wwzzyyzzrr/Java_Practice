PostgreSQL NOT条件

PostgreSQL NOT条件与WHERE子句一起使用以否定查询中的条件。

语法：

SELECT column1, column2, ..... columnN    
FROM table_name    
WHERE [search_condition] NOT [condition];

我们来看一下表“EMPLOYEES”，具有以下数据。

查询那些地址不为NULL的记录信息，执行以下查询：

SELECT *  
FROM EMPLOYEES  
WHERE address IS NOT NULL ;

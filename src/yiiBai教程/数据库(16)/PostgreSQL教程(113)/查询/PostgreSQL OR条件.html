PostgreSQL OR条件

PostgreSQL OR条件与WHERE子句一起使用，以从表中的一列或多列列中选择唯一数据。

语法

SELECT column1, column2, ..... columnN    
FROM table_name    
WHERE [search_condition]    
OR [search_condition];

我们来看一下表“EMPLOYEES”，具有以下数据。

查询名字是Minsu或者地址为Noida员工信息，执行以下查询：

SELECT *  
FROM  EMPLOYEES 
WHERE NAME = 'Minsu'  
OR ADDRESS = 'Noida';

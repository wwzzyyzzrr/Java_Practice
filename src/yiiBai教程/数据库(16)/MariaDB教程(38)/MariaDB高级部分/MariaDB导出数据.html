MariaDB导出数据

在使用数据库的过程中，我们免不了要备份数据。下面来看如何在MariaDB如何导出数据。

1. 使用HeidiSQL导出MariaDB数据库

在前面的章节中，我们一直在使用一个名称为testdb的数据库，这个数据库里面有一些表。如下所示 - 

MariaDB [testdb]> show tables;
+------------------+
| Tables_in_testdb |
+------------------+
| scores           |
| students         |
| subjects         |
| t1               |
| teachers         |
+------------------+
5 rows in set (0.11 sec)

现在打开HeidiSQL，导入所有这些表和数据。

打开连接后，工作界面如下 -

右键单击左列中的数据库名称“testdb”，然后选择“导出数据脚本(Q)”：

当点击“导出数据脚本(Q)”时，它将创建一个如下所示的图中所示：

选择要导出数据库的位置，并指定一个名称，如上图。现在点击“导出” 按钮。

它会创建一个这样的屏幕：

数据库现在被导出。 现在点击关闭按钮关闭屏幕。

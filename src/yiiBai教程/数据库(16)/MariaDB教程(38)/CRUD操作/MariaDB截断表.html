MariaDB截断表

在MariaDB数据库中，TRUNCATE TABLE语句用于从表中删除所有记录。它与没有WHERE子句的DELETE语句相同。

注意：如果使用TRUNCATE表语句，该表将被永久删除，无法回滚。

语法

TRUNCATE [TABLE] [database_name.]table_name;

示例：

假设我们想要截断students表，那么可以参考以下语句 - 

TRUNCATE TABLE testdb.students;

上面TRUNCATE查询成功执行。可以看到students表的记录已被永久删除。下面来查询students表中的数据，应该不会看到任何数据 - 

MariaDB [testdb]> TRUNCATE TABLE testdb.students;
Query OK, 0 rows affected (0.63 sec)

MariaDB [testdb]> select * from students;
Empty set (0.00 sec)

注意

语句TRUNCATE TABLE students相当于drop table students，然后再create table students ....，或者等效于delete from students，然后再设置自动增长值为：1，也就是执行语句：alter table students auto_increment = 1;

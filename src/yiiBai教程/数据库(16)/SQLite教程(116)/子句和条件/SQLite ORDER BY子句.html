SQLite ORDER BY子句

SQLite ORDER BY子句用于根据一个或多个列对所获取的数据按升序或降序进行排序(排序)。

语法

SELECT column-list   
FROM table_name   
[WHERE condition]   
[ORDER BY column1, column2, .. columnN] [ASC | DESC];

可以在ORDER BY子句中使用一个或多个列。所使用的列必须在列的列表中显示。

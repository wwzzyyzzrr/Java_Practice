SQL PERIOD_ADD()函数

PERIOD_ADD(P,N)

增加了N个月至周期P(格式为YYYYMM或YYYYMM)。返回格式YYYYMM的值。注意周期参数P不是日期值。

mysql> SELECT PERIOD_ADD(9801,2);
+---------------------------------------------------------+
| PERIOD_ADD(9801,2)                                      |
+---------------------------------------------------------+
| 199803                                                  |
+---------------------------------------------------------+
1 row in set (0.00 sec)

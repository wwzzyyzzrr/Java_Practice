SQL BIN(N)函数

BIN(N)

返回N的二进制值的字符串表示，其中N是一个长(BIGINT)数字。这等同于CONV(N，10,2)。返回NULL，如果N为NULL。

SQL> SELECT BIN(12);
+---------------------------------------------------------+
| BIN(12)                                                 |
+---------------------------------------------------------+
| 1100                                                    |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL CONV()函数

CONV(N,from_base,to_base)

CONV()函数的目的是在不同的数值基数之间进行数字转换。该函数返回值N从from_base到to_base转换的字符串。 最小基数为2，最大值为36。如果任何参数为NULL，则函数返回NULL。考虑下面的例子，其中数字5将从基数16向基数2转换：

SQL> SELECT CONV(5,16,2);
+---------------------------------------------------------+
| CONV(5,16,2)                                            |
+---------------------------------------------------------+
| 101                                                     |
+---------------------------------------------------------+
1 row in set (0.00 sec)

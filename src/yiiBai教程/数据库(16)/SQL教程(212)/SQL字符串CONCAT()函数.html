SQL字符串CONCAT()函数

CONCAT(str1,str2,...)

返回的结果是从串联参数字符串。可能有一个或多个参数。如果所有的参数都是非二进制字符串，结果是一个非二进制字符串。如果参数包括任何二进制串，结果是一个二进制串。一个数字参数被转换为等值的二进制字符串的形式;如果你想避免这种情况，你可以使用显式类型转换，如下面的例子：

SQL> SELECT CONCAT('My', 'S', 'QL');
+---------------------------------------------------------+
| CONCAT('My', 'S', 'QL')                                 |
+---------------------------------------------------------+
| SQL                                                   |
+---------------------------------------------------------+
1 row in set (0.00 sec)

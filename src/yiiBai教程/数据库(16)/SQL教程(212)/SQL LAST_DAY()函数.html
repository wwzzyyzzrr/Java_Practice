SQL LAST_DAY()函数

LAST_DAY(date)

需要一个日期或日期时间值，返回该月的最后一天对应的值。 如果该参数是无效的返回NULL。

mysql> SELECT LAST_DAY('2003-02-05');
+---------------------------------------------------------+
| LAST_DAY('2003-02-05')                                  |
+---------------------------------------------------------+
| 2003-02-28                                              |
+---------------------------------------------------------+
1 row in set (0.00 sec)

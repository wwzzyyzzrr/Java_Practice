SQL BIT_AND()函数

BIT_AND(expression)

BIT_AND函数返回的所有位在表达式的位与。 基本的前提是，如果两个对应位是相同的，那么一个位与操作返回1，而如果它们是不同的，位与操作将返回0。函数本身返回一个64位的整数值。如果没有匹配，那么它将返回18446744073709551615.下面的例子对由汽车制造商进行分组的价格列：

SQL> SELECT 
          MAKER, BIT_AND(PRICE) BITS
          FROM CARS GROUP BY MAKER
+---------------------------------------------------------+
|MAKER           BITS                                     |
+---------------------------------------------------------+
|CHRYSLER        512                                      |
|FORD            12488                                    |
|HONDA           2144                                     |
+---------------------------------------------------------+
1 row in set (0.00 sec)

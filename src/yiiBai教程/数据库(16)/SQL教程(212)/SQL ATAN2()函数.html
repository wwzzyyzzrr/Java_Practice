SQL ATAN2()函数

ATAN2(Y,X)

该函数返回两个参数：X和Y的反正切，它类似于Y / X的反正切，所不同的是两者的符号被用来查找象限的结果。

SQL> SELECT ATAN2(3,6);
+---------------------------------------------------------+
| ATAN2(3,6)                                              |
+---------------------------------------------------------+
| 0.46364760900081                                        |
+---------------------------------------------------------+
1 row in set (0.00 sec)

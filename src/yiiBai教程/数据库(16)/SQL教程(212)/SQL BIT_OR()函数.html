SQL BIT_OR()函数

BIT_OR(expression)

BIT_OR()函数返回表达式中所有的按位或。按位或函数的基本前提是，它返回0，如果相应的位匹配，1，如果他们不这样做。该函数返回一个64位的整数，如果没有匹配的行，则返回0。下面的例子执行上的CARS表，由MAKER分组的PRICE列：

SQL> SELECT 
          MAKER, BIT_OR(PRICE) BITS
          FROM CARS GROUP BY MAKER
+---------------------------------------------------------+
|MAKER           BITS                                     |
+---------------------------------------------------------+
|CHRYSLER        62293                                    |
|FORD            16127                                    |
|HONDA           32766                                    |
+---------------------------------------------------------+
1 row in set (0.00 sec)

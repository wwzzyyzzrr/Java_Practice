SQL ELT()函数

ELT(N,str1,str2,str3,...)

返回str1如果N= 1，str2 如果N= 2，等等。返回NULL如果N大于参数的数值小于1或更大。 ELT()是FIELD()的补充。

SQL> SELECT ELT(1, 'ej', 'Heja', 'hej', 'foo');
+---------------------------------------------------------+
| ELT(1, 'ej', 'Heja', 'hej', 'foo')                      |
+---------------------------------------------------------+
| ej                                                      |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL UNHEX()函数

UNHEX(str)

执行 HEX(str)的逆操作. 即，解释每个对十六进制数字的参数作为数字，并将其转换成由数字表示的字符。 所得字符返回为二进制字符串。

SQL> SELECT UNHEX('4D7953514C');
+---------------------------------------------------------+
| UNHEX('4D7953514C')                                     |
+---------------------------------------------------------+
| SQL                                                   |
+---------------------------------------------------------+
1 row in set (0.00 sec)
参数字符串中的字符必须是合法的十六进制数字: '0' .. '9', 'A' .. 'F', 'a' .. 'f'。如果UNHEX()遇到任何非十六进制数字的说法，它返回NULL。

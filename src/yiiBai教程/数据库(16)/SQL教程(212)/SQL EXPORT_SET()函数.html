SQL EXPORT_SET()函数

EXPORT_SET(bits,on,off[,separator[,number_of_bits]])

返回一个字符串，例如，对于每一个位值的位置，你会得到一个关于字符串和每一点的值没有设置，你会得到一个关闭的字符串。比特位被从右到左(从低阶到高阶比特)检查。字符串被添加到结果从左至右，由分隔符分离出的字符串(默认为逗号字符)。检查比特数是由number_of_bits(默认为64)给出。

SQL> SELECT EXPORT_SET(5,'Y','N',',',4);
+---------------------------------------------------------+
| EXPORT_SET(5,'Y','N',',',4)                             |
+---------------------------------------------------------+
| Y,N,Y,N                                                 |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL NOW()函数

NOW()

返回当前为'YYYY-MM-DD HH:MM:SS'或YYYYMMDDHHMMSS的日期和时间格式的一个值，根据函数是否用在字符串或数字语境中。该值表示在当前时区。

mysql> SELECT NOW();
+---------------------------------------------------------+
| NOW()                                                   |
+---------------------------------------------------------+
| 1997-12-15 23:50:26                                     |
+---------------------------------------------------------+
1 row in set (0.00 sec)

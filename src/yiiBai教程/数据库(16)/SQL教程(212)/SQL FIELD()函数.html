SQL FIELD()函数

FIELD(str,str1,str2,str3,...)

返回str 在str1, str2, str3, ...列表其中的索引(位置从1开始)。返回0，如果str没有找到。

SQL> SELECT FIELD('ej', 'Hej', 'ej', 'Heja', 'hej', 'foo');
+---------------------------------------------------------+
| FIELD('ej', 'Hej', 'ej', 'Heja', 'hej', 'foo')          |
+---------------------------------------------------------+
| 2                                                       |
+---------------------------------------------------------+
1 row in set (0.00 sec)

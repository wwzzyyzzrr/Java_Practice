SQL REPLACE()函数

REPLACE(str,from_str,to_str)

返回字符串str的字符串中所有出现的from_str字符串使用to_str 取代。REPLACE() 搜索from_str时进行区分大小写的匹配。

SQL> SELECT REPLACE('www.mysql.com', 'w', 'Ww');
+---------------------------------------------------------+
| REPLACE('www.mysql.com', 'w', 'Ww')                     |
+---------------------------------------------------------+
| WwWwWw.mysql.com                                        |
+---------------------------------------------------------+
1 row in set (0.00 sec)

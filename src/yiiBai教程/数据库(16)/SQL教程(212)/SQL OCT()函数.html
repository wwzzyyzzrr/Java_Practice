SQL OCT()函数

OCT(N)

返回N，其中N是一个很长很长(BIGINT)数字的八进制值的字符串表示。 这等同于CONV(N，10,8)。返回NULL，如果N为NULL。

SQL> SELECT OCT(12);
+---------------------------------------------------------+
| OCT(12)                                                 |
+---------------------------------------------------------+
| 14                                                      |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL CHAR()函数

CHAR(N,... [USING charset_name])

CHAR()解释每个参数n为一个整数，并返回包含这些整数的代码值中给出的字符的字符串。NULL值被跳过。

SQL> SELECT CHAR(77,121,83,81,'76');
+---------------------------------------------------------+
| CHAR(77,121,83,81,'76')                                 |
+---------------------------------------------------------+
| SQL                                                   |
+---------------------------------------------------------+
1 row in set (0.00 sec)

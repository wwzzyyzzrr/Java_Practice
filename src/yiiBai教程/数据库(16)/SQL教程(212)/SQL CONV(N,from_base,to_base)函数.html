SQL CONV(N,from_base,to_base)函数

CONV(N,from_base,to_base)

不同数值的基数之间的转换数字。返回数值N基数转换，从from_base到to_base的字符串表示。 返回NULL，如果任何参数是NULL。 自变量N被解释为一个整数，但也可以指定为整数或字符串。最小基数是2，最大基数是36。如果to_base是一个负数，N被看作一个带符号的数字。否则，N被当作无符号。 CONV()使用64位精度。

SQL> SELECT CONV('a',16,2);
+---------------------------------------------------------+
| CONV('a',16,2)                                          |
+---------------------------------------------------------+
| 1010                                                    |
+---------------------------------------------------------+
1 row in set (0.00 sec)

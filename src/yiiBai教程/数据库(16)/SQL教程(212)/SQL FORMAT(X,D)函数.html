SQL FORMAT(X,D)函数

FORMAT(X,D)

FORMAT()函数是用来按以下格式格式化数字X：###，###，###。##以小数D截断。下面的例子演示了FORMAT()函数的使用和输出：

SQL>SELECT FORMAT(423423234.65434453,2);
+---------------------------------------------------------+
| FORMAT(423423234.65434453,2)                            |
+---------------------------------------------------------+
| 423,423,234.65                                          |
+---------------------------------------------------------+
1 row in set (0.00 sec)

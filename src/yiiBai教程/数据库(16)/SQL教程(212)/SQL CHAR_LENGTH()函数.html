SQL CHAR_LENGTH()函数

CHAR_LENGTH(str)

返回字符测量字符串str的长度。一个多字节字符算作一个字符。这意味着，对于含52字节字符LENGTH()返回10，而CHAR_LENGTH()返回5。

SQL> SELECT CHAR_LENGTH("text");
+---------------------------------------------------------+
| CHAR_LENGTH("text")                                     |
+---------------------------------------------------------+
| 4                                                       |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL LEAST()函数

LEAST(N1,N2,N3,N4,......)

LEAST()函数与GREATEST()函数相反。其目的是为了从值列表(N1，N2，N3，等等)返回至小值项。下面的示例显示了LEAST()函数正确的使用和输出：

SQL>SELECT LEAST(3,5,1,8,33,99,34,55,67,43);
+---------------------------------------------------------+
| LEAST(3,5,1,8,33,99,34,55,67,43)                        |
+---------------------------------------------------------+
| 1                                                       |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL TIME_FORMAT()函数

TIME_FORMAT(time,format)

这是使用像DATE_FORMAT()函数，但格式字符串可以包含格式说明仅用于小时，分钟和秒。

如果时间值包含一个小时的部分大于23，%H和%k小时格式说明产生一个值大于0..23，通常范围更大。小时格式说明符产生小时的值为模数12。

mysql> SELECT TIME_FORMAT('100:00:00', '%H %k %h %I %l');
+---------------------------------------------------------+
| TIME_FORMAT('100:00:00', '%H %k %h %I %l')              |
+---------------------------------------------------------+
| 100 100 04 04 4                                         |
+---------------------------------------------------------+
1 row in set (0.00 sec)

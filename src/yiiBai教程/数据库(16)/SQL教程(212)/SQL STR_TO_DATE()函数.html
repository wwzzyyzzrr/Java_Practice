SQL STR_TO_DATE()函数

STR_TO_DATE(str,format)

这是DATE_FORMAT()函数的逆函数。它需要一个字符串str和一个格式字符串格式format参数。STR_TO_DATE()返回一个DATETIME值，如果格式字符串包含日期和时间部分或日期或时间值，如果字符串只包含日期或时间的部分。

mysql> SELECT STR_TO_DATE('04/31/2004', '%m/%d/%Y');
+---------------------------------------------------------+
| STR_TO_DATE('04/31/2004', '%m/%d/%Y')                   |
+---------------------------------------------------------+
| 2004-04-31                                              |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL TIMESTAMPADD()函数

TIMESTAMPADD(unit,interval,datetime_expr)

添加整数表达式间隔日期或日期时间表达式datetime_expr的日期时间。单位时间间隔由下式给出的单元参数，它应该是以下值之一： FRAC_SECOND, SECOND, MINUTE, HOUR, DAY, WEEK, MONTH, QUARTER 或 YEAR.

单位值可以使用关键字之一，如图所示或用SQL_TSI_前缀被指定。例如, DAY 和 SQL_TSI_DAY 都是非法的

mysql> SELECT TIMESTAMPADD(MINUTE,1,'2003-01-02');
+---------------------------------------------------------+
| TIMESTAMPADD(MINUTE,1,'2003-01-02')                     |
+---------------------------------------------------------+
| 2003-01-02 00:01:00                                     |
+---------------------------------------------------------+
1 row in set (0.00 sec)

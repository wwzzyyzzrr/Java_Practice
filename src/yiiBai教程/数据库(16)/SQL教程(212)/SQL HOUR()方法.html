SQL HOUR()方法

HOUR(time)

返回时间的小时部分。返回日期范围为0至23时小时的值。然而，时间值的范围实际上是要大得多，所以HOUR可以返回大于23的值。

mysql> SELECT HOUR('10:05:03');
+---------------------------------------------------------+
| HOUR('10:05:03')                                        |
+---------------------------------------------------------+
| 10                                                      |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL ORD()函数

ORD(str)

如果字符串str的最左边的字符是一个多字节字符返回该字符，用这个公式其组成字节的数值计算的代码：

  (1st byte code)
+ (2nd byte code . 256)
+ (3rd byte code . 2562) ...

如果最左边的字符不是一个多字节字符，ORD()返回相同的值如ASCII()函数。

SQL> SELECT ORD('2');
+---------------------------------------------------------+
| ORD('2')                                                |
+---------------------------------------------------------+
| 50                                                      |
+---------------------------------------------------------+
1 row in set (0.00 sec)

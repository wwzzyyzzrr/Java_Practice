SQL SEC_TO_TIME()函数

SEC_TO_TIME(seconds)

返回参数秒值，转化为小时，分钟和秒，如“HH:MM:SS'值或HHMMSS格式，根据函数是否用在字符串或数字语境中。

mysql> SELECT SEC_TO_TIME(2378);
+---------------------------------------------------------+
| SEC_TO_TIME(2378)                                       |
+---------------------------------------------------------+
| 00:39:38                                                |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL QUOTE()函数

QUOTE(str)

引用字符串产生一个结果可以作为在SQL语句中正确转义数据值。返回的字符串由单引号和单引号(')，反斜杠('')，ASCII NUL和控制-Z前面加一个反斜杠的每个实例封闭。如果参数为NULL，则返回值的词'NULL'不封闭单引号。

SQL> SELECT QUOTE('Don't!');
+---------------------------------------------------------+
| QUOTE('Don't!')                                        |
+---------------------------------------------------------+
| 'Don't!'                                               |
+---------------------------------------------------------+
1 row in set (0.00 sec)

注意：请检查您的安装有任何错误在使用此函数时，如果有问题那么请不要使用此函数。

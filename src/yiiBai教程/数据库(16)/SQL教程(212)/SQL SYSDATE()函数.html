SQL SYSDATE()函数

SYSDATE()

返回当前的日期和时间在'YYYY-MM-DD HH:MM:SS'或YYYYMMDDHHMMSS格式作为一个值，根据函数是否用在字符串或数字语境中。

mysql> SELECT SYSDATE();
+---------------------------------------------------------+
| SYSDATE()                                               |
+---------------------------------------------------------+
| 2006-04-12 13:47:44                                     |
+---------------------------------------------------------+
1 row in set (0.00 sec)

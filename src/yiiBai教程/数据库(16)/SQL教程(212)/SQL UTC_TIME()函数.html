SQL UTC_TIME()函数

UTC_TIME, UTC_TIME()

返回当前UTC时间在“HH：MM：SS'或HHMMSS格式的值， 根据函数是否用在字符串或数字语境中。

mysql> SELECT UTC_TIME(), UTC_TIME() + 0;
+---------------------------------------------------------+
| UTC_TIME(), UTC_TIME() + 0                              |
+---------------------------------------------------------+
| 18:07:53, 180753                                        |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL OCT(N)函数

OCT(N)

OCT()函数返回N的八进制数字符串表示，这相当于使用CONV(N,10,8)。

SQL>SELECT OCT(12);
+---------------------------------------------------------+
| OCT(12)                                                 |
+---------------------------------------------------------+
| 14                                                      |
+---------------------------------------------------------+
1 row in set (0.00 sec)

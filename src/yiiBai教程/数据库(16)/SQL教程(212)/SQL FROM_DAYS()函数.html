SQL FROM_DAYS()函数

FROM_DAYS(N)

给定一个天数N，并返回一个日期值。

mysql> SELECT FROM_DAYS(729669);
+---------------------------------------------------------+
| FROM_DAYS(729669)                                       |
+---------------------------------------------------------+
| 1997-10-07                                              |
+---------------------------------------------------------+
1 row in set (0.00 sec)

使用FROM_DAYS()谨慎旧日期。它不打算使用与之前的公历(1582年)的到来值。

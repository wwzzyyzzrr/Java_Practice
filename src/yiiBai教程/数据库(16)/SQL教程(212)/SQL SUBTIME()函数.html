SQL SUBTIME()函数

SUBTIME(expr1,expr2)

SUBTIME() 将返回expr1 ，expr2表示为相同的格式expr1值。expr1是一个时间或日期时间表达式，expr2是一个时间。

mysql> SELECT SUBTIME('1997-12-31 23:59:59.999999',
    -> '1 1:1:1.000002');
+---------------------------------------------------------+
| SUBTIME('1997-12-31 23:59:59.999999'...                 |
+---------------------------------------------------------+
| 1997-12-30 22:58:58.999997                              |
+---------------------------------------------------------+
1 row in set (0.00 sec)

SQL FORMAT()函数

FORMAT(X,D)

格式数字X如'＃，###，###。##“的格式，四舍五入至小数点后D位，并返回结果作为字符串。如果D为0，则结果没有小数点或小数部分。

SQL> SELECT FORMAT(12332.123456, 4);
+---------------------------------------------------------+
| FORMAT(12332.123456, 4)                                 |
+---------------------------------------------------------+
| 12,332.1235                                             |
+---------------------------------------------------------+
1 row in set (0.00 sec)

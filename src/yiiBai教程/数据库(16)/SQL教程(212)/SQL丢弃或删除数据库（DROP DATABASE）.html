SQL丢弃或删除数据库（DROP DATABASE）

SQL DROP DATABASE语句用来丢弃在SQL模式中现有数据库。

语法

DROP DATABASE语句的基本语法如下：

DROP DATABASE DatabaseName;

数据库名称在RDBMS中是唯一的。

例子：

如果要删除现有的数据库<testDB>，那么DROP DATABASE语句将如下所示：

SQL> DROP DATABASE testDB;

注意: 此操作之前要小心使用，因为通过删除现有的数据库将导致存储在数据库中的完整信息丢失。

确保你操作任何数据库之前有管理员权限。 一旦数据库被删除，可以检查它的数据库列表如下：

SQL> SHOW DATABASES;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| AMROOD             |
| TUTORIALSPOINT     |
| mysql              |
| orig               |
| test               |
+--------------------+
6 rows in set (0.00 sec)

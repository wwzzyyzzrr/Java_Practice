MySQL导入示例数据库

在本教程中，将学习如何在MySQL命令行下将MySQL示例数据库导入到MySQL数据库服务器中。 在本教程之后，您将yiibaidb示例数据库导入到MySQL服务器中以实践和学习MySQL。

第一步： 从MySQL示例数据库文章中下载示例数据库(yiibaidb)，有关示例数据库的结构，请参考： http://www.yiibai.com/mysql/sample-database.html

第二步： 将下载的文件解压缩到临时文件夹中。为了简单起见，我们将把它解压缩到D:\worksp，如下所示 -

第三步： 连接到MySQL服务器并创建数据库

D:\software\mysql-5.7.18-winx64\bin>mysql -hlocalhost -uroot -p

创建数据库 - 

mysql> CREATE DATABASE IF NOT EXISTS yiibaidb DEFAULT CHARSET utf8 COLLATE utf8_general_ci;

mysql> use yiibaidb;

导入数据 - 

mysql> use yiibaidb;
mysql> source D:/worksp/yiibaidb.sql;

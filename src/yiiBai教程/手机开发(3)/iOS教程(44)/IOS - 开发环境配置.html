iOS - Xcode 安装

1. 下载 Xcode 最新版本，从以下网址： https://developer.apple.com/downloads/

2. 双击Xcode的dmg文件。

3. 你会发现一个设备安装和打开。

4. 这里将两个项目的窗口即显示Xcode应用程序和应用程序“文件夹的快捷方式。

5. 拖动Xcode的应用程序，它会被复制到您的应用程序。

6. 现在Xcode将可从中你可以选择并运行其它程序的一部分。

也可以从Mac App商店下载Xcode中，然后安装。

Interface Builder（界面生成器）

Interface Builder工具它可以方便地创建UI界面。有一组丰富的UI元素被开发用于。刚才拖放到UI视图。我们将了解添加UI元素，创建布局和动作的UI元素在下面即将到来的页面。

对象库中由整个必要的UI元素在右下角。它们的文件扩展名xibs通常被称为用户界面。 xibs中的每一个都被链接到相应的视图控制器。

iOS模拟器

iOS的模拟器实际上由两种类型的设备，即iPhone和iPad不同版本。 iPhone版本包括iPhone（正常），iPhone的视网膜，iPhone5，iPad有iPad和iPad的视网膜。 iPhone模拟器显示如下。

可以模拟在iOS模拟器玩弄的经度和纬度的影响应用程序的位置。也可以模拟内存警告，并在模拟器上的通话状态。可以能够使用对于大多数用途模拟器。但不能测试设备的功能，例如加速度计。所以可能总是需要一个iOS设备的彻底测试应用程序在所有方面和场景。

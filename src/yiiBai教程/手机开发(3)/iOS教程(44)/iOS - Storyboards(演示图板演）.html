iOS - Storyboards(演示图板演）

简介

演示图板在 iOS 5 中才有介绍，当我们用演示图板，部署目标应该是iOS5.0或更高版本。演示图板帮助我们视觉流动的画面，在接口 MainStoryboard.storyboard 帮助并创建所有应用程序屏幕。它还有助于减少推/呈现视图控制器编码。

涉及的步骤

1. 创建一个单独视图应用程序，并确保选择 演示图板 复选框，同时创建应用程序。

2. 选择MainStoryboard.storyboard，在这里你可以找到单一视图控制器。现在添加一个视图控制器，更新视图控制器，如下所示。

3. 现在，让我们连接两个视图控制器。对于这个 "show modal" 按钮的名称上右击，将其拖动到如下图所示，在左侧视图控制器的右视图控制器。

4. 现在选择的模态显示三个选项，如下图所示。

5. 更新 ViewController.h 如下所示：

#import <UIKit/UIKit.h>

@interface ViewController : UIViewController

-(IBAction)done:(UIStoryboardSegue *)seque;

@end

6. 更新 ViewController.m  如下所示：

#import "ViewController.h"

@interface ViewController ()

@end

@implementation ViewController

- (void)viewDidLoad
{
    [super viewDidLoad];	
}

- (void)didReceiveMemoryWarning
{
    [super didReceiveMemoryWarning];
    // Dispose of any resources that can be recreated.
}
-(IBAction)done:(UIStoryboardSegue *)seque{
    [self.navigationController popViewControllerAnimated:YES];
}

@end

7. 选择“MainStoryboard.storyboard”，并右键点击“Exit ”按钮，在右侧视图控制器选择和连接的后退按钮，如下图所示。

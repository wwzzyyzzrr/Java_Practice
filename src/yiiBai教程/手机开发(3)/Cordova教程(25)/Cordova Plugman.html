Cordova Plugman

Cordova plugman是用于安装和管理插件的有用命令行工具。如果你希望你的应用程序能够在一个特定的平台上运行，应该使用 plugman。如果想创建跨平台应用程序，你应该使用cordova-cli，这将在不同平台上修改插件。

第1步 - 安装Plugman

打开命令提示符窗口，然后运行下面的代码片段安装 plugman。 
D:\worksp\cordova\CordovaProject>npm install -g plugman

第2步 - 安装插件

为了向你展示如何使用安装 Cordova 插件，我们将使用plugman安装摄像头插件为一个例子。 
D:\worksp\cordova\CordovaProject>plugman install --platform android --project platforms\android  --plugin cordova-plugin-camera
   plugman uninstall --platform android --project platforms\android --plugin cordova-plugin-camera

这里有三个所需的参数，如上述所示。

  • --platform − 我们使用的平台 (android, ios, amazon-fireos, wp8, blackberry10).

  • --project − 项目建成所在的路径。在我们的示例中，platforms\android目录。

  • --plugin − 我们要安装的插件。

如果设置了有效的参数，在命令提示符窗口应该输出以下。

其他方法

可以以类似的方式使用卸载方法。 
D:\worksp\cordova\CordovaProject>plugman uninstall --platform android --project platforms\android --plugin cordova-plugin-camera
命令提示符控制台会有以下输出。

Plugman提供了可用于一些其他方法。如下面的表中。
SN 方法 & 描述
1

install

用于安装 Cordova 插件
2

uninstall

用于卸载 Cordova 插件
3

fetch

用于复制 Cordova插件的具体位置
4

prepare

用于更新配置文件，以帮助JS模块的支持
5

adduser

用于添加用户帐户到注册表中
6

publish

用于发布插件到注册表中
7

unpublish

用于从注册表取消发布插件
8

search

用于在注册表中搜索插件
9

config

使用注册表设置配置
10

create

用于创建自定义插件
11

platform

用于添加或删除平台自定义创建的插件

附加命令

如果你被卡住了，可以随时使用 plugman-help 命令。版本可以通过使用 plugman-v 进行检查。要搜索插件可以使用plugman搜索，可以使用plugman配置组注册表命令修改注册表插件。

注意

由于Cordova通常用于跨平台开发，我们在接下来的教程将使用 Cordova CLI 来安装插件，而不是Plugman。

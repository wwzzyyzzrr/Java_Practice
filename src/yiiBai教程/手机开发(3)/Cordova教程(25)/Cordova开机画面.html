Cordova开机画面

这个插件是用于显示应用程序启动开机画面。

第1步 - 安装开机画面插件

开机画面插件可以通过在命令提示符窗口运行下面的代码来安装。 
C:\Users\username\Desktop\CordovaProject>cordova plugin add cordova-plugin-splashscreen

第2步 - 添加开机画面

添加开机画面与添加其他 Cordova 插件不同。我们需要打开 config.xml 并在 widget 元素中添加下面的代码片段。

首先片段是 SplashScreen. 它有值属性，该属性是 images 的名字 platform/android/res/drawable(目录). 我们在本实例中使用 Cordova提供默认 screen.png图像，但你可能会想增加自己的图片。重要的是要添加图像的纵向和横向视图，并覆盖不同的屏幕尺寸。 

<preference name = "SplashScreen" value = "screen" /> 

第二个片段，我们需要添加的是：SplashScreenDelay。我们设定 3000 三秒钟后隐藏开机画面。 

<preference name = "SplashScreenDelay" value = "3000" /> 

最后 preference 是可选的。如果它的值设置为true，图像不会被拉伸以适应屏幕。如果它被设置为false，它会被拉伸。 

<preference name = "SplashMaintainAspectRatio" value = "true" />

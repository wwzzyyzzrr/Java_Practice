处理后退按钮

我们通常希望使用Android后退按钮处理一些应用功能，就类似回到之前的屏幕。为了能够实现自己的功能，首先需要按下后退按钮时，禁用退出应用程序。 

    
document.addEventListener("backbutton", onBackKeyDown, false);

function onBackKeyDown(e) {
   e.preventDefault();
   alert('"后退按钮"被点击啦!');
} 

现在，当我们按下原生的Android后退按钮，警告提示就会出现在屏幕上，而不是退出应用程序。这是通过使用 e.preventDefault()完成。

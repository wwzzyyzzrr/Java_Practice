Meteor在手机上运行

在本章中，我们将学习如何在Android设备上运行你的应用程序。最近Meteor刚刚添加此功能适用于Windows操作系统，所以我们需要更新 Meteor 应用到 1.3测试版。

注

在写的时候本教程的时候1.3版本 Meteor 处于测试阶段。

由于我们要使用 Meteor 最新版本，可以通过在提示符窗口中运行下面的命令代码来更新。 
C:\Users\Administrator\Desktop\meteorApp>meteor update --release 1.3-beta.11

第1步 - 安装Android SDK

按照下表中的链接安装Android SDK。
SN 软件 描述
1 Java Development Kit & Android SDK 我们需要Android SDK，以便能够在移动环境中运行 Meteor 应用程序。如果你没有安装它，可以看看我们的Android环境设置教程。 

第2步- 添加Android平台

接下来我们需要Android平台添加到我们的项目中。 
C:\Users\Administrator\Desktop\meteorApp>meteor add-platform android

第3步 - 在运行Android模拟器的应用

现在，我们可以在Android模拟器的运行应用程序。我们将在该行的末尾使用 --verbose 命令，以便能够在启动过程中确定可能的错误。 

C:\Users\Administrator\Desktop\meteorApp>meteor run android --verbose

运行Android设备的应用

由于Android模拟器是缓慢的，所以更好的选择是直接在你的设备上运行应用程序。可以通过连接装置到您的计算机，使开发模式和USB调试去做。

这个过程对特定设备有些不同。需要找到内部版本号在设置/关于并点击。你会得到开发者和开发人员选项将被解锁的通知。

再次搜索并通过设置并启用USB调试。
现在，您可以从命令提示符下在移动设备上运行Meteor应用程序。 
C:\Users\Administrator\Desktop\meteorApp>meteor run android-device

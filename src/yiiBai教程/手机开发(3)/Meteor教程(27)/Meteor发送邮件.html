Meteor发送邮件

当你需要从Meteor应用程序发送电子邮件时使用这个包。

第1步 - 添加包

打开命令提示符窗口，然后运行以下-  
C:\Users\Administrator\Desktop\meteorApp>meteor add email

步骤2 - Mailgun帐户

我们需要创建账户: 点击这里. 这是 Meteor 应用程序默认的电子邮件提供商。

您登录后，打开Domains标签并点击下面的域名沙盒URL。它会打开新的页面，在这里我们可以找到默认SMTP登录名(Default SMTP Login)和默认密码(Default Password)。我们将需要为 MAIL_URL环境变量创建这两个。 

发送Email

要建立有效的 MAIL_URL，只需插入您的Mailgun凭证代替：YOUR_DEFAULT_SMTP_LOGIN 和YOUR_DEFAULT_PASSWORD. 

if (Meteor.isServer) {

   Meteor.startup( function() {
      process.env.MAIL_URL = "smtp://YOUR_DEFAULT_SMTP_LOGIN:YOUR_DEFAULT_PASSWORD@smtp.mailgun.org:587";
		
      Email.send({
         to: "toemailadress@email.com",
         from: "fromemailadress@email.com",
         subject: "Meteor Email",
         text: "The email content..."
      });
		
   });
}
当你运行应用程序，电子邮件将被发送到您的地址。

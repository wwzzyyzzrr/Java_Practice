Meteor核心API

在本教程中，我们将介绍学习Meteor核心API。
如果你想限制代码只在服务器或客户端可以使用下面的代码运行 -

meteorApp.js 

if (Meteor.isClient) {
   // Code running on client...
}

if (Meteor.isServer) {
   // Code running on server...
}
您可以限制代码运行，只有在应用程序使用Cordova捆绑时可使用。 
if (Meteor.isCordova) {
   // Code running on Cordova...
}
一些插件需要等到服务器和DOM已准备就绪。您可以使用下面的代码等待直至一切都开始。 
Meteor.startup(function () {
   // Code running after platform is ready...
});
还有其他核心API的方法。您可以在下表中查看。
S.No. 方法和详细
1

Meteor.wrapAsync(function)

用于包装异步代码并将其转换成同步。
2

Meteor.absoluteUrl([path], [options])

用于生成绝对URL指向的应用程序。
3

Meteor.settings

用于设置部署配置。
4

Meteor.publish(name, function)

用于发布记录客户端。

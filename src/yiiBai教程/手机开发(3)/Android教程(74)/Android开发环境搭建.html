Android开发环境搭建

可以在以下任一操作系统上进行Android应用程序开发 -

  • Microsoft Windows XP或更高版本。
  • 带有Intel芯片的Mac OS X 10.5.8或更高版本。
  • Linux包括GNU C库2.7或更高版本。

所有开发Android应用程序所需的工具都是免费的，可以从网上下载。 以下是开始进行Android应用程序编程之前所需的软件列表。

  • Java JDK5或更高版本
  • Android Studio(IDE)

如果使用Windows机器，那么这些组件使得在执行基于Java的应用程序开发时变得简单。下面来看看看如何进行设置所需的环境。

安装和设置Java开发工具包(JDK)

可以从Oracle的Java站点 - 下载最新版本的Java JDK。将在下载的文件中找到有关安装JDK的说明，请按照给出的说明安装和配置设置。

有关JDK的安装，请参考：http://www.yiibai.com/java/java_environment_setup.html

Android IDE(Android Studio)

有很多先进的技术可用于开发Android应用程序，当前主要使用的是 Android Studio。下面来看看如果在Windows 10 上安装 Android Studio 。

Android Studio是Android应用程序开发的官方IDE。它基于IntelliJ IDEA，可以下载最新版本的Android Studio(当前最新版本是：Android Studio 3.0.1)。官方下载的页面如下所示 -

Gerrit环境安装

在使用Gerrit之前，必须安装Git并做一些基本的配置更改。以下是在不同平台上安装Git客户端的步骤：

安装Git客户端

Linux
可以使用软件包管理工具在Linux上安装Git。例如，如果使用Fedora，那么可以使用：

sudo yum install git

如果您使用基于Debian的发行版，如Ubuntu，那么使用以下命令：

sudo apt-get install git

Windows
可以从Git网站下载Git在Windows上安装。打开链接：msysgit.github.io，点击下载按钮。

Mac
可以通过使用以下命令，在Mac上安装Git：

brew install git

也可以通过从Git网站下载Git来安装。下载链接：http://git-scm.com/download/mac 。

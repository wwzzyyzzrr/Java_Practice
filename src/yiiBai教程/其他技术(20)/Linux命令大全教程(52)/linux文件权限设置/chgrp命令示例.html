chgrp命令示例

在lunix系统里，文件或目录的权限是以拥有者及所诉群组来管理。可以使用chgrp指令取变更文件与目录所属群组，这种方式采用群组名称或群组识别码都可以。chgrp命令就是change group的缩写。要被改变的组名必须要在/etc/group文件内存在才行。

1．命令格式

chgrp [选项] [组] [文件]

2．命令功能

chgrp命令可采用群组名称或群组识别码的方式改变文件或目录的所属群组。使用权限是超级用户。

3．命令参数

必要参数:

  • -c 当发生改变时输出调试信息
  • -f 不显示错误信息
  • -R 处理指定目录以及其子目录下的所有文件
  • -v 运行时显示详细的处理信息
  • --dereference 作用于符号链接的指向，而不是符号链接本身
  • --no-dereference 作用于符号链接本身

选择参数:

  • --reference=<文件或者目录>
  • --help 显示帮助信息
  • --version 显示版本信息

4．使用实例：

实例1：改变文件的群组属性

命令：

chgrp -v bin run.py

输出：

[yiibai@localhost scripts]$ ll
total 0
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:10 run.sh
[yiibai@localhost scripts]$ chgrp -v bin run.py
chgrp: changing group of ‘run.py’: Operation not permitted
failed to change group of ‘run.py’ from yiibai to bin
[yiibai@localhost scripts]$ su root
Password:
[root@localhost scripts]# chgrp -v bin run.py
changed group of ‘run.py’ from yiibai to bin
[root@localhost scripts]# ll
total 0
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai bin    0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:10 run.sh
[root@localhost scripts]#

说明：将run.py文件由yiibai群组改为bin群组

实例2：根据指定文件改变文件的群组属性

命令：

chgrp --reference=run.py run.pl

输出：

[root@localhost scripts]# ll
total 0
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai bin    0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:10 run.sh
[root@localhost scripts]# chgrp --reference=run.py run.pl
[root@localhost scripts]# ll
total 0
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai bin    0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai bin    0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:10 run.sh
[root@localhost scripts]#

说明：改变文件 run.pl 的群组属性，使得文件run.pl的群组属性和参考文件run.py的群组属性相同

实例3：改变指定目录以及其子目录下的所有文件的群组属性

命令演示及输出：

[root@localhost yiibai]# ll scripts/
total 0
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai bin    0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai bin    0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai yiibai 0 Feb 17 02:10 run.sh
[root@localhost yiibai]# chgrp -R bin scripts/
[root@localhost yiibai]# ll scripts/
total 0
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:10 run.sh
[root@localhost yiibai]#

说明：改变指定目录以及其子目录下的所有文件的群组属性

实例4：通过群组识别码改变文件群组属性

命令：

chgrp -R 99 scripts/

输出：

[root@localhost yiibai]# ll scripts/
total 0
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai bin 0 Feb 17 02:10 run.sh
[root@localhost yiibai]# chgrp -R 99 scripts/
[root@localhost yiibai]# ll scripts/
total 0
-rw-rw-r--. 1 yiibai nobody 0 Feb 17 02:11 run.exe
-rw-rw-r--. 1 yiibai nobody 0 Feb 17 02:11 run.pl
-rw-rw-r--. 1 yiibai nobody 0 Feb 17 02:10 run.py
-rw-rw-r--. 1 yiibai nobody 0 Feb 17 02:10 run.sh
[root@localhost yiibai]#

说明：通过群组识别码改变文件群组属性，99 为 nobody 群组的识别码，具体群组和群组识别码可以查看/etc/group文件中的内容。

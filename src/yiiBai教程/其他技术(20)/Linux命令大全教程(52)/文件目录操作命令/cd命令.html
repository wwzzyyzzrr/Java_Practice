cd命令

Linux cd 命令是Linux中最基本的命令语句，其他的命令语句要进行操作，都是建立在使用 cd 命令上的。
所以，学习Linux 常用命令，首先就要学好 cd 命令的使用方法技巧。

1. 命令格式

cd [目录名]

2. 命令功能

切换当前目录至dirName

3. 常用范例

示例1：进入系统根目录

cd /

执行和输出：

yiibai@ubuntu:~$ cd /
yiibai@ubuntu:/$

说明：进入系统根目录,上面命令执行完后拿ls命令看一下，当前目录已经到系统根目录了。
命令：
cd ..或者 cd .. //
执行和输出：

yiibai@ubuntu:/$ cd ~
yiibai@ubuntu:~$ pwd
/home/yiibai
yiibai@ubuntu:~$ cd ..
yiibai@ubuntu:/home$ cd ..//
yiibai@ubuntu:/$

说明：
进入系统根目录可以使用“ cd .. ”一直退，就可以到达根目录
命令：
cd ../.. //
执行和输出：

yiibai@ubuntu:~$ pwd
/home/yiibai
yiibai@ubuntu:~$ cd ../.. //
yiibai@ubuntu:/$ pwd
/
yiibai@ubuntu:/$

说明：使用 cd 命令实现进入当前目录的父目录的父目录。

示例2：使用 cd 命令进入当前用户主目录

“当前用户主目录”和“系统根目录”是两个不同的概念。进入当前用户主目录有两个方法。

命令：

cd ~

执行和输出：

yiibai@ubuntu:/usr/local$ pwd
/usr/local
yiibai@ubuntu:/usr/local$ cd ~
yiibai@ubuntu:~$ pwd
/home/yiibai
yiibai@ubuntu:~$

例3：跳转到指定目录

cd /usr/local/src

执行和输出：

yiibai@ubuntu:~$ pwd
/home/yiibai
yiibai@ubuntu:~$ cd /usr/local/src
yiibai@ubuntu:/usr/local/src$ pwd
/usr/local/src

说明：
跳转到指定目录，从根目录开始，目录名称前加 / ,当前目录内的子目录直接写名称即可

例4：返回进入此目录之前所在的目录

cd -

执行和输出：

yiibai@ubuntu:/usr/local/src$ pwd
/usr/local/src
yiibai@ubuntu:/usr/local/src$ cd -
/home/yiibai
yiibai@ubuntu:~$ pwd
/home/yiibai
yiibai@ubuntu:~$ cd -
/usr/local/src
yiibai@ubuntu:/usr/local/src$

例5：把上个命令的参数作为cd参数使用

cd !$

执行和输出：

yiibai@ubuntu:/usr/local/src$ cd !$
cd /usr/local/src
yiibai@ubuntu:/usr/local/src$ cd !$
cd /usr/local/src
yiibai@ubuntu:/usr/local/src$ cd ~
yiibai@ubuntu:~$ cd !$
cd ~
yiibai@ubuntu:~$

Fortran Cycle语句

cycle语句使循环跳过它的主体的其余部分，并立即重新测试其条件在声明之前。

流程图

例子

program cycle_example     
implicit none      

   integer :: i     
   
   do i = 1, 20          
   
      if (i == 5) then 
         cycle          
      end if         
      
   print*, i      
   end do  
   
end program cycle_example

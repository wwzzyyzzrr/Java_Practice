Erlang内嵌if语句

有时，会有有多个 if 语句嵌入彼此内部，在其他编程语言这是比较常见。在 Erlang 中也是这样的。
下图是嵌套 if 语句流程的示意图。

这里是一个示例程序如下所示 -

示例 

-module(helloworld). 
-export([start/0]). 

start() -> 
   A = 4, 
   B = 6, 
   if 
      A < B ->
      
         if 
            A > 5 -> 
               io:fwrite("A is greater than 5"); 
            true -> 
               io:fwrite("A is less than 5")
         end;
      true -> 
         io:fwrite("A is greater than B") 
   end.
以下是上述程序需要说明的一些关键点 -
  • 当第一个 if 语句条件评估(计算)为真，那么它会开始第二 if 语句条件评估(计算)
上面的代码的输出结果是 - 
A is less than 5

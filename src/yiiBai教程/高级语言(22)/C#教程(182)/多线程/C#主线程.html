C#主线程

在进程中创建的第一个线程称为主线程。它第一个开始，最后一个结束。

下面来看看 C# 中主线程的一个例子。参考以下示例代码 - 

using System;  
using System.Threading;  
public class ThreadExample  
{  
    public static void Main(string[] args)  
    {  
        Thread t = Thread.CurrentThread;  
        t.Name = "MainThread";  
        Console.WriteLine(t.Name);  
    }  
}

执行上面示例代码，得到以下结果 - 

MainThread

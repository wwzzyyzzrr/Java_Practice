std::setiosflags()函数

C++函数std::resetiosflags的行为类似成员 setf 使用mask作为参数在作为操纵器插入/提取（在输入流或输出流上插入/提取）的流上被调用。

它用于由参数 mask 指定的格式来设置标志。

声明

以下是std::setiosflags函数的声明。

setiosflags (ios_base::fmtflags mask);

参数

mask − 掩码表示要设置的标志。fmtflags是位掩码（bitmask）类型。

返回值

它返回未指定（unspecified）。 此函数仅用作流操纵器。

示例

在下面的例子中演示了setiosflag函数的使用。

#include <iostream>
#include <iomanip>

int main () {
   std::cout << std::hex;
   std::cout << std::setiosflags (std::ios::showbase | std::ios::uppercase);
   std::cout << 100 << std::endl;
   return 0;
}

ios::copyfmt()函数

将rhs的所有内部成员的值（状态标志和关联的流缓冲区除外）复制到 *this 的相应成员。

声明

以下是ios::copyfmt函数的声明。

ios& copyfmt (const ios& rhs);

参数

rhs − Stream对象，其成员将被复制到 *this。

示例

在下面的例子中演示了ios::copyfmt函数的使用。

#include <iostream>
#include <fstream>

int main () {
   std::ofstream filestr;
   filestr.open ("test.txt");

   std::cout.fill ('*');
   std::cout.width (10);
   filestr.copyfmt (std::cout);

   std::cout << 40;
   filestr << 40;

   return 0;
}

std::resetiosflags()函数

C++函数std::resetiosflags的行为类似成员 unsetf 使用mask作为参数在作为操纵器插入/提取（在输入流或输出流上插入/提取）的流上被调用。

它用于取消由参数 mask 指定的格式标志。

声明

它的以下是std::resetiosflagss函数的声明。

示例

下面的例子解释了 resetiosflags 函数的功能。

#include <iostream>
#include <iomanip>

int main () {
   std::cout << std::hex << std::setiosflags (std::ios::showbase);
   std::cout << 100 << std::endl;
   std::cout << std::resetiosflags(std::ios::showbase) << 100 << std::endl;
   return 0;
}

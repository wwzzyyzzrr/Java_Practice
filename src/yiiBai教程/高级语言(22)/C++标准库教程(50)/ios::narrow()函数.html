ios::narrow()函数

它用于指缩小字符，并使用当前在流中的语言环境对象的ctype :: narrow （如果这样的等价存在）返回c的等效变换，否则返回dfault。

此函数设计用于使用不同（更宽）字符类型的basic_ios的实例化：请参阅basic_ios :: narro。

声明

以下是ios :: narrow函数的声明。

char narrow (char c, char dfault) const;

参数

  • c − 用于“缩小”的字符。

  • dfault − 如果c没有标准等效值，则返回字符。

返回值

如果有设置c则返回c。否则它返回默认值。

异常

强保证 - 如果抛出异常，流中没有任何更改。

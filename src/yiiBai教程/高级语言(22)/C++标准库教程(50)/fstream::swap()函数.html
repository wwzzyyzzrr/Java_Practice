fstream::swap()函数

它交换fstream对象x和y的值。

声明

下面是fstream:: swap的声明。

C++11

template <class charT, class traits>
  void swap (basic_fstream<charT,traits>& x, basic_fstream<charT,traits>& y);

参数

x,y − 与basic_ostream对象类型相同（即，具有相同的模板参数，charT和traits）。

示例

下面的例子解释了 fstream 中的 swap 函数。

#include <fstream>

int main () {
   std::fstream foo;
   std::fstream bar ("test.txt");

   swap(foo,bar);

   foo << "Yiibai tutorials";

   foo.close();

   return 0;
}

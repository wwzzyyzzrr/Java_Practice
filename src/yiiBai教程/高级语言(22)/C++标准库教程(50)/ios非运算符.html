ios非运算符

它用于检查评估流（非）。 这相当于调用成员失败。

声明

下面是ios::operator！函数的声明 。

bool operator!() const;

返回值

如果设置了failbit或badbit 则返回 True ， 否则返回 false.

示例

在下面的例子中解释了 ios::operator！ 函数功能。

#include <iostream>     
#include <fstream>      

int main () {
  std::ifstream is;
  is.open ("test.txt");
  if (!is)
    std::cerr << "Error opening 'test.txt'/n";
  return 0;
}

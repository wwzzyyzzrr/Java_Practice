ios::fill()函数

它用于获取/设置填充字符。填充字符是用于填充空格的字符到字段宽度在输出插入函数时。

声明

下面是ios::fill函数的声明。

get (1)     char fill() const;
set (2)     char fill (char fillch);

上述第一种形式（1）返回填充字符。

上述第二种形式（2）将fillch设置为新的填充字符，并返回在之前调用使用的填充字符。

参数

  • fillch − 新的填充字符。

返回值

  • 调用前的填充字符的值。

示例

下面的例子演示了ios::fill函数。

#include <iostream>     

int main () {
   char prev;

   std::cout.width (10);
   std::cout << 40 << '/n';

   prev = std::cout.fill ('x');
   std::cout.width (10);
   std::cout << 40 << '/n';

   std::cout.fill(prev);

   return 0;
}

编译和运行上面的程序，将产生以下结果 -

 40

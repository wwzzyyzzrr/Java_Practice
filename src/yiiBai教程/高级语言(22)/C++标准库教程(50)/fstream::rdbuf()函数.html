fstream::rdbuf()函数

它返回一个指向内部filebuf对象的指针。

声明

下面是fstream::rduf()函数的声明。

C++11

filebuf* rdbuf() const;

返回值

它返回一个指向内部 filebuf 对象的指针。

示例

在下面的例子中解释了 fstream::rdbuf() 函数。

#include <fstream>
#include <cstdio>

int main () {
   std::fstream src,dest;
   src.open ("test.txt");
   dest.open ("copy.txt");

   std::filebuf* inbuf  = src.rdbuf();
   std::filebuf* outbuf = dest.rdbuf();

   char c = inbuf->sbumpc();
   while (c != EOF) {
      outbuf->sputc (c);
      c = inbuf->sbumpc();
   }

   dest.close();
   src.close();

   return 0;
}

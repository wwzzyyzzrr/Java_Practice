C语言#define指令

#define预处理器指令用于定义常量或微替代。它可以使用任何基本数据类型。

语法：

#define token value

下面来看看一个使用#define来定义一个常量的例子。

#include <stdio.h>  
#define PI 3.14  
void main() {  
   printf("%f",PI);  
}

执行上面代码，得到以下结果 - 

3.140000

我们来看一个#define创建一个宏的例子。

#include <stdio.h>  
#define MIN(a,b) ((a)<(b)?(a):(b))  
void main() {  
   printf("Minimum between 10 and 20 is: %d\n", MIN(10,20));    
}

执行上面代码，得到以下结果 - 

Minimum between 10 and 20 is: 10

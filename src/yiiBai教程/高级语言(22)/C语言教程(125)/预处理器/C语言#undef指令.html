C语言#undef指令

#undef预处理指令用于取消通过#define定义的常量或宏。

语法：

#undef token

我们来看一个简单的例子来了解如何定义一个常量。

#include <stdio.h>  
#define PI 3.14  
#undef PI  
main() {  
   printf("%f",PI);  
}

执行上面示例代码，得到以下结果 - 

Compile Time Error: 'PI' undeclared

#undef指令用于将预处理常量定义取消定义，以便您可以再次声明常量。

我们来看一个例子，在定义和取消定义数字变量。 但在未定义之前，它被平方变量使用。

#include <stdio.h>  
#define number 15  
int square=number*number;  
#undef number  
main() {  
   printf("%d",square);  
}

执行上面示例代码，得到以下结果 - 

255

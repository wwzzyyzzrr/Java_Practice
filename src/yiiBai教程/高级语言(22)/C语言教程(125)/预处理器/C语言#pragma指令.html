C语言#pragma指令

#pragma预处理指令用于向编译器提供其他信息。 #pragma指令由编译器用于提供机器或操作系统功能。

语法：

#pragma token

不同的编译器可以提供不同的#pragma指令的使用。

turbo C++编译器支持以下#pragma指令。

#pragma argsused  
#pragma exit  
#pragma hdrfile  
#pragma hdrstop  
#pragma inline  
#pragma option  
#pragma saveregs  
#pragma startup  
#pragma warn

下面来看看一个简单使用#pragma预处理器指令的示例。创建一个源文件：pragma.c，其代码如下所示 - 

#include<stdio.h>  
#include<conio.h>  

void func();

#pragma startup func  
#pragma exit func  

void main() {
    printf("\nI am in main");

}

void func() {
    printf("\nI am in func");
}

执行上面示例代码，得到以下结果 - 

I am in main

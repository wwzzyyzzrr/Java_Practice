C语言strcat()函数

strcat(first_string, second_string)函数连接两个字符串，结果返回到first_string。

使用示例

创建一个源文件：string_strcat.c，其代码如下所示 - 

#include <stdio.h>  
void main()  
{  
   char ch[32]={'y', 'i', 'i', 'b', 'a','i', '\0'};  
   char ch2[10]={'.', 'c','o','m','\0'};  
   strcat(ch,ch2);  
   printf("Value of first string is: %s",ch);  
}

执行上面示例代码，得到以下结果 - 

Value of first string is: yiibai.com

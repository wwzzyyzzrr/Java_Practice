Hello World程序实例

这是学习GO编程语言的第一个程序，将打印经典的“hello world”消息。这里是完整的源代码。要运行程序，将代码放在hello-world.go文件中并使用go run。

所有的示例代码，都放在 F:\worksp\golang 目录下。

安装Go编程环境请参考：http://www.yiibai.com/go/go_environment.html

有时如果想要把程序构建成二进制文件。可以使用go build命令来做到这一点。然后可以直接执行构建好的二进制文件。

现在可以运行并构建Go程序，以了解如何编写，构建和运行Go语言程序。

hello-world.go的完整代码如下所示 -

package main
import "fmt"
func main() {
    fmt.Println("hello world")
}

执行上面代码，将得到以下输出结果 - 

$ go run hello-world.go
hello world

有时如想把程序构建成二进制文件。可以使用go build来做到这一点。

$ go build hello-world.go
$ ls
hello-world    hello-world.go

然后可以直接执行构建好的二进制。

$ ./hello-world
hello world

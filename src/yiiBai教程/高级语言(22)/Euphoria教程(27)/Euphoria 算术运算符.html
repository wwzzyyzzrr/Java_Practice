Euphoria 算术运算符

下面的简单示例程序演示了算术运算符。复制并粘贴以下Euphoria 程序在test.ex文件并运行这个程序：

#!/home/euphoria-4.0b2/bin/eui

integer a = 10
integer b = 20

printf(1, "a + b = %d\n" , (a + b) )
printf(1, "a - b = %d\n" , (a - b) )
printf(1, "a * b = %d\n" , (a * b) )
printf(1, "b / a = %d\n" , (b / a) )
printf(1, "+a   = %d\n" ,  (+a) )
printf(1, "-a   = %d\n" ,  (-a) )

这将产生以下结果：

a + b = 30
a - b = -10
a * b = 200
b / a = 2
+a   = 10
-a   = -10

注: 其他的Euphoria 数据类型例如Euphoria 支持所有上述操作序列，原子和对象。

C++回文程序实例

回文数字是一种反向后也相同的数字(从左边读与从右边读都是同一个数字)。 例如：121,34543,343,131,4894这些都是回文数。

回文数算法

  • 从用户输入获取数字
  • 将数字保存在临时变量中
  • 反转数字
  • 将临时数字与反转数字进行比较
  • 如果两个数字相同，则打印回文数字
  • 否则打印不是回文数

下面来看看看C++中如何实现回文的一个程序。 在这个程序中，将从用户得到一个输入，并检查数是否是回文。

#include <iostream>  
using namespace std;  
int main()  
{  
  int n,r,sum=0,temp;    
  cout<<"Enter the Number=";    
  cin>>n;    
  temp=n;    
     while(n>0)    
    {    
     r=n%10;    
     sum=(sum*10)+r;    
     n=n/10;    
    }    
    if(temp==sum)    
        cout<<"Number is Palindrome.";    
    else    
        cout<<"Number is not Palindrome.";   
  return 0;  
}

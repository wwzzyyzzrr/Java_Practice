C++打印字母表三角

三角形可以通过字母或数字生成，有不同的三角形可以打印。 在这个C++程序中，我们将打印字母三角形。

下面来看看看打印字母三角形的C++示例。

#include <iostream>  
using namespace std;  
int main()  
{  
    char ch='A';    
    int i, j, k, m;    
    for(i=1;i<=5;i++)    
    {    
        for(j=5;j>=i;j--)    
            cout<<" ";    
        for(k=1;k<=i;k++)    
            cout<<ch++;    
            ch--;    
        for(m=1;m<i;m++)    
            cout<<--ch;    
        cout<<"\n";    
        ch='A';    
    }    
    return 0;  
}

执行上面代码，得到以下结果 -

     A
    ABA
   ABCBA
  ABCDCBA
 ABCDEDCBA

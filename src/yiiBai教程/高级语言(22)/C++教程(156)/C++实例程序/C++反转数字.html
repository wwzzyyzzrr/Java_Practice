C++反转数字

在C++中，可以使用循环和算术运算符反转数字。 在这个程序中，我们从用户输入获取数字并反转该数字。

下面来看看一个简单的C++示例来反转一个给定的数字。

#include <iostream>  
using namespace std;  
int main()  
{  
  int n, reverse=0, rem;    
  cout<<"Enter a number: ";    
  cin>>n;    
  while(n!=0)    
  {    
     rem=n%10;      
     reverse=reverse*10+rem;    
     n/=10;    
  }    
  cout<<"Reversed Number: "<<reverse<<endl;     
  return 0;  
}

执行上面代码，得到以下结果 - 

Enter a number: 234  
Reversed Number: 432

C++ continue语句

C++ continue语句用于继续循环。它继续程序的当前流程，并在指定条件下跳过剩余的代码。 在内循环的情况下，它仅继续内循环，即：跳过内循环，继续执行下一个内循环。

jump-statement;      
continue;

C++ Continue语句示例

#include <iostream>  
using namespace std;  
int main()  
{  
    for(int i=1;i<=10;i++){      
        if(i==5){      
            continue;      
        }      
        cout<<i<<"\n";      
    }
    return 0;
}

执行结果如下所示 -

1
2
3
4
6
7
8
9
10

C++使用内循环continue声明

C++ continue语句只有在内循环中使用，continue语句时继续内循环。

#include <iostream>  
using namespace std;  
int main()  
{  
    for(int i=1;i<=3;i++){        
        for(int j=1;j<=3;j++){        
            if(i==2&&j==2){        
                continue;        
            }        
            cout<<i<<" "<<j<<"\n";                  
        }        
    }
    return 0;
}

执行结果如下所示 -

1 1
1 2
1 3
2 1
2 3
3 1
3 2
3 3

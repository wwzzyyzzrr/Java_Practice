Swift比较运算符

下表列出了所有 Swift 语言支持的关系运算符。假设变量A=10，变量B=20，则：

操作符 描述 示例
== 检查两个操作数的值是否相等; 如果是，则条件变为真 (A == B) 非 true.
!= 检查两个操作数的值是否相等; 如果值不相等，则条件变为真 (A != B) 是 true.
> 如果左操作数的值大于右操作数的值; 则条件变为真 (A > B) 非 true.
< 如果左操作数的值小于右操作数的值; 则条件变为真 (A < B) 是 true.
>= 如果左操作数的值大于或等于右操作数的值;则条件变为真 (A >= B) 非 true.
<= 如果左操作数的值小于或等于右操作数的值; 则条件变为真 (A <= B) 是 true.

Objective-C 函数返回指针

正如我们在最后一章所看到的，Objective-C编程语言允许从一个函数返回一个数组，Objective-C 类似的方式，允许从一个函数返回一个指针。要做到这一点，必须声明一个函数返回一个指针在下面的例子：

int * myFunction()
{
.
.
.
}

第二点要记住的是，返回一个局部变量的地址在函数之外它不是好想法，所以必须将局部变量定义为静态（static ）变量。

现在，考虑下面的函数，这将产生10个随机数，并返回它们使用数组名，它代表一个指针，即，第一个数组元素的地址。

#import <Foundation/Foundation.h>

/* function to generate and return random numbers. */
int * getRandom( )
{
   static int  r[10];
   int i;
 
   /* set the seed */
   srand( (unsigned)time( NULL ) );
   for ( i = 0; i < 10; ++i)
   {
      r[i] = rand();
      NSLog(@"%d
", r[i] );
   }
 
   return r;
}
 
/* main function to call above defined function */
int main ()
{
   /* a yiibaier to an int */
   int *p;
   int i;

   p = getRandom();
   for ( i = 0; i < 10; i++ )
   {
       NSLog(@"*(p + [%d]) : %d
", i, *(p + i) );
   }
 
   return 0;
}

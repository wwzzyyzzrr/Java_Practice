Objective-C 从函数返回数组

Objective-C编程语言不允许返回整个数组作为参数传递给函数。但是，您可以返回一个指针到一个数组中没有索引指定数组的名字。将在下一章节中学习指针，这样就可以跳过本章，直到你明白 Objective-C中的指针的概念。

如果想从一个函数返回一维数组，就必须声明一个函数返回一个指针在下面的例子：

int * myFunction()
{
.
.
.
}

第二点要记住的是Objective-C中不提倡的地址返回一个局部变量在函数之外，所以必须将局部变量定义为静态变量。

现在，考虑下面的函数，这将产生10个随机数，并将其退回使用数组，调用这个函数如下：

#import <Foundation/Foundation.h>

@interface SampleClass:NSObject

- (int *) getRandom;

@end

@implementation SampleClass

/* function to generate and return random numbers */
- (int *) getRandom
{
  static int  r[10];
  int i;

  /* set the seed */
  srand( (unsigned)time( NULL ) );
  for ( i = 0; i < 10; ++i)
  {
     r[i] = rand();
     NSLog( @"r[%d] = %d
", i, r[i]);

  }

  return r;
}

@end

/* main function to call above defined function */
int main ()
{
   /* a yiibaier to an int */
   int *p;
   int i;

   SampleClass *sampleClass = [[SampleClass alloc]init];
   p = [sampleClass getRandom];
   for ( i = 0; i < 10; i++ )
   {
       NSLog( @"*(p + %d) : %d
", i, *(p + i));
   }

   return 0;
}

Objective-C 指针的数组

这是最有可能的是，你可能不理解本章的内容，直到通过 Objective-C 中的指针相关的章节。

因此，假如你有一点认识，在Objective-C编程语言的指针，让我们开始：数组名是一个常量指针数组的第一个元素。因此声明：

double balance[50];

balance是一个指针，指向与平衡[0]，这是数组balance 第一个元素的地址。因此，下面的程序片段分配p为balance地址的第一个元素：

double *p;
double balance[10];

p = balance;

它是合法的，使用数组名作为常量指针，反之亦然。因此， *(balance + 4)是一种合法的方式访问数据balance[4]。 

一旦储存 p 中的第一个元素的地址，可以访问数组元素使用 *p, *(p+1), *(p+2)等。下面的例子来说明上述讨论的所有概念：

#import <Foundation/Foundation.h>

int main ()
{
   /* an array with 5 elements */
   double balance[5] = {1000.0, 2.0, 3.4, 17.0, 50.0};
   double *p;
   int i;

   p = balance;
 
   /* output each array element's value */
   NSLog( @"Array values using yiibaier
");
   for ( i = 0; i < 5; i++ )
   {
       NSLog(@"*(p + %d) : %f
",  i, *(p + i) );
   }

   NSLog(@"Array values using balance as address
");
   for ( i = 0; i < 5; i++ )
   {
       NSLog(@"*(balance + %d) : %f
",  i, *(balance + i) );
   }
 
   return 0;
}

在上面的例子中，p是一个指针，指向double，这意味着它可以存储双精度类型的变量地址。一旦我们有了p的地址，然后 *p 会给我们提供的地址存储在p值，因为我们已经在上面的例子显示。

Objective-C 指向指针的指针

指针的指针是多个间接或指针链的一种形式。正常情况下，指针包含一个变量的地址。当我们定义了一个指向指针的指针，第一个指针包含的地址的第二指针，它指向包含实际的值的位置，如下图所示。

这是一个指向指针的指针的变量必须在声明等。这是通过在它的名字前面放置一个附加的星号。例如，以下是声明一个指向int类型的指针的声明：

int **var;

当一个目标值间接指向由一个指向指针的指针，访问该数值，需要星号运算符被应用两次，如下面的例子中所示：

#import <Foundation/Foundation.h>
 
int main ()
{
   int  var;
   int  *ptr;
   int  **pptr;

   var = 3000;

   /* take the address of var */
   ptr = &var;

   /* take the address of ptr using address of operator & */
   pptr = &ptr;

   /* take the value using pptr */
   NSLog(@"Value of var = %d
", var );
   NSLog(@"Value available at *ptr = %d
", *ptr );
   NSLog(@"Value available at **pptr = %d
", **pptr);

   return 0;
}

上面的代码编译和执行时，它会产生以下结果：

2013-09-14 00:41:45.687 demo[22087] Value of var = 3000
2013-09-14 00:41:45.687 demo[22087] Value available at *ptr = 3000
2013-09-14 00:41:45.687 demo[22087] Value available at **pptr = 3000

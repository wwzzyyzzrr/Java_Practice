EDOM - C语言宏

C库宏EDOM表示域发生的错误，如果输入参数是外域的数学函数的定义，并设置errno为EDOM。

声明

以下是EDOM 宏的声明。

#define EDOM some_value

参数

  • NA

返回值

  • NA

实例

下面的例子演示了如何使用EDOM宏。

#include <stdio.h>
#include <errno.h>
#include <math.h>

int main()
{
   double val;

   errno = 0;
   val = sqrt(-10);
   if(errno == EDOM) 
   {
      printf("Invalid value 
");
   }
   else 
   {
      printf("Valid value
");
   }
   
   errno = 0;
   val = sqrt(10);
   if(errno == EDOM) 
   {
      printf("Invalid value
");
   }
   else 
   {
      printf("Valid value
");
   }
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

Invalid value
Valid value

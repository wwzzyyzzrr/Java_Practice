isxdigit() - C函数

C库函数int isxdigit(int c)检查传递的字符是否是十六进制数字。

声明

以下是isxdigit（）函数的声明。

int isxdigit(int c);

参数

  • c -- 这是要检查的字符。

返回值

这个函数返回一个非零值（true）如果c是十六进制数字，否则返回零（false）

例子

下面的例子演示了如何使用isxdigit（）函数。

#include <stdio.h>
#include <ctype.h>

int main()
{
   char var1[] = "tuts";
   char var2[] = "0xE";
  
   if( isxdigit(var1[0]) )
   {
     printf("var1 = |%s| is hexadecimal character
", var1 );
   }
   else
   {
     printf("var1 = |%s| is not hexadecimal character
", var1 );
   }
   
   if( isxdigit(var2[0] ))
   {
     printf("var2 = |%s| is hexadecimal character
", var2 );
   }
   else
   {
     printf("var2 = |%s| is not hexadecimal character
", var2 );
   }
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

var1 = |tuts| is not hexadecimal character
var2 = |0xE| is hexadecimal character

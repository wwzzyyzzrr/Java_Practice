feof() - C库函数

以下是 feof()函数的声明。

int feof(FILE *stream)

参数

  • stream -- 这是一个文件 FILE 对象的标识流的指针。

返回值

这个函数返回一个非零值时，与该流关联的文件结束指示器设置，否则返回零。

实例

下面的例子显示feof()函数的用法。

#include <stdio.h>

int main ()
{
   FILE *fp;
   int c;
   int n = 0;
  
   fp = fopen("file.txt","r");
   if(fp == NULL) 
   {
      perror("Error in opening file");
      return(-1);
   }
   while(!feof(fp))
   {
      c = fgetc(fp);
      printf("%c", c);
   }
  
   fclose(fp);
   
   return(0);
}

假设我们有一个文本文件file.txt，它具有以下的内容。此文件将被作为我们的示例程序输入：

This is yiibai.com

让我们编译和运行上面的程序，这将产生以下结果：

This is yiibai.com

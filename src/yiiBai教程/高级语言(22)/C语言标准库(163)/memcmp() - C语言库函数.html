memcmp() - C语言库函数

C库函数 int memcmp(const void *str1, const void *str2, size_t n)) 的前n个字节的内存区域str1和存储区str2中比较。

声明

以下是memcmp() 函数的声明。

int memcmp(const void *str1, const void *str2, size_t n)

参数

  • str1 -- 这就是指针的内存块。

  • str2 -- 这就是指针的内存块。

  • n -- 这是要比较的字节数。

返回值

  • 如果返回值<0，则表明str1小于str2

  • 如果返回值，如果> 0，则表明str2小于str1

  • 如果返回值= 0，则表明str1等于str2

实例

下面的例子演示了如何使用 memcmp() 函数。

#include <stdio.h>
#include <string.h>

int main ()
{
   char str1[15];
   char str2[15];
   int ret;

   memcpy(str1, "abcdef", 6);
   memcpy(str2, "ABCDEF", 6);

   ret = memcmp(str1, str2, 5);

   if(ret > 0)
   {
      printf("str1 is less than str2");
   }
   else if(ret < 0) 
   {
      printf("str2 is less than str1");
   }
   else 
   {
      printf("str1 is equal to str2");
   }
   
   return(0);
}

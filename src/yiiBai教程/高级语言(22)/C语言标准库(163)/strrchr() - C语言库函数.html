strrchr() - C语言库函数

C库函数char *strrchr(const char *str, int c) 搜索最后出现的字符串中的字符c（unsigned char类型）指向参数str。  

声明

以下是strrchr() 函数的声明。

char *strrchr(const char *str, int c)

参数

  • str -- 这是C字符串。

  • c -- 这是定位字符。它通过其int 转换，但它在内部转换为char。

返回值

这个函数返回一个指针，指向str中最后一次出现的字符。如果没有找到值，函数返回一个空指针。

例子

下面的例子显示strrchr() 函数的用法。

#include <stdio.h>
#include <string.h>

int main ()
{
   int len;
   const char str[] = "http://www.yiibai.com/html";
   const char ch = '.';
   char *ret;

   ret = strrchr(str, ch);

   printf("String after |%c| is - |%s|
", ch, ret);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

String after |.| is - |.com|

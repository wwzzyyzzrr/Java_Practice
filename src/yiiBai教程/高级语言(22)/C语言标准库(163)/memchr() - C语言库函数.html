memchr() - C语言库函数

C库函数void *memchr(const void *str, int c, size_t n) 搜索参数str指向的字符串的前n个字节中第一次出现的字符c（unsigned char类型）。

声明

以下是声明了memchr() 函数。

void *memchr(const void *str, int c, size_t n)

参数

  • str -- 这是指针内存块执行搜索。

  • c -- 这是要传递一个int值，但功能进行搜索使用无符号字符型转换这个值每字节一个字节。

  • n -- 这是要分析的字节数。

返回值

这个函数返回一个指针到匹配的字节或NULL，如果字符不会出现在给定的内存区域。

例子

下面的例子显示了memchr() 函数的用法。

#include <stdio.h>
#include <string.h>

int main ()
{
   const char str[] = "http://www.yiibai.com/html";
   const char ch = '.';
   char *ret;

   ret = memchr(str, ch, strlen(str));

   printf("String after |%c| is - |%s|
", ch, ret);

   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

String after |.| is - |.yiibai.com|

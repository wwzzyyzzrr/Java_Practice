ldexp() - C函数

C库函数double ldexp(double x, int exponent)返回x乘以2指数的幂。

声明

以下是ldexp()函数的声明。

double ldexp(double x, int exponent)

参数

  • x -- 这是有效浮点值表示。

  • exponent -- 这是一个指数值。

返回值

这个函数返回 x * 2 exp

实例

下面的例子显示ldexp()函数的用法。

#include <stdio.h>
#include <math.h>

int main ()
{
   double x, ret;
   int n;

   x = 0.65;
   n = 3;
   ret = ldexp(x ,n);
   printf("%f * 2^%d = %f
", x, n, ret);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

0.650000 * 2^3 = 5.200000

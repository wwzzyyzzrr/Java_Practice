ceil() - C函数

C库函数double ceil(double x) 返回大于或等于x的最小整数值。

声明

以下是ceil() 函数的声明。

double ceil(double x)

参数

  • x -- 这是浮点值。

返回值

这个函数返回不小于x的最小整数值。

例子

下面的例子显示ceil()函数的用法。

#include <stdio.h>
#include <math.h>

int main ()
{
   float val1, val2, val3, val4;

   val1 = 1.6;
   val2 = 1.2;
   val3 = 2.8;
   val4 = 2.3;

   printf ("value1 = %.1lf
", ceil(val1));
   printf ("value2 = %.1lf
", ceil(val2));
   printf ("value3 = %.1lf
", ceil(val3));
   printf ("value4 = %.1lf
", ceil(val4));
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

value1 = 2.0
value2 = 2.0
value3 = 3.0
value4 = 3.0

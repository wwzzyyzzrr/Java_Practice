mblen() - C语言库函数

C库函数 int mblen(const char *str, size_t n) 返回参数str指向多字节字符的长度。

声明

以下是 mblen() 函数的声明。

int mblen(const char *str, size_t n)

参数

  • str -- 这是多字节字符的第一个字节的指针。

  • n -- 这是要检查的字符长度的最大数目的字节。

返回值

mblen()函数返回解析多字节序列由str开始，如果一个非空宽字符被确认的字节数。它返回0，如果一个空宽字符被确认。返回-1，如果遇到无效的多字节序列，或者如果它不能解析一个完整的多字节字符。

例子

下面的例子显示 mblen() 函数的用法。

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main()
{
   int len;
   char *pmbnull  = NULL;
   char *pmb = (char *)malloc( MB_CUR_MAX );
   wchar_t *pwc = L"Hi";
   wchar_t *pwcs = (wchar_t *)malloc( sizeof( wchar_t ));

   printf("Converting to multibyte string
");
   len = wcstombs( pmb, pwc, MB_CUR_MAX);
   printf("Characters converted %d
", len);
   printf("Hex value of first multibyte character: %#.4x
", pmb);
   
   len = mblen( pmb, MB_CUR_MAX );
   printf( "Length in bytes of multibyte character %x: %u

   len = mblen( pmb, MB_CUR_MAX );
   printf( "Length in bytes of multibyte character %x: %u
", pmb, len );
   
   return(0);
}

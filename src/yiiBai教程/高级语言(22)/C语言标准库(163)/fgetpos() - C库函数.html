fgetpos() - C库函数

C库函数 int fgetpos(FILE *stream, fpos_t *pos) 获取当前文件位置的数据流，并将其写入到 pos。

声明

以下是声明的fgetpos()函数。

int fgetpos(FILE *stream, fpos_t *pos)

参数

  • stream -- 这是一个文件对象的标识流的指针。

  • pos -- 这就是指针一个为fpos_t的对象。

返回值

该函数成功返回零，否则在错误的情况下返回非零值。

例子

下面的例子显示 fgetpos()函数的用法。

#include <stdio.h>

int main ()
{
   FILE *fp;
   fpos_t position;

   fp = fopen("file.txt","w+");
   fgetpos(fp, &position);
   fputs("Hello, World!", fp);
  
   fsetpos(fp, &position);
   fputs("This is going to override previous content", fp);
   fclose(fp);
   
   return(0);
}

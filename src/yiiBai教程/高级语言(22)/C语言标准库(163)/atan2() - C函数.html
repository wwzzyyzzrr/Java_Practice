atan2() - C函数

C库函数double atan2(doubly y, double x) 返回y / x的弧度圆弧切线，根据这两个值，以确定正确的象限上的标志。 

声明

以下是atan2() 函数的声明。

double atan2(doubly y, double x)

参数

  • x -- 这是浮点值，表示x坐标。

  • y -- 这是浮点值同比y坐标。

返回值

这个函数返回的主要y / x的反正切，在区间 [-pi,+pi] 弧度.

实例

下面的例子显示atan2()函数的用法。

#include <stdio.h>
#include <math.h>

#define PI 3.14159265

int main ()
{
   double x, y, ret, val;

   x = -7.0;
   y = 7.0;
   val = 180.0 / PI;

   ret = atan2 (y,x) * val;
   printf("The arc tangent of x = %lf, y = %lf ", x, y);
   printf("is %lf degrees
", ret);
  
   return(0);
}

The arc tangent of x = -7.000000, y = 7.000000 is 135.000000 degrees

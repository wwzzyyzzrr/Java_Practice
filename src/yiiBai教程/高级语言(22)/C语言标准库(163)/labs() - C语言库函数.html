labs() - C语言库函数

C库函数long int labs(long int x)返回x的绝对值。  

声明

以下是声明labs()函数。

long int labs(long int x)

参数

  • x -- 这是整数值。

返回值

这个函数返回x的绝对值x.

例子

下面的例子显示使用 labs() 函数。

#include <stdio.h>
#include <stdlib.h>

int main ()
{
   long int a,b;

   a = labs(65987L);
   printf("Value of a = %ld
", a);

   b = labs(-1005090L);
   printf("Value of b = %ld
", b);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

Value of a = 65987
Value of b = 1005090

ldiv() - C语言库函数

C库函数 div_t div(long int numer, long int denom) 除以numer (numerator) by denom (denominator).

声明

以下是 ldiv() 函数的声明。

div_t div(long int numer, long int denom)

参数

  • numer -- 这是分子。

  • denom -- 这是分母。

返回值

这个函数返回值定义的结构在<cstdlib>，其中有两个成员。用于 ldiv_t:long quot; long rem;

例子

下面的例子显示ldiv() 函数的用法。

#include <stdio.h>
#include <stdlib.h>

int main ()
{
   ldiv_t output;

   output = ldiv(100000L, 30000L);

   printf("Quotient = %ld
", output.quot);

   printf("Remainder = %ld
", output.rem);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

Quotient = 3
Remainder = 10000

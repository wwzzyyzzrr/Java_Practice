ctime() - C语言库函数

C库函数 char *ctime(const time_t *timer) 返回一个字符串，它表示 localtime 基于的参数 timer。

返回的字符串的格式如下： Www Mmm dd hh:mm:ss yyyy ，Www 为周一至周五, Mmm 当月字母, dd 月份中的一天， hh:mm:ss 是时间, yyyy 为年份.

声明

以下是声明的ctime() 函数。

char *ctime(const time_t *timer)

参数

  • timer -- 这是一个time_t对象，它包含一个日历时间的指针。

返回值

这个函数返回一个字符串，包含人类可读的格式的日期和时间信息。

例子

下面的例子显示ctime() 函数的用法。

#include <stdio.h>
#include <time.h>

int main ()
{
   time_t curtime;

   time(&curtime);

   printf("Current time = %s", ctime(&curtime));

   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

Current time = Mon Aug 13 08:23:14 2012

puts() - C语言库函数

C语言库函数 int puts(const char *str) 将一个字符串写入到stdout，但不包括空字符。一个换行符被追加到输出。

声明

以下是 puts() 函数的声明。

int puts(const char *str)

参数

  • str -- 这是C字符串的写入。

返回值

如果成功，返回非负值。错误则函数返回EOF。

例子

下面的例子显示使用puts()  函数。

#include <stdio.h>
#include <string.h>

int main()
{
   char str1[15];
   char str2[15];

   strcpy(str1, "yiibai");
   strcpy(str2, "compileonline");

   puts(str1);
   puts(str2);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

yiibai
compileonline

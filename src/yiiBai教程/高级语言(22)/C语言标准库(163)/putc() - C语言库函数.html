putc() - C语言库函数

C语言库函数 int putc(int char, FILE *stream) 写入字符到指定的流，并将流的指针位置的参数指定一个字符（unsigned char类型）。

声明

以下是 putc() 函数的声明。

int putc(int char, FILE *stream)

参数

  • char -- 这是要写入的字符。字符通过作为整型转换。

  • stream -- 这是一个文件对象，确定要写入字符流的指针。

返回值

这个函数返回写入的字符为unsigned char转换为int或EOF错误。

例子

下面的例子显示putc() 函数的用法。

#include <stdio.h>

int main ()
{
   FILE *fp;
   int ch;

   fp = fopen("file.txt", "w");
   for( ch = 33 ; ch <= 100; ch++ ) 
   {
      putc(ch, fp);
   }
   fclose(fp);
   
   return(0);
}

让我们编译和运行上面的程序，这将在当前目录下创建一个文件file.txt，这将有以下内容：

!"#$%&'()*+,-./0123456789:;<=>?@ABCDEFGHIJKLMNOPQRSTUVWXYZ[]^_`abcd

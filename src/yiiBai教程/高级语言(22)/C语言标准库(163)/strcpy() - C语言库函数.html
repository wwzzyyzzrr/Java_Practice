strcpy() - C语言库函数

C库函数char *strcpy(char *dest, const char *src)  复制src指向的字符串到dest。

声明

以下是声明的strcpy() 函数。

char *strcpy(char *dest, const char *src)

参数

  • dest -- 这就是指针的内容将被复制到目标数组。

  • src -- 这是要复制的字符串。

返回值

这返回一个指向目标字符串dest

例子

下面的例子显示strcpy() 函数的用法。

#include <stdio.h>
#include <string.h>

int main()
{
   char src[40];
   char dest[100];
  
   memset(dest, '\0', sizeof(dest));
   strcpy(src, "This is yiibai.com");
   strcpy(dest, src);

   printf("Final copied string : %s
", dest);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

Final copied string : This is yiibai.com

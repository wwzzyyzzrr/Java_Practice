abs() - C语言库函数

C库函数int abs(int x) 返回x的绝对值。

声明

以下是 abs() 函数的声明。

int abs(int x)

参数

  • x -- 这是整数值。

返回值

这个函数返回x的绝对值。

例子

下面的例子演示了如何使用 abs() 函数。

#include <stdio.h>
#include <stdlib.h>

int main ()
{
   int a, b;

   a = abs(5);
   printf("value of a = %d
", a);

   b = abs(-10);
   printf("value of b = %d
", b);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

value of a = 5
value of b = 10

fputc() - C语言库函数

C语言库函数 int fputc(int char, FILE *stream) 写入字符到指定的流，并将流的位置指示器的参数指定一个字符（unsigned char类型）。

声明

以下是声明的 fputc() 函数。

int fputc(int char, FILE *stream)

参数

  • char -- 这是要写入的字符。这是通过作为整型转变。

  • stream -- 这是一个文件对象，确定要写入字符流的指针。

返回值

如果没有错误，已经写入相同的字符被返回。如果出现错误，则返回EOF并设置错误指示灯。

实例

下面的例子演示了如何使用 fputc() 函数。

#include <stdio.h>

int main ()
{
   FILE *fp;
   int ch;

   fp = fopen("file.txt", "w+");
   for( ch = 33 ; ch <= 100; ch++ )
   {
      fputc(ch, fp);
   }
   fclose(fp);

   return(0);
}

让我们编译和运行上面的程序，这将在当前目录下创建一个文件file.txt，这将有以下内容：

!"#$%&'()*+,-./0123456789:;<=>?@ABCDEFGHIJKLMNOPQRSTUVWXYZ[]^_`abcd

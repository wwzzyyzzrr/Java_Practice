log() - C函数

C库函数double log(double x) 返回x的自然对数（基-E对数）。

声明

以下是log()函数的声明。

double log(double x)

参数

  • x -- 这是浮点值。

返回值

这个函数返回自然对数。

实例

下面的例子演示了如何使用log()函数。

#include <stdio.h>
#include <math.h>

int main ()
{
   double x, ret;
   x = 2.7;

   /* finding log(2.7) */
   ret = log(x);
   printf("log(%lf) = %lf", x, ret);
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

log(2.700000) = 0.993252

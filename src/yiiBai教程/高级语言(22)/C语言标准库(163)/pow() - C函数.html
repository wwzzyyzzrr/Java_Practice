pow() - C函数

C库函数double pow(double x, double y)返回x的power提高到y即 i.e. xy.

声明

以下是pow()函数的声明。

double pow(double x, double y)

参数

  • x -- 这是值浮点为底。

  • y -- 这就是浮动的点的power值。

返回值

这个函数返回x的y次方的结果。

例子

下面的例子演示了如何使用pow()函数。 

#include <stdio.h>
#include <math.h>

int main ()
{
   printf("Value 8.0 ^ 3 = %lf
", pow(8.0, 3));

   printf("Value 3.05 ^ 1.98 = %lf", pow(3.05, 1.98));
   
   return(0);
}

让我们编译和运行上面的程序，这将产生以下结果：

Value 8.0 ^ 3 = 512.000000
Value 3.05 ^ 1.98 = 9.097324

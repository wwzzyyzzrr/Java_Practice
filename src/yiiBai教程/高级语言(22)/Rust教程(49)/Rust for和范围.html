Rust for和范围

for构建了一个可用于迭代的Iterator，一个懒惰的值生成器(后文详述)。 一个来创建一个迭代的最简单的方法是使用范围表示法。 a..b 将从a(含)产生值到b(不含)的一个递增步骤。

让我们使用for写一个FizzBuzz，来代替while。

 

fn main() {
    // `n` will take the values: 1, 2, ..., 100 in each iteration
    for n in 1..101 {
        if n % 15 == 0 {
            println!("fizzbuzz");
        } else if n % 3 == 0 {
            println!("fizz");
        } else if n % 5 == 0 {
            println!("buzz");
        } else {
            println!("{}", n);
        }
    }
}

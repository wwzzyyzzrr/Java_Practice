D语言do...while循环

与for和while循环不同，它们都是在循环的顶部测试循环条件，而D编程语言的do ... while循环中检查其状态放在循环的底部。

do ... while循环类似于while循环，不同的是一个do ... while循环是保证至少执行一次。

语法

D编程语言一个do...while的循环语法是：

do
{
   statement(s);

}while( condition );

请注意，条件表达式出现在循环的结尾，所以在循环语句执行一次之前的测试条件。

如果条件为true，控制流跳回起来执行，并且在循环语句再次执行。此过程重复进行，直到给定的条件为false。

例子：

import std.stdio;

int main ()
{
   /* local variable definition */
   int a = 10;

   /* do loop execution */
   do
   {
       writefln("value of a: %d", a);
       a = a + 1;
   }while( a < 20 );
 
   return 0;
}

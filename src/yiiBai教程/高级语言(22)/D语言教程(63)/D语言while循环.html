D语言while循环

while循环在D编程语言循环语句多次，只要给定的条件为真时，可执行一个目标语句多次。

语法

在D编程语言中的while循环的语法是：

while(condition)
{
   statement(s);
}

在这里，statement(s)可以是单个语句或语句块。该条件condition可以是任何表达式，并且真正是任何非零值。循环迭代，当条件为真。

当条件为假，程序控制传递给行紧跟在循环后。

流程图：

在这里，while循环的关键点是循环可能不会永远运行。当条件进行测试，结果为假，循环体将被跳过，在while循环之后的第一个语句将被执行。

例如：

import std.stdio;
 
int main ()
{
   /* local variable definition */
   int a = 10;

   /* while loop execution */
   while( a < 20 )
   {
      writefln("value of a: %d", a);
      a++;
   }
 
   return 0;
}

让我们编译和运行上面的程序，这将产生以下结果：

value of a: 10
value of a: 11
value of a: 12
value of a: 13
value of a: 14
value of a: 15
value of a: 16
value of a: 17
value of a: 18
value of a: 19

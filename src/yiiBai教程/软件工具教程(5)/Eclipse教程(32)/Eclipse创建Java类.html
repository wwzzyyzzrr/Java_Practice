Eclipse创建Java类

打开新建Java类向导

可以使用“新建Java类”向导来创建Java类。Java类向导可以以不同的方式调用 -

  • 通过单击文件菜单并选择： New → Class。
  • 右键单击软件包资源管理器并选择：New → Class。
  • 通过点击类下拉按钮(下拉按钮)和选择类 ()。

在启动 New Java Class 向导之前，如果可能，请选择要在其中创建类的包，以便向导可以自动填充包名称。创建类请参考下图 -

使用新的Java类向导

当java类向导出现 -

  • 确保源文件夹和包是否正确。
  • 输入类的名称。
  • 选择适当的类修饰符。
  • 输入超类名称或单击“浏览(Browse)”按钮以搜索现有类。
  • 单击添加(Add)按钮以选择此类实现的接口。
  • 检查并修改方法存根和注释相关的复选框。
  • 单击完成(Finish)按钮。

查看新创建的Java类

新创建的类应该出现在包资源管理器视图和一个用来修改新类的java编辑器实例。它应该出现在编辑器区域。

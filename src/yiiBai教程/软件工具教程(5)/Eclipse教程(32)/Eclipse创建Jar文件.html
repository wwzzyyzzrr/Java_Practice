Eclipse创建Jar文件

打开Jar文件向导

Jar文件向导可用于将项目的内容导出到jar文件中。打开Jar文件向导 -

  • 在包资源管理器中选择要导出的项目。 如果要导出项目中的所有类和资源，只需选择项目。
  • 单击 文件(File) 菜单，然后选择 导出(Export) 。
  • 在导出向导第一页的过滤器文本框中键入JAR。
  • 在Java类别下选择JAR文件。
  • 单击下一步(Next)。

右键项目名称，在弹出的菜单中选择： Export，如下图所示 -

选择 JAR file -

使用Jar文件向导

在JAR文件指定页面 -

  • 输入JAR文件名和文件夹。
  • 默认是只导出类。如果也要导出源代码，请单击 导出Java源文件和资源 复选框。
  • 单击下一步(Next)以更改JAR打包选项。
  • 单击下一步(Next)更改JAR Manifest规范。
  • 单击完成(Finish)。

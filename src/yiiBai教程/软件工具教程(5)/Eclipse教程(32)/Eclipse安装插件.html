Eclipse安装插件

查找和安装插件
Eclipse平台为Eclipse IDE提供基础，并由一些插件组合而成，旨在使用其他插件进行扩展。

一共有几百个插件可用。每个插件向Eclipse添加更多功能。可以通过搜索Eclipse Market place(http://marketplace.eclipse.org/) 找到一个提供某些功能的插件。 例如，可以找到允许 Eclipse 用作Python IDE的插件。

从Eclipse IDE中，可以使用Eclipse Marketplace对话框搜索市场查找插件，可以通过单击帮助(Help)菜单，然后在弹出的子菜单中选择Eclipse Marketplace来调用该对话框。要安装 “Python For Eclipse”，请参考下图 -

在上面的截图中，要为Eclipse安装PyDev-Python IDE，只需点击安装(Install)按钮。在出现的对话框中直接安装插件即可。

也可以使用从帮助菜单(Help → Install New Software)访问的安装新软件菜单项安装插件。

在这种情况下，必须知道可以从其下载插件的远程站点的URL。可以通过单击添加按钮提供此URL。

安装对话框显示远程站点中可用的插件组件，并允许您安装它们。

然后进入下一步，然后同意安装协议之类的就可以了。

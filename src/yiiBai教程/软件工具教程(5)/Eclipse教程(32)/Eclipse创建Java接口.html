Eclipse创建Java接口

打开新的Java接口向导

新建Java接口向导可用于创建新的Java接口。有很多方法可以打开这个向导 -

  1. 点击文件菜单并选择 New → Interface。
  2. 在软件包浏览器中右键单击并选择 New → Interface。
  3. 单击工具栏中的类下拉按钮(Java Interface Class)并选择 Interface(接口按钮)。

在启动New Java Interface向导之前，如果可能，请选择要在其中创建接口的包，以便向导可以自动为您填写包名称。

使用新的Java接口向导

当java 接口向导出现后，需要确定以下几项内容 -

  • 确保源文件夹和包是否正确。
  • 输入接口名称。
  • 单击添加(Add)按钮选择扩展接口。
  • 如果希望生成注释，请选中生成注释复选框。
  • 单击完成(Finish)按钮。

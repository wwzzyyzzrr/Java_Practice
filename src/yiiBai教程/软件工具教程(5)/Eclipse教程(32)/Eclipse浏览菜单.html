Eclipse浏览菜单

典型的Eclipse菜单

Eclipse窗口的菜单栏上提供的典型菜单是 -

  • 文件菜单
  • 编辑菜单
  • 导航菜单
  • 搜索菜单
  • 项目菜单
  • 运行菜单
  • 窗口菜单
  • 帮助菜单

插件(Plug-ins)可以添加新菜单和菜单项。 例如，当java编辑器打开时将会看到Source菜单，当XML编辑器打开时将会看到Design菜单。

编号 菜单名称 描述
1 File 文件菜单允许打开文件进行编辑，关闭编辑器，保存编辑器内容和重命名文件。 除此之外，它还允许导入和导出工作区内容和关闭Eclipse。
2 Edit 编辑菜单显示复制和粘贴等项目。
3 Source 仅当打开java编辑器时，“Source”菜单才可见。它提供了一些与编辑java源代码相关的有用的菜单项。
4 Navigate 导航菜单用来快速查找资源并导航到它们。
5 Search “Search”菜单显示允许在工作空间中搜索包含特定数据的文件的项目。
6 Project 与构建项目相关的菜单项可以在项目菜单上找到。
7 Run 运行菜单上的菜单项允许在运行模式或调试模式下启动程序。 它还提供了菜单项，用于调试代码。
8 Window 窗口菜单可用来打开和关闭视图和透视图。它还允许您调出“首选项”对话框。
9 Help 帮助菜单可用于显示帮助窗口，Eclipse Marketplace视图或安装新插件。 关于Eclipse菜单项提供了版本信息。

自定义菜单

菜单上的可见菜单项取决于使用自定义透视框完成的已安装插件和自定义。

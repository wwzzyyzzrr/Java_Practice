Eclipse创建Java包

打开“新建Java包”向导

可以使用“新建Java包”向导创建Java包。Java包向导可以以不同的方式打开 -

  • 通过单击文件菜单，然后选择 新建→软件包(New → Package)。
  • 通过右键单击软件包资源管理器中，选择 新建→软件包(New → Package)。
  • 通过单击工具栏中的包图标( ).

如果要创建子包，请在打开Java包向导之前选择父包，以便名称字段有一个默认值。

使用新的Java包向导

当Java包向导出现 -

  • 输入/确认源文件夹名称。
  • 输入包名称。
  • 单击完成(Finish)按钮。

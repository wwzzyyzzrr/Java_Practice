Eclipse Java构建路径

设置Java构建路径

在编译Java项目以发现依赖类时，将使用Java构建路径。它由以下项目组成 -

  • 源文件夹中的代码。
  • Jars和类文件夹与项目相关联。
  • 由此项目引用的项目导出的类和库。

可以通过使用Java项目 属性 对话框的Java构建路径页面来查看和修改Java构建路径。

要打开Java项目属性对话框，右键单击包资源管理器(Package Explorer)视图中的Java项目，然后选择 属性 菜单项。 在左侧树中选择 Java Build Path。

开发Java应用程序时遇到的一个常见要求是将现有jar添加到java构建路径，比如要连接数据库的 JDBC Jar 包。 这可以使用 Libraries 选项卡完成。 在 Libraries 选项卡中，如果 jar 已经在 Eclipse 工作区中了，则单击 Add JARs，或者如果该 jar 位于文件系统中的其他位置，则单击 Add External JARs。

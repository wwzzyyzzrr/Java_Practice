版本控制集成(Git,SVN)

IntelliJ IDEA支持Git，CVS，Mercurial，Perforce及其它等等。这起决于您的 版本控制使用，在文件中指定找开 File | Settings | Version Control。这可能增加当前项目或现有项目作为新的VCS根。

除了一般的VCS设置，在这里您可以配置所有对应于要使用指定的VCS。

IDE允许您添加，删除，提交，回复，管理分支，看历史，推，拉，以及很多其他的事情。有些操作是通过 版本控制工具： VCS | Show Changes View 配置使用。

其他的可以从VCS菜单的相应点来调用。

要了解更多关于VCS也可查看阅读IntelliJ IDEA的版本控制和观看版本控制的视频教程。

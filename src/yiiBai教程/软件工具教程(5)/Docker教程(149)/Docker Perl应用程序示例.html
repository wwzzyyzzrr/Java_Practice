Docker Perl应用程序示例

在这个例子中，将演示如何创建一个Perl应用程序并使用Docker执行，此示例包括以下步骤。

1.创建一个目录用来组织应用程序文件

yiibai@ubuntu:~$ mkdir -p /home/yiibai/docker/perl-docker-app

2.更改进入工作目录

yiibai@ubuntu:~$ cd /home/yiibai/docker/perl-docker-app

3.创建一个 Dockerfile 文件

FROM perl:5.20  
COPY . /var/www/php  
WORKDIR /var/www/php  
CMD [ "perl", "index.pl"]

4.创建一个Perl文件: index.pl

print "Hello from perl file\n";

5.创建Docker映像

现在，创建这个Scala应用程序的Docker映像。 以下命令用于创建Docker映像。

yiibai@ubuntu:~/docker/perl-docker-app$ docker build -t perl-app .

6.运行Scala Docker映像

yiibai@ubuntu:~/docker/perl-docker-app$ sudo docker run perl-app

运行docker映像后，执行perl脚本，并在控制台屏幕中打印一条消息，如下所示 -

yiibai@ubuntu:~/docker/perl-docker-app$ sudo docker run perl-app
Hello from perl file

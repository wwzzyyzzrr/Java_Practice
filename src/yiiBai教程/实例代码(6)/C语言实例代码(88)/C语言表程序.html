C语言表程序

使用C编程语言显示表或多或少与计数相似。我们仅使用一次迭代，并将其与要打印的表的值进行比较。

实现

现在我们看看程序的实际实现 -

#include <stdio.h>

int main() {
   int i, j, n;

   n = 3;
   j = 1;

   for(i = n; i <= (n*10); i+=n) {
      printf("%3d  x %2d  =  %3d\n", n, j, i);
      j++;
   }

   return 0;
}

执行上面示例代码，得到以下结果 -

  3  x  2  =    6
  3  x  3  =    9
  3  x  4  =   12
  3  x  5  =   15
  3  x  6  =   18
  3  x  7  =   21
  3  x  8  =   24
  3  x  9  =   27
  3  x 10  =   30

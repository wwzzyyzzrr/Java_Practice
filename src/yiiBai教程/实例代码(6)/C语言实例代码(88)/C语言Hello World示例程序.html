C语言Hello World示例程序

大多数开发者学习编程语言时，一般都会从“Hello World”代码开始。此程序在执行时打印“Hello World”。这个简单的例子演示如何编写C语言程序，以及了解它构造和执行。

首先打开Visual Studio 2015，创建一个名称为：helloworld的Win32 Console Application 的工程，如下图所示 -

点击“OK”，进入下一步，如下图所示 -

在新弹出的对话框中，在 Empty project 项上打上勾，去除选中 Security Development lifecycle(SDL) checks， 然后点击Finished， 如下图所示 -

现在创建工程已经完成了，点击主菜单“View” -> Solution Explorer，调出界面如下 -

在“Source Files”目录下创建一个源文件：helloworld.c，其代码实现如下 - 

#include <stdio.h>

int main() {
   printf("Hello World!\n");
   return 0;
}

现在整个工作区看起来如下所示 -

如何执行上面代码？

点击主菜单Debugs->Start Without Debugging，运行上面helloworld.c代码，得到 以下结果 -

注：这个程序本身很简单，就打印输出一句话，但是在这里我们演示了如何使用 Visual Studio 来创建C语言应用程序工程，编写代码，编译和执行程序。多说一句，编程需要多上机编写编译代码，只有通过亲自编写编译运行代码才真正体会到其中的问题和乐趣。

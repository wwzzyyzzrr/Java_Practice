JSP入门程序

在这一节中，我们来学习如何使用Eclipse创建一个JSP项目。创建JSP项目的工具有很多，常用主流的IDE工具有：

  • MyEclipse
  • Eclipse(Eclipse Jee Oxygen)
  • NetBeans

这里使用Eclipse创建一个简单的JSP项目，并发布到使用Eclipse发布到Tomcat容器中运行。

要在Eclipse IDE中创建JSP并使用Tomcat服务器运行，请参考以下几个步骤：

  • 创建动态Web项目
  • 创建一个jsp页面
  • 启动tomcat服务器并部署项目
  • 测试运行结果

1.创建动态Web项目

首先打开Eclipse，然后创建动态Web项目，请单击菜单文件(File) -> 新建(New) -> 动态Web项目(Dynameic Web Project) -> 填写项目名称(例如：helloworld) -> 完成(Finish)。

填写项目名称为：helloworld，然后下一步 -

配置项目的Java类编译目录，保持默认，点击下一步 -

配置Web模块设置，打勾选中生成web.xml部署描述符，点击完成(Finish) -

在Eclipse IDE中创建JSP文件

要创建一个jsp文件，请打开项目浏览(点击+图标) -> 右键单击WebContent -> New -> -> Other… -> Web -> jsp -> 填写jsp文件名(例如：index.jsp) -> 下一步 -> 完成。

选择：Web -> JSP File 如下图所示 -

下一步，填空文件名称：index.jsp ，如下图所示 -

现在创建JSP文件，下面我们来写一些代码简单的JSP代码。修改为如下 -

文件：index.jsp - 

<%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>第一个JSP应用程序</title>
</head>
<body>
    <div style="margin: auto; text-align: center;">
        <h2>第一个JSP应用程序</h2>
        <%
            String str = "JSP Hello World ~";
            out.print(str);
        %>
    </div>
</body>
</html>

启动服务器并部署项目

启动服务器并单步部署项目右键单击项目 -> 运行方式 -> 在服务器上运行 -> 选择tomcat server -> next -> addAll ->完成。

注意： 如果第一次使用Eclipse来编写JSP或动态网站项目，那么可能还没有配置过Tomcat服务器，请参考以下步骤来配置。

点击Eclipse上方的菜单：Windows -> References -> Server -> Runtime Environments，如下图所示 -

并点击右上方的Add，然后选择：Apache->Apache Tomcat v8.5 ，如下所示 -

下一步，点击 Browse… 选择Apache Tomcat的安装位置，如下图所示 -

这里安装的Tomcat v8.5是一个解压包(可通过： http://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/tomcat-8/v8.5.23/bin/apache-tomcat-8.5.23-windows-x64.zip 下载)，并解压到： D:\software\apache-tomcat-8.5.23 目录。

因此，配置选择的Apache Tomcat安装目录如下图所示 -

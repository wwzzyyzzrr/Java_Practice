JavaServer Faces提供丰富的组件库来定义应用程序的体系结构。

它包括以下内容：

丰富的用于指定用户界面组件的状态和行为的类：

  • 一种渲染模型，定义如何以各种方式呈现组件。
  • 一种定义如何将数据转换器注册到组件上的转换模型。
  • 一个定义如何处理组件事件的事件和监听器模型。
  • 一种验证模型，用于定义如何将验证器注册到组件上。

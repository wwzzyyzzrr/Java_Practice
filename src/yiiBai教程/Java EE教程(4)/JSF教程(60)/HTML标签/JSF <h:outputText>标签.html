JSF <h:outputText>标签

它用于渲染纯文本。 如果存在“styleClass”，“style”，“dir”或“lang”属性，则渲染“span”元素。 如果存在“styleClass”属性，则将其值作为“class”属性的值。

JSF <h:outputText> 标签示例1

<h:outputText value="hello"></h:outputText>

JSF将其作为纯文本呈现

hello

JSFJSF <h：outputText>标签示例2：

<h:outputText value="hello" lang="en" style="color: red"></h:outputText>

JSF将其呈现为HTML跨标签 - 

<span style="color: red" lang="en">hello</span>

JSF <h:outputText>标签属性

属性 描述
value 它保存此组件的当前值。
id 它是此组件的标识符，此ID必须是唯一的。您可以使用它来访问CSS和JS文件中的HTML元素。
style 它用于为组件应用CSS。
class 它给组件的类名，它用于从CSS和JS文件访问组件。
lang 它用于指定语言，它有助于使网页本地化。

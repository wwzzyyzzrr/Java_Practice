JSF输出脚本

h:outputScript标签呈现类型为“text/javascript”的“script”类型的HTML元素。此标签将外部JavaScript文件添加到JSF页面。

以下JSF标签 - 

<h:outputScript library="js" name="help.js" />

被渲染成以下HTML代码 - 

<script type="text/javascript" 
   src="/helloworld/javax.faces.resource/help.js.jsf?ln=js"></script>

示例

以下是文件：common.js 中的代码 - 

document.write('yiibai.com');

以下是文件：UserBean.java 中的代码 - 

package com.yiibai;

import javax.faces.bean.ManagedBean;
import javax.faces.bean.SessionScoped;

@ManagedBean(name="user")
@SessionScoped
public class UserBean{

}

以下是文件：index.xhtml 中的代码 - 

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" 
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml"   
      xmlns:h="http://java.sun.com/jsf/html">
    <h:head></h:head>
    <h:body>

      <h:outputScript library="js" name="common.js" target="head" />

    </h:body>

</html>

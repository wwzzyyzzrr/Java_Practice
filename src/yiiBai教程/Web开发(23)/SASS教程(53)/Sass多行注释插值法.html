Sass多行注释插值法

多行注释中插值解决在CSS结果。可以指定大括号中的变量或属性名称。

语法 

$var : "value";
/* multiline comments #{$var} */

示例

下面的例子演示了在SCSS文件多行注释使用插值： 
<html>
<head> <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> <title>SASS多行注释 - www.yiibai.com</title>
   <link rel="stylesheet" type="text/css" href="style.css"/>
</head>
<body>
  <h1>Welcome to YiiBai</h1>
  <p>This is an example for Interpolation in SASS.</p>
</body>
</html> 

接下来，创建文件 style.scss.

style.scss 

$version: "7.8";
/* Framework version for the generated CSS is #{$version}. */
可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件修改CSS： 
sass --watch C:\Ruby22-x64\style.scss:style.css
接着执行上面的命令，它会自动创建style.css文件，如下面的代码：

style.css 

/* Framework version for the generated CSS is 7.8. */

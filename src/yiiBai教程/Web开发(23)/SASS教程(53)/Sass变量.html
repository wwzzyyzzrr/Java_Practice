Sass变量

程序员使用变量来表示数据，例如,作为数值，字符或存储器地址。变量最重要性是，在整个样式表中的变量可重新使用所存储的值。

语法 

$variable_name : some value;
变量定义的美元符号($)，并使用分号(;)结束。

示例

下面的例子演示了SCSS文件中的变量的使用： 
<html>
<head>  <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> <title>变量 - www.yiibai.com</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
   <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
   <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
   <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
</head>
<body>
   <div class="container">
   <h1>SASS变量使用示例</h1>
   <p>Sass is an extension of CSS that adds power and elegance to the basic language.</p>
   </div>
</body>
</html>

接下来创建一个文件：style.scss.

style.scss 

$txtcolor:#008000;
$fontSize: 20px;

p{
   color:$txtcolor;
   font-size:$fontSize;
}
你可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件来修改CSS： 
sass --watch C:\Ruby22-x64\style.scss:style.css
接着执行上面的命令，它会自动创建style.css文件，下面的代码：

style.css 

p {
  color: #008000;
  font-size: 20px;
}

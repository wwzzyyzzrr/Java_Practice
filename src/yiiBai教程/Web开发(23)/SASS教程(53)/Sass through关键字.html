Sass through关键字

@for指令使用 through 关键字，包括<start> 和 <end>这两个值的范围

语法 

@for $var from <start> through <end>
语法简要说明如下。

  • $var: 它代表了变量的名称如 $i.

  • <start> 和 <end>: 这些SassScript表达式将返回整数。如果<start>大于 <end>则计数器变量递减，当<start>比小 <end>计数器变量会增加。

示例

下面的例子演示了使用@for指令及 through关键字： 
<html>
<head><meta charset="utf-8"> <title>Control Directives & Expressions</title>
   <link rel="stylesheet" type="text/css" href="style.css"/>
</head>
<body>
   <p class="p1">This is first line.</p>
   <p class="p2">This is second line.</p>
   <p class="p3">This is third line.</p>
   <p class="p4">This is fourth line.</p>
</body>
</html>
接下来，创建文件style.scss。

style.scss 

@for $i from 1 through 4 {
  .p#{$i} { padding-left : $i * 10px; }
}
可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件修改CSS： 
sass --watch C:\Ruby22-x64\style.scss:style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.p1 {
  padding-left: 10px; }

.p2 {
  padding-left: 20px; }

.p3 {
  padding-left: 30px; }

.p4 {
  padding-left: 40px; }

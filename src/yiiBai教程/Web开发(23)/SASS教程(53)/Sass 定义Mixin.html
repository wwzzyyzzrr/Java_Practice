Sass 定义Mixin

@mixin指令用于定义混入，它包括任选的变量和参数中的mixin名称后。

示例

下面的例子演示了在SCSS文件使用@mixin：

sample.html 

<html>
<head><meta charset="utf-8"> <title> Mixin示例-www.yiibai.com</title>
   <link rel="stylesheet" type="text/css" href="sample.css"/>
</head>
<body>
<div class="cont">
   <h1>Example using include</h1>
   <h3>Directive is used to define the Mixins, it includes variables and argument optionally.</h3>
</div>
</body>
</html>
接下来，创建文件 sample.scss。

sample.scss 

@mixin style {
.cont{
	color: #77C1EF;
    }
}
@include style;
可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件来修改CSS： 
sass --watch C:\Ruby22-x64\sample.scss:sample.css
接着执行上面的命令，它会自动创建 sample.css 文件，下面的代码：

sample.css 

.cont {
  color: #77C1EF;
  }

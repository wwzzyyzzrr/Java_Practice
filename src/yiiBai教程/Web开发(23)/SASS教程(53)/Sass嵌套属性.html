Sass嵌套属性

使用嵌套的属性就可以避免重写CSS多次。例如，使用字体使用一些属性，如font-family, font-size, font-weight and font-variant等命名空间。在正常的CSS每次写这些属性需要使用的命名空间。使用SASS嵌套属性命名空间只写一次。

示例

下面的例子演示使用SCSS文件嵌套属性： 
<html>
<head> <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> <title>内嵌属性 - www.yiibai.com</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
   <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
   <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
   <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
</head>
<body>
   <div class="container">
   <h1>内嵌属性示例</h1>
   <p class="line">SASS stands for Syntactically Awesome Stylesheet</p>
   </div>
</body>
</html>

接下来产生一个文件：style.scss.

style.scss 

.line {
  font: {
    family: Lucida Sans Unicode;
    size:20px;
    weight: bold;
    variant: small-caps;
  }
}
你可以告诉SASS看文件，并随时使用SASS下面的命令更新修改的CSS文件： 
sass --watch C:\Ruby22-x64\style.scss:style.css
接下来执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.line {
  Lucida Sans Unicode;
  font-size: 20px;
  font-weight: bold;
  font-variant: small-caps;
}

Sass函数

SASS支持通过提供一种使用普通的CSS函数语法，指定一些关键字参数使用函数。

语法 

p {
  color: hsl($hue: 0, $saturation: 50%, $lightness: 50%);
}
HSL代表色调，饱和度和亮度这是更直观的，用于通过使用饱和度和亮度创建套匹配的颜色。

  • hue: 它代表的颜色，如120红色，240绿色，290为柔和的紫等

  • saturation: 它是一个百分比值，增加颜色的饱和度。

  • lightness: 它是一个百分比值而减小颜色的亮度。

示例

下面的例子演示了SCSS文件中使用函数： 
<html>
<head><meta charset="UTF-8"> <title>Functions Example</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
   <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
   <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
   <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
</head>
<body>
   <div class="container">
   <h2>Example using Functions</h2>
   <p>SASS stands for Syntactically Awesome Stylesheet..</p>
   </div>
</body>
</html>
接下来，创建文件style.scss。

style.scss

使用它在SASS代码定义hsl函数在下面SCSS代码。 
p {
  color: hsl(290,60%,70%);
}
可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件修改CSS： 
sass --watch C:\Ruby22-x64\style.scss:style.css
接着执行上面的命令，它会自动创建style.css文件，下面的代码：

style.css 

p {
  color: #d185e0;
}

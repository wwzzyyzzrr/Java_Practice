Sass布尔运算符

可以通过使用与，或，非操作符来执行 Sass 脚本布尔运算。

示例

下面的例子演示了SCSS文件使用布尔操作： 
<html>
<head><meta charset="UTF-8"> <title>Boolean操作 -www.yiibai.com</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
   <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
   <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
   <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
</head>
<body>
   <h3>布尔操作示例</h3>
   <p class="bool">SASS stands for Syntactically Awesome Stylesheet..</p>
</body>
</html>
接下来，创建文件style.scss。

style.scss 

$age:20;
.bool {
    @if ($age > 10 and $age < 25) {
        color: green;
   }
}
可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件来修改CSS： 
sass --watch C:\Ruby22-x64\style.scss:style.css
接着执行上面的命令，它会自动创建 style.css 文件，如下面的代码：

style.css 

.bool {
  color: green;
}

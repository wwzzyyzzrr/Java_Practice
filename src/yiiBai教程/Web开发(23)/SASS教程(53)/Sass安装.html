Sass安装

本文提供安装Ruby过程说明，它用于执行SASS文件。

SASS系统要求

  • 操作系统 : 跨平台

  • 浏览器支持 : IE (IE 8+), Firefox, Google Chrome, Safari, Opera

  • 编程语言: Ruby

Ruby的安装

Step(1): 如果使用的是Linux系统，请打开URL链接：https://www.ruby-lang.org/en/downloads/, 会看到一个屏幕，如下所示：


如果是Window系统，请打开： http://rubyinstaller.org/downloads/ ，下载当前稳定版本的exe文件。界面如下所示：

Step(2): 接下来，在系统上运行来安装Ruby。

Step(3): 接下来，添加Ruby的bin文件夹到PATH用户变量和系统变量以使用 gem 命令工作。

Step(4): 在系统中打开命令提示符，然后输入以下命令行。 

gem source -a http://rubygems.org/  
gem sources -a http://ruby.taobao.org/ 
gem install sass

Step(5): 接下来，会看到屏幕下方SASS 安装成功提示(国内线路不稳定，需要多试几次)。

实例

下面是SASS的一个简单的例子，在C:\Ruby22-x64下面创建一个Hello.html的网页文件，内容如下。 
<html>
<head> <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> <title> 导入sass示例</title>
   <link rel="stylesheet" type="text/css" href="style.css"/>
</head>
<body>
   <h1>SASS简单示例</h1>
   <h3>Welcome to Yiibai Yiibai</h3>
</body>
</html> 

现在我们将创建文件style.scss，这是类似于CSS文件，有一个不同的是，它会被保存.scss为扩展名。.html 和 .scss 两个文件是在Ruby文件夹中创建的。可以保存您.scss文件放入文件夹 ruby\lib\sass\ 

h1{
   color: #AF80ED;
}

h3{
   color: #DE5E85;
}
可以告诉SASS文件，并随时使用下面的命令文件修改更新SASS的CSS： 
sass --watch C:\Ruby22-x64\style.scss:style.css


当运行上面的命令，它会自动创建style.css文件。无论何时更改SCSS文件，style.css文件将得到自动更新。
当运行上面的命令,style.css文件将产生下面的代码：

style.css 

h1 {
  color: #AF80ED;
  }
h3 {
  color: #DE5E85;
  }

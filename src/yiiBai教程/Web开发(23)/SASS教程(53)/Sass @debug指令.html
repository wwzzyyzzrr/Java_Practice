Sass @debug指令

@debug指令检测错误，并显示SassScript表达值到标准错误输出流。

示例

下面是保存扩展为.scss，这类似于CSS文件的样式表文件。

debug.scss 

$font-sizes: 10px + 20px;
$style: (
  color: #bdc3c7
);

.container{
  @debug $style;
  @debug $font-sizes;
}
可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件来修改CSS： 
sass --watch C:\Ruby22-x64\debug.scss:debug.css
当你运行上面的命令，它会自动创建debug.css文件。无论何时更改SCSS文件， debug.css 文件将得到自动更新。

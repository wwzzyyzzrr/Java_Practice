Sass包含mixin

@include 指令用于包含mixin到文档中。mixin名称取自可选的参数传递给它。由mixin定义的样式可以包含到当前的规则。

示例

下面的例子演示了使用，包含 mixin 在 SCSS 文件中：

sample.html 

<html>
<head><meta charset="utf-8"> <title>Sass Mixin 示例</title>
   <link rel="stylesheet" type="text/css" href="sample.css"/>
</head>
<body>
<div class="cont">
   <h2>Example using include</h2>
   <h3>Different Colors</h3>
   <ul>
   	<li>Red</li>
   	<li>Green</li>
   	<li>Blue</li>
   </ul>
  </div>
</body>
</html>
接下来，创建文件sample.scss。

sample.scss 

@mixin style {
.cont{
	background-color: #77C1EF;
	color: #ffffff;
    }
h3 {
	color: #ffffff;
	}
}
@include style;
可以告诉SASS监视文件，并随时使用下面的命令更新SASS文件来修改CSS： 
sass --watch C:\Ruby22-x64\sample.scss:sample.css
接着执行上面的命令，它会自动创建 sample.css 文件，下面的代码：

sample.css 

.cont {
  background-color: #77C1EF;
  color: #ffffff;
 }

h3 {
  color: #ffffff;
 }

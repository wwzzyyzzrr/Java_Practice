XML注释

本章介绍了如何XML注释在XML文档工作。 XML注释类似于HTML注释。这些评论添加为注释或行理解的XML代码的目的.

注释可以用于包括相关的链接信息和术语。它们是可见的，只有在源代码;未在XML代码。注释可以出现在XML代码的任何地方.

句法

XML注释有以下句法:

  <!-------Your comment----->

注释以<！ - 和结尾 - >。您可以添加文字说明作为字符之间意见。不能嵌套内另一个注释.

示例

下面的示例演示如何使用的XML文档注释:

<?xml version="1.0" encoding="UTF-8" ?>
<!---Students grades are uploaded by months---->
<class_list>
   <student>
      <name>Tanmay</name>
      <grade>A</grade>
   </student>
</class_list>

<! -  和  - >之间的任何文本字符被视为注释.

XML注释规则

需要以下规则应遵循的XML注释:

  • 注释可以XML声明之前不会出现。

  • 注释可以出现在文档中的任意位置.

  • 注释不能在属性值将出现.

  • 在另一个注释注释不能嵌套.

XML树结构

一个XML文档始终是描述性的。树状结构是通常被称为XML树和起着很容易地描述任何XML文档中起重要作用。

树状结构包含根(父)要素，子要素等。通过采用树状结构，你可以知道所有后续的分行及支行从根开始。解析开始于开始和向下移动一个元素的第一分支，并进一步向叶节点.

例子

下面的例子演示了简单的XML树形结构:

<?xml version="1.0"?>
<Company>
  <Employee>
      <FirstName>Tanmay</FirstName>
      <LastName>Patil</LastName>
      <ContactNo>1234567890</ContactNo>
      <Email>tanmaypatil@xyz.com</Email>
      <Address>
           <City>Bangalore</City>
           <State>Karnataka</State>
           <Zip>560212</Zip>
      </Address>
  </Employee>
</Company>

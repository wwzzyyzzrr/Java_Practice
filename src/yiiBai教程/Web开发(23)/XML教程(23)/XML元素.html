XML元素

XML元素可以被定义为XML的积木。元素可以表现为容器，以容纳文本，元素，属性，媒体对象或所有的这些.

每个XML文档包含一个或多个元素，其范围是任一分隔的由开始和结束标记，或者为空元素，用一个空元素标签.

语法

以下是XML元素语法:

<element-name attribute1 attribute2>
....content
</element-name>

那么

  • XML元素 是元素的名称 name 开始和结束标记必须匹配.

  • ATTRIBUTE1，attribute2 是由空格分隔的元素的属性。属性定义了元素的属性。它关联的名称与价值，这是字符的字符串。属性被写为：

    name = "value"

    名后跟一个等号(=)和双引号(“”)或单引号('')，引号内的字符串值。

空元素

空元素(没有内容的元素)的语法如下:

<name attribute1 attribute2.../>

使用各种XML元素的XML文档的例子:

<?xml version="1.0"?>
<contact-info>
   <address category="residence">
      <name>Tanmay Patil</name>
      <company>YiiBai</company>
      <phone>(011) 123-4567</phone>
   <address/>
</contact-info>

XML元素规则

需要以下规则应遵循的XML元素:

  • 元素名称可以包含任何字母数字字符。在名称中允许的唯一的标点符号是连字符( - )，在得分(_)和句点(。).

  • 名称是区分大小写的。例如，地址，地址，和地址是不同的名称.

  • 开始和结束标记必须必须相同。

  • 一个元素它是一个容器可以包含文本或元素在上述例子中看到的.

开始之前，需要了解或使用Windows或Unix。 Windows或Unix的工作知识，使得它更容易学习HTML。

你需要熟悉：

  • 如何创建目录和文件。

  • 如何通过不同的目录中导航。

  • 使用的浏览器，如Internet Explorer或Firefox等在互联网上浏览的基本认识

  • 开发简单的Web页面使用HTML或XHTML的基本理解。

如果不熟悉的HTML和XHTML的话，建议先通过HTML教程或XHTML教程。HTML或XHTML学习完成再继续学习本教程。

CSS是什么?

层叠样式表，如CSS，称为是一个简单的设计旨在简化制作网页的过程中的语言。

CSS处理的外表和感觉的网页的一部分。使用CSS，可以控制文字的颜色，字体的风格，段落之间的间距，列的大小和布局，背景颜色或图像的使用，以及各种其他效果。

CSS是很容易学习和理解，但它提供了一个HTML文档演示功能强大的控制权。最常用标记语言HTML或XHTML，CSS结合。

CSS的优点：

  • CSS节省时间 - 写CSS一次，然后重复使用同一张纸上多个HTML页面。可以定义每个HTML元素的样式，并将其应用到许多网页。

  • 页面加载速度更快 - 如果正在使用CSS，不需要每次写HTML标签的属性。只要写一个CSS规则的标签，并适用于所有出现该标记。所以更少的代码意味着更快的下载时间。

  • 易于维护 - 为了使全局变化，简单地改变风格，并会自动更新所有网页中的所有元素。

  • 高级样式到HTML - CSS有一个更广泛的阵列比HTML属性，所以可以远胜HTML页面的HTML属性。

  • 多种设备兼容性 - 允许对内容进行优化，以一个以上的移动设备不同的样式表。通过使用相同的HTML文档中，不同版本的网站，可以为手持设备，如PDA和手机或打印。

  • 全局Web标准 - 现在的HTML属性被弃用，并且它被推荐使用CSS。因此，它是一个好主意，开始所有的HTML页面中使用CSS以使其兼容未来的浏览器。

谁创造和维护CSS？

CSS是通过一组人 W3C 的 CSS工作组内创建和维护。 CSS工作组创建文件要求的规格。当规范已经讨论，并正式批准由W3C的成员，提供一些建议。

这些批准的规格被称为W3C的建议，因为有无法控制的实际执行的语言。独立的公司和组织创建该软件。

注：万维网联盟W3C是一组关于互联网如何工作的，以及应该如何发展提出建议。

CSS 版本:

层叠样式表级别1（CSS1）出来了W3C的建议在1996年12月。这个版本介绍CSS语言以及简单的可视化格式模型，所有的HTML标签。

CSS2成为W3C推荐的1998年5月的基础上CSS1。这个版本增加了对特定媒体的样式表的支持，例如打印机和听觉设备，可下载字体元素定位和表格。

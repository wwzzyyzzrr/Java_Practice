网站发布和开发工具

作为一个初学者，需要了解一个网站开发者开发网站必需的一些工具，了解这些工具的主要用途。

首先，应该有一个基本的互联网包，具有良好的上传速度，最低1Mbps。 其次，应该有一台电脑与任何操作系统(可以是Linux，iOS X或Windows)。 那么硬件规格应该是最小的 - 1-2 GB的RAM，处理器至少应该是一个双核心和硬盘必须在80 GB左右。

下一步是 - 应该有一个文本编辑器，可以编写代码。 一些网站开发最好的文本编辑器如下所述。

Text Sublime

Text Sublime可以从 - http://www.sublimetext.com/ 下载。 这个编辑器中的文本根据其功能改变了参数的颜色。

Notepad++

另一个流行的文本编辑器是Notepad++，可以从 - http://notepad-plus-plus.org/ 下载。

EditPad

在最受欢迎的文本编辑列表中的下一个是 - EditPad，这是另一个免费的工具。 它可以从 - http://www.editpadlite.com/ 下载。

注意 - 应该总是使用一个你感觉更舒适(习惯)的文本编辑器。

XAMPP＆LAMPP

如果您专门开发动态网页时，需要在计算机中安装另一个工具 - 用于Windows的XAMPP，或用于Linux系统的LAMPP。 它有各种产品，如

  • Web Apache
  • MariaDB与phpMyAdmin数据库管理
  • PHP程序支持
  • Perl程序支持。

对于想要编写ASP的开发人员，应该有Microsoft提供的Visual Studio。 它可以从 - http://www.microsoft.com/web/ 下载。

如果你是一个初学者，并且你打算开发一个平面(静态)网站，那么应该使用一个名为WYSIWYG的Visual HTML文本编辑器。 最常用的编辑器之一是:Macromedia Dreamweaver。 它可以很容易地从以下链接下载 - http://www.adobe.com/products/dreamweaver.html 。

因为开发好的网站程序还需要发布到服务器上，所以还需要的另一个工具是FTP客户端，用于将文件传输到服务器上。 一般来说，这些FTP客户端是免费的，可以很容易地从互联网上下载。

WinSCP 可以从 http://winscp.net/eng/docs/guide_install 下载。
FileZilla 可以从 http://filezilla-project.org/download.php 下载。

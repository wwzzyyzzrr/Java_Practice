1. 为什么要创建一个自己的网站

创建一个网站对于每个人都有自己的原因，假设你是互联网积极分子热衷于关注和研究互联网，创建一个自己的网站让你对互联网的理解更加深刻。对于热爱写文章和日志的朋友，那是一个与分享自己知识见识和生活点点滴滴的地方。对于玩自媒体的朋友，个人网站不仅能实现自我营销，还可以降低自我营销成本，下面分别列出创建一个自己的网站的好处和坏处。

1.1. 好处

  • 创建个人的一个网上名片。可以将自己的个人网站定义为自己的个人简历，传统简历不能非常完整地展示一个人的个人经历和工作案例，但是个人网站可以做到。

  • 自我营销。对于文学创作者、营销策划人员、新媒体自媒体工作者而言，个人网站十分必要。自媒体、新媒体平台是一旦账号被封或者被盗，之前的努力可能就白费了。个人网站就不会陷入这样的困境了。

  • 降低自我营销成本。在新媒体自媒体等平台的广告投放价格是十分昂贵的，即便是需要使用自己的域名，一整年的花销加起来也几百元。

  • 记录人生。除了前面那些比较功利的意义和作用，个人网站也是非常好的记录人生的工具。入住其它共公博客平台，也要承担平台关闭后数据遗失的风险。个人网站则很好地规避了这一点，并且能更好地保护好自己的数据和文章。

  • 挑战自我。尝试着自己去做个网站，其实也是对自己的一个挑战。很多人，包括文章开头那个小伙伴都觉得做网站是件非常困难的事情，我们可能不会写代码，可能不会做UI设计，甚至于我们都不知道什么是切图什么是HTML。但是只要你会使用网络搜索，做个网站早就没有那么难了。

1.2. 坏处

  • 费钱，至少要购买域名和服务器。
  • 费神，还需要懂一点网络或IT技术基础，有时服务被攻击或宕机时需要处理。
  • 费时，自己的网站或博客是要花时间不断地学文章内容的，并不是拿来摆设的。

相比于创建个人网站的好处以及创建个人网站的决心，这些不算什么事，并且也值得我们去折腾。

2. 定位

有关如何定位自己的个人网站，这得从创建个人网站的目的(初衷)。常见的一些个人网站的目的：

  • 记录日志，比如：小孩生长记录日志，宠物日志，花卉日志等等。
  • 网络文章写作，比如：连载小说，自媒体等
  • 专用于各种优惠码，打折返现等
  • 其他因人而异
  • ……

3.3 网站备案

根据工信部《互联网信息服务管理办法》(国务院 292 号令)，网站在未完成备案之前，不能指向大陆境内服务器开通访问。如果您的网站托管在中国大陆节点服务器，或者开通 CDN 服务，就必须申请 ICP(互联网内容提供商)备案。若网站服务器为非中国大陆节点，则不用申请备案。

ICP 包含两种类型：

1. ICP 备案： 对于单纯的资讯性并且不涉及直销的网站，需要申请 ICP 备案。ICP 备案可通过网站托管服务器提供商或网络空间服务商(如电信、移动、联通等)进行申请。阿里云 ICP 备案操作可分为：首次备案、接入备案、新增网站备案(原备案不在阿里云) 和 新增网站备案(原备案在阿里云)。注意：个人网站属于这一类型。

2. 经营性 ICP 许可证： 对于商品或服务的在线平台或第三方卖方，需获得商用许可证。要申请经营性 ICP 许可证，您必须先有一个有效的 ICP 备案号。经营性备案需用户自己到当地管理局相关部门进行办理，服务商可协助提供资质以及接入协议。

有关网站备案要，可通过：http://beian.aliyun.com/ 了解，需要注意的是：不同服务器提供商都有自己的备案接入。不可以跨服务商备案，也就是说：买了阿里的服务器不能在阿里云之外的服务商备案。

4.4 服务器环境安装

服务器的安装比较简单，这里主要介绍：Window系统和Linux系统服务器，对于那些不太熟悉IT技术的朋友，建议在购买服务器时选择Window系统，这样可以避开不必要的麻烦。而如果有些IT技术基础，又想了解和学习Linux系统的话，建议安装CentOS或Ubuntu。

Window安装PHP环境：WampServer等集成环境安装，搜索下载WampServer软件，安装到一个目录，比如：C:/wamp。

4.5 安装博客系统

用来做网站或个人博客的免费系统的系统实在太多了，这些系统经过多年的发展，都十分健壮和强大。下面介绍几种主流的免费博客系统。

  1. Wordpress ：当之无愧的王者，国内国外都有众多用户，插件，模板数量众多，seo方面做得也非常好。
  2. bo-blog：国内php技术的知名博客系统，有很多漂亮的中国化的模板。
  3. Z-BlogPHP：好一阵子没有更新系统代码，不影响使用。
  4. emlog : emlog是一款基于PHP和MySQL的功能强大的博客及CMS建站系统。致力于为您提供快速、稳定，且在使用上又极其简单、舒适的内容创作及站点搭建服务。

推荐： Wordpress，有很多用户，有什么问题都能比较容易找到解决办法。

安装很简单，下载系统的源代码，并解压代码文件到安装WampServer的安装目录(如前面所述的：C:/Wamp 目录)之下，通过在服务器上的浏览器访问：http://localhost 进行安装。

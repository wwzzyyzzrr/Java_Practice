HTML 是什么?

在开始学习之前:

开始之前，重要的是应该知道Windows或Unix。Windows或Unix的工作知识，使得它更容易学习HTML。

您应该熟悉:

  • 基本的文字处理，使用任何文本编辑器。

  • 如何创建目录和文件。

  • 如何通过不同的目录中导航。

  • 使用的浏览器，如Internet Explorer或Firefox等在互联网上浏览的基本认识

HTML简介：

HTML代表超文本标记语言，它是最广泛使用的语言写的Web页。正如它的名字所暗示的，HTML是一种标记语言。

  • 超文本是指到的网页（HTML文件）的方式，是连在一起的。当点击一个Web页面中的链接，使用的是超文本。

  • 标记语言描述HTML如何工作。一种标记语言，只需“标记”标签告诉Web浏览器如何构建它来显示一个文本文件。

最初，HTML开发的意图定义结构的文件，如标题，段落，列表和促进科学的研究人员之间的信息共享等等。

使用HTML需要做的是了解什么类型的标记来得到想要的结果。

创建HTML文档：

创建一个HTML文档是很简单的。要开始编写HTML代码，只需要两件事情：一个简单的文本编辑器和网页浏览器。记事本是最基本的简单的文本编辑器，可能会编写一个标准的HTML。

下面是简单的步骤来创建一个基本的HTML文档：

  • 打开记事本或其他文本编辑器。

  • 在顶部的页输入<html>.

  • 下一行，缩进五个空格，现在添加开放头标记：<head>.

  • 下一行，缩进十个空格，然后键入 <title> </title>.

  • 进入到下一行，缩进五个空格的边际和插入的闭合头标记： </head>.

  • 从下一行，边距的五个间隔键入<body>.

  • 现在下拉另一行并低于键入结束标记： </body>.

  • 最后，进入到下一行并键入</html>.

  • 在“文件”菜单上，选择“另存为”。

  • 在“保存类型”选项中，选择“所有文件”。

  • 将文件命名为 template.html.

  • 点击“保存”。

现在基本的HTML文档，看到一些结果把下面的代码在标题和正文标签。

<html>
<head>
<title>This is document title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>Document description goes here.....</p>
</body>
</html>

现在已经创建了一个HTML页面，可以使用Web浏览器打开这个HTML文件，看到的结果。希望理解网页是什么，但他们是简单的HTML文件，其中的一些内容，可以使用Web浏览器呈现。

这里 <html>, <head>,...<p>, <h1> 等. 它们叫 HTML 标签。 HTML标记HTML文档和构建模块在随后的章节中，我们将学习所有的HTML标签。

注：一个HTML文件可以扩展为.html 或 .htm。所以，可以使用其中之一。

HTML文档结构：

<html>和</html>标记HTML文档的开始和结束。这些标记告诉浏览器，整个文件是由HTML中。这两个标记内部，文档将被分为两部分：

  • <head>...</head> 元素，其中包含的信息的文件，如文件的标题，作者在此标签内的文件等信息不显示外面。

  • <body>...</body> 元素，其中包含的文件，在屏幕上看到的真实内容。

HTML标签和元素：

HTML语言是一种标记语言，我们使用了许多标签标记文本。在上面的例子中已经看到<html>，<body>等被称为HTML标记或HTML元素。

每个标签由标签名，有时还会置于打开和关闭括号（< 和 >）的标签属性，所有的可选列表。最简单的标签是什么比一个适当的名称括在括号中，，如<head>和<i>更多。更复杂的标签包含一个或多个属性，这些属性的标签的指定或修改的行为。

根据标准的HTML标签和属性名称是不区分大小写。有没有效果差异之间的<head>，<HEAD>，<HEAD>，甚至<HeaD>; 它们都是等效的。但与XHTML的情况是重要的：目前所有的标准标签和属性名称以小写。

HTML是宽松？

所有的浏览器有一个很好的处理，他们不会给出任何错误，如果还没有把任何HTML标签或属性正确。他们会忽略该标签或属性，将仅适用于正确的标签和属性，然后显示结果。

我们不能说，HTML是宽容的，因为这仅仅是一种标记语言，并要求格式文件。

WordPress垃圾邮件防护

在本章中，我们将学习如何保护WordPress博客或网站的垃圾邮件。请确保你的WordPress脚本更新为最新的稳定版本。WordPress预装了反垃圾邮件解决方案 - Akismet。

可以激活 Akismet，但应该有一个WordPress的API密钥。可以在 WordPress 官方网站进行注册。密钥会发送到你的邮箱。

要为您的网站或博客启用 Akismet 垃圾邮件防护插件，请按照下述的简单步骤。


步骤（1）：进入WordPress管理区 插件 -> 安装插件。会看到屏幕如下图所示。


步骤（2）：点击激活按钮，如下图所示：


步骤（3）：Akismet在插件将被启用，然后单击启用Akismet账户按钮，如下图所示：


步骤（4）：点击获取您的API密钥来获得一个新的密钥或手动输入，如果已经有API密钥表示如下。


步骤（5）：如果没有API密钥，然后，点击获得Akismet API密钥表进一步操作，如下图所示：


步骤（6）：填写了所需的字段，然后点击注册按钮，如下图所示：


步骤（7）：一旦完成注册过程。将在您注册的邮件ID获得API密钥。手动输入API密钥，然后点击使用此密钥按钮，如下图所示。


步骤（8）：如果输入了正确的API密钥，它会验证，会得到一个确认信息，如下图所示。


步骤（9）：从现在开始，你的博客使用Akismet防垃圾邮件功能。能够检查的评论，在您的博客的垃圾邮件，从博客管理区的垃圾邮件->评论手动标记评论。


可以保持跟踪有多少垃圾帖子已经被Akismet停止，可以保护自己的帖子，博客，评论等。此外，还可以防止垃圾邮件发送者，可能会损害您的网站。

WordPress重设密码

在本章中，我们将了解如何在WordPress中重置您的密码。

在这里，我们要在wordpress重新密码，如下所述的两种方法。

  • 用户

  • 忘记密码

让我们来看看用户部分。

步骤(1)：登录到你的 WordPress 管理面板仪表板，然后单击 用户 -> 所有用户。


步骤（2）：当你点击所有用户，得到的用户列表的页面。从这个选择要重设的密码，然后点击编辑。

当你点击编辑得到下面的页面。

现在，让我们看看在忘记密码部分重置密码。


步骤（3）：在这里可以输入您的新密码或生成包含7个字符。一旦完成了设置新密码，点击更新个人资料。

步骤（3.1）：当您尝试登录到你的WordPress管理面板，并且您忘记了密码，需要重新设置密码的页面上显示以下消息。


步骤（3.2）：在这里点击忘记密码？下面的页面时，更新您的电子邮件，以获得新的密码。


步骤（3.3）：当你点击已发送的电子邮件地址，得到一个消息链接用以重新设置密码。

步骤（3.4）：登陆并查看电子邮件，打开（WordPress站点）密码重置邮件，点击所提供的链接。

步骤（3.5）：输入新密码，再确认是否需要，然后点击重置密码。

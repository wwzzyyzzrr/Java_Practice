Wordpress小工具管理

在本章中，我们将学习有关Widget(小工具)管理。Widget是执行特定功能的小方块。这些是用来设计和结构控制 wordpress 主题。

  • 它们可以帮助您添加内容和功能。

  • 它们可以很容易地拖动并放置在小工具区域。

  • 小工具不同主题也不尽相同。它们在每个主题可能不相同。

步骤(1)：点击 外观 -> 小工具。


步骤（2）：当你点击这个你会得到一个窗口小工具页面。

在这里，我们有三种类型的部件和一个边栏，让我们来看看这些细节：

  • 可用小工具：可以使用这些添加到你的侧边栏。

  • 非活动侧栏（未使用）：这些不使用，可以从窗口小部件列表永久删除。

  • 非活动窗口小部件：删除小部件的侧边栏，但保持它的设置。

  • 边栏主：在此处添加任何部件将出现在网站上。

  • 管理在定制：回到定制页面。

步骤（3）：在主边栏拖放。在此处添加任何插件，显示在网站。

Javascript String.blink()方法

这个方法会导致一个字符串闪烁，好像它是BLINK 标签。

语法

string.blink( )


下面是参数的详细信息：

  • NA:

返回值:

  • 返回字符串带有<blink>标记。

例子：

<html>
<head>
<title>JavaScript String blink() Method</title>
</head>
<body>
<script type="text/javascript">

var str = new String("Hello world");

alert(str.blink());

</script>
</body>
</html>


这将产生以下结果：

<blink>Hello world</blink>

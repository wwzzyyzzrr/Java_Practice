Javascript Date.getHours()方法

javascript Date.getHours()方法按照本地时间在指定日期的返回小时。由 getHours 返回的值是0和23之间的整数。

语法

Date.getHours()

下面是参数的详细信息：

  • NA

返回值:

按照本地时间返回指定日期的小时。

例子：

<html>
<head>
<title>JavaScript getHours Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date("December 25, 1995 23:15:00");
   document.write("getHours() : " + dt.getHours() ); 
</script>
</body>
</html>

这将产生以下结果：

getHours() : 23

Javascript Date.getMonth()方法

javascript日期的getMonth()方法按照本地时间在指定日期返回的月份。通过得到月返回的值是0-11；0对应于一月，1对应2月，等等之间的整数。

语法

Date.getMonth()

下面是参数的详细信息：

  • NA

返回值:

按照本地时间返回指定日期的月份

例子：

<html>
<head>
<title>JavaScript getMonth Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date( "December 25, 1995 23:15:00" );
   document.write("getMonth() : " + dt.getMonth() ); 
</script>
</body>
</html>

这将产生以下结果：

getMonth() : 11

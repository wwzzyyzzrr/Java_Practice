Javascript Date.getMilliseconds()方法

javascript Date.getMilliseconds()方法按照本地时间返回指定日期的毫秒数。通过getMilliseconds返回的是一个介于0和999的值。

语法

Date.getMilliseconds()

下面是参数的详细信息：

  • NA

返回值:

按照本地时间返回指定日期的毫秒数。

例子：

<html>
<head>
<title>JavaScript getMilliseconds Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date( );
   document.write("getMilliseconds() : " + dt.getMilliseconds() ); 
</script>
</body>
</html>

这将产生以下结果：

getMilliseconds() : 578

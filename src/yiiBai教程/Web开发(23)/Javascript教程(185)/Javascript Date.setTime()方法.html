Javascript Date.setTime()方法

javascript Date.setTime()方法设置Date对象通过毫秒数自1970年1月1日00:00:00 UTC表示的时间。

语法

Date.setTime(timeValue)

注：括号内外的数据是可选的

下面是参数的详细信息：

  • timeValue : 表示自1970年1月00:00:00 UTC起的一个整数(毫秒数)。

返回值:

NA

例子：

<html>
<head>
<title>JavaScript setTime Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date( "Aug 28, 2008 23:30:00" );
   dt.setTime( 5000000 );
   document.write( dt ); 
</script>
</body>
</html>

这将产生以下结果：

Thu Jan 1 06:53:20 UTC+0530 1970

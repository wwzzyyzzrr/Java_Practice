Javascript Array.toSource()方法

JavaScript数组 toSource()方法返回表示数组的源代码的字符串。这种方法是通过Mozilla来支持。

语法

array.toSource();

下面是参数的详细信息：

  • NA

返回值:

返回表示该数组的源代码的字符串。

例子：

<html>
<head>
<title>JavaScript Array toSource Method</title>
</head>
<body>
<script type="text/javascript">
var arr = new Array("orange", "mango", "banana", "sugar");

var str = arr.toSource();
document.write("Returned string is : " + str );

</script>
</body>
</html>

这将产生以下结果：

Returned string is : ["orange", "mango", "banana", "sugar"]

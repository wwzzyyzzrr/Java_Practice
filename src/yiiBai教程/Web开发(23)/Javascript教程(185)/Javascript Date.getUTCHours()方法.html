Javascript Date.getUTCHours()方法

按照通用时间，javascript Date.getUTCHours()方法返回的时间在指定的日期。由getUTCHours返回的值是0和23之间的整数。

语法

Date.getUTCHours()

下面是参数的详细信息：

  • NA

返回值:

按照通用时间返回的时间在指定的日期。

例子：

下面的例子打印的当前时间变量hrs的时间部分。

<html>
<head>
<title>JavaScript getUTCHours Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date();
   document.write("getUTCHours() : " + dt.getUTCHours() ); 
</script>
</body>
</html>

这将产生以下结果为印度时区：

getUTCHours() : 14

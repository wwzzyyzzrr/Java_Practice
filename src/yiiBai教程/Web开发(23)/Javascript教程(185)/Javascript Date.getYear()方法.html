Javascript Date.getYear()方法

javascript Date.getYear()方法返回按照通用时间在指定日期的年份。getYear不再使用，已使用getFullYear方法取代。

通过了getYear返回的值是当前年份减去1900(在JavaScript1.2)及返回或者是2位或4位数字的年份，例如，如果年份是2026年，返回的值是2026所以测试此函数之前，请确保JavaScript的版本。

语法

Date.getYear()

这里是参数的细节:

  • NA

返回值:

按照通用时间返回指定日期的年份。

例子：

<html>
<head>
<title>JavaScript getYear Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date();
   document.write("getYear() : " + dt.getYear() ); 
</script>
</body>
</html>

这将产生以下结果2012年：

getYear() : 212

Javascript Date.constructor属性

javascript日期constructor属性返回一个引用到创建实例的prototype数组函数。

语法

date.constructor

下面是参数的详细信息：

  • NA

返回值:

返回创建此对象的实例的函数。

例子：

<html>
<head>
<title>JavaScript Date constructor Property</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date();
   document.write("dt.constructor is : " + dt.constructor); 
</script>
</body>
</html>

这将产生以下结果：

dt.constructor is: function Date() { [native code] }

Javascript Date.getUTCDate()方法

JavaScript Date.getUTCDate()方法按照通用时间在指定的日期返回月份的一天。通过getUTCDate返回的值是1到31之间的整数。

语法

Date.getUTCDate()

下面是参数的详细信息：

  • NA

返回值:

返回按照通用时间指定日期的月份的一天。

例子：

<html>
<head>
<title>JavaScript getUTCDate Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date( "December 25, 1995 23:15:20" );
   document.write("getUTCDate() : " + dt.getUTCDate() ); 
</script>
</body>
</html>

这将产生以下结果为印度时区：

getUTCDate() : 25

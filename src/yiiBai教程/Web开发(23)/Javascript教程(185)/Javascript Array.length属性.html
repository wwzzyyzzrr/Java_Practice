Javascript Array.length属性

JavaScript数组的length属性返回一个无符号，32位整数，指定数组中元素的个数。

语法

array.length


下面是参数的详细信息：

  • NA

返回值:

返回数组的长度

例子：

<html>
<head>
<title>JavaScript Array length Property</title>
</head>
<body>
<script type="text/javascript">
   var arr = new Array( 10, 20, 30 );
   document.write("arr.length is : " + arr.length); 
</script>
</body>
</html>


这将产生以下结果：

arr.length is : 3

Javascript Math.toSource()方法

这个方法返回字符串“Math”。但这种方法不能在许多浏览器使用，如IE浏览器。

语法

Math.toSource() ;

下面是参数的详细信息：

  • NA

返回值:

返回字符串 "Math".

例子：

<html>
<head>
<title>JavaScript Math toSource() Method</title>
</head>
<body>
<script type="text/javascript">

var value = Math.toSource( );
document.write("Value : " +  value ); 
  
</script>
</body>
</html>

这将产生以下结果：

Value : Math

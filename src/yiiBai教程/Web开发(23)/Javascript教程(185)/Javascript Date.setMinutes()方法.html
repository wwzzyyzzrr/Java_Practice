Javascript Date.setMinutes()方法

javascript Date.setMinutes()方法按照本地时间的设置分钟为一个指定日期

语法

Date.setMinutes(minutesValue[, secondsValue[, msValue]])

注：括号内外观和数据是可选的

下面是参数的详细信息：

  • minutesValue : 0到59之间的整数，表示分钟

  • secondsValue : 0到59之间的整数，表示秒。如果指定secondsValue参数，则还必须指定minutesValue

  • msValue : 一个数字0和999之间，表示毫秒。如果指定msValue参数，则还必须指定minutesValue和secondsValue

如果不指定secondsValue和msValue参数，从getSeconds返回的值以及getMilliseconds方法被使用

返回值:

NA

例子：

<html>
<head>
<title>JavaScript setMinutes Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date( "Aug 28, 2008 23:30:00" );
   dt.setMinutes( 45 );
   document.write( dt ); 
</script>
</body>
</html>

这将产生以下结果：

Thu Aug 28 23:45:00 UTC+0530 2008

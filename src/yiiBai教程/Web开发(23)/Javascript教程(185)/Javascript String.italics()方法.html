Javascript String.italics()方法

这种方法会导致一个字符串是斜体，就好像它是在<i>标签

语法

string.italics( )


下面是参数的详细信息：

  • NA

返回值:

  • 返回字符串带有<i>标签

例子：

<html>
<head>
<title>JavaScript String italics() Method</title>
</head>
<body>
<script type="text/javascript">

var str = new String("Hello world");

alert(str.italics( 3 ));

</script>
</body>
</html>


这将产生以下结果：

<i>Hello world</i>

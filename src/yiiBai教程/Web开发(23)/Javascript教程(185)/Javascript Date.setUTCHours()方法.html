Javascript Date.setUTCHours()方法

javascript Date.setUTCHours()方法按照通用时间设置小时在指定的日期。

语法

Date.setUTCHours(hoursValue[, minutesValue[, 
                                     secondsValue[, msValue]]])

注：括号内外的数据是可选的

下面是参数的详细信息：

  • hoursValue : 0到23之间的整数，表示小时。

  • minutesValue : 0到59之间的整数，表示分钟。

  • secondsValue : 0到59之间的整数，表示秒。如果指定secondsValue参数，则还必须指定minutesValue。

  • msValue : 一个数字在0和999之间，表示毫秒。如果指定msValue参数，则还必须指定minutesValue和secondsValue。

如果不指定minutesValue，secondsValue和msValue参数，从getUTCMinutes，getUTCSeconds，并getUTCMilliseconds方法返回的值被使用。

如果指定的参数是预期范围之外，调用setUTCHours尝试相应地更新Date对象的最新信息。例如，如果secondsValue使用100 ，该分钟的将被加1 (min + 1)递增，并且40将用于秒。

返回值:

NA

例子：

<html>
<head>
<title>JavaScript setUTCHours Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date( "Aug 28, 2008 23:30:00" );
   dt.setUTCHours( 15 );
   document.write( dt ); 
</script>
</body>
</html>

这将产生以下结果：

Thu Aug 28 20:30:00 UTC+0530 2008

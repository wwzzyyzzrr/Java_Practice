Javascript Date.getUTCMonth()方法

javascript Date.getUTCMonth()方法返回按照通用时间在指定日期的月份。由getUTCMonth返回的值是0和11之间的整数对应的月份。 0代表一月，1代表二月，2表 示三月份，依此类推。

语法

Date.getUTCMonth()

下面是参数的详细信息：

  • NA

返回值:

按照通用时间返回指定日期的月份。

例子：

下面的例子打印当前时间变量hrs的月份部分。

<html>
<head>
<title>JavaScript getUTCMonth Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date();
   document.write("getUTCMonth() : " + dt.getUTCMonth() ); 
</script>
</body>
</html>

这将产生以下结果：

getUTCMonth() : 7

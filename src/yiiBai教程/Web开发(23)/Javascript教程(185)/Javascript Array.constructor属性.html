Javascript Array.constructor属性

JavaScript数组构造constructor属性返回的引用，创建实例的原型数组函数

语法

array.constructor


下面是参数的详细信息：

  • NA

返回值:

返回创建此对象的实例的功能。

例子：

<html>
<head>
<title>JavaScript Array constructor Property</title>
</head>
<body>
<script type="text/javascript">
   var arr = new Array( 10, 20, 30 );
   document.write("arr.constructor is:" + arr.constructor); 
</script>
</body>
</html>


这将产生以下结果：

arr.constructor is: function Array() { [native code] }

Javascript Date()函数

JavaScript的Date()方法返回今天的日期和时间，并且不需要任何对象被调用。

语法

Date()

下面是参数的详细信息：

  • NA

返回值:

返回今天的日期和时间

例子：

<html>
<head>
<title>JavaScript Date Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = Date();
   document.write("Date and Time : " + dt ); 
</script>
</body>
</html>

这将产生以下结果：

Date and Time : Wed Aug 27 2008 20:12:50 GMT+0530 
                (India Standard Time)

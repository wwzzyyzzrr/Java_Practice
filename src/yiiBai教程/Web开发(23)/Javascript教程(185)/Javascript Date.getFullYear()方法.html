Javascript Date.getFullYear()方法

javascript Date.getFullYear()方法按照本地时间返回指定日期的年份。通过getFullYear返回的值是一个绝对的数字。对于1000年和9999之间的日期，getFullYear返回一个4位数字，例如2008年。

语法

Date.getFullYear()

下面是参数的详细信息：

  • NA

返回值:

按照本地时间返回指定日期的年份。

例子：

<html>
<head>
<title>JavaScript getFullYear Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date("December 25, 1995 23:15:00");
   document.write("getFullYear() : " + dt.getFullYear() ); 
</script>
</body>
</html>

这将产生以下结果：

getFullYear() : 1995

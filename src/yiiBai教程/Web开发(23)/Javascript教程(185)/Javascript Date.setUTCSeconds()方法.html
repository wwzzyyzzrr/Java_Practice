Javascript Date.setUTCSeconds()方法

javascript Date.setUTCSeconds()方法按照通用时间设置秒指定日期。

语法

Date.setUTCSeconds(secondsValue[, msValue])

注：括号内外的数据是可选的

下面是参数的详细信息：

  • secondsValue : 0到59之间的整数，表示秒。

  • msValue : 一个数字0和999之间，表示毫秒。

如果不指定msValue参数，那么使用从getUTCMilliseconds方法返回的值。

如果指定的参数是预期范围之外，调用setUTCSeconds尝试相应地更新Date对象的最新信息。例如，如果secondsValue使用100 ，存储在日期对象的分钟将加1，而40将用于秒。

返回值:

NA

例子：

<html>
<head>
<title>JavaScript setUTCSeconds Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date( "Aug 28, 2008 13:30:00" );
   dt.setUTCSeconds( 65 );
   document.write( dt ); 
</script>
</body>
</html>

这将产生以下结果：

Thu Aug 28 13:31:05 UTC+0530 2008

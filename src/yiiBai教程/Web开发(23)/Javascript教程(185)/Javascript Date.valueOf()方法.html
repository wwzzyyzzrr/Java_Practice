Javascript Date.valueOf()方法

该方法返回一个Date对象为数字数据类型，以毫秒为单位自午夜1970年1月1日UTC的数的原始值。

语法

Date.valueOf()

下面是参数的详细信息：

  • NA

返回值:

返回Date对象的原始值。

例子：

<html>
<head>
<title>JavaScript valueOf Method</title>
</head>
<body>
<script type="text/javascript">
   var dateobject = new Date(1993, 6, 28, 14, 39, 7);
   document.write( dateobject.valueOf() ); 
</script>
</body>
</html>

这将产生2008年8月29日，结果如下：

743850547000

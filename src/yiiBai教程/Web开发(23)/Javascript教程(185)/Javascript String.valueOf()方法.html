Javascript String.valueOf()方法

该方法返回一个String对象的原始值。

语法

string.valueOf( )


下面是参数的详细信息：

  • NA

返回值:

  • 返回一个String对象的原始值。

例子：

<html>
<head>
<title>JavaScript String valueOf() Method</title>
</head>
<body>
<script type="text/javascript">

var str = new String("Hello world");

document.write(str.valueOf( ));
</script>
</body>
</html>


这将产生以下结果：

Hello world

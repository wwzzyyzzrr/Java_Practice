Javascript Date.getUTCMinutes()方法

javascript Date.getUTCMinutes()方法按照通用时间在指定日期返回分钟。通过getUTCMinutes返回的值是0到59之间的一个整数。

语法

Date.getUTCMinutes()

下面是参数的详细信息：

  • NA

返回值:

按照通用时间返回分钟，在指定的日期。

例子：

下面的例子打印的当前时间变量hrs的分钟部分。

<html>
<head>
<title>JavaScript getUTCMinutes Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date();
   document.write("getUTCMinutes() : " + dt.getUTCMinutes() ); 
</script>
</body>
</html>

这将产生以下结果为印度时区：

getUTCMinutes() : 32

Javascript RegExp.source属性

source是正则表达式对象的只读字符串属性。它包含了正则表达式模式的文本。这个文本不包括在正则表达式文本中使用的分隔斜线，并且它不包括在“g”，“i” 和“m” 属性。

语法

RegExpObject.source

下面是参数的详细信息：

  • NA

返回值:

返回用于模式匹配的文本

例子：

<html>
<head>
<title>JavaScript RegExp source Property</title>
</head>
<body>
<script type="text/javascript">
   var str = "Javascript is an interesting scripting language";
   var re = new RegExp( "script", "g" );

   re.test(str);
   document.write("The regular expression is : " +  re.source); 
</script>
</body>
</html>

这将产生以下结果：

The regular expression is : script

Javascript Boolean对象

Boolean对象表示两个值： "true" 或 "false".

Syntax:

创建一个 boolean 对象:

var val = new Boolean(value);

如果value参数被省略或为0, -0, null, false, NaN, undefined,或空字符串(“”)，该对象的初始值为false。

Boolean 方法

这里是每种方法及其描述的列表

方法 描述

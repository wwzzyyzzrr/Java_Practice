Javascript String.length属性

此属性返回字符串中的字符数。

语法

string.length

下面是参数的详细信息：

  • A string

返回值:

返回字符串中的字符数。

例子：

<html>
<head>
<title>JavaScript String length Property</title>
</head>
<body>
<script type="text/javascript">
   var str = new String( "This is string" );
   document.write("str.length is:" + str.length); 
</script>
</body>
</html>

这将产生以下结果：

str.length is:14

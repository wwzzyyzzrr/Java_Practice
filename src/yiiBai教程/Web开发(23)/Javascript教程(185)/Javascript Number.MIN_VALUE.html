Javascript Number.MIN_VALUE

Number.MIN_VALUE属性属于静态Number对象。这代表了尽可能小的正数的JavaScript可以工作的常量。

该常量的实际值是 5 x 10-324

语法

您可以使用以下语法访问属性：

var val = Number.MIN_VALUE;

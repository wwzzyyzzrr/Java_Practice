Javascript Date.getDate()方法

按照本地时间的javascript date.getDate()方法返回月份的一天，在指定日期。由getDate返回的值是1到31之间的整数。

语法

Date.getDate()

下面是参数的详细信息：

  • NA

返回值:

返回今天的日期和时间

例子：

<html>
<head>
<title>JavaScript getDate Method</title>
</head>
<body>
<script type="text/javascript">
   var dt = new Date("December 25, 1995 23:15:00");
   document.write("getDate() : " + dt.getDate() ); 
</script>
</body>
</html>

这将产生以下结果：

getDate() : 25

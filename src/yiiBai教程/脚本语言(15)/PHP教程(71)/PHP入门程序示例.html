PHP入门程序示例

在本文中，将创建一个简单的PHP示例。创建一个文件并写入HTML标签 + PHP代码，并使用.php扩展名保存此文件。

所有PHP代码都在php标签：<?php 和 ?> 之间。 PHP标签的语法如下：

<?php   
//your code here  
?>

下面来看看一个简单的PHP示例，这里将使用PHP的echo命令编写一些文本。将其保存到 first.php 文件之中，并将 first.php 放到 D:\wamp\www 目录中 。如下所示 -

<!DOCTYPE>  
<html>  
<body>  
<?php  
    echo "<h2>Hello First PHP</h2>";  
?>  
</body>  
</html>

PHP echo语句

PHP echo是一个语言结构(也可以叫作语句)，不是一个函数，所以不需要使用括号。 但是如果要使用多个参数，则需要使用括号。

PHP echo的语法如下：

void echo ( string $arg1 [, string $... ] )

PHP的echo语句可以用来打印字符串，多行字符串，转义字符，变量，数组等。

PHP echo：打印字符串

文件：echo1.php -

<?php  
    echo "Hello by PHP echo";  
?>

执行上面代码得到输出如下：

Hello by PHP echo

PHP echo：打印多行字符串

File: echo2.php

<?php  
    echo "Hello by PHP echo  
this is multi line  
text printed by   
PHP echo statement  
";  
?>

执行上面代码得到输出如下：

Hello by PHP echo this is multi line text printed by PHP echo statement

PHP echo：打印转义字符

文件：echo3.php

<?php  
    echo "Hello escape \"sequence\" characters";  
?>

执行上面代码得到输出如下：

Hello escape "sequence" characters

PHP echo：打印变量值

文件：echo4.php 

<?php  
    $msg="Hello Yiibai PHP";  
    echo "Message is: $msg";    
?>

执行上面代码得到输出如下：

Message is: Hello Yiibai PHP

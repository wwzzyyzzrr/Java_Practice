PHP可变长度参数函数

PHP支持可变长参数函数。这让我们可以在函数中传递0，1,...或n个参数。 为此，您需要在参数名称前使用3个省略号(点)。

3点(...)概念是从PHP 5.6开始实现的可变长参数。下面来看看一个简单的例子PHP变长参数函数。

<?php  
function add(...$numbers) {  
    $sum = 0;  
    foreach ($numbers as $n) {  
        $sum += $n;  
    }  
    return $sum;  
}  

echo add(1, 2, 3, 4);  
?>

输出结果如下 -

10

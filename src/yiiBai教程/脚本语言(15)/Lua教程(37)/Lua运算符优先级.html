Lua运算符优先级

运算符优先级来确定条件的表达式中的分组。这会影响一个表达式如何计算。某些运算符的优先级高于其他。

例子

试试下面的例子就明白了在Lua编程语言的运算符所有的优先级：

a = 20
b = 10
c = 15
d = 5

e = (a + b) * c / d;-- ( 30 * 15 ) / 5
print("Value of (a + b) * c / d is   :",e )

e = ((a + b) * c) / d; -- (30 * 15 ) / 5
print("Value of ((a + b) * c) / d is :",e )

e = (a + b) * (c / d);-- (30) * (15/5)
print("Value of (a + b) * (c / d) is :",e )

e = a + (b * c) / d;  -- 20 + (150/5)
print("Value of a + (b * c) / d is   :",e )

当建立并执行上面的程序它会产生以下结果：

Value of (a + b) * c / d is   :	90
Value of ((a + b) * c) / d is :	90
Value of (a + b) * (c / d) is :	90
Value of a + (b * c) / d is   :	50

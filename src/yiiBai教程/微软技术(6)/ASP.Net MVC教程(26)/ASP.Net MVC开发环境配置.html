ASP.Net MVC开发环境配置

MVC开发工具包含在Visual Studio 2012及更高版本中。 它也可以安装在Visual Studio 2010 SP1 / Visual Web Developer 2010 Express SP1上。 如果您使用Visual Studio 2010，则可以使用 Web Platform Installer 安装MVC 4

Microsoft提供了Visual Studio的免费版本，该版本还包含SQL Server，可以从 http://www.visualstudio.com 下载，在本教程中使用的是 Visual Studio 2017的社区版本。

Visual Studio 2017社区版本(下载： http://www.visualstudio.com/downloads/ )作为开发工具。

请注意，需要有效的互联网连接才能安装快速版。

下载完成后，按照安装软件的常用方法，双击软件包，启动安装界面，同意协议，选择安装目录，然后在线下载安装，至到完成。

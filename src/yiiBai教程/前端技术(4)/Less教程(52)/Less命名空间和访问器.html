Less命名空间和访问器

它是用来分组混入通用名称。使用命名空间就可以避免冲突的名称，并从外封装组混入。

示例

下面的例子演示了使用命名空间和访问修饰符在 Less 文件： 
<html>
<head>
   <title>Less Namespaces and Accessors</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
   <h1>Example using Namespaces and Accessors</h1>
   <p class="myclass">LESS enables customizable, manageable and reusable style sheet for web site.</p>
</body>
</html>

接下来, 创建一个文件 style.less.

style.less 

.class1 {
  .class2 {
    .val(@param) {
      font-size: @param;
      color:green;
    }
  }
}

.myclass {
  .class1 > .class2 > .val(20px);
}
你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css文件，下面的代码：

style.css 

.myclass {
  font-size: 20px;
  color: green;
}

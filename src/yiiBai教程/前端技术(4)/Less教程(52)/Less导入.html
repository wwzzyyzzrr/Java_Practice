Less导入

它是用于导入 LESS 或 CSS 文件的内容。

示例

下面的例子演示了使用导入 LESS 文件： 
<html>
<head>
   <title>Less Importing</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
   <h1>Example using Importing</h1>
   <p class="myclass">LESS enables customizable, manageable and reusable style sheet for web site.</p>
   <p class="myclass1">It allows reusing CSS code and writing LESS code with same semantics.</p>
   <p class="myclass2">LESS supports creating cleaner, cross-browser friendly CSS faster and easier.</p>
</body>
</html>
接下来，创建文件 myfile.less。

myfile.less 

.myclass{
    color: #FF8000;
}

.myclass1{
    color: #5882FA;
}
接下来，创建文件 style.less。

style.less 

@import "http://www.yiibai.com/less/myfile.less";
.myclass2
{
color: #FF0000;
} 

这将从路径  http://www.yiibai.com/less/myfile.less 导入 myfile.less 到 style.less 文件

你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css文件，下面的代码：

style.css 

.myclass {
  color: #FF8000;
}
.myclass1 {
  color: #5882FA;
}
.myclass2 {
  color: #FF0000;
}

Less传递规则集到混合

分离的规则集包含一个普通的规则集，如属性，嵌套规则集，变量声明，混合类型等。 它存储在一个变量和包含在其他的结构，其中所述规则集的所有的属性将被复制到该结构。

示例

下面的例子是通过规则设定混合使用在 Less 文件：

passing_ruleset.html 

<!doctype html>
<head><meta charset="UTF-8"> <link rel="stylesheet" href="style.css" type="text/css" />
</head>
<body>
<div class="cont">
   <h2>Welcome to Yiibai Yiibai</h2>
   <p>专注于IT技术教程和实例</p>
</div>
</body>
</html>
接下来，创建文件 style.less

style.less 

@detached-ruleset: {
    .mixin() {
        "Comic Sans MS";
        background-color: #AA86EE;
    }
};

.cont {
    @detached-ruleset();
    .mixin();
}
你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.cont {
  "Comic Sans MS";
  background-color: #AA86EE;
}

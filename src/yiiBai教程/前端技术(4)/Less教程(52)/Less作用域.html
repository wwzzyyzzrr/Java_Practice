Less作用域

变量的作用域指定可用变量的位置。变量将从本地范围搜查，如果它们不提供，则编译器会从父范围搜索。

示例

下面的例子演示了使用命名空间和访问修饰符在Less文件： 
<html>
<head>
   <title>Less Scope</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
   <h1>Example using Scope</h1>
   <p class="myclass">LESS enables customizable, manageable and reusable style sheet for web site.</p>
</body>
</html>
接下来，创建文件 style.less。

style.less 

@var: @a;
@a: 15px;

.myclass {
  font-size: @var;
  @a:20px;
  color: green;
}
你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css文件，下面的代码：

style.css 

.myclass {
  font-size: 20px;
  color: green;
}

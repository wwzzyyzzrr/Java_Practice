Less重复检测

它不能检测选择器的重复。

示例

下面的例子演示了使用重复检测在Less文件：

extend_syntax.html 

<!doctype html>
<head>
<link rel="stylesheet" href="style.css" type="text/css" />
</head>
<body>
<div class="cont-main">
   <p>Welcome to Yiibai Yiibai</p>
   <div class="style">
     <ul>
       <li>SASS </li>
       <li>LESS</li>
     </ul>
   </div>
</div>
</body>
</html>
接下来，创建文件 style.less

style.less 

.cont-main,
.style {
  "Comic Sans MS";
  font-size: 30px;
}
.cont-main{
  font-size: 30px;
}
.cont:extend(.cont-main, .style) {}
你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.cont-main,
.style,
.cont,
.cont {
  "Comic Sans MS";
  font-size: 30px;
}
.cont-main,
.cont {
  font-size: 30px;
}

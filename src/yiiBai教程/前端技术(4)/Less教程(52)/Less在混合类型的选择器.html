Less在混合类型的选择器

混合类型(mixins)可以包含不仅是属性，而且它们还可以含有选择器。

示例

下面的例子演示了 Less 文件中的 mixin 选择器的使用： 
<html>
<head><meta charset="UTF-8"> <link rel="stylesheet" href="style.css" type="text/css" />
  <title>Selectors in Mixins</title>
</head>
<body>
<h2>Welcome to Yiibai Yiibai</h2>
<a href="http://www.yiibai.com/">易百教程</a>
</body>
</html>
接下来，创建文件 style.less。

style.less 

.mixin() {
  &:hover {
    background: #FFC0CB;
  }
}
a {
  .mixin();
}
你可以编译 style.less 使用下面的命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

a:hover {
  background: #FFC0CB;
}

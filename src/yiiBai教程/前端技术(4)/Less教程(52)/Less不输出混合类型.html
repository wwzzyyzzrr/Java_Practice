Less不输出混合类型

可以创建一个 mixin 并且它可以由简单在括号后，它将消失在输出中。

示例

下面的例子演示了 Less 文件中使用混合类型： 
<html>
<head>
  <link rel="stylesheet" href="style.css" type="text/css" />
  <title>LESS Mixins</title>
</head>
<body>
<div class="myclass">
<h1>Welcome to Yiibai Yiibai</h1>
<p>LESS is a CSS pre-processor that enables customizable, manageable and reusable style sheet for web site.</p>
</div>
</body>
</html>
接下来，创建文件 style.less。

style.less 

.a(){
  padding-left: 100px;
}
.myclass{
  background : #64d9c0;
  .a;
}
你可以编译 style.less 使用下面的命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.myclass {
  background: #64d9c0;
  padding-left: 100px;
}

Less扩展内部规则集

&:extend(selector) 语法可以规则集体里面放。它将扩展到规则集的每一个选择器的快捷方式。

示例

下面的例子演示了使用扩展内部规则在LESS文件中设置：

extend_syntax.html 

<!doctype html>
<head>
   <link rel="stylesheet" href="style.css" type="text/css" />
</head>
<body>
   <div class="style">
      <h1>Welcome to Yiibai Yiibai</h1>
      <div class="container">
         <h2>Hello!!!!!</h2>
      </div>
   </div>
</body>
</html>
接下来，创建文件style.less

style.less 

.container,
.style {
     &:extend(.img);
}
.img{
    font-style: italic;
    background-color: #7B68EE;
 }
你可以编译style.less文件使用以下命令来生成 style.css文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.img,
.container,
.style {
  font-style: italic;
  background-color: #7B68EE;
}

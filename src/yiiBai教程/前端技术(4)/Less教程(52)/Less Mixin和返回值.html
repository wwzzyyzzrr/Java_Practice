Less Mixin和返回值

mixin类似于函数，并且在一个 mixin 定义的行为作为它的返回值的变量。

示例

下面的例子演示了使用混合&返回值在Less文件： 
<html>
<head>
  <link rel="stylesheet" href="style.css" type="text/css" />
  <title>Mixins & return values</title>
</head>
<body>
<div class="myclass">
<h2>Welcome to Yiibai Yiibai</h2>
<p>LESS is a CSS pre-processor that enables customizable, manageable and reusable style sheet for web site.</p>
</div>
</body>
</html>
接下来，创建文件 style.less。

style.less 

.padding(@x, @y) {
  @padding: ((@x + @y) / 2);
}

.myclass{
  .padding(80px, 120px);  // call to the mixin
  padding-left: @padding; //  returns the value
}
你可以编译style.less使用下面的命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.myclass { padding-left:100px;}

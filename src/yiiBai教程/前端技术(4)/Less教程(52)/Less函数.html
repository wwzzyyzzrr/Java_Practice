Less函数

Less映射JavaScript代码配合值，并使用预先定义的函数在样式表处理HTML元素方面。它提供了多种函数来操作的颜色，如round函数，floor函数，ceil函数，百分比等函数

示例

下面的例子演示了使用在Less文件的函数： 
<html>
<head>
   <title>Less Functions</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
   <h1>Example using Functions</h1>
   <p class="mycolor">LESS enables customizable, manageable and reusable style sheet for web site.</p>
</body>
</html>

接下来，创建一个文件 style.less.

style.less 

@color: #FF8000;
@width:1.0;
.mycolor{
color: @color;
 width: percentage(@width);
}
你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.mycolor {
  color: #FF8000;
  width: 100%;
}

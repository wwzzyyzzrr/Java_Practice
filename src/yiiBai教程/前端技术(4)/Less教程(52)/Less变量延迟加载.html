Less变量延迟加载

在延迟加载，即使没有声明的变量也可以直接使用。

示例

下面的例子演示了如何在Less文件使用变量的延迟加载： 
<html>
<head>
  <link rel="stylesheet" href="style.css" type="text/css" />
  <title>LESS Lazy Loading</title>
</head>
<body>
<h2>Welcome to Yiibai Yiibai</h2>
<p>LESS is a CSS pre-processor.</p>
</body>
</html>
接下来，创建文件 style.less。

style.less 

p {
  font-size: @a;
  color: #ca428b;
}
@a: @b;
@b: 25px;
你可以编译 style.less 使用下面的命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css文件，下面的代码：

style.css 

p {
  font-size: 25px;
  color: #ca428b;
}

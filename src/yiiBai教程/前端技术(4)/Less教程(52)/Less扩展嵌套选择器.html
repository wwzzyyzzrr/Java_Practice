Less扩展嵌套选择器

嵌套选择使用扩展选择器匹配。

示例

下面的例子演示了使用在 Less 文件扩展嵌套选择：

extend_syntax.html 

<!doctype html>
<head>
	<link rel="stylesheet" href="style.css" type="text/css" />
</head>
<body>
<div class="style">
	<h3>Hello, You guys!</h3>
</div>
<p class="img">Welcome to Yiibai Yiibai</p>
</body>
</html>
接下来，创建文件 style.less

style.less 

.style {
  h3{
	color: #BF70A5;
	font-size: 30px;
}
}
.img:extend(.style h3){}
你可以编译style.less文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.style h3,
.img {
  	color: #BF70A5;
 	font-size: 30px;
}

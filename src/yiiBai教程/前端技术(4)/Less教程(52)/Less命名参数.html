Less命名参数

mixin用自己的名字为参数的值，而不是使用位置。 参数没有任何顺序用于放置值，可通过名称引用。命名参数的结果更容易阅读，并提供了清晰的代码。

示例

下面的例子演示了 Less 文件中使用命名参数： 
<!doctype html>
<head>
   <title>Named Parameters</title>
   <link rel="stylesheet" href="style.css" type="text/css" />
</head>
<body>
   <h2>Example of Named Parameters</h2>
   <p class="class1">Hello World...</p>
   <p class="class2">Welcome to Yiibai Yiibai ...</p>
</body>
</html>
接下来，创建文件 style.less。

style.less 

.mixin(@color: black; @fontSize: 10px) {
  color: @color;
  font-size: @fontSize;
}
.class1 {
  .mixin(@fontSize: 20px; @color: #F5A9D0);
}
.class2 {
  .mixin(#F79F81; @fontSize: 20px);
}
你可以编译 style.less 使用下面的命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

.class1 {
  color: #F5A9D0;
  font-size: 20px;
}
.class2 {
  color: #F79F81;
  font-size: 20px;
}

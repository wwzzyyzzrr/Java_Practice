Less作用域及扩展@media

内部媒体声明，扩展应被写入。该扩展只存在于同一个媒体声明中选择器匹配。扩展出现在媒体声明中不匹配存在嵌套的声明选择器。

示例

下面的例子演示了使用范围界定内部媒体扩展 LESS 文件：

extend_syntax.html 

<!doctype html>
<head>
   <link rel="stylesheet" href="style.css" type="text/css" />
</head>
<body>
  <h2>Example using extend inside media</h2>
  <img src="http://www.yiibai.com/less/images/nature.jpg" class="style">
</body>
</html>
接下来，创建文件 style.less

style.less 

@media screen {
  .style {
    width:500px;
  }
  @media (min-width: 1023px) {
    .style {
      width:500px;
    }
  }
}
.cont:extend(.style) {}
你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

@media screen {
  .style,
  .cont {
    width: 500px;
  }
}
@media screen and (min-width: 1023px) {
  .style,
  .cont {
    width: 500px;
  }
}

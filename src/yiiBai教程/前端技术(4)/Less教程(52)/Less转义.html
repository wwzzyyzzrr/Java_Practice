Less转义

它动态地构建选择和使用属性或变量的值作为任意字符串。

示例

下面的例子演示了使用转义在Less文件： 
<html>
<head>
   <title>Less Escaping</title>
   <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
   <h1>Example using Escaping</h1>
   <p>LESS enables customizable, manageable and reusable style sheet for web site.</p>
</body>
</html>

接下来，创建一个文件 style.less.

style.less 

p {
  color: ~"green";
}
你可以编译style.less文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css文件，下面的代码：

style.css 

p {
  color: green;
}
凡是里面写〜“some_text”将被编译Less代码CSS代码后，显示为 some_text。

Less缺省变量

默认的变量设置仅在它尚未设置变量时。 此功能不是必须的，因为变量可以通过之后定义，它们就很容易地覆盖。

示例

下面的例子演示了如何使用的默认变量在Less文件： 
<html>
<head>
  <link rel="stylesheet" href="style.css" type="text/css" />
  <title>LESS Default Variables</title>
</head>
<body>
<h1>Welcome to Yiibai Yiibai</h1>
<p>LESS is a CSS pre-processor that enables customizable, manageable and reusable style sheet for web site.</p>
</body>
</html>
接下来，创建文件 style.less。

style.less 

@import "http://www.yiibai.com/less/lib.less"; // first declaration of @color
@color: green; // this will override @color defined previously
p{
 color : @color;
}
style.less这将从下面代码路径  http://www.yiibai.com/less/lib.less 导入lib.less文件：

lib.less 

@color: blue;
你可以编译 style.less 使用下面的命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css文件，下面的代码： 

style.css 

p {
  color: green;
}

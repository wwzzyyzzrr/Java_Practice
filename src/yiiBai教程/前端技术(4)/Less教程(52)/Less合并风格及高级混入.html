Less合并风格及高级混入

使用扩展，我们可以将特定选择器的相同的样式到其他选择器。

示例

下面的例子描述了使用相结合的方式混入在LESS 文件：

extend_syntax.html 

<!doctype html>
<head><meta charset="UTF-8"> <link rel="stylesheet" href="style.css" type="text/css" />
</head>
<body>
<div class="cont">
  <h2>Welcome to Yiibai Yiibai</h2>
  <p>专注于IT教程和实例.</p>
   <ul class="nav">
     <li>HTML</li>
     <li>SASS</li>
     <li>LESS</li>
   </ul>
</div>
</body>
</html>
接下来，创建文件 style.less：

style.less 

ul.nav > li {
   background-color: #6DE6E6;
  color: black;
}
.cont {
  &:extend(ul.nav > li);
}
你可以编译 style.less 文件使用以下命令来生成 style.css 文件： 
lessc style.less style.css
接着执行上面的命令，它会自动创建 style.css 文件，下面的代码：

style.css 

ul.nav > li,
.cont {
  background-color: #6DE6E6;
  color: black;
}

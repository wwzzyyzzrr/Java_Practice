QC常用功能

常用的功能：

在Infragistics网格视图或树结构惠普应用生命周期管理（ALM）显示数据，用户可以以各种不同的方式，如重新安排列，筛选，排序和分组数据。

也可以将文件附加到这些网格记录，搜索记录中特定的文本，以及查看各种实体的历史。以下功能将被理解为确保用户能够无缝地与QC工作

点击这些链接中的每一个更多地了解常见的功能。

常见的功能 描述
Filter Dialog Box 使用户能够执行基于定义的标准过滤器ALM数据。
Users Dialog Box 使用户能够从用户列表中选择一个用户名。
Find Dialog Box 使用户可以搜索特定的记录在树视图或网格视图
Replace Dialog Box 使用户能够查找和替换的特定字段值在树上或网格。
History Tab 使用户能够查看到当前选定的工作项所做的更改的列表。
Send Email Dialog 使质量中心的用户发送一封电子邮件关于实体的信息给其他用户。
Attachments Dialog 使用户可以添加和管理附件的特定记录。
Snapshot Dialog 使用户能够捕捉和被测附上您的应用程序的屏幕截图，以在ALM的实体。
Column chooser Dialog 允许用户选择他们希望看到在网格视图中的列。
Set Defaults Dialog 使用户能够在ALM的某些字段设置默认值。
Alerts and Flags Dialog 使用户能够查看的警报所选记录的列表，并允许他们创建，查看和修改跟进标志，以提醒自己跟进的问题
Favorites Dialog 允许用户创建和组织您的收藏夹。

测试结果

执行测试后，测试人员可以在查看结果的"Test Runs"，其中包含的选项卡，它可以让测试人员分析测试实例运行和测试集运行的结果模块。 

转到"Test Runs" ，在“Testing”标签，并在打开的窗口如下图所示：

下面是测试运行选项卡的功能。

  • Delete - 使测试人员来删除一个特定的测试运行，并可以根据用户的权限进行控制。

  • Details - 在打开的详细信息选项卡显示每个步骤的执行细节。

  • Continue Manual Run - 使测试人员继续与手动运行，如果暂停插图中。

  • Goto Test Instance - 此选项将测试实例下的测试实验室模块。

  • comments - 允许测试人员添加注释到试运行。

  • Report - 显示步骤，其状态运行的详细信息。

  • Attachments - 使测试人员能够添加附件为特定的执行。

  • Run ID - HP-ALM自动分配一个唯一的ID运行（日期时间戳），以便它可以很容易地跟踪。

  • Exec Date - 使测试人员跟踪的基础上执行日期的测试执行。

  • Exec Time - 使测试人员跟踪的基础上执行时间的测试执行。

  • Host - 显示在其上执行的测试主机系统。

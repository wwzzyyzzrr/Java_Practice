项目用户

该模块可以帮助我们增加有效用户列表的整体ALM的用户列表。此外，我们还可以指定每个项目的用户到用户组。每个用户组都有一个特定的权限来执行某些任务。

点击 "Add user"按钮，在项目中的用户模块。

The "Add user" dialog opens. The project administrator has to key in the user name inorder to add users successfully.

成功添加用户后，选择要向其会员等级有要编辑的用户。选择的会员级别，然后点击“>>”按钮。

项目管理员还可以使用"Remove user"按钮，在项目区域中删除一个用户。记住这只是从项目区中删除用户，它仍然会保留用户的HP-ALM站点管理。
